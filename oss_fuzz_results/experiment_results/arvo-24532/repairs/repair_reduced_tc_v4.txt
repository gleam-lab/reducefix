template <typename T>
void hb_vector_t<T>::resize(int new_capacity) {
    if (new_capacity <= capacity_) {
        return; // 不需要调整大小
    }

    // 计算新的内存大小
    size_t new_size = new_capacity * sizeof(T);

    // 分配新的内存
    T* new_data = static_cast<T*>(malloc(new_size));
    if (!new_data) {
        throw std::bad_alloc(); // 内存分配失败

    // 复制现有数据到新内存
    for (size_t i = 0; i < count_; ++i) {
        new (new_data + i) T(data_[i]); // 使用 placement new 构造对象

        data_[i].~T(); // 显式调用析构函数

}

1. 检查新的容量是否小于等于当前容量，如果是，则不需要调整大小。
2. 动态分配新的内存块。
3. 使用 placement new 将现有数据复制到新内存块中。
4. 显式调用析构函数销毁旧数据。
5. 释放旧的内存块。
6. 更新数据指针和容量。

通过这些步骤，我们可以确保在调整向量大小时不会发生段错误。
