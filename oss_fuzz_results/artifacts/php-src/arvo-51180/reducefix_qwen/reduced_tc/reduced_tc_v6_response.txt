The crash is a **heap-use-after-free** in `update_errors_warnings`, triggered when calling `timelib_error_container_dtor(*last_errors)` **after** potentially assigning that same pointer to `DATEG(last_errors)`. If `DATEG(last_errors)` holds the only reference to the error container, and we free `*last_errors`, then `DATEG(last_errors)` becomes a dangling pointer — leading to use-after-free when accessed later.

### 🔍 Root Cause:
In this code:
```c
if ((*last_errors)->warning_count || (*last_errors)->error_count) {
    DATEG(last_errors) = *last_errors;  // 👈 Stores pointer without copying
}
timelib_error_container_dtor(*last_errors);  // 👈 Frees memory pointed to by *last_errors
```

If `*last_errors` is non-null and has errors/warnings, it's stored **by reference** into `DATEG(last_errors)`, but then immediately **freed**, leaving `DATEG(last_errors)` pointing to freed memory. Later access (e.g., in `timelib_error_container_dtor(DATEG(last_errors))`) causes the use-after-free.

### ✅ Fix:
We must **copy** the error container before storing it in `DATEG(last_errors)`, so we don't store a pointer to memory about to be freed.
