### Approaches
1. **Graph Representation**: Represent the graph using adjacency lists where each node points to its neighbors along with the edge weights.
2. **Dijkstra's Algorithm**: Use a priority queue to implement Dijkstra's algorithm for finding the shortest paths from vertex 1 to all other vertices. The priority queue will help efficiently retrieve the next vertex to process based on the current shortest distance.
3. **Distance Initialization**: Initialize the distance array with a sufficiently large value (like `LLONG_MAX`) to represent infinity. The distance from vertex 1 to itself is set to the vertex weight `A_1`.
4. **Relaxation**: For each vertex processed, relax all outgoing edges. Update the distance to the neighboring vertices if a shorter path is found through the current vertex.
5. **Output**: After processing all vertices, print the distances from vertex 1 to all other vertices from 2 to N.

### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

const ll INF = LLONG_MAX;

struct Edge {
    int to;
    ll weight;
};

struct Node {
    int vertex;
    ll distance;
    bool operator<(const Node& other) const {
        return distance > other.distance;
    }
};

void solve() {
    int N, M;
    cin >> N >> M;
    vector<ll> A(N + 1);
    for (int i = 1; i <= N; ++i) {
        cin >> A[i];
    }
    vector<vector<Edge>> graph(N + 1);
    for (int i = 0; i < M; ++i) {
        int U, V;
        ll B;
        cin >> U >> V >> B;
        graph[U].push_back({V, B});
        graph[V].push_back({U, B});
    }

    vector<ll> dist(N + 1, INF);
    dist[1] = A[1];
    priority_queue<Node> pq;
    pq.push({1, dist[1]});

    while (!pq.empty()) {
        Node current = pq.top();
        pq.pop();
        int u = current.vertex;
        ll current_dist = current.distance;
        if (current_dist > dist[u]) continue;
        for (const Edge& edge : graph[u]) {
            int v = edge.to;
            ll new_dist = current_dist + edge.weight + A[v];
            if (new_dist < dist[v]) {
                dist[v] = new_dist;
                pq.push({v, dist[v]});
            }
        }
    }

    for (int i = 2; i <= N; ++i) {
        cout << dist[i] << (i == N ? "\n" : " ");
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    solve();
    return 0;
}
```

### Explanation
1. **Graph Representation**: The graph is stored using an adjacency list where each entry contains the destination vertex and the edge weight.
2. **Priority Queue**: The priority queue is used to always process the vertex with the currently smallest known distance, which is crucial for the efficiency of Dijkstra's algorithm.
3. **Distance Initialization**: The distance array is initialized to `LLONG_MAX` (infinity) except for the starting vertex (vertex 1), which is set to its own vertex weight.
4. **Relaxation Process**: For each vertex extracted from the priority queue, the algorithm checks all outgoing edges. If a shorter path to a neighboring vertex is found, the distance is updated, and the vertex is added to the priority queue for further processing.
5. **Output**: The distances from vertex 1 to all other vertices are printed, ensuring correct formatting with spaces between values and a newline at the end.

This approach ensures that the solution is efficient and correct, handling all edge cases and large input sizes as specified in the problem constraints.
