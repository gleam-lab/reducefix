To solve this problem efficiently, we can use Dijkstra's algorithm, which is well-suited for finding the shortest paths in a weighted graph. However, since our graph is unweighted except for the edges, we need to adjust Dijkstra's algorithm to account for both vertex and edge weights.

Here's the corrected code using Dijkstra's algorithm:

```cpp
#include <bits/stdc++.h>
using namespace std;

typedef pair<long long, int> pii;

void dijkstra(vector<vector<pii>>& adj, vector<long long>& dist, vector<long long>& a) {
    priority_queue<pii, vector<pii>, greater<pii>> pq;
    pq.push({a[0], 1});
    dist[1] = a[0];

    while (!pq.empty()) {
        auto [currentDist, node] = pq.top();
        pq.pop();

        if (currentDist > dist[node]) continue;

        for (auto [neighbor, edgeWeight] : adj[node]) {
            long long newDist = currentDist + a[neighbor - 1] + edgeWeight;
            if (newDist < dist[neighbor]) {
                dist[neighbor] = newDist;
                pq.push({newDist, neighbor});
            }
        }
    }
}

int main() {
    ios::sync_with_stdio(false); cin.tie(0);

    int n, m;
    cin >> n >> m;

    vector<long long> a(n);
    for (int i = 0; i < n; ++i) cin >> a[i];

    vector<vector<pii>> adj(n + 1);
    for (int j = 0; j < m; ++j) {
        int u, v, w;
        cin >> u >> v >> w;
        adj[u].push_back({v, w});
        adj[v].push_back({u, w});
    }

    vector<long long> dist(n + 1, LLONG_MAX);
    dijkstra(adj, dist, a);

    for (int i = 2; i <= n; ++i) {
        cout << dist[i] << " ";
    }

    return 0;
}
```

### Explanation:
1. **Graph Representation**: We represent the graph using an adjacency list where each node points to its neighbors along with the edge weight.
2. **Dijkstra's Algorithm**:
   - Initialize a distance array `dist` with `LLONG_MAX` and set `dist[1]` to `a[0]` because the starting point is vertex 1.
   - Use a min-priority queue to always expand the least costly path first.
   - For each node, update the distances to its neighbors if a cheaper path is found.
3. **Output**: Print the distances from vertex 1 to all other vertices from index 2 to N.

This approach ensures that we find the minimum weight path from vertex 1 to every other vertex in the graph efficiently.
