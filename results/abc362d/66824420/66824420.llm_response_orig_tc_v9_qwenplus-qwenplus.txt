The issue with the original code lies in the way it computes the path cost and initializes distances. The main problems are:

1. **Initialization of Distance Array**: It uses `1E10+0` as the initial distance, which is too small for some cases (e.g., paths involving many high-cost edges/nodes). This leads to incorrect relaxation in Dijkstra's algorithm.

2. **Incorrect Distance Updates**: When computing the total path cost, the current node's value (`a[node]`) is already included in the accumulated path cost. However, in the original code, the vertex weight (`a[it.first-1]`) is added again during relaxation, effectively double-counting the vertex weights unless the path starts exactly at 1.

3. **Vertex Weight Inclusion**: The starting point includes `A_1` correctly, but other nodes shouldn't include their own vertex weight again when traversing an edge from a neighbor.

---

### Solution Strategy

- Use Dijkstra's algorithm on the graph since all edge/vertex weights are non-negative.
- Initialize `dist[1] = A[0]`, i.e., the weight of the starting vertex.
- For each edge traversal from node `u` to node `v`, the cost is:
  ```
  dist[u] + edge_weight(u,v) + A[v]
  ```
  But this will be compared only to the existing value of `dist[v]`. Hence, we must not add `A[v]` again if we've already visited `v`.

So the correct update should be:
```
total_cost = dist[u] + edge_weight
if (total_cost < dist[v]) {
    dist[v] = total_cost;
}
```

This assumes that the vertex weight is added only once when you first reach the node.

#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(0);

    int n, m;
    cin >> n >> m;
    vector<long long> A(n);
    for (int i = 0; i < n; ++i) {
        cin >> A[i];
    }

    vector<vector<pair<int, long long>>> adj(n + 1);
    for (int j = 0; j < m; ++j) {
        int u, v;
        long long b;
        cin >> u >> v >> b;
        adj[u].push_back({v, b});
        adj[v].push_back({u, b});
    }

    // Use a large enough "infinity" value
    const long long INF = numeric_limits<long long>::max() / 2;
    vector<long long> dist(n + 1, INF);
    dist[1] = A[0]; // Starting from node 1, include its vertex weight

    priority_queue<pair<long long, int>, vector<pair<long long, int>>, greater<>> pq;
    pq.push({dist[1], 1});

    while (!pq.empty()) {
        auto [cost, u] = pq.top();
        pq.pop();

        if (cost > dist[u]) continue;

        for (auto &[v, edge_cost] : adj[u]) {
            long long new_cost = cost + edge_cost;
            if (new_cost < dist[v]) {
                dist[v] = new_cost;
                pq.push({new_cost, v});
            }
        }
    }

    for (int i = 2; i <= n; ++i) {
        cout << dist[i] << " ";
    }
    cout << '\n';

    return 0;
}
