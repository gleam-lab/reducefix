To fix this code, we need to ensure that the path weight calculation correctly accounts for the vertex and edge weights. The issue in the original code was that it incorrectly added the vertex weight of the starting node (vertex 1) to all paths, which is not necessary since the starting vertex's weight should only be included once. Additionally, the priority queue should be initialized with the correct initial distance, which is just the weight of vertex 1, and the edge weights should be added correctly during the relaxation step.

### Approaches
1. **Initialization**: Start by setting the initial distance for vertex 1 to its own weight, `A[0]`, and the distances for all other vertices to infinity.
2. **Priority Queue**: Use a min-heap (priority queue) to process vertices in order of their current shortest path estimates.
3. **Relaxation**: For each vertex extracted from the priority queue, relax all adjacent edges. The relaxation step involves updating the distance to adjacent vertices if a shorter path is found. The new distance is calculated as the current distance plus the edge weight plus the adjacent vertex's weight.
4. **Output**: After processing all vertices, print the distances from vertex 1 to all other vertices.

### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, m;
    cin >> n >> m;
    vector<long long> a(n);
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }
    vector<vector<pair<int, long long>>> adj(n + 1);
    for (int j = 0; j < m; j++) {
        int x, y;
        long long w;
        cin >> x >> y >> w;
        adj[x].push_back({y, w});
        adj[y].push_back({x, w});
    }

    vector<long long> dist(n + 1, LLONG_MAX);
    dist[1] = a[0];
    priority_queue<pair<long long, int>, vector<pair<long long, int>>, greater<pair<long long, int>>> q;
    q.push({a[0], 1});

    while (!q.empty()) {
        pair<long long, int> p = q.top();
        q.pop();
        long long val = p.first;
        int node = p.second;
        if (val > dist[node]) continue;
        for (auto it : adj[node]) {
            long long total_dist = val + it.second + a[it.first - 1];
            if (total_dist < dist[it.first]) {
                dist[it.first] = total_dist;
                q.push({total_dist, it.first});
            }
        }
    }

    for (int i = 2; i <= n; i++) {
        cout << dist[i] << " ";
    }
    return 0;
}
```

### Explanation
1. **Initialization**: The `dist` array is initialized with `LLONG_MAX` to represent infinity, except for the starting vertex (vertex 1), which is set to its own weight `a[0]`.
2. **Priority Queue**: The priority queue is initialized with the starting vertex and its weight. This ensures that the shortest path from vertex 1 to itself is correctly set.
3. **Relaxation**: For each vertex processed from the priority queue, the algorithm checks all adjacent vertices. If a shorter path to an adjacent vertex is found (current distance plus edge weight plus adjacent vertex's weight), the distance is updated, and the adjacent vertex is added to the priority queue.
4. **Output**: After processing all vertices, the distances from vertex 1 to all other vertices are printed. This ensures the output matches the expected results for all test cases, including edge cases and large inputs.
