
The problem requires finding the minimum weight path from vertex 1 to every other vertex in a connected undirected graph. The weight of a path includes both the weights of the vertices and the edges.

### Approach
1. **Graph Representation**: Use an adjacency list to represent the graph.
2. **Dijkstra's Algorithm**: Use Dijkstra's algorithm to find the shortest path from the source (vertex 1) to all other vertices. The weight of each vertex `i` in the path will be `a[i] + distance[i]` where `distance[i]` is the shortest distance from vertex 1 to vertex `i`.
3. **Output**: For each vertex from 2 to N, output the sum of its weight and its distance from vertex 1.

#include <bits/stdc++.h>
using namespace std;
#define int long long
const int INF = 1e18;

vector<pair<int, int>> adj[200005];  // Adjacency list for the graph
int dist[200005];  // Distance array
int a[200005];  // Vertex weights

void dijkstra(int src) {
    priority_queue<pair<int, int>, vector<pair<int, int>>, greater<pair<int, int>>> pq;  // Min-heap for Dijkstra's
    pq.push({0, src});
    dist[src] = a[src];  // Distance to the source itself is its weight

    while (!pq.empty()) {
        int d = pq.top().first;
        int u = pq.top().second;
        pq.pop();

        if (dist[u] < d) continue;  // Skip if already processed with better distance

        for (auto &edge : adj[u]) {
            int v = edge.first;
            int weight = edge.second;
            if (dist[u] + weight + a[v] < dist[v]) {
                dist[v] = dist[u] + weight + a[v];
                pq.push({dist[v], v});
            }
        }
    }
}

int main() {

    cin >> n >> m;

    for (int i = 1; i <= n; i++) {
        cin >> a[i];
    }

    for (int i = 0; i < m; i++) {

        cin >> u >> v >> w;
        adj[u].push_back({v, w});
        adj[v].push_back({u, w});
    }

    fill(dist, dist + n + 1, INF);  // Initialize distances to infinity
    dijkstra(1);  // Run Dijkstra's from vertex 1

### Explanation
- **Graph Representation**: The graph is represented using an adjacency list, where `adj[i]` contains pairs of vertices and edge weights that are connected to vertex `i`.
- **Dijkstra's Algorithm**: The algorithm uses a min-heap to always expand the shortest path first. It updates the distances and continues until all vertices are processed.
- **Output**: The distances are printed for all vertices from 2 to N. The distance to the source itself is also its weight, hence `dist[src] = a[src]`.
