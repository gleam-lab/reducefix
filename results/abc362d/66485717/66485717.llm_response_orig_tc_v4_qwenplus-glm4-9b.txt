
### Problem Analysis

The problem is to find the minimum weight of a path from vertex 1 to every other vertex in a given undirected graph. The weight of a path includes both the weights of the vertices and the edges traversed. The graph is simple (no loops or multiple edges between the same pair of vertices) and connected (there is a path between any pair of vertices).

### Solution Approach

The solution can be approached using Dijkstra's algorithm, which is well-suited for finding the shortest paths from a source vertex to all other vertices in a graph with non-negative edge weights.

1. **Graph Representation**: Use an adjacency list to represent the graph. Each vertex will have a list of its neighbors along with the edge weights connecting them.

2. **Initialization**:
   - Initialize a distance array `d` with `inf` values, indicating that all vertices are initially unreachable from the source vertex.
   - Set the distance to the source vertex (vertex 1) as the sum of its vertex weight and 0 edge weight (since no edges are traversed yet).

3. **Dijkstra's Algorithm**:
   - Use a priority queue to always expand the least costly vertex first. The priority queue stores pairs of (current distance, vertex).
   - For each vertex, update the distances to its neighbors if a shorter path is found.
   - Continue until all vertices are processed or the priority queue is empty.

4. **Output**: For each vertex from 2 to N, output the minimum distance found for that vertex.

#include <iostream>
#include <vector>
#include <queue>
#include <limits>
using namespace std;

const int MAX_N = 200000 + 10;
vector<pair<int, int>> adj[MAX_N]; // Adjacency list to store (neighbor, weight)
vector<long long> dist(MAX_N, numeric_limits<long long>::max()); // Distance array initialized to max long long
priority_queue<pair<long long, int>, vector<pair<long long, int>>, greater<pair<long long, int>>> pq; // Min-heap priority queue

void dijkstra(int src) {
    dist[src] = 0; // Distance to the source vertex is 0
    pq.push({0, src});
    
    while (!pq.empty()) {
        auto [d, u] = pq.top();
        pq.pop();
        
        if (d > dist[u]) continue; // Skip if we've found a better path already
        
        for (auto [v, weight] : adj[u]) {
            long long alt = d + weight + dist[v]; // Calculate alternative path distance
            if (dist[v] > alt) {
                dist[v] = alt;
                pq.push({alt, v});
            }
        }
    }
}

int main() {
    int n, m;
    cin >> n >> m;
    
    for (int i = 1; i <= n; ++i) {
        cin >> dist[i]; // Read vertex weights
    }
    
    for (int i = 0; i < m; ++i) {
        int u, v, weight;
        cin >> u >> v >> weight; // Read edges
        adj[u].push_back({v, weight});
        adj[v].push_back({u, weight});
    }
    
    dijkstra(1); // Run Dijkstra's algorithm from vertex 1
    
    for (int i = 2; i <= n; ++i) {
        cout << dist[i] << " "; // Output the minimum distance to each vertex starting from 2
    }
    cout << endl;
    
    return 0;
}
