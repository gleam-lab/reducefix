To solve this problem correctly, we need to compute the **minimum path weight** from vertex `1` (vertex 0 in 0-based indexing) to every other vertex in a **weighted undirected graph**, where both **vertex weights and edge weights contribute** to the total path cost.

The key mistake in your code is how you're calculating the cost of paths. Specifically:

- You're including `val[v]` only once at the start (`dist[0] = 0`) but not accounting for it on the destination side when pushing into the priority queue.
- Also, you're adding `val[u]` again at the end (`dist[i] + val[i]`) when outputting, which leads to **double-counting** the destination vertex's value.

---

### ✅ Fix Summary

- Initialize `dist[v]` as `INF`.
- Start Dijkstra with `dist[0] = A[0]`, because any path from node 0 must include its own vertex weight.
- In each edge, represent the movement from node `v` to node `u` as:  
  `new_cost = dist[v] + B_uv + A[u]`
- Do **not** add `A[i]` again during output.

Here is the corrected full C++20 code:

```cpp
#include <iostream>
#include <vector>
#include <queue>
#include <set>
#include <map>
#include <algorithm>
using namespace std;
typedef long long ll;

const ll INF = 1e18;

void solve() {
    int n, m;
    cin >> n >> m;

    vector<ll> A(n);
    for (int i = 0; i < n; ++i) cin >> A[i];

    vector<vector<pair<int, ll>>> adj(n);
    for (int i = 0; i < m; ++i) {
        int u, v, w;
        cin >> u >> v >> w;
        u--; v--;
        adj[u].push_back({v, w});
        adj[v].push_back({u, w});
    }

    vector<ll> dist(n, INF);
    priority_queue<pair<ll, int>, vector<pair<ll, int>>, greater<>> pq;

    dist[0] = A[0];
    pq.push({dist[0], 0});

    while (!pq.empty()) {
        auto [d, u] = pq.top(); pq.pop();
        if (d > dist[u]) continue;

        for (auto [v, w_edge] : adj[u]) {
            ll new_cost = dist[u] + w_edge + A[v];
            if (new_cost < dist[v]) {
                dist[v] = new_cost;
                pq.push({dist[v], v});
            }
        }
    }

    for (int i = 1; i < n; ++i)
        cout << dist[i] << " ";
    cout << "\n";
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr); cout.tie(nullptr);

    solve();

    return 0;
}
```

### ✅ Explanation:
- The initial distance from node 0 to itself is just `A[0]`.
- When moving from `u` to `v`, the total cost added includes:
  - The current shortest path to `u`: `dist[u]`
  - The edge weight: `w_edge`
  - The vertex weight of `v`: `A[v]`
- This ensures that all nodes' weights are counted exactly once when visited.
- No double counting occurs because each node's weight is only added when entering it via an edge.

- **Space:** `O(N + M)` — adjacency list and distance array.
