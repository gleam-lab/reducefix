The issue in your code is related to how the path weight is computed. In particular, you're not correctly accounting for the inclusion of both vertex and edge weights in the total path cost.

---

### **Key Mistake**
In your current graph model:

- You added the vertex weight (`val[v]`) into the edge weight when creating adjacency lists:
  ```cpp
  adj[a].push_back({b, 1ll * w + val[a]});
  adj[b].push_back({a, 1ll * w + val[b]});
  ```

This effectively includes the vertex weight `val[a]` or `val[b]` every time a node is reached through an edge, which leads to **double-counting** of vertex weights in paths longer than one step.

For example, in a path `1 -> u -> v`, the total cost becomes:
```
(val[1] + B_1u) + (val[u] + B_uv) = val[1] + val[u] + B_1u + B_uv
```
But when reaching node `v`, the correct total should be:
```
val[1] + B_1u + val[u] + B_uv + val[v]
```
So the destination node's weight must be included once at the **end** of the path, but not as part of the edge weight from its incoming edges.

---

### **Fix Summary**

1. Store only edge weights in the adjacency list.
2. Initialize `dist[0] = A[0]` since starting node’s own weight must always be counted.
3. When updating distances, use `dist[v] + B_j` to reach `u`.
4. At the end, add `A[i]` to each `dist[i]` before outputting.

---

### ✅ Corrected Code

```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

const ll INF = 1e18;

void solve() {
    int n, m;
    cin >> n >> m;

    vector<ll> A(n);
    for (int i = 0; i < n; ++i)
        cin >> A[i];

    vector<vector<pair<int, ll>>> adj(n);
    for (int i = 0; i < m; ++i) {
        int u, v;
        ll b;
        cin >> u >> v >> b;
        u--, v--;
        adj[u].push_back({v, b});
        adj[v].push_back({u, b});
    }

    vector<ll> dist(n, INF);
    priority_queue<pair<ll, int>, vector<pair<ll, int>>, greater<>> pq;

    dist[0] = A[0]; // Start with vertex 1 (index 0), include its weight
    pq.push({dist[0], 0});

    while (!pq.empty()) {
        auto [d, u] = pq.top();
        pq.pop();

        if (d > dist[u]) continue;

        for (auto [v, b] : adj[u]) {
            if (dist[v] > dist[u] + b) {
                dist[v] = dist[u] + b;
                pq.push({dist[v], v});
            }
        }
    }

    for (int i = 1; i < n; ++i) {
        cout << dist[i] + A[i] << " ";
    }
    cout << '\n';
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    solve();
}
```

- **Correct Path Weight Calculation**: Vertex weight is only applied once at the final destination.
- **Efficient Dijkstra**: Uses `priority_queue` correctly with O(M log N) complexity.
- **No Double Counting**: Vertex weight is not added prematurely in edge costs.
