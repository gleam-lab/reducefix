To fix this code, we need to address several issues. The primary problem is that the initial implementation incorrectly handles the calculation of the path weight, which should include the sum of the vertex weights and edge weights along the path. The current implementation also has some issues with the Dijkstra's algorithm setup, such as incorrect initialization of distances and improper handling of the priority queue.

### Approaches
1. **Initialization**: The distance from vertex 1 to itself should be the weight of vertex 1 (`A_1`). Other vertices should be initialized to a very large value (infinity).
2. **Priority Queue**: Use a min-heap (priority queue) to efficiently extract the vertex with the current minimum distance. Each extracted vertex's adjacent vertices are then processed.
3. **Relaxation**: For each adjacent vertex, update the distance if a shorter path is found. The new distance should be the sum of the current vertex's distance, the adjacent vertex's weight (`A_v`), and the edge weight (`B_j`).

### Solution Code
```cpp
#include<bits/stdc++.h>
using namespace std;
#define int long long 
const int INF = 1e18;
vector<pair<int,int>> G[200005];
int dis[200005];
int a[200005];
int n, m;

void dij(int u) {
    priority_queue<pair<int,int>, vector<pair<int,int>>, greater<pair<int,int>>> q;
    q.push({a[u], u});
    dis[u] = a[u];
    while (!q.empty()) {
        int current_dist = q.top().first;
        int u = q.top().second;
        q.pop();
        if (current_dist > dis[u]) continue;
        for (auto [v, w] : G[u]) {
            if (dis[v] > dis[u] + a[v] + w) {
                dis[v] = dis[u] + a[v] + w;
                q.push({dis[v], v});
            }
        }
    }
}

signed main() {
    scanf("%lld%lld", &n, &m);
    for (int i = 1; i <= n; i++) {
        scanf("%lld", &a[i]);
    }
    for (int i = 1; i <= m; i++) {
        int u, v, w;
        scanf("%lld%lld%lld", &u, &v, &w);
        G[u].push_back({v, w});
        G[v].push_back({u, w});
    }
    fill(dis, dis + n + 1, INF);
    dij(1);
    for (int i = 2; i <= n; i++) {
        if (dis[i] == INF) {
            cout << "-1 ";
        } else {
            cout << dis[i] << ' ';
        }
    }
    cout << "\n";
    return 0;
}
```

### Explanation
1. **Initialization**: The distance array `dis` is initialized to `INF` (a very large value), except for the starting vertex (vertex 1), which is initialized to its weight `A_1`.
2. **Priority Queue**: A min-heap is used to process vertices in order of increasing distance. This ensures that the shortest path to each vertex is found first.
3. **Edge Relaxation**: For each vertex extracted from the heap, the algorithm checks all adjacent vertices. If a shorter path is found (current distance plus vertex weight plus edge weight), the adjacent vertex's distance is updated and it is added to the heap.
4. **Output**: For each vertex from 2 to N, the algorithm prints the computed shortest path distance. If a vertex remains unreachable (distance remains `INF`), it prints `-1` (though the problem guarantees the graph is connected, so this might not be necessary).

This approach ensures that the shortest path is computed efficiently using Dijkstra's algorithm, with a time complexity of O((N + M) log N), which is optimal for the given constraints.
