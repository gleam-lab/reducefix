
### Problem Analysis
The problem is about finding the minimum weight of a path from vertex 1 to every other vertex in a given undirected graph. The weight of a path is defined as the sum of the weights of the vertices and edges that appear on the path. This can be approached using a modified Dijkstra's algorithm where vertex weights are considered as part of the graph edges.

### Solution Strategy
The solution involves using Dijkstra's algorithm to find the shortest paths from the source vertex (vertex 1) to all other vertices. The complexity of the algorithm can be managed by using a priority queue (min-heap) to always expand the least costly node first.

### Key Steps in the Solution
1. **Initialization**:
   - Read the number of vertices `N` and edges `M`.
   - Initialize arrays and data structures required for the graph representation and for Dijkstra's algorithm.
   - Set up the adjacency list for the graph.

2. **Graph Representation**:
   - Each vertex `i` has an associated weight `A_i`.
   - Each edge connects two vertices `U_j` and `V_j` with a weight `B_j`.
   - The adjacency list `g` is used to store the graph, where `g[i]` contains pairs of vertices `(j, weight)` that are directly connected to vertex `i`.

3. **Modified Dijkstra's Algorithm**:
   - Use a priority queue to keep track of the vertices to explore, prioritizing the vertex with the lowest accumulated cost.
   - Initialize distances `dis[i]` to a very high value (`LONG_LONG_MAX`) for all vertices except the source vertex, which is initialized to 0.
   - Process the graph using the priority queue:
     - For the current vertex `u`, explore all its neighbors `v`. The cost to reach `v` through `u` is `dis[u] + A[u] + B[u]`.
     - If this cost is less than the currently known cost to reach `v`, update `dis[v]` and push `v` with the new cost into the priority queue.
   - Continue until the priority queue is empty.

4. **Output**:
   - After computing the shortest paths, output the minimum path weights from vertex 1 to all other vertices from 2 to N.

#include <queue>
#include <vector>
#include <cstdio>
#include <cstring>
#include <climits>
using namespace std;

typedef long long ll;
typedef pair<int, ll> pii; // Using ll for the second value to handle large weights
const int N = 2e5 + 5;

int n, m;
ll a[N];
ll dis[N];
bool vis[N];
vector<pii> g[N]; // Adjacency list with weights

void dijkstra() {
    memset(vis, false, sizeof(vis));
    for (int i = 1; i <= n; i++)
        dis[i] = LLONG_MAX;
    dis[1] = 0;
    priority_queue<pii, vector<pii>, greater<pii>> q; // Min-heap
    q.push({0, 1}); // Start with vertex 1 and cost 0

    while (!q.empty()) {
        auto [cost, u] = q.top();
        q.pop();
        if (vis[u]) continue;
        vis[u] = true;
        for (auto [v, w] : g[u]) {
            ll new_cost = dis[u] + a[u] + w;
            if (dis[v] > new_cost) {
                dis[v] = new_cost;
                q.push({new_cost, v});
            }
        }
    }
}

int main() {
    scanf("%d %d", &n, &m);
    for (int i = 1; i <= n; i++)
        scanf("%lld", &a[i]);
    for (int i = 1; i <= m; i++) {
        int u, v, w;
        scanf("%d %d %d", &u, &v, &w);
        g[u].push_back({v, w});
        g[v].push_back({u, w});
    }
    dijkstra();
    for (int i = 2; i <= n; i++)
        printf("%lld ", dis[i]);
    return 0;
}
