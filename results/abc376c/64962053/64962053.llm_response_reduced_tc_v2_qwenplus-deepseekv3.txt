### Correct Approach
1. **Sort Both Arrays**: Sorting the toys and boxes allows us to efficiently compare their sizes.
2. **Check Placement Feasibility**: We need to ensure that each toy can be placed in a box that is at least as large as the toy. Since we have one extra box (the one we purchase), we can choose the smallest possible \( x \) that allows all toys to be placed appropriately.
3. **Determine Minimum \( x \)**:
   - The key observation is that the largest toy must be placed in either the largest existing box or the new box \( x \).
   - Similarly, the second largest toy must be placed in the second largest existing box or \( x \), and so on.
4. **Algorithm**:
   - Sort both the toys and the boxes in non-decreasing order.
   - For each toy from the largest to the smallest, check if it can be placed in the corresponding box (from the largest to the smallest). If not, the new box \( x \) must be at least as large as the toy.

#include <bits/stdc++.h>
using namespace std;

int main() {
    int n;
    cin >> n;
    vector<int> A(n);
    vector<int> B(n - 1);

    for (int i = 0; i < n; ++i) {
        cin >> A[i];
    }
    for (int i = 0; i < n - 1; ++i) {
        cin >> B[i];
    }

    sort(A.begin(), A.end());
    sort(B.begin(), B.end());

    // Check if we can place all toys without purchasing a new box
    bool ok = true;
    for (int i = 0; i < n - 1; ++i) {
        if (A[i] > B[i]) {
            ok = false;
            break;
        }
    }
    if (ok) {
        cout << A.back() << endl;
        return 0;
    }

    // Now, we need to purchase a new box x.
    // The minimal x is the minimal value such that when added to B, each toy <= box.
    // We need to find the minimal x >= the largest toy that cannot be placed in existing boxes.
    // The largest such toy is the first one in A (sorted) that is > corresponding B[i].

    int x = -1;
    for (int i = 0; i < n; ++i) {
        if (i < n - 1) {
            if (A[i] > B[i]) {
                if (x == -1) {
                    x = A[i];
                } else {
                    // More than one toy cannot be placed in existing boxes.
                    cout << -1 << endl;
                    return 0;
                }
            }
        } else {
            if (x == -1) {
                x = A[i];
            }
        }
    }

    // Now, check if with x, all toys can be placed.
    vector<int> new_B = B;
    new_B.push_back(x);
    sort(new_B.begin(), new_B.end());

    ok = true;
    for (int i = 0; i < n; ++i) {
        if (A[i] > new_B[i]) {
            ok = false;
            break;
        }
    }

    if (ok) {
        cout << x << endl;
    } else {
        cout << -1 << endl;
    }

    return 0;

This code first checks if no new box is needed. If a new box is needed, it looks for the minimal x that can accommodate all toys. If more than one toy cannot be placed in existing boxes, it returns -1. Otherwise, it checks if the chosen x allows all toys to be placed. If not, it returns -1.
