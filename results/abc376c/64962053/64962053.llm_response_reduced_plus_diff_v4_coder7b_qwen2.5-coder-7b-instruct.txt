2. **Calculate the Total Box Size**: Sum up the sizes of all existing boxes.
3. **Check if the Largest Toy Fits**: If the largest toy size is greater than the total box size, then we need at least one additional box of size equal to the largest toy size.
4. **Output the Result**: If the largest toy fits, output `-1`. Otherwise, output the size of the largest toy as the minimum additional box size.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n;
    cin >> n;
    vector<int> va(n);
    vector<int> vb(n - 1);

    for (int i = 0; i < n; i++) {
        cin >> va[i];
    }
    for (int i = 0; i < n - 1; i++) {
        cin >> vb[i];
    }

    // Sort the toy sizes
    sort(va.begin(), va.end());
    sort(vb.begin(), vb.end());

    // Calculate the total size of existing boxes
    long long totalBoxSize = accumulate(vb.begin(), vb.end(), 0LL);

    // Check if the largest toy can fit in any of the existing boxes
    if (va[n - 1] <= totalBoxSize) {
        cout << "-1" << endl;
    } else {
        cout << va[n - 1] << endl;
    }

    return 0;
}
```

### Explanation:
1. **Sorting**: We sort both the toy sizes (`va`) and the box sizes (`vb`).
2. **Total Box Size Calculation**: We use `accumulate` to sum up the sizes of all existing boxes.
3. **Comparison**: We compare the size of the largest toy (`va[n - 1]`) with the total size of the existing boxes. If the largest toy fits, we output `-1`; otherwise, we output the size of the largest toy.
