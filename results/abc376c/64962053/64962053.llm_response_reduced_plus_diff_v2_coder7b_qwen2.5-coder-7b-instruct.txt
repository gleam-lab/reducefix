#include <bits/stdc++.h>

    // Read input sizes
    for(int i=0;i<n;++i) cin >> a[i];

    // Sort both arrays
    sort(a.begin(), a.end());
    sort(b.begin(), b.end());

    // Initialize variables
    int i = 0, j = 0;
    while(i < n && j < n-1) {
        // If current toy fits in the current box
        if(a[i] <= b[j]) {
            ++i;
        } 
        // If toy doesn't fit, try next box
        else {
            ++j;
        }
    }

    // Check if we've successfully placed all toys
    if(i == n) {
        cout << b[j] << '\n';
    } 
    else {
        cout << "-1\n";
    }

    return 0;
}

This solution works as follows:

1. **Reading Input:** We read the number of toys `n` followed by their sizes `a[i]` and the sizes of the existing boxes `b[i]`.

2. **Sorting Arrays:** Both the array of toy sizes (`a`) and the array of box sizes (`b`) are sorted to facilitate efficient placement.

3. **Placement Logic:** We use two pointers `i` and `j`. The pointer `i` iterates over the toys, and `j` iterates over the existing boxes. For each toy, we check if it can fit into the current box. If it does, we move to the next toy (`i++`). If it doesn't, we try the next box (`j++`).

4. **Check Completion:** After attempting to place all toys, if `i` equals `n`, it means all toys have been successfully placed, and we output the size of the last used box (`b[j]`). If not, it means it's impossible to place all toys, and we output `-1`.
