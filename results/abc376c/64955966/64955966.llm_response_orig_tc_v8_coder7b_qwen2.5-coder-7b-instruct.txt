This code sorts both arrays, then iterates through them checking if the current toy can fit in the current box. If at any point it cannot, it outputs -1 and returns. If it goes through all toys without finding an issue, it outputs the size of the last toy. This ensures we always use the smallest possible box that fits all toys. The time complexity is O(N log N) due to the sorting operations.
