```cpp
#include<bits/stdc++.h>
#define ll long long
#define yes cout<<"YES\n"
#define no cout<<"NO\n"
#define sp ' '
using namespace std;

void solve(){
    ll N;
    cin >> N;
    vector<ll> a(N), b(N-1);
    for(int i = 0; i < N; i++) cin >> a[i];
    for(int i = 0; i < N-1; i++) cin >> b[i];

    sort(a.begin(), a.end());
    sort(b.begin(), b.end());

    ll max_a = *max_element(a.begin(), a.end());
    ll min_b = *min_element(b.begin(), b.end());

    if(max_a > min_b){
        cout << -1 << endl;
    } else {
        cout << max_a << endl;
    }
}

int main(){
    ios::sync_with_stdio(0);
    cin.tie(0);
    solve();
    return 0;
}
```

### Explanation
1. **Input Reading**: The code reads the number of toys and their sizes, as well as the sizes of the boxes.
2. **Sorting**: Both the array of toy sizes (`a`) and the array of box sizes (`b`) are sorted.
3. **Finding Minimum Box Size**: The maximum size of any toy (`max_a`) is compared with the minimum size of any box (`min_b`). If `max_a` is greater than `min_b`, it means there's no box big enough to hold the largest toy, so the output is `-1`. Otherwise, the largest toy's size is the minimum box size needed, which is printed.
