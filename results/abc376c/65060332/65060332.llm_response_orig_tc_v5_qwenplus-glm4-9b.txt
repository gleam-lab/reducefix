
To solve this problem, we need to find the minimum box size \( x \) such that every toy can be placed in a box whose size is not less than the toy's size, and no box can contain two or more toys. The key insight is that \( x \) must be at least the size of the largest toy that cannot be placed in any of the \( N-1 \) existing boxes.

Here's a step-by-step breakdown of the approach:

1. For each box \( B_i \), find the smallest toy that cannot fit in it. This can be done efficiently by sorting the toys by size and iterating through the sorted list with the current box size in mind.

2. Determine the maximum of these smallest toys that couldn't fit in any box. This maximum size is the minimum \( x \) we are looking for.

Here's how you can implement this algorithm:

```cpp
#include<bits/stdc++.h>
#define ll long long
using namespace std;

const ll N = 2e5 + 5;
ll a[N], b[N];

int main() {
    ll n;
    cin >> n;
    for (ll i = 0; i < n; ++i) cin >> a[i];
    for (ll i = 0; i < n - 1; ++i) cin >> b[i];

    sort(a, a + n);
    sort(b, b + n - 1);

    ll maxToyThatCouldNotFit = -1;
    for (ll i = 0, j = 0; i < n; ++i) {
        // Find the next box that can fit the current toy a[i]
        while (j < n - 1 && b[j] < a[i])
            ++j;

        // If we have run out of boxes, the current toy a[i] is the largest toy
        // that could not fit in any box, so set maxToyThatCouldNotFit to a[i]
        if (j == n - 1) {
            maxToyThatCouldNotFit = max(maxToyThatCouldNotFit, a[i]);
        }
    }

    // If there was a toy that could not fit in any box, print its size as the answer
    // Otherwise, print -1
    cout << (maxToyThatCouldNotFit == -1 ? -1 : maxToyThatCouldNotFit);
    return 0;
}
```

This solution first sorts the toy sizes \( A \) and the box sizes \( B \), then iterates through the sorted toy sizes while searching for the first box that can't fit the current toy. If we reach the end of the boxes without finding a fitting one, we know we have found the maximum toy size that can't be placed, and that is our answer. If all toys can be placed in the existing boxes, the answer is `-1`.
