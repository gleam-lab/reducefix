### Explanation
1. **Sorting**: Both the toy sizes and box sizes are sorted to apply a greedy matching strategy.
2. **Two-pointer Technique**: We iterate through both sorted arrays to match each toy to the smallest possible box that can accommodate it. This ensures optimal use of existing boxes.
3. **Unmatched Toys**: Toys that cannot be matched to any existing box are collected in a list. If this list contains more than one toy, it's impossible to place all toys with the new box, hence the answer is `-1`.
