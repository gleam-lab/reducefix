```cpp
//<<In the name of Allah, The Most Gracious and The Most Merciful>>
#include<bits/stdc++.h>
using namespace std;
#define ll long long

void solve()
{
    ll n;
    cin >> n;
    vector<ll> a(n), b(n - 1);
    for (ll i = 0; i < n; ++i)
        cin >> a[i];
    for (ll i = 0; i < n - 1; ++i)
        cin >> b[i];
    
    sort(a.begin(), a.end());
    sort(b.begin(), b.end());

    ll c = 0, ans = a[0];
    for (ll i = n - 1, j = n - 2; i >= 0 && j >= 0;) {
        if (a[i] <= b[j]) {
            i--, j--;
        } else {
            c++, ans = max(ans, a[i]), i--;
        }
    }

    if (c > 1)
        cout << -1 << endl;
    else
        cout << ans << endl;
}

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(0);
    cout.tie(0);
    solve();
}
```

### Explanation of Changes:
1. **Use of `vector` instead of arrays**: This change enhances readability and makes managing dynamic sizes easier.
2. **Input/Output Optimization**: Added `ios_base::sync_with_stdio(false)` and `cin.tie(0); cout.tie(0);` to speed up I/O operations.
3. **Fixed Variable Naming**: Changed `x` to `ans` to clearly indicate the variable holds the answer.
4. **Corrected Logic for Box Purchase Condition**: Ensured that the loop continues until both indices reach their limits, ensuring all conditions are checked correctly.
5. **Edge Case Handling**: Correctly handles scenarios where multiple boxes might need to be purchased to accommodate all toys, returning `-1` as required.
