```cpp
#include<bits/stdc++.h>
using namespace std;
#define ll long long

void solve() {
    ll n;
    cin >> n;
    vector<ll> a(n), b(n - 1);
    for (ll i = 0; i < n; i++) {
        cin >> a[i];
    }
    for (ll i = 0; i < n - 1; i++) {
        cin >> b[i];
    }
    
    // Sort both arrays to facilitate pairing
    sort(a.begin(), a.end());
    sort(b.begin(), b.end());
    
    // Initialize variables
    ll max_b = *max_element(b.begin(), b.end()); // Find the maximum box size
    ll min_a = *min_element(a.begin(), a.end()); // Find the minimum toy size
    
    // If the largest box is smaller than the smallest toy, it's impossible to fit any toy
    if (max_b < min_a) {
        cout << -1 << endl;
        return;
    }
    
    // The answer will be the size of the smallest box that can hold the largest toy
    ll ans = ceil((double)a.back() / b.size()); // Calculate the minimum box size needed
    
    cout << ans << endl;
}

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    solve();
}
```

### Explanation of Changes:
1. **Vector Usage**: Changed arrays `a` and `b` to vectors for better flexibility and ease of operations like finding the maximum and minimum elements.
2. **Efficient Calculation**: Directly calculated the minimum box size required using `ceil((double)a.back() / b.size())`, which ensures we get the smallest box that can accommodate the largest toy.
3. **Edge Case Handling**: Added a check at the beginning to handle the case where the largest box is smaller than the smallest toy, making it impossible to fit any toy.
4. **Optimization**: Used built-in functions like `sort`, `max_element`, and `min_element` for cleaner and potentially faster code execution.
5. **I/O Optimization**: Added `ios_base::sync_with_stdio(false)` and `cin.tie(NULL)` to optimize I/O operations.
