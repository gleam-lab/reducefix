```cpp
#include<bits/stdc++.h>
using namespace std;
#define ll long long

void solve()
{
    ll n;
    cin >> n;
    vector<ll> a(n), b(n - 1);
    for (ll i = 0; i < n; i++) {
        cin >> a[i];
    }
    for (ll i = 0; i < n - 1; i++) {
        cin >> b[i];
    }
    
    // Sort both arrays to facilitate pairing
    sort(a.begin(), a.end());
    sort(b.begin(), b.end());
    
    // Initialize variables to track the number of boxes needed
    ll maxBoxSize = 0, count = 0;
    
    // Iterate through the sorted arrays to determine the maximum box size required
    for (ll i = n - 1, j = n - 2; i >= 0 && j >= 0;) {
        if (a[i] <= b[j]) {
            i--;
        } else {
            count++;
            maxBoxSize = max(maxBoxSize, a[i]);
            i--;
        }
    }
    
    // If we need more than one additional box, output -1
    if (count > 1) {
        cout << -1 << endl;
    } else {
        // Otherwise, output the size of the largest box needed
        cout << maxBoxSize << endl;
    }
}

int main() {
    solve();
    return 0;
}
```

### Explanation
1. **Input Reading**: The code reads the number of toys and their sizes along with the sizes of the existing boxes.
2. **Sorting**: Both arrays `a` (toy sizes) and `b` (box sizes) are sorted to facilitate efficient pairing.
3. **Pairing Logic**: We iterate from the end of both arrays, trying to place the largest remaining toy into the largest available box that can fit it. If a toy cannot fit into any existing box, we increment a counter and record the size of the toy as the potential new box size.
4. **Result Calculation**: If more than one new box is needed, we output `-1`. Otherwise, we output the size of the largest box required.

This approach ensures that we find the minimum possible box size efficiently while handling the constraints effectively.
