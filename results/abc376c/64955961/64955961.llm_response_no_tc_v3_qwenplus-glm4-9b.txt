
```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int findMinimumBoxSize(int n, vector<int>& a, vector<int>& b) {
    vector<int> maxOfLeft(n);
    maxOfLeft[0] = a[0];

    // Calculate max of the left part
    for (int i = 1; i < n; i++) {
        maxOfLeft[i] = max(maxOfLeft[i - 1], a[i]);
    }

    // Binary search for the smallest x
    int low = 0, high = 1e9 + 1;
    while (high - low > 1) {
        int mid = (low + high) / 2;
        int i = 0, j = 0;
        int count = 0;
        // Check if all toys can be placed in boxes of size <= mid
        while (i < n && j < n - 1) {
            if (maxOfLeft[i] <= b[j] || a[i] <= mid) {
                i++;
            } else {
                count++;
                j++;
            }
        }
        // Adjust binary search bounds
        if (count <= n - 1) {
            high = mid;
        } else {
            low = mid;
        }
    }
    return (count <= n - 1) ? low : -1;
}

int main() {
    int n;
    cin >> n;
    vector<int> a(n), b(n - 1);
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }
    for (int i = 0; i < n - 1; i++) {
        cin >> b[i];
    }

    int result = findMinimumBoxSize(n, a, b);
    cout << result << endl;
    return 0;
}
```

### Explanation:

1. The function `findMinimumBoxSize` is created to encapsulate the logic for finding the minimum box size.
2. The array `maxOfLeft` is used to store the maximum value of toy sizes from the start to the current index. This helps in quickly determining the largest toy that can be placed in the first `i` boxes without exceeding the size `x`.
3. A binary search is performed on the possible box sizes between `1` and `1e9 + 1` to find the smallest box size `x` that allows all toys to be placed in boxes.
4. If the count of toys that cannot be placed in the existing boxes is less than or equal to `n - 1`, then the current middle value of the binary search is a valid candidate for `x`.
5. The main function reads the input, calls the `findMinimumBoxSize` function, and prints the result.
