```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios_base::sync_with_stdio(false);cin.tie(NULL);cout.tie(NULL);

    int n;
    cin >> n;
    vector<int> a(n), b(n-1);
    
    for(int i=0;i<n;i++) cin>>a[i];
    for(int i=0;i<n-1;i++) cin>>b[i];

    sort(a.begin(), a.end());
    sort(b.begin(), b.end());

    long long sum = 0;
    for(int i=0;i<n-1;i++) sum += b[i];
    
    if(sum >= a[n-1]) cout << a[n-1] << "\n";
    else cout << "-1\n";

    return 0;
}
```
This solution calculates the total size of all boxes except the last one (`sum`) and compares it with the size of the largest toy (`a[n-1]`). If the sum of the smaller boxes is greater than or equal to the size of the largest toy, then the largest toy can be placed in any of the available boxes without exceeding their sizes. Therefore, the minimum box size needed would be the size of the largest toy itself. If the sum of the smaller boxes is less than the size of the largest toy, it means we cannot place the largest toy in any of the available boxes because they do not have enough space. In this case, we output `-1`.
