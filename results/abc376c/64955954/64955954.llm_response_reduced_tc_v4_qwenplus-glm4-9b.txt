2. **Checking for Fit**: Starting with the smallest box that can fit the smallest toy, we need to check if all subsequent toys can fit into the existing boxes or a new box of the same size. If any toy cannot fit, we increment the box size and try again.

3. **Finding Minimum Box Size**: We need to find the smallest box size that can fit all toys. If no such box size exists, we should return `-1`.

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    int n;
    cin >> n;
    vector<int> toys, boxes;
    for (int i = 0; i < n; ++i) {
        int size;
        cin >> size;
        toys.push_back(size);
    }
    for (int i = 0; i < n - 1; ++i) {
        int size;
        cin >> size;
        boxes.push_back(size);
    }

    // Sort the toy and box sizes
    sort(toys.begin(), toys.end());
    sort(boxes.begin(), boxes.end());

    // Binary search for the minimum box size
    int left = 0, right = 1e9 + 1; // Set the upper bound higher than any possible box size
    while (left < right) {
        int mid = left + (right - left) / 2;
        int i = 0, j = 0;
        bool canFit = true;

        // Check if all toys can fit into the boxes of size mid
        while (i < n && j < n - 1) {
            if (toys[i] <= boxes[j]) {
                ++i;
            }
            ++j;
        }

        // If all toys fit, try a smaller box size
        if (i == n) {
            right = mid;
        } else {
            left = mid + 1;
        }
    }

    // If the left boundary is still greater than the right, it means no box can fit
    if (left > right) {
        cout << "-1" << endl;
    } else {

    }

    return 0;
}

### Explanation

1. **Input and Sorting**: We read the sizes of the toys and boxes and sort them.

2. **Binary Search**: We use binary search to find the smallest box size that can fit all toys. This is done by checking if all toys can fit into a box of a given size.

3. **Checking Fit**: We use a two-pointer technique to determine if the toys can fit. The first pointer `i` iterates over the toys and the second pointer `j` iterates over the boxes. We increment `i` when a toy can fit and `j` when we move to the next box. If we can iterate through all toys (`i == n`), then all toys fit.

4. **Output**: Finally, we print the smallest box size that can fit all toys or `-1` if no such box size exists.
