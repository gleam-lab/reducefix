bool canPlace(vector<int>& toys, vector<int>& boxes) {

    int m = boxes.size();

    // Sort toys and boxes in descending order
    sort(toys.begin(), toys.end(), greater<int>());
    sort(boxes.begin(), boxes.end(), greater<int>());

    // Iterate through the sorted toys
    for (int i = 0; i < n; ++i) {
        // If current toy cannot fit into any available box, return false
        if (toys[i] > boxes.back()) {
            return false;
        }
        // Remove the largest box that can fit the current toy
        while (!boxes.empty() && boxes.back() >= toys[i]) {
            boxes.pop_back();
        }
    }

    return true;
}

1. **Sorting**: Both the `toys` and `boxes` vectors are sorted in descending order. This ensures that we always try to fit the largest remaining toy into the largest possible box.

2. **Placement Check**: We iterate through the sorted list of toys. For each toy, we check if there is at least one box available that can hold it. If no such box is found, it means we need an additional box, and we return `-1`.

3. **Output**: If it's possible to place all toys using the existing boxes, we output the size of the smallest toy that needs to be placed. If not, we output `-1`.
