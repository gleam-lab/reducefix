
```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main() {
    int n;
    cin >> n;
    vector<int> va(n), vb(n);
    for (int i = 0; i < n; i++) {
        cin >> va[i];
    }
    for (int i = 0; i < n - 1; i++) {
        cin >> vb[i];
    }

    // Instead of sorting in descending order, we need to find the smallest x
    // that can fit all toys into boxes.
    // We can use a two-pointer technique to find the minimum x.
    sort(vb.rbegin(), vb.rend()); // Sort the box sizes in descending order for easier iteration.

    int start = 0; // Start index for va
    int end = n - 1; // End index for va
    bool canFit = true;

    while (start <= end) {
        if (vb[start] >= va[start]) {
            start++;
        } else if (vb[start] >= va[end]) {
            end--;
        } else {
            canFit = false;
            break;
        }
    }

    if (canFit) {
        if (end < start) {
            cout << vb[start] << endl;
        } else {
            cout << vb[start] + 1 << endl;
        }
    } else {
        cout << -1 << endl;
    }

    return 0;
}
```

### Explanation

1. We sort the box sizes in descending order to make it easier to iterate over them.
2. We use a two-pointer approach to find the smallest box size that can fit all toys:
   - `start` pointer iterates over the toys from the largest to the smallest.
   - `end` pointer iterates over the boxes from the largest to the smallest.
3. If the largest box at `start` index can fit the smallest toy at `start` index, we move to the next toy.
4. If the largest box at `start` index can fit the largest toy at `end` index, we move to the next box.
5. If the largest box at `start` index cannot fit the largest toy at `end` index, it means we cannot fit all toys into any box, so we set `canFit` to false and break out of the loop.
6. If all toys can be placed in boxes, we check if all toys are placed in boxes that are smaller than or equal to their size. If so, the minimum box size is `vb[start]`. If not, we need an additional box of size `vb[start] + 1`.
7. If it is not possible to fit all toys into the boxes, we return `-1`.
