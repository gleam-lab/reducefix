The provided code calculates the cost of moving from one point to the next by summing up the distances. However, it does not account for the return trip to the origin, which is necessary to get the total cost of the journey.

1. Initialize the `px` and `py` variables to the origin coordinates, i.e., `px = 0` and `py = 0`.
2. Traverse through the list of points and for each point, calculate the distance from the last point using the `hypot()` function.
3. Add this distance to the `ans` variable.
4. After traversing through all points, add the distance from the last point back to the origin to the `ans`.
5. Print the total cost `ans`.
