### Approaches
1. **Calculate the Euclidean Distance**: For each move from point \((a, b)\) to point \((c, d)\), compute the Euclidean distance \(\sqrt{(a - c)^2 + (b - d)^2}\).
2. **Accumulate the Total Cost**: Sum all the distances from the origin to the first point, between consecutive points, and back from the last point to the origin.
3. **Output with High Precision**: Ensure the final result is printed with enough decimal places to meet the precision requirement. Using `std::setprecision` with a sufficiently high value (like 20) ensures this.

### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    long double total_cost = 0;
    int N;
    cin >> N;
    int nowX = 0, nowY = 0;
    int nextX, nextY;

    for (int i = 0; i < N; ++i) {
        cin >> nextX >> nextY;
        long double dx = nowX - nextX;
        long double dy = nowY - nextY;
        long double distance = sqrt(dx * dx + dy * dy);
        total_cost += distance;
        nowX = nextX;
        nowY = nextY;
    }

    long double dx = nowX - 0;
    long double dy = nowY - 0;
    long double distance = sqrt(dx * dx + dy * dy);
    total_cost += distance;

    cout << fixed << setprecision(20) << total_cost << endl;

    return 0;
}
```

### Explanation
1. **Initialization**: The total cost is initialized to zero, and the current position is set to the origin \((0, 0)\).
2. **Reading Input**: The number of points \(N\) is read, followed by the coordinates of each point.
3. **Distance Calculation**: For each point, the Euclidean distance from the current position to the next point is calculated and added to the total cost. The current position is then updated to the next point.
4. **Return to Origin**: After visiting all points, the distance from the last point back to the origin is calculated and added to the total cost.
5. **Output**: The total cost is printed with high precision (20 decimal places) to ensure the output meets the required accuracy.

This approach ensures that the solution is both correct and efficient, handling all edge cases and providing the required precision for the output.
