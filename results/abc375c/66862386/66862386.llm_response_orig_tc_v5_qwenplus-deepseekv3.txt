### Approaches
1. **Calculate Distances**: For each point, compute the Euclidean distance from the current position to the next point. The initial position is the origin (0, 0).
2. **Sum Distances**: Accumulate the distances between each consecutive point and finally add the distance from the last point back to the origin.
3. **Precision Handling**: Ensure the output is printed with enough decimal places to avoid scientific notation and match the expected precision. We'll use `std::fixed` and `std::setprecision` to control the output format.

### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    long double cost = 0;
    long double ans = 0;
    int N;
    long double nowX = 0, nowY = 0;
    long double nextX, nextY;

    cin >> N;

    for (int i = 0; i < N; i++) {
        cin >> nextX >> nextY;
        cost = sqrtl((nowX - nextX) * (nowX - nextX) + (nowY - nextY) * (nowY - nextY));
        nowX = nextX;
        nowY = nextY;
        ans += cost;
    }

    cost = sqrtl(nowX * nowX + nowY * nowY);
    ans += cost;

    cout << fixed << setprecision(15) << ans << endl;

    return 0;
}
```

### Explanation
1. **Initialization**: Variables are initialized to store the cumulative distance (`ans`), the current position (`nowX`, `nowY`), and the next point (`nextX`, `nextY`).
2. **Input Handling**: The number of points `N` is read first. Then, for each point, its coordinates are read.
3. **Distance Calculation**: For each point, the Euclidean distance from the current position to the next point is calculated using `sqrtl` for precision. The current position is updated to the next point after each calculation.
4. **Return to Origin**: After processing all points, the distance from the last point back to the origin is calculated and added to the total distance.
5. **Output**: The total distance is printed with high precision (15 decimal places) using `fixed` and `setprecision` to ensure the output format matches the expected result without scientific notation.

This approach ensures that the distances are calculated accurately and the output is formatted correctly to meet the problem's precision requirements.
