### Solution Code

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    long double ans = 0;
    int N;
    int nowX = 0, nowY = 0;
    int nextX, nextY;

    cin >> N;

    for (int i = 0; i < N; i++) {
        cin >> nextX >> nextY;
        long double dx = nextX - nowX;
        long double dy = nextY - nowY;
        ans += sqrtl(dx * dx + dy * dy);
        nowX = nextX;
        nowY = nextY;
    }

    long double dx = 0 - nowX;
    long double dy = 0 - nowY;
    ans += sqrtl(dx * dx + dy * dy);

    cout << fixed << setprecision(15) << ans << endl;

    return 0;
}
```

### Explanation
1. **Input Handling**: The code reads the number of points `N` and then reads each point's coordinates.
2. **Distance Calculation**: For each point, the Euclidean distance from the current position to the next point is calculated and added to the total distance `ans`.
3. **Return to Origin**: After visiting all points, the distance from the last point back to the origin is calculated and added to `ans`.
4. **Output Precision**: The result is printed with `fixed` notation and a precision of 15 decimal places to ensure the output matches the expected format exactly, avoiding scientific notation.

This approach ensures the algorithm is both correct and efficient, with a time complexity of O(N) where N is the number of points, which is optimal for this problem. The output formatting guarantees the result meets the required precision and format specifications.
