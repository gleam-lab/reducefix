```cpp
#include <iostream>
#include <cmath>

int main() {
    int N;
    std::cin >> N;

    // Initialize the starting position at the origin
    double startX = 0.0, startY = 0.0;
    double totalCost = 0.0;

    // Process each point
    for (int i = 0; i < N; ++i) {
        double nextX, nextY;
        std::cin >> nextX >> nextY;

        // Calculate the Euclidean distance to the next point
        double dx = nextX - startX;
        double dy = nextY - startY;
        double currentCost = std::sqrt(dx * dx + dy * dy);

        // Add the cost to the total
        totalCost += currentCost;

        // Update the current position
        startX = nextX;
        startY = nextY;
    }

    // Calculate the cost to return to the origin
    double finalCost = std::sqrt(startX * startX + startY * startY);
    totalCost += finalCost;

    // Output the total cost
    std::cout << totalCost << std::endl;

    return 0;
}
```

### Explanation of Changes:
1. **Initialization**: The starting position is initialized at the origin `(0.0, 0.0)` instead of `0`.
2. **Loop Variable Naming**: Changed `operation` to `N` as per the problem statement.
3. **Euclidean Distance Calculation**: Used `std::sqrt` for calculating the Euclidean distance between the current and next points.
4. **Total Cost Accumulation**: Added the cost of moving to the next point and the final cost of returning to the origin to `totalCost`.
5. **Output**: Printed the total cost directly without any unnecessary intermediate variables.

This solution ensures that the algorithm correctly calculates the total movement cost based on the Euclidean distance between consecutive points and includes the cost of returning to the origin.
