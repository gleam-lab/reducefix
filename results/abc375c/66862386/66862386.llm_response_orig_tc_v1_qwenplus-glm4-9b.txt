
The error in your code is due to a few issues:

1. You are using `powl` to calculate the square root, which requires the use of `pow` instead.
2. Your calculation of the distance between points is incorrect. You're taking the square root of the square of the distance, which is not necessary. Instead, you should simply sum up the squares of the differences in x and y, and then take the square root of the sum to get the Euclidean distance.
3. The final calculation of the distance from the last point back to the origin is incorrect. You should not recalculate it separately. Instead, it should be included in the loop.
4. The loop should iterate over `operation` (which is essentially `N`), but you're using `i < operation` instead. 

Here's the corrected code:

```cpp
#include <iostream>
#include <cmath> // For sqrt and pow

using namespace std;

int main() {
    int N;
    cin >> N;

    long double x = 0, y = 0;
    long double distance = 0;
    long double total_cost = 0;

    for (int i = 0; i < N; i++) {
        int nextX, nextY;
        cin >> nextX >> nextY;

        distance = sqrt(pow(nextX - x, 2) + pow(nextY - y, 2));

        total_cost += distance;

        x = nextX;
        y = nextY;
    }

    // Include the return journey to the origin
    total_cost += sqrt(pow(x, 2) + pow(y, 2));

    cout << total_cost << endl;

    return 0;
}
```

This will give you the correct total cost of moving from the origin to each point and back to the origin. The code now calculates the distance between each point correctly, adds it to the total cost, and at the end includes the cost of returning to the origin.
