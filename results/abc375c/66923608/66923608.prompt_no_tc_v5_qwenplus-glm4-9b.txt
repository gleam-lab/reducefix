### Problem Statement

Takahashi is at the origin on a two-dimensional coordinate plane.

The cost for him to move from point (a, b) to point (c, d) is \sqrt{(a - c)^2 + (b - d)^2}.

Find the total cost when he starts at the origin, visits N points (X\_1, Y\_1), \ldots, (X\_N, Y\_N) in this order, and then returns to the origin.

### Constraints

* 1 \leq N \leq 2 \times 10^5
* -10^9 \leq X\_i, Y\_i \leq 10^9
* All input values are integers.

---

### Input

The input is given from Standard Input in the following format:

```
N
X_1 Y_1
\vdots
X_N Y_N

```

### Output

Print the answer.
