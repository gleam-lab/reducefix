```cpp
#include <iostream>
#include <iomanip>
#include <cmath>
using namespace std;

int main() {
    int n;
    cin >> n;
    long long x = 0, y = 0;
    double total_cost = 0.0;
    
    for (int i = 0; i < n; ++i) {
        long long xi, yi;
        cin >> xi >> yi;
        double dx = x - xi;
        double dy = y - yi;
        total_cost += sqrt(dx * dx + dy * dy);
        x = xi;
        y = yi;
    }
    
    double dx = x;
    double dy = y;
    total_cost += sqrt(dx * dx + dy * dy);
    
    cout << fixed << setprecision(15) << total_cost << endl;
    return 0;
}
```

1. **Data Types**: The coordinates are read into `long long` variables to handle the large input range (-\(10^9\) to \(10^9\)) without overflow.
2. **Distance Calculation**: The distance between consecutive points is calculated using the Euclidean distance formula. The differences in coordinates are converted to `double` before squaring to avoid integer overflow.
3. **Accumulation**: The total cost is accumulated in a `double` variable to maintain sufficient precision throughout the calculations.
4. **Final Distance**: After visiting all points, the distance back to the origin is added to the total cost.
5. **Output**: The result is printed with a precision of 15 decimal places to meet the problem's accuracy requirements.
