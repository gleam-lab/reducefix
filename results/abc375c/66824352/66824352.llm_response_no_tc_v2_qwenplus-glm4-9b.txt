
```cpp
#include <iostream>
#include <cmath>

using namespace std;

int main() {
    int n;
    cin >> n;
    double x = 0.0, y = 0.0, total_cost = 0.0;

    for (int i = 0; i < n; ++i) {
        int X, Y;
        cin >> X >> Y;
        total_cost += sqrt((X - x) * (X - x) + (Y - y));
        x = X;
        y = Y;
    }

    total_cost += sqrt(x * x + y * y);
    cout << fixed << setprecision(10) << total_cost << endl;

    return 0;
}
```

### Explanation:
1. The total cost calculation was initially incorrect because it was not storing the previous position correctly after each move, causing a double count. The fixed code correctly accumulates the distances between consecutive points.
2. The corrected code calculates the distance between two points using the Euclidean distance formula without needing to store all points in an array.
3. The code now initializes `x` and `y` to `0.0` to represent the origin and calculates the total cost as it reads the input points.
4. After processing all points, the code adds the distance from the last point back to the origin, ensuring the total trip cost includes the return journey.
5. The output is formatted with a precision that meets the problem's requirements.
