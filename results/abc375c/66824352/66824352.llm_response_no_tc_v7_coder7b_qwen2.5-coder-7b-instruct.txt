```cpp
#include <iostream>
#include <algorithm>
#include <iomanip>
#include <cmath>
#include <vector>
using namespace std;

int main() {
    int n;
    double totalCost = 0;
    cin >> n;
    vector<pair<int, int>> points(n);
    
    // Read all points
    for (int i = 0; i < n; ++i) {
        cin >> points[i].first >> points[i].second;
    }

    // Calculate the cost of moving from one point to another
    for (size_t i = 0; i < n - 1; ++i) {
        totalCost += sqrt(pow(points[i].first - points[i + 1].first, 2) +
                          pow(points[i].second - points[i + 1].second, 2));
    }

    // Add the cost of returning to the origin
    totalCost += sqrt(pow(points[n - 1].first, 2) + pow(points[n - 1].second, 2));

    // Print the total cost with high precision
    cout << fixed << setprecision(10) << totalCost << endl;
    return 0;
}
```

### Explanation of Fixes:
1. **Data Structure**: Changed `pair<int, int> a[n]` to `vector<pair<int, int>> points(n)` for better flexibility and ease of use.
2. **Variable Initialization**: Removed unnecessary initialization of `x` and `y`.
3. **Loop for Cost Calculation**: Correctly looped through the points to calculate the distance between consecutive points and added the final distance back to the origin.
4. **Output Precision**: Used `fixed` and `setprecision(10)` to ensure the output matches the required precision.
