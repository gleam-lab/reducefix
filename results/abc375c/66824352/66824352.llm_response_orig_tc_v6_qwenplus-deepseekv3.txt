### Solution Code
```cpp
#include <iostream>
#include <iomanip>
#include <cmath>
using namespace std;

int main() {
    int n;
    cin >> n;
    double total_cost = 0.0;
    double prev_x = 0.0, prev_y = 0.0;

    for (int i = 0; i < n; ++i) {
        double x, y;
        cin >> x >> y;
        double dx = x - prev_x;
        double dy = y - prev_y;
        total_cost += sqrt(dx * dx + dy * dy);
        prev_x = x;
        prev_y = y;
    }

    // Return to origin
    double dx = 0.0 - prev_x;
    double dy = 0.0 - prev_y;
    total_cost += sqrt(dx * dx + dy * dy);

    cout << fixed << setprecision(15) << total_cost << endl;
    return 0;
}
```

### Explanation
1. **Initialization**: The variables `prev_x` and `prev_y` are initialized to 0.0, representing the origin. The `total_cost` is initialized to 0.0 to accumulate the total distance.
2. **Loop Through Points**: For each point in the input, the code calculates the Euclidean distance from the current point to the previous point. This distance is added to `total_cost`.
3. **Return to Origin**: After visiting all points, the code calculates the distance from the last point back to the origin and adds this to `total_cost`.
4. **Output**: The total cost is printed with a precision of 15 decimal places to ensure accuracy, using `fixed` and `setprecision` from `<iomanip>`.

This approach ensures that all calculations are done in double precision, avoiding overflow and underflow issues, and correctly accumulates the total travel distance.
