```cpp
#include <iostream>
#include <vector>
#include <cmath>
#include <iomanip>
using namespace std;

double calculateDistance(int x1, int y1, int x2, int y2) {
    return sqrt(static_cast<double>(pow(x2 - x1, 2) + pow(y2 - y1, 2)));
}

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    int n;
    cin >> n;
    vector<pair<int, int>> points(n);

    for (int i = 0; i < n; ++i) {
        cin >> points[i].first >> points[i].second;
    }

    double totalCost = 0;

    // Calculate the cost of moving from one point to another
    for (size_t i = 1; i < points.size(); ++i) {
        totalCost += calculateDistance(points[i - 1].first, points[i - 1].second,
                                       points[i].first, points[i].second);
    }

    // Add the cost of returning to the origin
    totalCost += calculateDistance(points[0].first, points[0].second, 0, 0);

    cout << fixed << setprecision(10) << totalCost << endl;

    return 0;
}
```

### Explanation:
1. **Distance Calculation**: The `calculateDistance` function computes the Euclidean distance between two points using the formula \( \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2} \). We cast one of the terms to `double` to ensure floating-point division rather than integer division.

2. **Reading Input**: The program reads the number of points and their coordinates into a vector of pairs.

3. **Total Cost Calculation**:
   - The loop iterates through the points starting from the second point and calculates the distance from each point to the previous one.
   - After the loop, it adds the distance from the last point back to the origin.

4. **Output**: The total cost is printed with up to 10 decimal places of precision.

This approach ensures that the code handles large inputs efficiently and correctly calculates the total cost of the journey.
