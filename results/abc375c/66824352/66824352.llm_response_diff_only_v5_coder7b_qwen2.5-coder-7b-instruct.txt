```cpp
#include <iostream>
#include <iomanip>
#include <cmath>
using namespace std;

int main() {
    int n;
    double c = 0;
    cin >> n;
    pair<int, int> prev = {0, 0}; // Starting at the origin
    for (int i = 0; i < n; ++i) {
        int x, y;
        cin >> x >> y;
        c += sqrt(pow(prev.first - x, 2) + pow(prev.second - y, 2)); // Calculate distance to next point
        prev = {x, y}; // Update current position
    }
    c += sqrt(pow(prev.first, 2) + pow(prev.second, 2)); // Return to origin
    cout << setprecision(10) << c << endl;
    return 0;
}
```

### Explanation of Changes:
1. **Initialization**: The starting point was not initialized properly, so I updated it to `{0, 0}` representing the origin.
2. **Distance Calculation**: The formula used to calculate the distance between two points `(x1, y1)` and `(x2, y2)` should use `pow` instead of multiplying directly to ensure correct computation.
3. **Loop Optimization**: No significant optimization was needed as the loop logic was already efficient for the problem constraints.
4. **Output Precision**: The precision was set correctly to match the requirements.
