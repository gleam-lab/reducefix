```cpp
#include <iostream>
#include <cmath>
#include <iomanip>

int main() {
    std::ios::sync_with_stdio(false);
    std::cin.tie(nullptr);

    int N;
    std::cin >> N;

    double curX = 0, curY = 0;
    double total = 0; 

    for (int i = 0; i < N; ++i) {
        double x, y;
        std::cin >> x >> y;

        double dx = curX - x;
        double dy = curY - y;
        total += std::sqrt(dx * dx + dy * dy);

        curX = x; 
        curY = y;
    }
    total += std::sqrt(curX * curX + curY * curY); // 原点へ戻る

    std::cout << std::fixed << std::setprecision(10) << total << '\n';

    return 0;
}
```

### Explanation of Changes:
1. Changed data types: Used `double` instead of `long long` for coordinates and distance calculations to handle floating-point numbers accurately.
2. Calculated Euclidean distance correctly: Used `std::sqrt()` function to calculate the square root which is essential for the distance formula.
3. Printed result with high precision: Utilized `std::fixed` and `std::setprecision(10)` to ensure the output matches the required accuracy.
4. Removed unnecessary string conversion: Directly printed the calculated total distance as a double.
