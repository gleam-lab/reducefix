```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int N;
    cin >> N;

    vector<pair<double, double>> points(N);
    for (int i = 0; i < N; ++i) {
        cin >> points[i].first >> points[i].second;
    }

    double total = 0.0;
    for (int i = 0; i < N - 1; ++i) {
        double dx = points[i + 1].first - points[i].first;
        double dy = points[i + 1].second - points[i].second;
        total += sqrt(dx * dx + dy * dy);
    }

    // Return to origin
    double dx = points[0].first;
    double dy = points[0].second;
    total += sqrt(dx * dx + dy * dy);

    printf("%.10f\n", total);

    return 0;
}
```

### Explanation:
1. **Reading Points**: We read the number of points `N` and store each point's coordinates in a `vector` of pairs.
2. **Calculating Distances**: For each pair of consecutive points, we calculate the Euclidean distance using the formula `sqrt((x2 - x1)^2 + (y2 - y1)^2)` and accumulate it in `total`.
3. **Returning to Origin**: After visiting all points, we calculate the distance back to the origin and add it to `total`.
4. **Output**: Finally, we print the total cost with up to 10 decimal places using `printf`.

This approach ensures that we handle large numbers and provide accurate floating-point results. The time complexity is O(N), which is efficient for the given constraints.
