```cpp
#include <iostream>
#include <cmath>
#include <string>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int N;
    cin >> N;

    double curX = 0, curY = 0;
    double total = 0;

    for (int i = 0; i < N; ++i) {
        long long x, y;
        cin >> x >> y;

        double dx = curX - x;
        double dy = curY - y;
        total += sqrt(dx * dx + dy * dy);

        curX = x; curY = y;
    }
    total += sqrt(curX * curX + curY * curY); // 原点へ戻る

    printf("%.10f\n", total);

    return 0;
}
```

Explanation of Changes:
1. Changed `long long` to `double` for coordinates and distances to handle large numbers more accurately and avoid overflow issues.
2. Used `sqrt` function to calculate the Euclidean distance between points.
3. Removed unnecessary conversion to `__int128` as it was not needed and could lead to overflow.
4. Used `printf` with a precision of 10 to ensure the output matches the required accuracy.
5. Simplified the code structure for better readability and performance.
