```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    long double cost = 0;
    long double ans = 0;
    int N;
    int nowX = 0, nowY = 0;
    int nextX, nextY;

    cin >> N;

    for (int i = 0; i < N; i++) {
        cin >> nextX >> nextY;

        // Calculate the Euclidean distance squared to avoid computing square roots multiple times
        long double dx = nowX - nextX;
        long double dy = nowY - nextY;
        cost = dx * dx + dy * dy;

        ans += sqrtl(cost);  // Take the square root of the squared distance

        nowX = nextX;
        nowY = nextY;
    }

    // Return to the origin
    cost = nowX * nowX + nowY * nowY;
    ans += sqrtl(cost);

    cout << fixed << setprecision(10) << ans << endl;  // Print with high precision

    return 0;
}
```

### Explanation of Fixes
1. **Reading Input**: Correctly read `N` as the number of points.
2. **Distance Calculation**: Compute the squared Euclidean distance `(dx * dx + dy * dy)` instead of taking the square root immediately. This avoids redundant computations and potential numerical inaccuracies.
3. **Accumulating Cost**: Accumulate the distance to the `ans` variable after converting it back to the actual distance using `sqrtl`.
4. **Returning to Origin**: After visiting all points, calculate the distance from the last point back to the origin and add it to `ans`.
5. **Output Formatting**: Use `fixed` and `setprecision(10)` to ensure the output matches the required precision.
