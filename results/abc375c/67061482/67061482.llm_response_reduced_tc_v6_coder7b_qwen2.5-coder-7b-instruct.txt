```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(0);

    double a, b, c, d, cou;
    int N;
    cin >> N;
    a = 0;
    b = 0;
    cou = 0;
    for(int i = 0; i < N; i++){
        cin >> c >> d;
        cou += sqrt((a-c)*(a-c) + (b-d)*(b-d));
        a = c;
        b = d;
    }
    cou += sqrt(a*a + b*b);
    printf("%.10f\n", cou); // Ensuring the output has up to 10 decimal places
}
```

### Explanation of Changes
1. **Input/Output Optimization**: Added `ios_base::sync_with_stdio(false)` and `cin.tie(0)` to optimize I/O operations.
2. **Output Formatting**: Used `printf` with format specifier `%.10f` to ensure the output matches the expected precision up to 10 decimal places.
3. **Code Readability**: Kept the rest of the logic unchanged as it was already correct but needed minor adjustments for optimization and precision control.
