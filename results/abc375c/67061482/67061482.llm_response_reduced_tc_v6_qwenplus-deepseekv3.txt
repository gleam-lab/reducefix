```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    double a = 0, b = 0, c, d, cou = 0;
    int N;
    cin >> N;
    for (int i = 0; i < N; i++) {
        cin >> c >> d;
        cou += sqrt((a - c) * (a - c) + (b - d) * (b - d));
        a = c;
        b = d;
    }
    cou += sqrt(a * a + b * b);
    cout << fixed << setprecision(15) << cou << endl;
    return 0;
}
```

1. **Initialization**: The variables `a` and `b` are initialized to 0, representing the origin (0, 0). The variable `cou` accumulates the total distance traveled.
2. **Loop Through Points**: For each point `(c, d)` read from input, the Euclidean distance from the current position `(a, b)` to `(c, d)` is calculated and added to `cou`. The current position is then updated to `(c, d)`.
3. **Return to Origin**: After visiting all points, the Euclidean distance from the last point back to the origin is calculated and added to `cou`.
4. **Output**: The total distance `cou` is printed with a precision of 15 decimal places using `fixed` and `setprecision(15)` to ensure the output matches the required precision.
