
```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
using ld = long double;

int main() {
    int N;
    cin >> N;
    
    ll x = 0, y = 0;
    ld total_cost = 0.0;
    
    for (int i = 0; i < N; ++i) {
        ll xi, yi;
        cin >> xi >> yi;
        total_cost += sqrt(pow(xi - x, 2) + pow(yi - y, 2));
        x = xi;
        y = yi;
    }
    
    total_cost += sqrt(pow(x, 2) + pow(y, 2)); // Return to origin
    cout << fixed << setprecision(12) << total_cost << endl;
    return 0;
}
```

### Explanation:
1. I've corrected the type of `total_cost` to `ld` to maintain precision required for the final result.
2. Changed the variable names from `a`, `b`, `c`, `d` to `x`, `y`, `xi`, `yi` for clarity.
3. I've used the `pow` function to correctly calculate the square of the distance.
4. The output is formatted to 12 decimal places to ensure precision requirements of the problem statement are met.
5. The loop calculates the cumulative distance while moving from one point to the next and finally adds the distance from the last point back to the origin.
