```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    int n;
    cin >> n;

    vector<pair<long long, long long>> points(n);
    for(int i = 0; i < n; ++i){
        cin >> points[i].first >> points[i].second;
    }

    long long total_cost = 0;

    // Calculate the cost of moving from the origin to the first point
    total_cost += abs(points[0].first) + abs(points[0].second);

    // Calculate the cost of moving between consecutive points
    for(int i = 0; i < n-1; ++i){
        total_cost += abs(points[i+1].first - points[i].first) + abs(points[i+1].second - points[i].second);
    }

    // Calculate the cost of returning to the origin from the last point
    total_cost += abs(points[n-1].first) + abs(points[n-1].second);

    cout << total_cost << endl;

    return 0;
}
```

### Explanation:
1. **Input Handling**: The code reads the number of points `n` and then reads each point's coordinates into a vector of pairs.
2. **Cost Calculation**:
   - The cost to move from the origin `(0, 0)` to the first point is calculated as the sum of the absolute values of the x and y coordinates of the first point.
   - For each subsequent point, the cost to move from the current point to the next point is calculated similarly.
   - Finally, the cost to return from the last point back to the origin is calculated.
3. **Output**: The total cost is printed.

This approach ensures that the code handles the problem correctly within the constraints and efficiently calculates the required distances.
