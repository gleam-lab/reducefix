```cpp
#include <iostream>
#include <cmath>
#include <iomanip>

int main() {
    int n;
    std::cin >> n;
    
    long double x = 0, y = 0, prev_x = 0, prev_y = 0;
    long double total_cost = 0;

    for (int i = 0; i < n; ++i) {
        std::cin >> prev_x >> prev_y;
        total_cost += std::hypot(prev_x - x, prev_y - y);
        x = prev_x;
        y = prev_y;
    }

    // Add the cost of returning to the origin
    total_cost += std::hypot(x, y);

    std::cout << std::fixed << std::setprecision(15) << total_cost << std::endl;
    return 0;
}
```

### Explanation
1. **Reading Input**: The program reads the number of points `n` followed by `n` pairs of coordinates.
2. **Initial Position**: The starting position `(x, y)` is initialized to `(0, 0)`.
3. **Cost Calculation**:
   - For each point, calculate the Euclidean distance from the current position to the new point using `std::hypot`.
   - Accumulate this distance in `total_cost`.
   - Update the current position to the new point.
4. **Return to Origin**: After processing all points, add the cost of returning to the origin `(0, 0)`.
5. **Output**: Print the total cost with high precision using `std::fixed` and `std::setprecision`.

This approach ensures that the solution handles large inputs efficiently and accurately calculates the total cost with the required precision.
