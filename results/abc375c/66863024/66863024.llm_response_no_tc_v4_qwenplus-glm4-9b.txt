
```cpp
#include <iostream>
#include <cmath>
#include <vector>

using namespace std;

int main() {
    int N;
    cin >> N;

    double x = 0, y = 0;
    double total_cost = 0;

    for (int i = 0; i < N; ++i) {
        int X, Y;
        cin >> X >> Y;
        total_cost += sqrt((X - x) * (X - x) + (Y - y) * (Y - y));
        x = X;
        y = Y;
    }

    total_cost += sqrt(x * x + y * y); // Adding the cost to return to the origin
    cout << fixed << setprecision(10) << total_cost << endl; // Set precision to 10 decimal places

    return 0;
}
```

### Explanation
1. The code begins by reading the integer `N`, which represents the number of points Takahashi is going to visit.
2. Two variables `x` and `y` are initialized to 0, which represent the current position of Takahashi.
3. The loop runs `N` times, in each iteration:
   - It reads the next coordinates `(X, Y)`.
   - It computes the cost of moving from the current position `(x, y)` to the new position `(X, Y)` and adds it to `total_cost`.
   - It updates the current position `(x, y)` to `(X, Y)`.
4. After the loop, the code computes the final cost of moving back to the origin `(0, 0)` and adds it to `total_cost`.
5. The result is printed with a fixed decimal precision of 10 to meet the output requirements.
