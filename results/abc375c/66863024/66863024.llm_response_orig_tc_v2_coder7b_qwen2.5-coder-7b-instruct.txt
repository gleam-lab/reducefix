```cpp
#include <bits/stdc++.h>

using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.precision(15);
    
    int n;
    cin >> n;
    
    vector<pair<long double, long double>> points(n + 1);
    points[0] = {0, 0}; // Origin
    
    for (int i = 1; i <= n; ++i) {
        cin >> points[i].first >> points[i].second;
    }
    
    long double total_cost = 0;
    
    for (int i = 1; i <= n; ++i) {
        long double dx = points[i].first - points[i - 1].first;
        long double dy = points[i].second - points[i - 1].second;
        total_cost += sqrtl(dx * dx + dy * dy);
    }
    
    long double dx = points[n].first - points[0].first;
    long double dy = points[n].second - points[0].second;
    total_cost += sqrtl(dx * dx + dy * dy);
    
    cout << total_cost << endl;
    
    return 0;
}
```

### Explanation:
1. **Input Handling**: We read the number of points `n` and store the coordinates of each point in a vector `points`. We also initialize the first element of the vector as the origin `(0, 0)`.

2. **Distance Calculation**: We iterate through the list of points and calculate the Euclidean distance between each pair of consecutive points. This is done using the formula \(\sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}\). The result is accumulated in `total_cost`.

3. **Return to Origin**: After visiting all the points, we calculate the distance from the last point back to the origin and add it to `total_cost`.

4. **Output**: Finally, we print the total cost with high precision to ensure accuracy.
