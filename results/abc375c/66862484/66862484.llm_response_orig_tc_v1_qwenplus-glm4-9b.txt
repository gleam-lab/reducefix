
### Problem Analysis

The problem requires calculating the total distance Takahashi travels on a 2D coordinate plane, starting from the origin, visiting N given points in a specific order, and finally returning to the origin. The distance between two points is calculated using the Euclidean distance formula, which is the square root of the sum of the squares of the differences in their coordinates.

### Solution Strategy

1. **Calculate Distances Sequentially**: As Takahashi moves from one point to the next, the distance to the next point is calculated by subtracting the current coordinates from the next point's coordinates, squaring these differences, summing them, and then taking the square root. This approach, however, can lead to precision issues with large integers due to floating-point inaccuracies.

2. **Use Integer Arithmetic for Accuracy**: Instead of using the square root function, which can introduce floating-point errors, we can keep track of the sum of the squares of the differences. This sum will directly give us the total Euclidean distance without the need for square root operations.

3. **Optimization by Tracking Previous Position**: We can optimize our solution further by tracking the cumulative sum of the coordinates (both x and y). As we move to each new point, we can adjust this cumulative sum and use it to calculate the distance directly from the previous point to the current point.

### Implementation

```cpp
#include <iostream>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int N;
    cin >> N;

    long long sumX = 0, sumY = 0; // cumulative sum of coordinates
    long long totalDistance = 0;  // total distance traveled

    for (int i = 0; i < N; ++i) {
        long long x, y;
        cin >> x >> y;

        // Calculate the distance from the previous point (sumX, sumY) to the current point (x, y)
        totalDistance += (x * x - sumX * x + (y * y - sumY * y));
        
        // Update the cumulative sums of coordinates
        sumX += x;
        sumY += y;
    }

    // Output the total distance as a string to handle large numbers
    cout << totalDistance << endl;

    return 0;
}
```

### Explanation

1. **Initialization**: We initialize `sumX` and `sumY` to track the cumulative x and y coordinates. `totalDistance` is initialized to zero to accumulate the total distance.

2. **Loop through Points**: For each point, we read its coordinates `(x, y)`. We then calculate the distance from the previous point using the formula `(x * x - sumX * x + (y * y - sumY * y))`. This formula effectively calculates the squared distance directly without taking the square root, thus avoiding precision issues.

3. **Update Coordinates**: After processing each point, we update `sumX` and `sumY` to include the current point's coordinates.

4. **Output**: Finally, we output the total distance, which is the sum of squared distances between consecutive points plus the distance from the last point back to the origin. This result is guaranteed to be an integer and can be printed directly without further formatting concerns.
