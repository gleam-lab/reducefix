```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int N;
    cin >> N;

    long long curX = 0, curY = 0;   // 現在位置（原点）
    double total = 0;             // 総コスト (double)

    for (int i = 0; i < N; ++i) {
        long long x, y;
        cin >> x >> y;

        long long dx = curX - x;
        long long dy = curY - y;
        total += sqrt(dx * dx + dy * dy);  // 距離を計算し加算

        curX = x; curY = y;
    }

    // 最後の点から原点へ戻る
    total += sqrt(curX * curX + curY * curY);

    /* --- 出力 ---
       整数のまま文字列化して出力すれば誤差 0。 */
    printf("%.10f\n", total);  // 少数点以下10桁まで出力
    return 0;
}
```

This solution uses `double` instead of `__int128` for the total cost calculation to handle floating-point arithmetic correctly. The distance formula inside the loop calculates the Euclidean distance between the current position and the next point, which is then added to the total cost. Finally, it calculates the distance back to the origin before printing the result with 10 decimal places precision using `printf`.
