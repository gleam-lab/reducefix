### Approaches

2. **Processing Queries**: For each query:
   - If there's a wall at the given position, destroy it.
   - If there's no wall, find the first walls in the four directions (up, down, left, right) and destroy them.
3. **Efficient Lookup**: Use data structures like sets for each row and column to efficiently find the nearest walls in the four directions.

### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int H, W, Q;
    cin >> H >> W >> Q;

    vector<set<int>> row_walls(H + 2);
    vector<set<int>> col_walls(W + 2);

    for (int i = 1; i <= H; ++i) {
        for (int j = 1; j <= W; ++j) {
            row_walls[i].insert(j);
            col_walls[j].insert(i);
        }
    }

    int total_walls = H * W;

    while (Q--) {
        int R, C;
        cin >> R >> C;

        if (row_walls[R].count(C)) {
            row_walls[R].erase(C);
            col_walls[C].erase(R);
            --total_walls;
        } else {
            // Look up
            auto it_up = col_walls[C].lower_bound(R);
            if (it_up != col_walls[C].begin()) {
                --it_up;
                int r = *it_up;
                row_walls[r].erase(C);
                col_walls[C].erase(r);
                --total_walls;
            }

            // Look down
            it_up = col_walls[C].lower_bound(R);
            if (it_up != col_walls[C].end()) {
                int r = *it_up;
                row_walls[r].erase(C);
                col_walls[C].erase(r);
                --total_walls;
            }

            // Look left
            auto it_left = row_walls[R].lower_bound(C);
            if (it_left != row_walls[R].begin()) {
                --it_left;
                int c = *it_left;
                row_walls[R].erase(c);
                col_walls[c].erase(R);
                --total_walls;
            }

            // Look right
            it_left = row_walls[R].lower_bound(C);
            if (it_left != row_walls[R].end()) {
                int c = *it_left;
                row_walls[R].erase(c);
                col_walls[c].erase(R);
                --total_walls;
            }
        }
    }

    cout << total_walls << "\n";
    return 0;
}
```

### Explanation
- **Initialization**: We use two arrays of sets, `row_walls` and `col_walls`, to keep track of wall positions. Each set in `row_walls[i]` contains column indices of walls in row `i`, and each set in `col_walls[j]` contains row indices of walls in column `j`.
- **Processing Queries**: For each query:
  - If the queried position has a wall, it is removed from both sets, and the total wall count is decremented.
  - If there's no wall, the code searches for the nearest walls in the four directions using `lower_bound` for efficient lookup. Each found wall is removed from the sets, and the total wall count is decremented accordingly.
- **Output**: After processing all queries, the remaining wall count is printed.

This approach ensures that each query is processed in logarithmic time relative to the number of walls in the row or column, making the solution efficient even for large grids and numerous queries.
