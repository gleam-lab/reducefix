- Each cell initially has a wall.
- For each query, you place a bomb at `(R_q, C_q)`.
  - If that cell still has a wall, destroy it.
  - Else, look in all four directions (up, down, left, right), and destroy **the first existing wall** in each direction (if any).

### Key Observations:

1. We must track for each row and column:
   - Which columns in a given row still have walls (`row[i]`).
   - Which rows in a given column still have walls (`col[j]`).

2. Using `std::set` for each row and column is appropriate because:
   - It allows efficient lookup, insertion, and deletion (`O(log N)` per operation).
   - It supports binary search operations like `upper_bound` and `lower_bound`.

3. The main mistake in the original code was in how it handled wall destruction when the bomb was placed on an already destroyed cell. Specifically:
   - It did not properly check for the nearest wall in all four directions.
   - It tried to store results in a fixed-size array (`er`) which led to confusion and missed updates.

### Fixes Applied:

1. **Corrected Wall Destruction Logic:**
   - For each direction (left, right, up, down), use `upper_bound` and `lower_bound` appropriately to find the closest wall in that direction.
   - Destroy those walls **simultaneously**, removing them from both `row` and `col`.

2. **Used Clearer Variable Names and Logic Flow:**
   - Instead of trying to pack everything into a single loop with ambiguous arrays, we extract positions clearly using set operations.

3. **Maintained Efficiency:**
   - Since total queries can be up to 200,000 and total cells up to 400,000, this approach remains within acceptable time limits due to logarithmic complexity per operation.

### Final Working Code:

```cpp
#include <bits/stdc++.h>
using namespace std;

#define ll long long

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(0);

    ll H, W, Q;
    cin >> H >> W >> Q;

    vector<set<ll>> row(H), col(W);

    for (ll i = 0; i < H; ++i) {
        for (ll j = 0; j < W; ++j) {
            row[i].insert(j);
            col[j].insert(i);
        }
    }

    while (Q--) {
        ll R, C;
        cin >> R >> C;
        R--; C--;

        if (row[R].find(C) != row[R].end()) {
            // There's a wall here; destroy it
            row[R].erase(C);
            col[C].erase(R);
        } else {
            // Try to destroy the first wall in each direction
            // Left
            auto itL = row[R].lower_bound(C);
            if (itL != row[R].begin()) {
                --itL;
                ll c2 = *itL;
                row[R].erase(c2);
                col[c2].erase(R);
            }

            // Right
            auto itR = row[R].upper_bound(C);
            if (itR != row[R].end()) {
                ll c2 = *itR;
                row[R].erase(c2);
                col[c2].erase(R);
            }

            // Up
            auto itU = col[C].lower_bound(R);
            if (itU != col[C].begin()) {
                --itU;
                ll r2 = *itU;
                col[C].erase(r2);
                row[r2].erase(C);
            }

            // Down
            auto itD = col[C].upper_bound(R);
            if (itD != col[C].end()) {
                ll r2 = *itD;
                col[C].erase(r2);
                row[r2].erase(C);
            }
        }
    }

    ll ans = 0;
    for (ll i = 0; i < H; ++i) {
        ans += row[i].size();
    }
    cout << ans << '\n';

    return 0;
}
```
