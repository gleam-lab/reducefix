    auto destroyWall = [&](int r, int c) {
        if (grid[r][c]) {
            grid[r][c] = false;
            ans--;
        } else {
            for (int i = r - 1; i >= 0; --i) {
                if (grid[i][c]) {
                    grid[i][c] = false;
                    ans--;

                }

                if (grid[i][c]) {
                    grid[i][c] = false;
                    ans--;
                    break;
                }
            }
            for (int j = c - 1; j >= 0; --j) {
                if (grid[r][j]) {
                    grid[r][j] = false;
                    ans--;
                    break;
                }
            }
            for (int j = c + 1; j < W; ++j) {
                if (grid[r][j]) {
                    grid[r][j] = false;
                    ans--;
                    break;
                }
            }
        }

        int R, C;
        cin >> R >> C;
        R--; C--;
        destroyWall(R, C);

This solution addresses the problem by using a 2D boolean array `grid` to represent the presence or absence of walls in each cell. The function `destroyWall` handles both scenarios: if the target cell contains a wall, it simply removes it; otherwise, it iterates through the adjacent cells in all four directions until it finds a wall and destroys it. This approach ensures that the constraints are respected and efficiently handles the large input sizes.
