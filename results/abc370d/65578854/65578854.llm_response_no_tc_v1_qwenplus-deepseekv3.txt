### Explanation
1. **Initialization**: We initialize sets `X` for each row and `Y` for each column. Each set initially contains all the column indices (for rows) or row indices (for columns) where walls are present.

2. **Processing Queries**: For each query:

3. **Counting Remaining Walls**: The total remaining walls is the initial count minus the number of walls removed during the queries. This is printed as the result.
