
### Problem Analysis

The problem involves a grid with walls, and we need to process a series of queries to destroy walls based on specific rules. The goal is to determine the number of remaining walls after all queries are processed.

### Solution Strategy

1. **Grid Initialization**: We initialize a grid representing walls. We will maintain a boolean grid `coo` where `coo[i][j]` is `true` if there is a wall at cell `(i, j)`, and `false` otherwise.

2. **Query Processing**: For each query, which specifies a cell `(R_q, C_q)`, we perform the following:
   - If there's a wall at `(R_q, C_q)`, we destroy it.
   - If there's no wall at `(R_q, C_q)`, we simulate the explosion by destroying the first wall in each direction (up, down, left, right).

3. **Efficient Wall Destruction**: To efficiently manage wall destruction, we use sets to keep track of the walls in each row and column. This allows us to quickly find and remove walls without iterating over the entire row or column.

4. **Counting Remaining Walls**: After processing all queries, we count the number of remaining walls by summing up the number of `true` values in the `coo` grid.

#include <iostream>
#include <vector>
#include <set>
using namespace std;

int main() {
    int h, w, q;

    vector<vector<bool>> coo(h + 1, vector<bool>(w + 1, true));

    // Initialize row and column sets
    for (int i = 1; i <= h; ++i) {

        if (!r_set.empty()) {

            r_set.erase(wall);
            col[wall][r].erase(r);
            coo[r][c] = false;
        } else {
            // Destroy walls in all four directions
            // Up
            for (int i : r_set) {
                if (i < r) {

                    r_set.erase(wall);
                    col[wall][r].erase(r);
                    coo[i][c] = false;
                }
            }
            // Down
            for (int i : r_set) {
                if (i > r) {

                    r_set.erase(wall);
                    col[wall][r].erase(r);
                    coo[i][c] = false;
                }
            }
            // Left
            for (int j : c_set) {
                if (j < c) {
                    int wall = j;
                    c_set.erase(wall);
                    row[r][wall].erase(r);
                    coo[r][j] = false;
                }
            }
            // Right
            for (int j : c_set) {
                if (j > c) {
                    int wall = j;
                    c_set.erase(wall);
                    row[r][wall].erase(r);
                    coo[r][j] = false;
                }

        }
