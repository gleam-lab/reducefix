To fix this code, we need to ensure that all four directions (up, down, left, right) are correctly handled when there is no wall at the bomb's position. The original code had some issues with how it handled the directions, particularly in checking the conditions for the first wall in each direction. We need to ensure that the correct walls are removed in each case.

### Approaches
1. **Data Structures**: Use sets to keep track of walls in each row (yoko) and each column (tate). This allows efficient insertion, deletion, and searching operations.
2. **Bomb Processing**: 
   - If there is a wall at the bomb's position, remove it.
   - If there is no wall, find the first wall in each of the four directions (up, down, left, right) and remove them if they exist.
3. **Efficiency**: The use of sets ensures that each operation (insertion, deletion, searching) is O(log n), making the overall complexity manageable within the constraints.

### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;
#define rep(i,n) for(int i = 0; i < (n); i++)
using ll = long long;

int main() {
    int h, w, q;
    cin >> h >> w >> q;
    vector<set<int>> tate(w), yoko(h);
    rep(i, h) rep(j, w) {
        tate[j].insert(i);
        yoko[i].insert(j);
    }
    rep(qi, q) {
        int r, c;
        cin >> r >> c;
        r--; c--;
        if (tate[c].count(r)) {
            tate[c].erase(r);
            yoko[r].erase(c);
        } else {
            // Up
            auto it_up = tate[c].lower_bound(r);
            if (it_up != tate[c].begin()) {
                it_up--;
                int tmp = *it_up;
                tate[c].erase(tmp);
                yoko[tmp].erase(c);
            }
            // Down
            auto it_down = tate[c].lower_bound(r);
            if (it_down != tate[c].end()) {
                int tmp = *it_down;
                tate[c].erase(tmp);
                yoko[tmp].erase(c);
            }
            // Left
            auto it_left = yoko[r].lower_bound(c);
            if (it_left != yoko[r].begin()) {
                it_left--;
                int tmp = *it_left;
                yoko[r].erase(tmp);
                tate[tmp].erase(r);
            }
            // Right
            auto it_right = yoko[r].lower_bound(c);
            if (it_right != yoko[r].end()) {
                int tmp = *it_right;
                yoko[r].erase(tmp);
                tate[tmp].erase(r);
            }
        }
    }
    int ans = 0;
    rep(i, w) ans += tate[i].size();
    cout << ans << endl;
    return 0;
}
```

### Explanation
1. **Initialization**: We initialize sets for each column (`tate`) and each row (`yoko`) to keep track of walls. Each cell is initially present in these sets.
2. **Processing Queries**: For each query:
   - If the specified cell has a wall, it is removed from both the column and row sets.
   - If there is no wall, we look in the four directions (up, down, left, right) for the first existing wall and remove those walls from the sets.
3. **Result Calculation**: After processing all queries, the total number of remaining walls is the sum of the sizes of all column sets (or row sets, as they are symmetric).

This approach efficiently handles the operations using set data structures, ensuring optimal performance for large inputs.
