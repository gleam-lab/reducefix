The original code attempts to manage vertical and horizontal sets of walls, but it has some issues, particularly when checking for the existence of walls in the four directions and handling the destruction of the first walls encountered.

### Approaches

1. **Data Structures**: Use two sets of sets for vertical and horizontal walls. Specifically:
   - `tate[j]` will store the rows (i) where a wall exists in column j.
   - `yoko[i]` will store the columns (j) where a wall exists in row i.
   
2. **Query Processing**: For each query:
   - If there's a wall at the query position, simply remove it.
   - If there's no wall, look for the nearest walls in the four directions (up, down, left, right) and remove them if they exist.

3. **Algorithm**:
   - **Up**: Find the largest row less than `R_q` in the column `C_q`.
   - **Down**: Find the smallest row greater than `R_q` in the column `C_q`.
   - **Left**: Find the largest column less than `C_q` in the row `R_q`.
   - **Right**: Find the smallest column greater than `C_q` in the row `R_q`.

### Solution Code

```cpp
#include <bits/stdc++.h>
using namespace std;
#define rep(i,n) for(int i = 0; i < (n); i++)
using ll = long long;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int h, w, q;
    cin >> h >> w >> q;
    vector<set<int>> tate(w + 1), yoko(h + 1);
    
    for (int i = 1; i <= h; ++i) {
        for (int j = 1; j <= w; ++j) {
            tate[j].insert(i);
            yoko[i].insert(j);
        }
    }
    
    while (q--) {
        int r, c;
        cin >> r >> c;
        
        if (tate[c].count(r)) {
            tate[c].erase(r);
            yoko[r].erase(c);
        } else {
            auto it_up = tate[c].lower_bound(r);
            if (it_up != tate[c].begin()) {
                --it_up;
                int i = *it_up;
                tate[c].erase(i);
                yoko[i].erase(c);
            }
            
            auto it_down = tate[c].upper_bound(r);
            if (it_down != tate[c].end()) {
                int i = *it_down;
                tate[c].erase(i);
                yoko[i].erase(c);
            }
            
            auto it_left = yoko[r].lower_bound(c);
            if (it_left != yoko[r].begin()) {
                --it_left;
                int j = *it_left;
                yoko[r].erase(j);
                tate[j].erase(r);
            }
            
            auto it_right = yoko[r].upper_bound(c);
            if (it_right != yoko[r].end()) {
                int j = *it_right;
                yoko[r].erase(j);
                tate[j].erase(r);
            }
        }
    }
    
    int ans = 0;
    for (int j = 1; j <= w; ++j) {
        ans += tate[j].size();
    }
    cout << ans << '\n';
    
    return 0;
}
```

### Explanation

1. **Initialization**: The code initializes sets for vertical (`tate`) and horizontal (`yoko`) walls and populates them with all possible walls initially present in the grid.
2. **Query Processing**: For each query:
   - If there's a wall at the query position `(r, c)`, it is removed from both `tate[c]` and `yoko[r]`.
   - If there's no wall at the query position, the code searches for the nearest walls in all four directions (up, down, left, right) using `lower_bound` and `upper_bound` to efficiently find the closest walls, which are then removed.
3. **Result Calculation**: After processing all queries, the total number of remaining walls is calculated by summing the sizes of all vertical sets `tate[j]` for each column `j`.

This approach efficiently handles each query in logarithmic time relative to the number of walls in the respective row or column, ensuring optimal performance for large grids and queries.
