### Solution
1. **Correct Destruction Logic**: For each direction, we need to find the nearest wall that is not obstructed. This involves:
   - **Up**: The first wall above the bomb's row in the same column.
   - **Down**: The first wall below the bomb's row in the same column.
   - **Left**: The first wall to the left of the bomb's column in the same row.
   - **Right**: The first wall to the right of the bomb's column in the same row.
2. **Efficient Searching**: Using sets to maintain the walls allows for efficient searching with `lower_bound` and `upper_bound` operations, ensuring logarithmic time complexity for each query.
