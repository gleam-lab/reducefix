2. **Tracking Walls**: Use a 2D array or hash set to track which cells still have walls.
3. **Processing Queries**: For each query:
   - If the cell has a wall, remove it and decrement the count.
   - If the cell has no wall, find the first walls in the four directions (up, down, left, right) and remove them if they exist, decrementing the count for each removed wall.
4. **Output**: After processing all queries, output the count of remaining walls.

### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int H, W, Q;
    cin >> H >> W >> Q;
    
    vector<vector<bool>> walls(H + 1, vector<bool>(W + 1, true));
    long long remaining = (long long)H * W;
    
    while (Q--) {
        int R, C;
        cin >> R >> C;
        
        if (walls[R][C]) {
            walls[R][C] = false;
            remaining--;
        } else {
            // Check up
            for (int i = R - 1; i >= 1; --i) {
                if (walls[i][C]) {
                    walls[i][C] = false;
                    remaining--;
                    break;
                }
            }
            // Check down
            for (int i = R + 1; i <= H; ++i) {
                if (walls[i][C]) {
                    walls[i][C] = false;
                    remaining--;
                    break;
                }
            }
            // Check left
            for (int j = C - 1; j >= 1; --j) {
                if (walls[R][j]) {
                    walls[R][j] = false;
                    remaining--;
                    break;
                }
            }
            // Check right
            for (int j = C + 1; j <= W; ++j) {
                if (walls[R][j]) {
                    walls[R][j] = false;
                    remaining--;
                    break;
                }
            }
        }
    }
    
    cout << remaining << '\n';
    
    return 0;
}
```

### Explanation
1. **Initialization**: The `walls` 2D vector is initialized to `true`, indicating all cells initially have walls. The `remaining` variable starts with the total number of cells, `H * W`.
2. **Processing Queries**: For each query:
   - If the specified cell has a wall (`walls[R][C]` is `true`), it is removed (set to `false`), and the `remaining` count is decremented.
   - If the cell has no wall, the algorithm checks in the four directions (up, down, left, right) for the nearest wall. The first wall found in each direction is removed, and `remaining` is decremented accordingly.
3. **Output**: After processing all queries, the total number of remaining walls is printed.

This approach efficiently tracks the walls and processes each query in linear time relative to the grid dimensions, ensuring correctness and optimal performance for the given constraints.
