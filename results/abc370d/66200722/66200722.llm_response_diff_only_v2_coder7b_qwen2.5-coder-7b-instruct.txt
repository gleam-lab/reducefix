This solution uses two vectors of vectors `row` and `col`. Each element of `row[i]` stores the indices where there are walls in the ith row. Similarly, each element of `col[j]` stores the indices where there are walls in the jth column. 

For each query, if there is no wall in the queried cell `(r,c)`, we add this cell into `row[r]` and `col[c]`. If there is already a wall in the queried cell `(r,c)`, we do nothing. After processing all queries, we calculate the number of remaining walls as `ans = total cells - sum of sizes of all `row` vectors - sum of sizes of all `col` vectors + number of rows + number of columns`. This is because each intersection of a row and a column has been counted twice (once in its corresponding `row` vector and once in its corresponding `col` vector), so we need to subtract them out and then add back the number of rows and columns. The final answer is `ans/2`.
