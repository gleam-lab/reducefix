1. **Data Structure**: You are using `set` data structures to keep track of the walls. This is great for maintaining unique elements, but checking the existence of a wall and removing walls are both O(log n) operations. Given the high constraints on the number of queries and grid size, this will be inefficient.

2. **Query Processing**: You are trying to find the first wall in each direction and remove it. This is a complex and potentially expensive operation with your current data structure.

3. **Output**: You need to output the total number of walls destroyed, but it looks like you might be double-counting some walls.

### Proposed Solution:

To optimize this problem, you can use a segment tree to efficiently manage and update the walls in each row and column. Here's an outline of the solution:

1. **Data Structure Preparation**:
   - Create two **segment trees** for each row and column. These segment trees will help efficiently find and remove walls in O(log n) time.
   - Initialize all segment trees with the total number of walls in each row and column.

2. **Processing Queries**:
   - For each query, check if the target cell `(R_q, C_q)` has a wall.
   - If it does, remove the wall and update the segment trees for both the row and column.
   - If it doesn't, find the closest walls up, down, left, and right and remove them.
   - Update the segment trees to reflect the removal of the walls.

3. **Calculating Remaining Walls**:
   - After processing all queries, calculate the total number of walls left by summing up the values in all segment trees.

#include <bits/stdc++.h>

// Segment tree data structure
struct SegTree {
    vector<int> tree;
    int n;
    function<int(int, int)> join;
    function<int(int, int)> merge;

    // Constructor
    SegTree(int n, function<int(int, int)> join, function<int(int, int)> merge)
        : n(n), join(join), merge(merge) {
        tree.assign(4 * n, 1);
    }

    // Build the tree
    void build(int node, int start, int end) {
        if (start == end) {
            tree[node] = 1;
        } else {
            int mid = (start + end) / 2;
            build(2 * node, start, mid);
            build(2 * node + 1, mid + 1, end);
            tree[node] = merge(tree[2 * node], tree[2 * node + 1]);
        }
    }

    // Update a range of elements
    void update(int l, int r, int val) {
        update(1, 1, n, l, r, val);
    }

    void update(int node, int start, int end, int l, int r, int val) {
        if (r < start || end < l) return;
        if (l <= start && end <= r) {
            tree[node] = val;
            return;
        }
        int mid = (start + end) / 2;
        update(2 * node, start, mid, l, r, val);
        update(2 * node + 1, mid + 1, end, l, r, val);
        tree[node] = merge(tree[2 * node], tree[2 * node + 1]);
    }

        if (r < start || end < l) return 0;

        int left = query(2 * node, start, mid, l, r);
        int right = query(2 * node + 1, mid + 1, end, l, r);

    }
