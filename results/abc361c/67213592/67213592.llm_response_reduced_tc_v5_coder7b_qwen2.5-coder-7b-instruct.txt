```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, k;
    cin >> n >> k;

    vector<int> v(n);
    for (int i = 0; i < n; ++i) {
        cin >> v[i];
    }

    sort(v.begin(), v.end());

    long long best = LLONG_MAX;
    for (int i = 0; i <= n - k; ++i) {
        best = min(best, static_cast<long long>(v[i + k - 1]) - v[i]);
    }

    cout << best << endl;

    return 0;
}
```

### Explanation:
1. **Sorting**: The first step is to sort the array `v`. This allows us to easily find the smallest difference between the largest and smallest elements after removing `K` elements.
2. **Sliding Window**: We use a sliding window approach to find the minimum possible value of the difference between the maximum and minimum values in the remaining array after removing `K` elements. The window size is `n - k`.
3. **Best Calculation**: For each possible starting index `i` of the window, we calculate the difference between the element at position `i + k - 1` (the maximum in the window) and the element at position `i` (the minimum in the window). We keep track of the minimum difference encountered.
4. **Output**: Finally, we print the best calculated difference.

This solution ensures that we efficiently find the minimum possible value of the difference using sorting and a sliding window technique, making it suitable for large input sizes up to \(2 \times 10^5\).
