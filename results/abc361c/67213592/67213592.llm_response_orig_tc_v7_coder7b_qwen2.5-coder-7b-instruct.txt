#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, k; 
    cin >> n >> k;
    vector<int> v(n);
    
    for(auto& val : v){
        cin >> val;
    }
    
    sort(v.begin(), v.end()); // Sort the array
    
    long long ans = v[n-1] - v[0]; // Initial answer is the difference between max and min values
    
    // Try removing k smallest elements
    for(int i=0; i<k; i++){
        ans = min(ans, v[n-k+i] - v[i]);
    }
    
    cout << ans << endl; // Print the final answer
    return 0;
}

1. **Sort the Array**: Sorting helps us easily find the smallest and largest elements.

3. **Try Removing Smallest Elements**: Instead of directly trying to maximize the minimum value or minimize the maximum value after removing `k` elements, it's more efficient to consider removing the `k` smallest elements. This approach works because removing the smallest elements will increase the minimum value of the remaining array, potentially reducing the overall difference.

4. **Calculate New Differences**: For each possible starting index `i` to remove `k` elements, calculate the new difference between the maximum and minimum values of the remaining array. Update the answer with the minimum difference found.

5. **Output the Result**: Finally, print the minimum difference obtained.
