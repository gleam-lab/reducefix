```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main(){
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n, k; 
    cin >> n >> k;
    vector<int> v(n);
    for(auto &val : v) {
        cin >> val;
    }
    sort(v.begin(), v.end());

    int min_diff = INT_MAX;
    for(int i = 0; i <= k; ++i) {
        min_diff = min(min_diff, v[n-k+i]-v[i]);
    }

    cout << min_diff;
}
```

This solution works as follows:
1. **Input Reading**: Read the values of \( N \) and \( K \), followed by the sequence \( A \).
2. **Sorting**: Sort the sequence \( A \) in non-decreasing order.
3. **Finding Minimum Difference**:
   - Initialize `min_diff` with a large value (`INT_MAX`).
   - Iterate over all possible ranges of length \( N-K+1 \) in the sorted array.
   - For each range, calculate the difference between the maximum and minimum values within that range.
   - Update `min_diff` with the smallest difference found.
4. **Output**: Print the minimum difference.

This approach ensures that we efficiently find the minimum possible value of the maximum value of the resulting sequence minus the minimum value of the resulting sequence after removing exactly \( K \) elements. The time complexity is \( O(N \log N) \) due to sorting, which is efficient given the constraints.
