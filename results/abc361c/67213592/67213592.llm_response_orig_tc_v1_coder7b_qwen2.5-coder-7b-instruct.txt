```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, k;
    cin >> n >> k;

    vector<int> v(n);
    for (auto& val : v) {
        cin >> val;
    }

    sort(v.begin(), v.end());

    long long min_diff = LLONG_MAX;

    for (int i = 0; i <= n - k; ++i) {
        min_diff = min(min_diff, static_cast<long long>(v[i + k - 1]) - v[i]);
    }

    cout << min_diff << endl;

    return 0;
}
```

### Explanation:
1. **Sorting**: The array `v` is sorted in ascending order. This allows us to easily find the smallest difference between any two elements after removing `K` elements.
   
2. **Sliding Window**: We use a sliding window approach to consider each possible subarray of length `n - k`. For each subarray, we calculate the difference between the maximum and minimum values in that subarray (`v[i + k - 1] - v[i]`). We keep track of the minimum difference encountered.

3. **Result**: The minimum difference found during the iteration is printed as the result.

This solution ensures that we efficiently find the minimum possible value of the maximum value minus the minimum value of the resulting sequence after removing `K` elements. The time complexity is \(O(N \log N)\) due to sorting, which is efficient enough given the constraints.
