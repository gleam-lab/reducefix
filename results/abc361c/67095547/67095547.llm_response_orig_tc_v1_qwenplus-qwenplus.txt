To solve this problem, we are given a sequence of integers and asked to remove exactly `K` elements such that the **maximum - minimum** of the resulting subsequence is minimized.

---

### Key Observations:

1. We must keep `(N - K)` elements from the array in their original order.
2. The goal is to minimize the difference between the max and min values among those `(N - K)` elements.
3. Since the relative order must be preserved, we're essentially looking for a **contiguous subsequence** of length `(N - K)` with the smallest possible spread (max - min).

---

### Efficient Strategy:

- A sliding window approach works well here:
  - Slide a window of size `(N - K)` over the array.
  - For each window, compute the max and min efficiently.
  - Track the minimum spread found.

- To compute min and max efficiently within each window, we can use two **monotonic deques**:
  - One for tracking maximums (`maxDeque`)
  - One for tracking minimums (`minDeque`)

This leads to an overall time complexity of **O(N)**, which is suitable for large inputs (`N <= 2 * 10^5`).

---

### Final Working Code:

```cpp
#include <bits/stdc++.h>
using namespace std;

#define int long long

void solve() {
    int N, K;
    cin >> N >> K;

    vector<int> A(N);
    for (int i = 0; i < N; ++i) {
        cin >> A[i];
    }

    int windowSize = N - K;

    deque<int> maxDeque, minDeque;
    int result = numeric_limits<int>::max();

    // Sliding window setup
    for (int i = 0; i < N; ++i) {
        // Maintain maxDeque: elements are in decreasing order
        while (!maxDeque.empty() && A[maxDeque.back()] <= A[i]) {
            maxDeque.pop_back();
        }
        maxDeque.push_back(i);

        // Maintain minDeque: elements are in increasing order
        while (!minDeque.empty() && A[minDeque.back()] >= A[i]) {
            minDeque.pop_back();
        }
        minDeque.push_back(i);

        // Remove out-of-window indices
        if (maxDeque.front() <= i - windowSize) maxDeque.pop_front();
        if (minDeque.front() <= i - windowSize) minDeque.pop_front();

        // Once window is valid, update result
        if (i >= windowSize - 1) {
            result = min(result, A[maxDeque.front()] - A[minDeque.front()]);
        }
    }

    cout << result << endl;
}

signed main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    solve();
    return 0;
}
```
