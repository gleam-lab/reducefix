#include <bits/stdc++.h>

using namespace std;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    int N, K;
    cin >> N >> K;
    vector<int> A(N);
    for (int i = 0; i < N; ++i) {
        cin >> A[i];
    }

    // The goal is to minimize the maximum value of B minus the minimum value of B
    // after removing exactly K elements. To achieve this, we want to remove
    // elements that are farthest from the median of the array. This is because
    // the difference between the maximum and minimum value of B is minimized
    // when the values around the median are removed.

    // First, sort the array to easily find the median and the elements around it.
    sort(A.begin(), A.end());

    // The median index is (N-1)/2 for a 0-based index.
    int medianIndex = (N - 1) / 2;

    // Calculate the difference between the maximum and minimum values of B
    // after removing K elements. We remove elements that are farthest from the median.
    int removedElements = min(K, 2 * medianIndex + 1 - K);
    int remainingElements = N - removedElements;

    // The minimum possible value of the maximum minus the minimum of B is now
    // the difference between the largest remaining element and the smallest remaining element.
    cout << A[medianIndex + remainingElements - 1] - A[medianIndex - remainingElements] << endl;

    return 0;
}

This fixed code reads the input for the number of elements `N` and the number of elements to remove `K`. It then sorts the array `A`. To minimize the difference between the maximum and minimum of the new sequence `B`, it removes elements as close to the median as possible. The median index is calculated, and then the number of elements to remove from either side of the median is determined based on the minimum of `K` and twice the index of the median plus one minus `K`. Finally, it calculates the difference between the largest and smallest remaining elements, which is the desired result.
