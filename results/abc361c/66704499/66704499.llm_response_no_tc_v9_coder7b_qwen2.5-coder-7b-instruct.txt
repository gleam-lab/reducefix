```cpp
#include <bits/stdc++.h>
using namespace std;

int nums[200000];

int main(void){
    int n, k;
    cin >> n >> k;

    for (int i = 0; i < n; i++){
        cin >> nums[i];
    }

    sort(nums, nums + n);

    int l = 0;
    int r = n - 1;

    while (k--) {
        // Calculate the difference between the current max and min values
        int diff = nums[r] - nums[l];
        // If removing the smallest element results in a smaller difference, do so
        if (diff >= nums[r] - nums[l+1]) {
            r--;
        } 
        // Otherwise, remove the largest element
        else {
            l++;
        }
    }

    cout << nums[r] - nums[l] << endl;

    return 0;
}
```

This solution optimizes the process of finding the minimum possible value of the difference between the maximum and minimum values of the resulting sequence B after removing K elements. It iteratively decides whether to remove the smallest or largest remaining element based on which action would result in a smaller difference between the max and min values of the sequence. This approach ensures that at each step, the decision is made to minimize the difference, leading to the optimal solution. The time complexity remains O(N log N) due to the sorting step, making it efficient even for large inputs within the given constraints.
