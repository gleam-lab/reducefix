```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, k;
    cin >> n >> k;
    vector<int> a(n);
    
    for(int i = 0; i < n; ++i) {
        cin >> a[i];
    }

    sort(a.begin(), a.end());

    long long min_diff = LLONG_MAX;
    for(int i = 0; i <= k; ++i) {
        min_diff = min(min_diff, static_cast<long long>(a[i + n - k - 1]) - a[i]);
    }

    cout << min_diff << endl;

    return 0;
}
```

### Explanation
1. **Input Handling**: The input is read efficiently using `cin` and stored in a vector `a`.
2. **Sorting**: The array `a` is sorted to facilitate finding the minimum difference between the maximum and minimum values of the resulting subarray.
3. **Finding Minimum Difference**: Iterate through all possible starting positions for the subarray of size `n-k`, calculate the difference between the maximum and minimum values in that subarray, and keep track of the minimum difference found.
4. **Output**: Print the minimum difference.

This approach ensures that we find the minimum possible value of the maximum value minus the minimum value of the resulting sequence after removing exactly `K` elements. The time complexity is \(O(N \log N)\), which is efficient given the constraints.
