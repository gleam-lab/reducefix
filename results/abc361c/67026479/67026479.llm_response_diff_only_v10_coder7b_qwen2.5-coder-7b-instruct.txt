```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, k;
    cin >> n >> k;
    vector<int> a(n);
    
    for (int i = 0; i < n; i++) {
        cin >> a[i];
    }
    
    sort(a.begin(), a.end());
    
    int min_diff = INT_MAX;
    
    for (int i = 0; i <= k; i++) {
        int diff = a[i + n - k - 1] - a[i];
        min_diff = min(min_diff, diff);
    }
    
    cout << min_diff << endl;
    
    return 0;
}
```

### Explanation
1. **Input Reading**: The code reads `n` and `k`, followed by the array `a`.
2. **Sorting**: The array `a` is sorted to facilitate finding the minimum difference between the maximum and minimum values of the resulting sequence after removing `k` elements.
3. **Finding Minimum Difference**:
   - The loop iterates over all possible starting positions `i` for the first element of the resulting sequence `B`.
   - For each position `i`, it calculates the difference between the element at position `i + n - k - 1` (the maximum value) and the element at position `i` (the minimum value).
   - It updates `min_diff` with the smallest difference found.
4. **Output**: Finally, the minimum difference is printed.

This solution efficiently handles the problem within the given constraints and ensures that the output is correct.
