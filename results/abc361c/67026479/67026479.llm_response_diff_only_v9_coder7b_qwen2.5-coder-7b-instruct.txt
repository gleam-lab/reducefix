```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, k;
    cin >> n >> k;
    vector<int> a(n);
    
    for (int i = 0; i < n; ++i) {
        cin >> a[i];
    }
    
    sort(a.begin(), a.end());
    
    int min_diff = INT_MAX;
    for (int i = 0; i <= k; ++i) {
        int diff = a[i + n - k - 1] - a[i];
        min_diff = min(min_diff, diff);
    }
    
    cout << min_diff << endl;
    
    return 0;
}
```

### Explanation of Changes
1. **Initialization of `min_diff`**: Changed the initialization of `min_diff` to `INT_MAX` to ensure that any valid difference found will be smaller.
2. **Loop Range Correction**: The loop should run from `0` to `k` inclusive because we need to consider all possible starting points for removing `K` elements.
3. **Correct Indexing in Difference Calculation**: Used `i + n - k - 1` to correctly calculate the index of the last element after removing `K` elements from the sorted array.

This corrected version ensures that the program finds the minimum possible value of the difference between the maximum and minimum values of the new sequence `B`.
