2. If K is odd (let's say `K = 2m + 1`), we want to remove `m + 1` elements that are the highest to minimize the range of B. We then need to consider the range of the remaining elements.

The correct code would involve iterating over all possible pairs of maximum and minimum elements (or the maximum and minimum after removing elements) and calculating the range for each case.

    }
    
    sort(A.begin(), A.end());
    
    // Initialize the minimum possible range to a large value
    long long min_range = LLONG_MAX;
    
    // Calculate the possible ranges for even and odd K
    if (K % 2 == 0) {
        // For even K, remove m elements from the top and m elements from the bottom
        int m = K / 2;
        for (int i = m; i < N - m; ++i) {
            min_range = min(min_range, (long long)A[i + m] - (long long)A[i - m]);
        }
