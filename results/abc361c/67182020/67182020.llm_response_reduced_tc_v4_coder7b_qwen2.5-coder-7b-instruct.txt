To solve this problem, we need to minimize the difference between the maximum and minimum values of the sequence after removing exactly \( K \) elements. Here's a step-by-step approach to achieve this:

1. **Sort the Sequence**: By sorting the sequence, we can easily identify the smallest and largest elements that contribute to the maximum and minimum values of the resulting sequence.

2. **Consider Two Scenarios**:
   - If \( K \) is even, we can always choose the middle \( K \) elements such that the difference between the maximum and minimum values is minimized.
   - If \( K \) is odd, we have two choices:
     - Remove the \( (K/2) \)-th smallest element and the \( (K/2 + 1) \)-th smallest element.
     - Remove the \( (K/2) \)-th smallest element and the \( (K/2) \)-th largest element.

3. **Calculate the Minimum Difference**: Compute the differences for both scenarios and take the minimum.

Here's the corrected C++ code implementing this approach:

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int N, K;
    cin >> N >> K;
    vector<int> A(N);
    for (int i = 0; i < N; ++i) {
        cin >> A[i];
    }

    sort(A.begin(), A.end());

    if (K % 2 == 0) {
        int mid = K / 2;
        cout << A[N - 1 - mid] - A[mid] << endl;
    } else {
        int mid = K / 2;
        cout << min(A[N - 1 - mid] - A[mid + 1], A[N - 2 - mid] - A[mid]) << endl;
    }

    return 0;
}
```

### Explanation:
1. **Sorting**: The sequence is sorted to easily access the smallest and largest elements.
2. **Even \( K \)**: For even \( K \), the optimal strategy is to remove the middle \( K \) elements, which minimizes the difference between the maximum and minimum values.
3. **Odd \( K \)**: For odd \( K \), we consider two scenarios:
   - Removing the \( (K/2) \)-th smallest and \( (K/2 + 1) \)-th smallest elements.
   - Removing the \( (K/2) \)-th smallest and \( (K/2) \)-th largest elements.
   We take the minimum of these two differences.

This solution ensures that we handle all edge cases and constraints efficiently.
