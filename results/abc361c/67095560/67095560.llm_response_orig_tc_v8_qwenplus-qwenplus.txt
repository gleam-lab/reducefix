To solve this problem optimally, we need to **minimize the difference between the maximum and minimum values** in a subsequence of length `N - K` from the original array `A`, while maintaining the **relative order** of the elements.

### Key Observations:
- We are allowed to remove exactly `K` elements.
- The resulting sequence must have `N - K` elements.
- The relative order of the remaining elements must be preserved.
- We want the minimum possible value of `max(B) - min(B)` for all such sequences.

### Strategy:
1. Since we're choosing `N-K` elements, and we care about minimizing the spread (max - min), it makes sense to look at **sliding windows** of size `N-K`.
2. A sliding window of size `N-K` gives us all valid subsequences of that length where the order is maintained.
3. For each such window, compute `max - min`. The answer is the **minimum of these values** across all windows.

### Optimized Implementation:
We can use a **Segment Tree** or **Monotonic Deque** to compute the minimum and maximum in each sliding window efficiently in `O(N)` time.

Here's the correct and efficient solution using a **monotonic deque** approach:

```cpp
#include <bits/stdc++.h>
using namespace std;

#define int long long

void solve() {
    int N, K;
    cin >> N >> K;

    vector<int> A(N);
    for (int i = 0; i < N; ++i)
        cin >> A[i];

    // Sliding window min and max using deques
    deque<int> minDeque, maxDeque;
    int result = numeric_limits<int>::max();

    for (int i = 0; i < N; ++i) {
        // Maintain min deque (increasing)
        while (!minDeque.empty() && A[minDeque.back()] >= A[i])
            minDeque.pop_back();
        minDeque.push_back(i);

        // Maintain max deque (decreasing)
        while (!maxDeque.empty() && A[maxDeque.back()] <= A[i])
            maxDeque.pop_back();
        maxDeque.push_back(i);

        // Remove elements out of the current window of size N-K
        if (i >= N - K - 1) {
            while (minDeque.front() <= i - N + K)
                minDeque.pop_front();
            while (maxDeque.front() <= i - N + K)
                maxDeque.pop_front();

            result = min(result, A[maxDeque.front()] - A[minDeque.front()]);
        }
    }

    cout << result << endl;
}

signed main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    solve();
    return 0;
}
```

### Time Complexity:
- O(N): Each element is pushed and popped from the deques at most once.
