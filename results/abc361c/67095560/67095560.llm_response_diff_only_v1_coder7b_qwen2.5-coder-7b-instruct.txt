```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main() {
    int n, k;
    cin >> n >> k;
    
    vector<int> a(n);
    for (int i = 0; i < n; ++i) {
        cin >> a[i];
    }
    
    // Sort the array
    sort(a.begin(), a.end());
    
    // The minimum possible value of max(B) - min(B) occurs when we remove the smallest k elements
    // and keep the rest in their original order.
    int result = a[n - 1] - a[k];
    
    cout << result << endl;
    
    return 0;
}
```

### Explanation:
1. **Input Reading**: Read `n` and `k`, followed by the elements of array `a`.
2. **Sorting**: Sort the array to easily identify the smallest and largest elements.
3. **Calculation**: The optimal strategy is to remove the smallest `k` elements because this minimizes the difference between the maximum and minimum values of the resulting array.
4. **Output**: Print the result which is the difference between the largest element (`a[n-1]`) and the `(k+1)`th smallest element (`a[k]`).

This solution ensures that the code runs efficiently within the given constraints.
