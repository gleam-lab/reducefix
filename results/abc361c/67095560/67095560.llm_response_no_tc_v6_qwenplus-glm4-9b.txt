
```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main() {
    int n, k;
    cin >> n >> k;
    vector<int> a(n);
    
    for (int i = 0; i < n; ++i) {
        cin >> a[i];
    }
    
    sort(a.begin(), a.end()); // Sort the array to easily access the minimum and maximum values
    
    // To minimize the difference between the max and min of the resulting array, we should aim to
    // remove elements that are extreme values (either very small or very large).
    // We can achieve this by removing the smallest `K/2` and the largest `K/2` elements, if possible.
    // If k is odd, we can remove the smallest `K//2` elements and the largest `K//2 + 1` elements.
    
    int left = K / 2;
    int right = n - K - left; // right is the count of elements to remove from the end
    
    // Calculate the difference between the max and min of the resulting array after removal
    int result = a[right] - a[left];
    
    cout << result << endl;
    
    return 0;
}
```

### Explanation:
1. **Input Reading**: The program reads the values of `n` and `k`, and then reads the array `a`.
2. **Sorting**: The array `a` is sorted to make it easy to access the smallest and largest elements.
3. **Calculation of Removals**: Since we want to minimize the difference between the maximum and minimum of the resulting array, we consider removing `K/2` elements from the start and `K/2` elements from the end. If `k` is odd, we remove one more element from the end.
4. **Result Calculation**: The difference between the maximum and minimum of the remaining elements is calculated as `a[right] - a[left]`, where `right` is the index of the element just before the last removed element and `left` is the index of the element just after the first removed element.
5. **Output**: The result is printed.
