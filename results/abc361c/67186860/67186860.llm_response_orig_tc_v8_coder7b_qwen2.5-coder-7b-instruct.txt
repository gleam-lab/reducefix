```cpp
#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    int n, k;
    std::cin >> n >> k;
    
    // Read the input array
    std::vector<int> arr(n);
    for (int i = 0; i < n; ++i) {
        std::cin >> arr[i];
    }
    
    // Sort the array
    std::sort(arr.begin(), arr.end());
    
    // Initialize the result with the maximum possible difference
    long long result = LLONG_MAX;
    
    // Iterate through all possible starting points of the subarray
    for (int i = 0; i <= n - k; ++i) {
        // Calculate the difference between the maximum and minimum values in the current subarray
        long long diff = arr[i + k - 1] - arr[i];
        
        // Update the result with the minimum difference found so far
        result = std::min(result, diff);
    }
    
    // Output the result
    std::cout << result << std::endl;
    
    return 0;
}
```

This solution works as follows:

1. **Reading Input**: We read the size of the array `n` and the number of elements to remove `k`, followed by the array elements themselves.
   
2. **Sorting the Array**: We sort the array to facilitate finding the smallest and largest elements efficiently.
   
3. **Finding the Optimal Subarray**: We iterate through all possible starting points of a subarray of size `n - k`. For each starting point, we calculate the difference between the maximum and minimum values in the subarray.
   
4. **Updating the Result**: We keep track of the minimum difference encountered during the iteration.

5. **Outputting the Result**: Finally, we print the minimum difference.

This approach ensures that we find the optimal subarray after removing `k` elements to minimize the difference between the maximum and minimum values. The time complexity of this solution is \(O(N \log N)\), which is efficient enough given the problem constraints.
