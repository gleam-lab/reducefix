```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false); cin.tie(nullptr);

    int n, k;
    cin >> n >> k;
    vector<long long> arr(n);
    for(int i = 0; i < n; ++i) cin >> arr[i];

    // Sort the array
    sort(arr.begin(), arr.end());

    // Calculate the difference between the maximum and minimum elements
    long long min_val = LLONG_MAX;
    long long max_val = LLONG_MIN;
    for(int i = 0; i <= n-k; ++i) {
        min_val = min(min_val, arr[i]);
        max_val = max(max_val, arr[i+k-1]);
    }

    // The result is the minimum difference found
    cout << max_val - min_val << '\n';

    return 0;
}
```

### Explanation
1. **Sorting**: The array `arr` is sorted so that we can easily find the smallest and largest elements after removing `K` elements.
2. **Sliding Window**: We use a sliding window approach to consider every possible subarray of size `N-K`. For each subarray, we calculate the difference between the maximum and minimum elements.
3. **Result**: The minimum difference found during the sliding window process is the result.

This solution ensures that we efficiently find the minimum possible value of the maximum value minus the minimum value after removing `K` elements from the array. It runs in \(O(N \log N)\) time due to the sorting step, which is efficient enough given the constraints.
