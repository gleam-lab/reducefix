```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  int n, k;
  cin >> n >> k;
  vector<long long> v(n);
  for (int i = 0; i < n; i++) {
    cin >> v[i];
  }

  sort(v.begin(), v.end());

  long long ans = LLONG_MAX;
  for (int i = 0; i <= n - k; i++) {
    ans = min(ans, v[i + k - 1] - v[i]);
  }

  cout << ans << endl;
  return 0;
}
```

### Explanation
1. **Input Handling**: Read `n` and `k`, followed by the elements of the array `v`.
2. **Sorting**: Sort the array `v` to facilitate finding the minimum difference between the maximum and minimum elements after removing `k` elements.
3. **Sliding Window Technique**: Iterate through the sorted array with a sliding window of size `n-k+1`. For each window, calculate the difference between the maximum and minimum elements (`v[i + k - 1] - v[i]`). Keep track of the minimum difference encountered.
4. **Output**: Print the minimum difference found.

This approach ensures that you efficiently find the optimal solution within the given constraints. The time complexity is \(O(N \log N)\) due to sorting, which is efficient enough for the problem constraints.
