
```cpp
#include <iostream>
#include <algorithm>
using namespace std;

typedef struct{int a, p, b, q;} p;
p A[100];
long long dp[100][10000];

int N, X;
long long cost(int i, int n){
    long long d = n / (A[i].a * A[i].b), rem = n % (A[i].a * A[i].b);
    return dp[i][rem] + d * (A[i].p * A[i].b);
}

bool f(int n){
    long long ct = 0;
    for(int i = 0; i < N; i++){
        ct += cost(i, n);
        if(ct > X) return false;
    }
    return true;
}

int main(){
    scanf("%d %d", &N, &X);

    for(int i = 0, a, b, p, q; i < N; i++){
        scanf("%d %d %d %d", &a, &p, &b, &q);
        A[i] = {a, p, b, q};
        dp[i][0] = 0;
        for(int j = 1; j < a * b; j++){
            dp[i][j] = min(dp[i][max(0, j - a)] + p, dp[i][max(0, j - b)] + q);
        }
    }

    int lo = 0, hi = 1e9;
    while(hi - lo > 1){
        int mid = (hi + lo) / 2;
        if(f(mid)) lo = mid;
        else hi = mid - 1;
    }

    if(lo != hi && f(lo + 1)) lo++;
    printf("%d\n", lo);
}
```

### Explanation:
1. **Data Structure and Input Handling:** The problem uses a struct `p` to store the dimensions and costs related to each part of the problem. The input for these is taken from the standard input using `scanf`.

2. **Dynamic Programming Table (`dp`):** The `dp` table is used to store the minimum cost for certain remainders when dividing the total number of items by the unit area of each part. It is filled based on the given rules for each part.

3. **Cost Calculation (`cost` function):** This function calculates the total cost for a given number of items `n` by summing up the costs calculated from the `dp` table.

4. **Binary Search for Solution:** The main function performs a binary search to find the smallest number of items `n` such that the total cost does not exceed `X`.

5. **Edge Cases:** The binary search works within the bounds of the problem constraints, ensuring that the solution is efficient and correct.
