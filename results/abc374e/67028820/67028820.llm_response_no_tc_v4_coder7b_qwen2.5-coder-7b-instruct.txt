```cpp
#include <bits/stdc++.h>
using namespace std;

#define ll long long

ll n, x;
vector<ll> a, b, q, p;

// Function to check if we can achieve at least 'mid' production capacity within the budget
bool canAchieveCapacity(ll mid) {
    ll totalCost = 0;
    for (int i = 0; i < n; ++i) {
        // Calculate the cost of using machine S_i or T_i to meet the required production
        ll useS = max(0LL, mid - (a[i] * (mid / a[i]))); // Remaining products after using S_i as much as possible
        ll useT = max(0LL, mid - (b[i] * (mid / b[i]))); // Remaining products after using T_i as much as possible
        
        // Calculate the cost for using S_i and T_i
        ll costS = useS * (q[i] / (double)b[i]); // Cost of using S_i to handle remaining products
        ll costT = useT * (p[i] / (double)a[i]); // Cost of using T_i to handle remaining products
        
        // Total cost for process i
        totalCost += min(useS * q[i], useT * p[i]) + costS + costT;
    }
    
    // Check if the total cost is within the budget
    return totalCost <= x;
}

signed main() {
    cin >> n >> x;
    a.resize(n), b.resize(n), p.resize(n), q.resize(n);
    for (int i = 0; i < n; ++i) cin >> a[i] >> p[i] >> b[i] >> q[i];

    // Binary search to find the maximum production capacity
    ll left = 0, right = 1e10;
    while (left < right) {
        ll mid = (left + right + 1) / 2;
        
        if (canAchieveCapacity(mid)) left = mid;
        else right = mid - 1;
    }

    cout << left << endl;

    return 0;
}
```

### Explanation:
- **Function `canAchieveCapacity`**: This function checks whether we can produce at least `mid` units of the product within the given budget `x`. It calculates the cost of using both types of machines for each process and sums up the total cost. If the total cost is within the budget, it returns `true`; otherwise, it returns `false`.

- **Binary Search**: The binary search is used to find the maximum possible production capacity. We start with `left = 0` and `right = 1e10` (an upper bound). In each iteration, we calculate the midpoint `mid` and check if it's feasible to produce at least `mid` units. If it is, we move the lower bound up (`left = mid`). If not, we move the upper bound down (`right = mid - 1`).

- **Output**: Finally, we print the maximum achievable production capacity.
