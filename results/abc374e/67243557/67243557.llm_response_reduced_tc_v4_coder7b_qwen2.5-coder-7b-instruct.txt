To solve this problem, we need to determine the maximum achievable production capacity within a given budget. The key steps involve calculating the cost and production capacity for each machine type and then using binary search to find the optimal production capacity.

Here's the corrected and optimized C++ code:

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

struct Machine {
    int a, b, p, q;
};

long long calculateCost(const Machine& m, long long capacity) {
    long long unitsS = capacity / m.a;
    long long unitsT = capacity / m.b;
    return unitsS * m.p + unitsT * m.q;
}

bool checkCapacity(int n, const vector<Machine>& machines, long long budget, long long targetCapacity) {
    for (const auto& m : machines) {
        long long cost = calculateCost(m, targetCapacity);
        if (cost > budget) {
            return false;
        }
    }
    return true;
}

int main() {
    int n;
    long long budget;
    cin >> n >> budget;
    
    vector<Machine> machines(n);
    for (int i = 0; i < n; ++i) {
        cin >> machines[i].a >> machines[i].p >> machines[i].b >> machines[i].q;
    }

    long long left = 0, right = 10000000000LL;
    while (left < right) {
        long long mid = (left + right + 1) / 2;
        if (checkCapacity(n, machines, budget, mid)) {
            left = mid;
        } else {
            right = mid - 1;
        }
    }

    cout << left << endl;
    return 0;
}
```

### Explanation:

1. **Struct Definition**: We define a `Machine` struct to hold the parameters for each machine type.
2. **Cost Calculation**: The `calculateCost` function computes the total cost to achieve a given capacity with a specific machine.
3. **Feasibility Check**: The `checkCapacity` function checks if it's possible to achieve at least the target capacity within the given budget using all machines.
4. **Binary Search**: We use binary search to find the maximum achievable capacity. The search range starts from 0 to a large value (`10^10`). For each midpoint (`mid`), we check if it's feasible to achieve that capacity. If it is, we move the lower bound up; otherwise, we move the upper bound down.
5. **Output**: Finally, we print the maximum achievable capacity.

This approach ensures that we efficiently find the optimal production capacity within the given constraints.
