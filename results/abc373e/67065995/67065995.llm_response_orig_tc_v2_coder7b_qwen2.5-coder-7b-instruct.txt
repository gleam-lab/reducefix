To solve this problem efficiently, we need to determine the minimum number of additional votes each candidate needs to guarantee their victory. The key insight is that a candidate \( i \) needs at least as many additional votes as the difference between the maximum votes any candidate currently has minus the votes candidate \( i \) has, plus one more vote to ensure they surpass the others even if the worst-case scenario happens.

Here's a step-by-step breakdown of the solution:

1. **Calculate the current total votes**: Sum up the votes each candidate has.
2. **Find the maximum votes any candidate currently has**: Identify the highest value in the array of votes.
3. **Compute the minimum additional votes needed for each candidate**:
   - For each candidate \( i \), calculate the difference between the maximum votes and the votes candidate \( i \) has.
   - Add one to this difference to ensure candidate \( i \) wins even if the worst-case scenario occurs (other candidates get no additional votes).
4. **Handle edge cases**: If a candidate already has more votes than the maximum votes plus one, they don't need any additional votes and should be marked as 0.

int main() {
    ios_base::sync_with_stdio(0);
    cin.tie(0);

    int n, m;
    long long k;
    cin >> n >> m >> k;
    vector<long long> a(n);
    for(int i = 0; i < n; ++i) {
        cin >> a[i];
        k -= a[i];
    }

    // Find the maximum votes any candidate currently has
    long long max_votes = *max_element(a.begin(), a.end());

    vector<long long> ans(n, -1);
    for(int i = 0; i < n; ++i) {
        if(a[i] >= max_votes + 1) {
            ans[i] = 0;
        } else {
            ans[i] = max_votes + 1 - a[i];
        }
    }

    for(auto x : ans) {
        cout << x << " ";
    }
    return 0;
}

- **Calculate Total Votes**: Subtract the sum of votes already counted from \( k \).
- **Find Maximum Votes**: Determine the maximum votes any candidate currently has using `max_element`.
- **Compute Minimum Additional Votes**: For each candidate, compute the minimum additional votes needed to guarantee their victory. If a candidate already has enough votes, set their result to 0.
