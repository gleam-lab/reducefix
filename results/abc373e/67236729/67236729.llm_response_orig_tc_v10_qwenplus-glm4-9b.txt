
The problem is to determine the minimum number of additional votes each candidate needs to guarantee their victory in an election. The victory condition is that the number of candidates who have received more votes than them is less than a given number M.

Here is an analysis and a solution to the problem:

### Analysis
1. **Current Votes**: Each candidate has a certain number of votes `A_i`.
2. **Total Votes**: There are `K` total votes.
3. **Remaining Votes**: The number of remaining votes is `K - sum(A_i)`.
4. **Victory Condition**: A candidate wins if the number of candidates who have more votes than them is less than M.

### Steps to Solve
1. **Sort Candidates**: Sort the candidates based on their current votes in decreasing order.
2. **Cumulative Votes**: Compute a cumulative sum of votes from the top. This helps in determining how many votes are needed to surpass each candidate.
3. **Calculate Votes Needed**: For each candidate, calculate how many more votes they need to ensure that no other candidates have more votes than them.

### Detailed Steps
1. **Sort**: Sort the candidates based on their votes in descending order.
2. **Cumulative Votes**: Calculate the cumulative sum of votes (`tmp`) and keep track of the number of candidates who have that many votes (`num`).
3. **Determine Votes Needed**: For each candidate, if the number of candidates who have more votes (`num - 1`) is less than M, they already have a guaranteed victory. Otherwise, calculate the votes needed to surpass the next candidate who has more votes than them.

### Implementation
Here is the implementation based on the above analysis:

```cpp
#include <bits/stdc++.h>
using namespace std;
using i64 = long long;
using u64 = unsigned long long;

int main() {
    i64 N, M, K;
    cin >> N >> M >> K;
    vector<i64> A(N);
    for (i64 &x : A) cin >> x;
    
    sort(A.begin(), A.end(), greater<i64>());
    
    vector<pair<i64, i64>> votes_needed(N);
    i64 tmp = 0, num = 0, remaining_votes = K - accumulate(A.begin(), A.end(), 0LL);
    
    for (i64 i = 0; i < N; ++i) {
        if (num > M) {
            i64 votes_to_surpass = A[i] + (M + 1 - num) * A[i];
            votes_needed[i] = {votes_to_surpass, tmp};
        } else {
            votes_needed[i] = {0, 0};
        }
        tmp += A[i];
        num++;
    }
    
    for (auto &v : votes_needed) {
        cout << v.second << ' ';
    }
    cout << '\n';
    
    return 0;
}
```

### Explanation
- The code first reads the input values.
- It sorts the array of votes in descending order.
- It then iterates over the sorted array, calculating the minimum votes needed for each candidate to ensure victory.
- Finally, it prints the result.

This solution efficiently calculates the minimum votes needed for each candidate using sorting and a single pass calculation of votes required to surpass other candidates.
