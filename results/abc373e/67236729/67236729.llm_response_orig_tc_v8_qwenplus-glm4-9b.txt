
The problem asks to calculate the minimum additional votes each candidate needs to guarantee their victory in an election. The election has `N` candidates with `K` votes cast so far. For a candidate to guarantee victory, the number of candidates who have received more votes than them must be less than `M`. 

Let's break down the solution:

1. **Input Parsing**: Read values of `N`, `M`, `K`, and the list of votes `A_i` each candidate has received so far.

2. **Initial Calculations**:
    - Calculate the number of votes left: `left_votes = K - sum(A_i)`.
    - Sort the list `A_i` in descending order.

3. **Iterate Over Candidates**:
    - For each candidate `i`, determine the minimum votes `x` they need to guarantee their victory.
    - Calculate `curM = M - i` because we consider the remaining `M - 1` candidates as potential competitors (`M` candidates in total).
    - The candidate needs to have the highest number of votes among the `M` candidates.
    - Calculate `x` as `x = (curM * x + 1 - A[i]) // curM`. This formula calculates the minimum additional votes required for the candidate to surpass the `curM`th candidate.
    - If `A[i] == curM * x`, then the candidate can already guarantee victory without further votes, so set `x = 0`.
    - If `x > left_votes`, then it's impossible for the candidate to guarantee victory, so set `x = -1`.

4. **Output**:
    - Print the calculated votes `x` for each candidate.

This approach ensures that each candidate gets evaluated based on the remaining votes and their position in the sorted list of votes received so far.

```pseudocode

# Sort the list A_i in descending order

# Calculate remaining votes

# Iterate over each candidate

    # If there are more candidates than allowed to win, set result to -1

        continue

    # Number of candidates to consider as competitors
    curM = M - i

    # Calculate minimum required votes to guarantee victory
    if A[i] < curM * (left_votes + 1 - A[i]) // curM:
        results[i] = (curM * (left_votes + 1 - A[i]) // curM) - A[i]
    else:
        results[i] = 0

# Print results
```
