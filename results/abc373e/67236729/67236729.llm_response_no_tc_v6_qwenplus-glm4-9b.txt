
The problem is to determine the minimum number of additional votes each candidate needs to guarantee their victory. We have the following constraints:

1. We have a total of \( N \) candidates.
2. We have \( K \) votes in total.
3. We know how many votes each candidate has received so far.
4. A candidate wins if fewer than \( M \) other candidates have received more votes than they have after all votes are counted.

The goal is to compute the minimum number of additional votes each candidate needs to guarantee their victory. If a candidate has already secured their victory, they need 0 additional votes. If it's impossible for a candidate to secure their victory, they need -1 additional votes.

### Approach

1. **Sort Candidates by Votes**: First, sort the candidates based on the votes they have received so far in descending order. This helps us easily determine which candidates are leading and who are trailing.
  
2. **Behind the Leader**: Identify the candidate with the highest number of votes and let's call this candidate the "leader". All candidates behind this leader are in the "behind the leader" group.

3. **Exactly M Candidates**: If the number of candidates behind the leader is exactly \( M \), then the leader has already secured their victory. No additional votes are needed.

4. **More than M Candidates**: If there are more than \( M \) candidates behind the leader, we need to calculate the minimum votes required to ensure the candidate's victory.

5. **Calculate Additional Votes**:
    - For each candidate behind the leader, calculate the difference between the number of votes they have and the number of votes the leader has.
    - Add these differences to the total votes that would go to the leading candidate if the votes were evenly distributed.

6. **Check Feasibility**: If the calculated additional votes exceed the remaining votes, output -1 for that candidate. Otherwise, output the calculated number of additional votes.

### Implementation

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <climits>

using namespace std;

int main() {
    int N, M, K;
    cin >> N >> M >> K;
    vector<long long> A(N);
    for (int i = 0; i < N; ++i) {
        cin >> A[i];
    }

    // Sort the votes in descending order
    sort(A.begin(), A.end(), greater<long long>());

    vector<long long> additional_votes(N, -1);
    long long sum_votes = 0;
    for (int i = 0; i < N; ++i) {
        sum_votes += A[i];
    }

    if (N - M == 0) {
        // If there's exactly one candidate behind the leader or no candidates behind the leader
        for (int i = 0; i < N; ++i) {
            additional_votes[i] = 0;
        }
    } else {
        long long votes_needed = 0;
        for (int i = M - 1; i < N; ++i) {
            votes_needed = max(votes_needed, A[0] - A[i]);
        }
        votes_needed = votes_needed * M + K - sum_votes;

        if (votes_needed > K) {
            votes_needed = -1; // Cannot guarantee victory
        } else {
            for (int i = M - 1; i < N; ++i) {
                additional_votes[i] = (A[0] - A[i]) * M;
            }
        }
    }

    for (int i = 0; i < N; ++i) {
        cout << additional_votes[i] << " ";
    }
    cout << endl;

    return 0;
}
```
