```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m;
    ll k;
    cin >> n >> m >> k;
    vector<ll> a(n);
    for(int i = 0; i < n; ++i) {
        cin >> a[i];
        k -= a[i];
    }

    // Calculate the maximum votes needed for a candidate to win
    ll max_votes_needed = *max_element(a.begin(), a.end()) + 1;
    ll total_votes_needed = (m - 1) * max_votes_needed;

    if(k < total_votes_needed) {
        // Not enough votes left to ensure anyone wins
        fill(a.begin(), a.end(), -1);
    } else {
        // Calculate the minimum additional votes needed for each candidate
        for(int i = 0; i < n; ++i) {
            ll needed = max(max_votes_needed - a[i], 0LL);
            if(needed > k) {
                a[i] = -1;
            } else {
                a[i] = needed;
                k -= needed;
            }
        }
    }

    // Adjust the results to ensure no candidate can overtake others
    for(int i = 0; i < n; ++i) {
        if(a[i] != -1) {
            for(int j = 0; j < n; ++j) {
                if(i != j && a[j] == -1) {
                    if(a[i] > a[j]) {
                        a[j] = a[i] - 1;
                        break;
                    }
                }
            }
        }
    }

    // Output the results
    for(ll v : a) {
        cout << v << " ";
    }
    return 0;
}
```

### Explanation:
1. **Initialization**: We read the input values and subtract the initial votes from `k`.
2. **Maximum Votes Needed**: We determine the maximum votes a candidate needs to win (`max_votes_needed`).
3. **Total Votes Needed**: We calculate the total votes required for `(m-1)` candidates to each get at least `max_votes_needed` votes.
4. **Check Feasibility**: If the remaining votes `k` are less than `total_votes_needed`, it's impossible for anyone to win, so we set all results to `-1`.
5. **Calculate Minimum Additional Votes**: For each candidate, we calculate the minimum additional votes needed to ensure they win. If this exceeds `k`, we set the result to `-1`.
6. **Adjust Results**: We adjust the results to ensure no candidate can overtake another after receiving the calculated votes.
7. **Output**: Finally, we print the results.
