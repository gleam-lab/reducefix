```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m;
    ll k;
    cin >> n >> m >> k;
    
    vector<ll> a(n);
    for(int i = 0; i < n; ++i) {
        cin >> a[i];
        k -= a[i];
    }
    
    // Calculate the maximum votes needed for a candidate to win
    vector<ll> max_votes(n, 0);
    for(int i = 0; i < n; ++i) {
        max_votes[i] = (k / (n - m)) + (k % (n - m) > i);
    }
    
    vector<ll> ans(n, -1);
    for(int i = 0; i < n; ++i) {
        if(a[i] + max_votes[i] >= k / (n - m)) {
            ans[i] = 0;
        } else {
            ans[i] = max_votes[i] - (a[i] - max_votes[i]);
        }
    }
    
    for(int i = 0; i < n; ++i) {
        cout << ans[i] << " ";
    }
    return 0;
}
```

### Explanation:
1. **Input Handling**: The code reads the input values for `n`, `m`, `k`, and the array `a` containing the initial vote counts for each candidate.
2. **Max Votes Calculation**: For each candidate, calculate the maximum number of votes they need to ensure they can still win even if all remaining votes are distributed optimally against them.
3. **Answer Calculation**: For each candidate, determine if they can meet or exceed the calculated maximum votes needed. If so, they don't need any additional votes (`ans[i] = 0`). Otherwise, calculate the minimum additional votes required.
4. **Output**: Print the results for all candidates.

This solution ensures that the constraints are handled efficiently and correctly determines the minimum additional votes needed for each candidate to win.
