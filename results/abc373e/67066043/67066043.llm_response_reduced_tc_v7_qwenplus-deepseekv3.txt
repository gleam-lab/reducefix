### Approaches
1. **Sorting and Prefix Sums**: First, sort the candidates' current votes. This helps in efficiently calculating the number of candidates that could potentially have more votes than the candidate in question.
2. **Binary Search**: For each candidate, perform a binary search to find the minimum number of additional votes (X) needed such that the candidate is elected. The binary search checks if the candidate can be among the top M candidates after receiving X votes.
3. **Condition Check**: For each candidate i, after adding X votes, ensure that the number of candidates with votes strictly greater than (A_i + X) is less than M. This involves checking if the (N - M + 1)-th largest vote count (after adjustments) is less than or equal to (A_i + X).

### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;
#define all(x) (x).begin(), (x).end()

int main() {
    ios_base::sync_with_stdio(0);
    cin.tie(0);
    int N, M;
    ll K;
    cin >> N >> M >> K;
    vector<ll> A(N);
    ll sum = 0;
    for (int i = 0; i < N; ++i) {
        cin >> A[i];
        sum += A[i];
    }
    ll remaining = K - sum;
    vector<int> ord(N);
    iota(all(ord), 0);
    sort(all(ord), [&](int i, int j) { return A[i] < A[j]; });
    vector<ll> sortedA = A;
    sort(all(sortedA));
    vector<ll> prefix(N + 1);
    for (int i = 0; i < N; ++i) {
        prefix[i + 1] = prefix[i] + sortedA[i];
    }
    vector<ll> ans(N, -1);
    for (int idx = 0; idx < N; ++idx) {
        int i = ord[idx];
        ll low = 0, high = remaining;
        ll best = -1;
        while (low <= high) {
            ll mid = (low + high) / 2;
            ll new_votes = A[i] + mid;
            int pos = upper_bound(all(sortedA), new_votes) - sortedA.begin();
            int needed = N - M;
            if (pos > needed) {
                best = mid;
                high = mid - 1;
            } else {
                int cnt = needed - pos + 1;
                ll total = (new_votes + 1) * cnt - (prefix[needed + 1] - prefix[pos]);
                if (total <= remaining) {
                    best = mid;
                    high = mid - 1;
                } else {
                    low = mid + 1;
                }
            }
        }
        ans[i] = best;
    }
    for (int i = 0; i < N; ++i) {
        cout << (ans[i] == -1 ? -1 : max(0LL, ans[i])) << ' ';
    }
    return 0;
}
```

1. **Input Handling**: The input is read, and the total votes and remaining votes are calculated.
2. **Sorting and Ordering**: The candidates' votes are sorted, and their original indices are stored to map back the results later.
3. **Prefix Sum Array**: A prefix sum array is created from the sorted votes to efficiently calculate sums of ranges.
4. **Binary Search for Each Candidate**: For each candidate, a binary search determines the minimum votes needed (X) such that after adding X votes, the candidate is in the top M candidates. The binary search checks if the candidate's new vote count ensures that no more than M-1 candidates have strictly more votes.
5. **Result Construction**: The results for each candidate are stored and printed, ensuring that already secured victories are marked with 0 and impossible scenarios with -1.
