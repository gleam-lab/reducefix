
From the problem statement, we need to find the minimum additional votes required for each candidate to guarantee their victory, given the constraints and the number of votes already counted. Let's break down the problem and approach it step by step:

### Key Observations:
1. A candidate will be elected if they receive more votes than at least M other candidates.
2. If we have a sorted list of votes, the Mth largest candidate is the one who will just miss being elected if the votes are distributed optimally.
3. The remaining votes can be distributed in any way that benefits the candidates who need them the most.

### Solution Approach:
1. **Sort the Candidates by Votes**: First, sort the candidates based on the number of votes they have received so far in descending order.
2. **Calculate Minimum Votes Needed**: For each candidate, calculate the minimum votes needed to guarantee their victory by checking the votes of the Mth largest candidate and how far behind the current candidate is.
3. **Determine Feasibility**: Check if the calculated votes can be provided from the remaining votes. If not, the candidate cannot be guaranteed victory.

### Detailed Steps:
1. Read the input values and initialize the candidate votes array `a`.
2. Sort the candidate array based on votes in descending order.
3. Create an auxiliary array `ac` to store cumulative sums of votes for quick lookups.
4. Iterate over each candidate:
   - Find the Mth largest candidate's votes by iterating from the end of the sorted list.
   - Calculate the votes the current candidate needs to guarantee victory.
   - Check if the votes can be provided from the remaining votes. If not, assign `-1` to the candidate.
   - If the votes can be provided, subtract the votes from the remaining count.
5. Print the results.

### Implementation:

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    long long n, m, K;
    cin >> n >> m >> K;
    vector<long long> a(n);
    for (long long &v : a) cin >> v;

    // Sort the candidates by votes in descending order
    sort(a.begin(), a.end(), greater<long long>());

    vector<long long> ac(n + 1, 0);
    for (int i = 0; i < n; ++i) {
        ac[i + 1] = ac[i] + a[i];
    }

    for (int i = 0; i < n; ++i) {
        long long votes_needed = 0;
        // Find the Mth largest candidate's votes
        int j = n - m;
        if (j >= 0) {
            votes_needed = a[j] - a[i];
        } else {
            // If the Mth largest candidate is not present, the smallest candidate wins
            votes_needed = 0;
        }

        // Check if the votes needed can be provided from the remaining votes
        if (votes_needed <= K) {
            cout << votes_needed << " ";
        } else {
            cout << "-1 ";
        }
        K -= votes_needed;
    }
    cout << endl;

    return 0;
}
```

This solution sorts the candidates and uses cumulative sums to quickly find the Mth largest candidate's votes, then calculates the votes needed and checks if they can be provided from the remaining votes. If they can't, it outputs `-1` for that candidate.
