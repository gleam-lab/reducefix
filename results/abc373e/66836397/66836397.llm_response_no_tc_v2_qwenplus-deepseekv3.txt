### Problem Analysis
The problem involves determining the minimum number of additional votes each candidate needs to secure their election, given that:
- There are N candidates.
- Each candidate i currently has A_i votes.
- There are K total votes, with some remaining (K - sum(A_i)).
- A candidate is elected if at most M-1 candidates have more votes than them.
- We need to find, for each candidate, the smallest X (between 0 and remaining votes) such that if they receive X more votes, they are guaranteed to be elected regardless of how the remaining votes are distributed among other candidates.

### Issues in the Original Code
1. **Binary Search Logic**: The binary search in the original code checks for a condition `f(i, mid)` to determine if `mid` votes are sufficient. The logic inside `f` seems complex and potentially incorrect.
2. **Handling of Edge Cases**: The code does not properly handle cases where candidates are already guaranteed to win (X=0) or cannot win (X=-1).
3. **Efficiency**: Given the constraints (N up to 2e5), the solution must be efficient, ideally O(N log N).

### Correct Approach
1. **Sorting**: Sort the current votes to efficiently determine how many candidates have more votes than a given threshold.
2. **Binary Search for Each Candidate**: For each candidate, perform a binary search to find the minimum X such that after adding X votes to candidate i, at most M-1 candidates can have more votes than candidate i, even if all remaining votes are given to those candidates.
3. **Mathematical Calculation**: For each candidate i, after adding X votes, the threshold for other candidates is `A_i + X`. The number of candidates that can potentially exceed this threshold must be less than M.

### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    ll N, M, K;
    cin >> N >> M >> K;
    vector<ll> A(N);
    ll total = 0;
    for (ll &a : A) {
        cin >> a;
        total += a;
    }
    ll rem = K - total;

    vector<ll> sortedA = A;
    sort(sortedA.begin(), sortedA.end());

    vector<ll> prefix(N + 1, 0);
    for (int i = 0; i < N; ++i) {
        prefix[i + 1] = prefix[i] + sortedA[i];
    }

    for (int i = 0; i < N; ++i) {
        ll current = A[i];
        ll low = 0, high = rem;
        ll ans = -1;

        while (low <= high) {
            ll mid = (low + high) / 2;
            ll new_votes = current + mid;
            auto it = upper_bound(sortedA.begin(), sortedA.end(), new_votes);
            ll cnt = sortedA.end() - it;

            if (cnt < M) {
                ans = mid;
                high = mid - 1;
            } else {
                ll needed = 0;
                ll pos = it - sortedA.begin();
                ll start = pos;
                ll end = N - (M - 1);
                if (end <= start) {
                    needed = 0;
                } else {
                    needed = (new_votes + 1) * (end - start) - (prefix[end] - prefix[start]);
                }
                if (needed <= rem - mid) {
                    low = mid + 1;
                } else {
                    ans = mid;
                    high = mid - 1;
                }
            }
        }

        cout << ans << " \n"[i == N - 1];
    }

    return 0;
}
```
