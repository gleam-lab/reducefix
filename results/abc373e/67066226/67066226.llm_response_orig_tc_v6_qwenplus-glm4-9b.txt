
The problem is asking to find the minimum number of additional votes candidate `i` needs to guarantee their victory. The victory is guaranteed if the number of candidates who have received more votes than them is less than `M`.

Let's approach the problem step by step:

1. **Sort the candidates by their votes**: This will allow us to determine the minimum number of additional votes each candidate needs by comparing them to the candidates closest to them in the sorted list.

2. **Calculate the number of votes each candidate needs to guarantee victory**: To guarantee victory without any competition, a candidate needs `max_votes - current_votes` additional votes, where `max_votes` is the highest vote count of all candidates.

3. **Adjust votes for the top `M` candidates**: Since there could be multiple candidates elected, we need to consider the top `M` candidates. We should not add more votes than needed to ensure one of these candidates gets the highest number of votes.

4. **Calculate the additional votes needed**: For each candidate, we need to ensure they have more votes than any of the top `M` candidates. We will do this by taking the difference between the votes needed to surpass the top `M` candidate and their current votes.

5. **Output the results**: If a candidate does not need any additional votes to surpass the competition, output `0`. If they cannot secure victory under any circumstances, output `-1`.

    vector<ll> a(n);
    for (int i = 0; i < n; i++) {
        cin >> a[i];
        k -= a[i];
    }
    
    // Sort the candidates by their votes
    sort(a.begin(), a.end());
    
    // Calculate the votes needed to surpass the top M candidates
    vector<ll> votes_needed(n);
    for (int i = 0; i < n; i++) {
        votes_needed[i] = a[n - 1 - i] - a[i] + 1;
    }
    
    // Sort the votes_needed for easy access
    sort(votes_needed.begin(), votes_needed.end());
    
    // Calculate the additional votes needed for each candidate
    vector<ll> additional_votes(n);
    for (int i = 0; i < n; i++) {
        // The minimum additional votes needed to guarantee victory
        ll min_votes_needed = votes_needed[m - 1] - a[i];
        
        // Ensure we do not exceed the remaining votes
        if (min_votes_needed > k) {
            additional_votes[i] = -1;
        } else {
            additional_votes[i] = min_votes_needed;
            k -= additional_votes[i];
        }
    }
    
    // Output the results
    for (ll additional_vote : additional_votes) {
        cout << additional_vote << ' ';
    }
    cout << '\n';
}
