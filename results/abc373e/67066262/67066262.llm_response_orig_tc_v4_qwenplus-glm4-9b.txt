
Given the problem statement, we need to determine the minimum number of additional votes each candidate requires to guarantee their election. Here's a step-by-step approach to solve the problem:

1. **Calculate votes needed to secure victory:** 
   For each candidate `i`, we first need to determine the total votes they need to secure their victory. This can be calculated as:
   \[
   \text{Votes needed} = A_i + (M - 1) \times \text{max votes of other candidates}
   \]
   Note that M is the number of candidates that can be elected.

2. **Determine the minimum additional votes required:** 
   Once we have the total votes required for each candidate to secure their victory, we need to determine how many additional votes they need from the remaining ballots. 
   \[
   \text{Additional votes needed} = \text{Votes needed} - A_i
   \]
   If the additional votes needed are greater than the remaining votes available (`K - \sum A_i`), then it is impossible for this candidate to secure their victory. In this case, the answer should be `-1`.

3. **Optimization considerations:** 
   Since the value of `K` can be very large, we need to be efficient in our calculations. A direct approach can be costly, so we may need to use pre-computation, sorting, or other optimization techniques.

#include <algorithm>
#include <iostream>

using namespace std;

typedef long long ll;

int main() {
    int n, m;
    ll k;
    cin >> n >> m >> k;
    vector<ll> A(n);
    ll votes_sum = 0;
    for (int i = 0; i < n; ++i) {
        cin >> A[i];
        votes_sum += A[i];
    }
    
    vector<ll> votes_needed(n);
    for (int i = 0; i < n; ++i) {
        // Calculation to determine the votes needed to secure the election
        // A_i + (M-1) * max votes of other candidates
        // Since we need at least as many votes as the maximum to be guaranteed,
        // we use max(A) as a placeholder for max votes of other candidates
        votes_needed[i] = A[i] + (m - 1) * A.back();
    }

    // Array to store the minimum additional votes needed
    vector<ll> additional_votes_needed(n);
    
    // Sort the votes needed in descending order
    sort(votes_needed.rbegin(), votes_needed.rend());
    
    // Iterate through the sorted votes needed and calculate minimum additional votes needed
    for (int i = 0; i < n; ++i) {
        if (votes_needed[i] > k - votes_sum) {
            // If the votes needed are more than what's left, set to -1
            additional_votes_needed[i] = -1;
        } else {
            // Calculate additional votes needed
            additional_votes_needed[i] = votes_needed[i] - A[i];
        }
    }
    
    // Output the result
    for (ll vote_needed : additional_votes_needed) {
        cout << vote_needed << " ";
    }

    return 0;
