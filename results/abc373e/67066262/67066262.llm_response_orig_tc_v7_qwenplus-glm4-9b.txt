### Problem Analysis

The problem is essentially about predicting the outcome of an election based on the current votes and then deciding the minimum votes needed for each candidate to ensure they are elected. The conditions are:

1. A candidate i will be elected if the number of candidates who have received more votes than them is less than M.
2. We need to determine the minimum votes candidate i needs to guarantee their election.
3. We need to do this for every candidate.

Given the constraints, this problem is computationally intensive, and the solution has to be optimized to handle large inputs efficiently.

### Solution Strategy

1. **Sorting Candidates**: First, we should sort candidates by their current votes in ascending order. This helps us to determine the number of votes needed by a candidate to be the top M or to surpass other candidates.

2. **Prefix Sum and Binary Search**: 
   - Calculate the prefix sum of the sorted votes to quickly determine the total votes received by the top M candidates.
   - Use binary search to determine the point where the remaining votes can guarantee a candidate's election.

3. **Handling Edge Cases**:
   - If a candidate is already in the top M, they need 0 additional votes.
   - If no candidate can be guaranteed election, the answer for that candidate is -1.

4. **Vote Calculation**:
   - For each candidate, calculate the minimum votes required using the prefix sum array and binary search as described above.

### Implementation Steps

1. **Input Parsing**: Read and parse the input values.
2. **Sorting**: Sort the candidates based on their current votes.
3. **Prefix Sum Calculation**: Compute the prefix sum for the sorted votes.
4. **Binary Search for Votes**:
   - For each candidate, use binary search to determine the minimum votes required to ensure victory.
   - Update the answer if the candidate is already guaranteed victory or find the minimum votes needed using the remaining votes.
5. **Output**: Print the results for each candidate.

typedef long long ll;

int main() {

    for (ll &a : A) cin >> a;
    vector<pair<ll, int>> candidates(A.begin(), A.end());
    sort(candidates.begin(), candidates.end());

    // Calculate prefix sums
    vector<ll> prefix(n);
    prefix[0] = candidates[0].first;
    for (int i = 1; i < n; i++) {
        prefix[i] = prefix[i - 1] + candidates[i].first;
    }

    vector<ll> ans(n, -1);
    for (int i = 0; i < n; i++) {
        // Binary search to find the minimum votes needed to ensure victory
        auto bsearch = [&](ll l, ll r) -> ll {
            while (l <= r) {
                ll mid = (l + r) / 2;
                ll num = (mid + 1) * (n - m - (i >= n - m)) - (prefix[n - m - 1] - (i >= n - m ? prefix[n - 2] : prefix[i - 1]));
                if (num >= k) return r = mid;
                l = mid + 1;
            }
            return l;
        };

        ll votes_needed = bsearch(0, k - (prefix[n - m - 1] - (i >= n - m ? prefix[n - 2] : prefix[i - 1])) + 1);
        if (votes_needed <= k - (prefix[n - m - 1] - (i >= n - m ? prefix[n - 2] : prefix[i - 1]))) {
            ans[i] = votes_needed;
        } else {
            ans[i] = -1;
        }
    }

    for (ll a : ans) cout << a << ' ';
