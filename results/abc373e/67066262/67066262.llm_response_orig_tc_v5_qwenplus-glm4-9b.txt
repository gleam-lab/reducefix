
The problem is asking to determine the minimum additional votes each candidate needs to be guaranteed to win the election, given that a candidate wins if the number of candidates who have received more votes is less than M.

### Analysis

The main idea is to:
1. Sort the candidates based on their votes to date.
2. Calculate how many votes are needed for each candidate to be guaranteed a win.
3. Determine the minimum number of additional votes they need to secure this.

### Steps:

1. **Sort the Candidates**: First, we need to sort the candidates based on the number of votes they have already received.

2. **Calculate Required Votes**: For each candidate, calculate the number of votes they need to guarantee a win. This is done by considering the votes of M-1 candidates that have more votes than or equal to them, since they are the ones that could potentially surpass the current candidate.

3. **Determine Additional Votes**: Subtract the votes the candidate has already received from the calculated required votes to determine the additional votes needed.

4. **Edge Cases**: If the candidate already has more votes than the number of remaining candidates plus M-1, they're guaranteed a win, so they need 0 additional votes. If even with the maximum possible votes from the remaining ballots a candidate cannot guarantee a win, assign -1 as the answer.

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

typedef long long ll;

int main() {
    int n, m, k;
    cin >> n >> m >> k;
    vector<ll> votes(n);
    
    // Read and subtract the votes that have already been counted.
    for (int i = 0; i < n; i++) {
        cin >> votes[i];
        k -= votes[i];
    }

    // Sort the candidates based on their votes in descending order.
    sort(votes.rbegin(), votes.rend());

    vector<ll> additional_votes(n);

    // Calculate the minimum additional votes needed for each candidate
    for (int i = 0; i < n; i++) {
        // Calculate the number of votes we need to take from the remaining candidates
        // to ensure this candidate wins.
        if (i < m - 1) {
            additional_votes[i] = votes[i + 1] - votes[i];
        } else {
            // If the remaining votes are not enough for the first m-1 candidates to
            // surpass this candidate's current vote count, then this candidate can
            // win even with the maximum votes from the remaining ballots.
            additional_votes[i] = k - votes[i] + votes[0];
        }
    }

    // Output the results
    for (int i = 0; i < n; i++) {
        if (additional_votes[i] <= k) {
            cout << additional_votes[i] << ' ';
        } else {
            cout << -1 << ' '; // Cannot guarantee victory, even with all remaining votes.
        }
    }

    return 0;
}
