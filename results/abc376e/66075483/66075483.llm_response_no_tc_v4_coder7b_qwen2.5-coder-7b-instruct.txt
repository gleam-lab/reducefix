```cpp
#include<iostream>
#include<algorithm>
#include<vector>
using namespace std;

#define ll long long

int main() {
    ios_base::sync_with_stdio(false);cin.tie(NULL);cout.tie(NULL);

    int t,n,k;
    cin>>t;
    while(t--){
        cin>>n>>k;
        vector<ll> a(n),b(n);
        for(int i=0;i<n;i++) cin>>a[i];
        for(int i=0;i<n;i++) cin>>b[i];

        // Sort based on A_i to maximize the product when multiplied with max(B_i) later
        sort(a.begin(),a.end());
        sort(b.begin(),b.end());

        // Calculate prefix sums for B_i to quickly get sum of any subarray
        vector<ll> ps(n+1,0);
        for(int i=1;i<=n;i++) ps[i]=ps[i-1]+b[i-1];

        ll ans=LLONG_MAX;
        for(int i=n-k+1;i>=0;i--){
            // For each possible subset of size K, calculate the product
            ll max_a=a[i+k-1]; // Maximum A_i in current subset
            ll sum_b=ps[n]-ps[i-1]; // Sum of corresponding B_i's
            ans=min(ans,max_a*sum_b);
        }

        cout<<ans<<'\n';
    }
    return 0;
}
```

### Explanation
1. **Sorting**: We first sort both arrays `a` and `b` to facilitate finding the maximum element in any subset and its corresponding sum efficiently.
2. **Prefix Sum Array**: We compute a prefix sum array `ps` for `b` to quickly calculate the sum of any contiguous subarray.
3. **Iterating Over Subsets**: For each possible starting index `i` of a subset of size `K`, we calculate the product of the maximum element in that subset (`a[i+k-1]`) and the sum of the corresponding elements in `b`.
4. **Updating Minimum Product**: We keep track of the minimum product encountered during this process.
5. **Output**: Finally, we output the minimum product for each test case.
