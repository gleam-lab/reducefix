### Approaches
1. **Sorting and Priority Queue**: 
   - First, we sort the pairs \((A_i, B_i)\) based on \(A_i\). This allows us to consider subsets where the maximum \(A_i\) is easily determined.
   - For each element in the sorted list, we consider it as the maximum \(A_i\) in the subset \(S\). The corresponding \(B_i\) values of the \(K-1\) smallest \(B_j\) (where \(j < i\)) are selected to minimize the sum \(\sum_{i \in S} B_i\).
   - We use a max-heap (priority queue) to keep track of the largest \(B_j\) values. By maintaining the heap size at \(K-1\), we ensure that the sum of the smallest \(B_j\) values (excluding the current \(B_i\)) is minimized when the current \(A_i\) is the maximum.

#include <iostream>
#include <vector>
#include <queue>
#include <algorithm>
using namespace std;
#define int long long

const int inf = 1e18;

signed main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int t;
    cin >> t;
    while (t--) {
        int n, k;
        cin >> n >> k;
        vector<pair<int, int>> elements(n);
        for (int i = 0; i < n; ++i) {
            cin >> elements[i].first;
        }
        for (int i = 0; i < n; ++i) {
            cin >> elements[i].second;
        }
        
        sort(elements.begin(), elements.end());
        
        priority_queue<int> max_heap;
        int sum = 0;
        int ans = inf;
        
        for (int i = 0; i < n; ++i) {
            int current_a = elements[i].first;
            int current_b = elements[i].second;
            
            if (max_heap.size() == k - 1) {
                ans = min(ans, current_a * (sum + current_b));
            }
            
            if (max_heap.size() < k - 1) {
                max_heap.push(current_b);
                sum += current_b;
            } else if (!max_heap.empty() && max_heap.top() > current_b) {
                sum -= max_heap.top();
                max_heap.pop();
                max_heap.push(current_b);
                sum += current_b;
            }
        }
        
        cout << ans << '\n';
    }
    
    return 0;
}

   - For each element, if the heap size is \(K-1\), the current \(A_i\) is considered as the maximum in the subset. The expression value is computed and the minimum is tracked.
   - If the heap size is less than \(K-1\), the current \(B_i\) is added to the heap. Otherwise, if the current \(B_i\) is smaller than the largest in the heap, it replaces the largest to minimize the sum.
4. **Output**: The minimum value of the expression for each test case is printed.

This approach ensures that we efficiently compute the desired minimum value by leveraging sorting and a priority queue to manage the subset elements optimally.
