### Approaches

Then, for each possible `A_i` as the maximum in the subset of size `K`, we need to select the `K-1` smallest `B_j` values from the first `i-1` elements to minimize the sum of `B` values in the subset. This ensures that the expression `(max A_i in subset) * (sum of B_i in subset)` is minimized.

2. **Maintaining Minimum Sum**: We use a max-heap (priority queue) to keep track of the largest `B` values encountered so far. If the size of the heap exceeds `K-1`, we remove the largest element, thus maintaining the smallest `K-1` elements. The sum of these elements plus the current `B_i` gives the sum for the subset where `A_i` is the maximum.

#include <iostream>

const int inf = 1e18;

signed main() {
    ios_base::sync_with_stdio(false);

    while (t--) {
        int n, k;
        cin >> n >> k;
        vector<pair<int, int>> ab(n);
        for (int i = 0; i < n; i++) {
            cin >> ab[i].first;
        }
        for (int i = 0; i < n; i++) {
            cin >> ab[i].second;
        }
        sort(ab.begin(), ab.end());
        priority_queue<int> pq;
        int sum = 0;
        int ans = inf;
        for (int i = 0; i < n; i++) {
            int current_sum = sum + ab[i].second;
            ans = min(ans, ab[i].first * current_sum);
            pq.push(ab[i].second);
            sum += ab[i].second;
            if (pq.size() > k - 1) {
                sum -= pq.top();
                pq.pop();
            }
        }
        cout << ans << '\n';
    }
    return 0;
}
