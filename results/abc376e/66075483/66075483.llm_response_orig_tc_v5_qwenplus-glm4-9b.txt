
The problem is asking for the minimum value of the expression:

\[ \left(\max_{i \in S} A_i\right) \times \left(\sum_{i \in S} B_i\right) \]

where \( S \) is a subset of \(\{1, 2, \dots, N\}\) with size \( K \).

Given the constraints, the problem is computationally intensive and requires an efficient approach.

### Analyzing the Problem:

The first observation is that if we want to minimize the expression, we should minimize both the maximum \( A_i \) in the subset \( S \) and the sum \( \sum_{i \in S} B_i \).

For the sum, we could choose the smallest \( K \) elements from \( B \) because adding larger \( B_i \) to the sum will increase the value. However, this would not always minimize the maximum \( A_i \).

For the maximum \( A_i \), we should consider that if we include \( A_i \) in \( S \), we should also consider the sum of \( B \) up to \( i \) because it determines the product.

### Approach:

1. **Sort and Compute Prefix Sums**: First, sort the array \( A \) in descending order and compute the prefix sums of \( B \) in ascending order.
2. **Sliding Window**: Use a sliding window technique to find the optimal \( K \) elements from \( B \) to minimize the sum while considering the maximum \( A_i \) from the sorted \( A \).
3. **Calculate the Expression**: For each valid subset \( S \), calculate the expression and keep track of the minimum value.

### Implementation:

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <numeric>

using namespace std;

int main() {
    int T;
    cin >> T;

    while (T--) {
        int N, K;
        cin >> N >> K;

        vector<int> A(N), B(N);
        for (int i = 0; i < N; ++i) {
            cin >> A[i];
        }
        for (int i = 0; i < N; ++i) {
            cin >> B[i];
        }

        // Sort A in descending order
        sort(A.begin(), A.end(), greater<int>());

        // Calculate prefix sums for B
        vector<int> prefix_sum(N, 0);
        partial_sum(B.begin(), B.end(), prefix_sum.begin());

        // Use dynamic programming to find the minimum possible value
        vector<long long> dp(N + 1, LLONG_MAX);
        dp[0] = 0; // Base case: no elements in the subset

        for (int i = 0; i < N; ++i) {
            if (i >= K) {
                dp[i + 1] = min(dp[i + 1], dp[i - K] + prefix_sum[i]);
            }
            dp[i + 1] = min(dp[i + 1], dp[i] + prefix_sum[i] - (i >= K ? prefix_sum[i - K] : 0));
        }

        // The minimum value of the expression
        cout << A[0] * dp[N] << endl;
    }

    return 0;
}
```

This solution sorts \( A \) and computes prefix sums for \( B \). It then uses a dynamic programming approach to find the minimum sum of \( K \) elements in \( B \) and calculates the expression using the maximum value from \( A \). This approach should be efficient enough given the problem constraints.
