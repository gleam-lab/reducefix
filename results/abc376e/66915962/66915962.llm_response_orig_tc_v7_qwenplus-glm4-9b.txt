
The problem asks to find the minimum value of the expression \((\max_{i \in S} A_i) \times (\sum_{i \in S} B_i)\) for a given subset \(S\) of \(\{1, 2, \dots, N\}\) of size \(K\). The goal is to minimize this expression by choosing an optimal subset \(S\).

### Approach

1. **Sorting and Seeking**: The first step is to sort the values \(A\) in descending order. This is because we want to maximize the maximum value of the subset \(S\), which is directly proportional to the first \(K\) elements of the sorted \(A\).

2. **Tracking the Sum**: We need to find the subset \(S\) that minimizes the product of the maximum and the sum of the subset. For this, we can consider the following:
   - For each element \(A_i\) in the sorted \(A\), we calculate the sum of elements \(B\) that are included in \(S\).
   - We can maintain a sorted list of the sums of \(B\) values for all possible subsets of size \(K\).

3. **Efficient Calculation**:
   - To efficiently find and update the sums of \(B\) for subsets as we iterate through \(A\), we can use a data structure like a segment tree or a binary indexed tree.
   - As we include each element \(A_i\) from the sorted \(A\) into our subset \(S\), we update the segment tree/binary indexed tree with the new sum and calculate the product for this subset.
   - We keep track of the minimum product found so far.

4. **Output**: After processing all elements of \(A\) for a test case, the minimum product found is the answer for that test case.

#include <iostream>
#include <vector>
#include <set>
#include <algorithm>
using namespace std;

int main() {
    int t;
    cin >> t;
    while (t--) {
        int n, k;
        cin >> n >> k;
        vector<int> A(n), B(n);
        for (int i = 0; i < n; ++i) {
            cin >> A[i];
        }
        for (int i = 0; i < n; ++i) {
            cin >> B[i];
        }
        
        // Sort A in descending order
        sort(A.begin(), A.end(), greater<int>());

        // Use a set to keep track of the sums of B values for subsets of size K
        multiset<long long> sums;

        // Iterate over A and calculate the sums of B for possible subsets
        for (int i = 0; i < n; ++i) {
            long long sumB = 0;
            for (int j = 0; j < n; ++j) {
                if (j >= i + 1 - k) {
                    sumB += B[j];
                }
            }
            sums.insert(sumB);
        }

        // Find the minimum product
        long long ans = LLONG_MAX;
        for (int i = 0; i < n; ++i) {
            long long maxA = A[i];
            auto it = sums.lower_bound(maxA * k);
            if (it != sums.begin()) {
                --it;
                ans = min(ans, maxA * (*it));
            }
        }

        cout << ans << '\n';
    }
    return 0;
}
