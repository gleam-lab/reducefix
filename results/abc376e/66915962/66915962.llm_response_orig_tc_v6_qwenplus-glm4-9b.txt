
The problem is asking us to find the minimum possible value of the expression:

\[
\left(\max\_{i \in S} A_i\right) \times \left(\sum\_{i \in S} B_i\right)
\]

where \( S \) is a subset of \( \{1, 2, \dots, N\} \) of size \( K \).

### Approach

To solve this problem, we can use the following approach:

1.  Create a list of tuples where each tuple contains \( A_i \) and \( B_i \) for each \( i \) from 1 to \( N \).
2.  Sort the list of tuples in descending order based on \( A_i \).
3.  Initialize a variable `sumB` to 0.
4.  Initialize a variable `maxA` to the first element of the sorted list.
5.  Iterate through the first \( K \) elements of the sorted list:
    *   For each element, add \( B_i \) to `sumB`.
    *   Update `maxA` to the maximum of `maxA` and \( A_i \).
6.  The minimum possible value of the expression is `maxA` multiplied by `sumB`.

### Code

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    int T;
    cin >> T;
    while (T--) {
        int N, K;
        cin >> N >> K;
        vector<pair<int, int>> p(N);
        for (int i = 0; i < N; i++) {
            cin >> p[i].first >> p[i].second;
        }
        sort(p.begin(), p.end(), greater<pair<int, int>>());
        int sumB = 0, maxA = p[0].first;
        for (int i = 0; i < K; i++) {
            sumB += p[i].second;
            maxA = max(maxA, p[i].first);
        }
        cout << maxA * sumB << endl;
    }
    return 0;
}
```
