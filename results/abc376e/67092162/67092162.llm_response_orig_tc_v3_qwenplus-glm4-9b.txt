
The problem requires finding the minimum possible value of the expression \((\max_{i \in S} A_i) \times (\sum_{i \in S} B_i)\) for a subset \(S\) of \(\{1, 2, \dots, N\}\) of size \(K\). This problem can be solved using a greedy approach along with some optimization techniques.

### Solution Approach

1. **Pairing and Sorting**: 
   - First, we create a list of pairs where each pair consists of elements from `A` and `B` corresponding to the same index. This allows us to consider the contribution of both arrays \(A\) and \(B\) when selecting a subset \(S\).
   - We sort this list in descending order based on the first element of each pair (i.e., elements from array `A`). This ensures that for any subset of size \(K\), we always consider the top \(K\) elements (which are the largest ones) from `A`.

2. **Sliding Window Technique**:
   - After sorting the list of pairs, we use a sliding window of size \(K\) to compute the sum of the second element of each pair in the window (i.e., elements from array `B`).
   - For each window, we calculate the product of the maximum element from `A` in that window and the sum of `B` in that window. This gives us the value of the expression for that particular subset.
   - We keep track of the minimum value of this expression as we slide the window from the start to the end of the list.

3. **Optimization**:
   - Since we are dealing with multiple test cases and large inputs, it's crucial to optimize our solution. The sorting step is \(O(N \log N)\) for each test case, and the sliding window step is \(O(N)\) for each test case. Hence, the total complexity for each test case is \(O(N \log N)\), which is efficient given the constraints.

#include <algorithm>

using namespace std;

int compare(const pair<int, int>& a, const pair<int, int>& b) {
    return a.first > b.first;  // Sort primarily by A_i in descending order
}

int main() {
    int t;
    cin >> t;

    while (t--) {
        int n, k;
        cin >> n >> k;
        
        vector<pair<int, int>> a_b;
        
        // Read A and B and store as pairs
        for (int i = 0; i < n; ++i) {
            int a_i, b_i;
            cin >> a_i >> b_i;
            a_b.emplace_back(a_i, b_i);
        }
        
        // Sort pairs by A_i in descending order
        sort(a_b.begin(), a_b.end(), compare);
        
        vector<int> prefix_sum(n + 1, 0);
        
        // Calculate prefix sums of B
        for (int i = 0; i < n; ++i) {
            prefix_sum[i + 1] = prefix_sum[i] + a_b[i].second;
        }
        
        int min_value = INT_MAX;
        
        // Evaluate the expression using a sliding window
        for (int i = 0; i <= n - k; ++i) {
            int max_a = a_b[i].first; // max A_i in the current window
            int sum_b = prefix_sum[i + k] - prefix_sum[i]; // sum B_i in the current window
            min_value = min(min_value, max_a * sum_b);
        }
        
        cout << min_value << endl;
    }
    
    return 0;
}
