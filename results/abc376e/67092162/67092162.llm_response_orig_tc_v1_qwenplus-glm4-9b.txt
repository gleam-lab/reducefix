\[
\left(\max\_{i \in S} A_i\right) \times \left(\sum\_{i \in S} B_i\right)
\]

where \( S \) is a subset of size \( K \) of the indices from 1 to \( N \).

### Approach

1. **Sort the Pairs**: First, we combine the sequences A and B into a list of pairs where each pair consists of an element from A and the corresponding element from B. We sort this list in descending order based on the elements from A, because we want to maximize the maximum element of the subset.

2. **Initial Calculation**: We calculate the sum \( \text{sum} \) of elements from B for the first \( K \) elements of the sorted list (as these are the largest \( K \) elements in A). We also maintain a priority queue (max-heap) of the elements from B for these \( K \) elements.

3. **Iterate and Update**: We iterate over the rest of the sorted list and for each element, we check if adding this element to our subset will decrease the overall value of the expression. We do this by comparing the current maximum of the subset (which is the top element of the max-heap) with the element from the current pair. If the element from the current pair is larger, we replace the current maximum, update the sum and the max-heap, and then recalculate the potential new minimum value of the expression.

4. **Output the Result**: After processing all elements, we output the minimum value found.
