To solve this problem efficiently, we need to minimize the expression \((\max_{i \in S} A_i) \times (\sum_{i \in S} B_i)\) for a subset \(S\) of size \(K\) from the sequences \(A\) and \(B\). The key insight is to consider the largest elements in \(A\) and match them with the smallest elements in \(B\) to minimize the product.

            b.push_back({x, i});
        }

        sort(a.begin(), a.end());

        int max_a = 0;
        int sum_b = 0;
        for (int i = 0; i < k; i++) {
