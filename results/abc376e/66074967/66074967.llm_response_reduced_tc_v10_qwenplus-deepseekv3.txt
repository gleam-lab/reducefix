2. **Prefix Sum and Priority Queue**: For each element in the sorted list, treat it as the maximum A_i in the subset. Then, select the K-1 smallest B_i values from the elements before it (since A_i is sorted, all previous A_i's are smaller or equal). 3. **Calculate the Expression**: For each element, compute the expression (A_i) * (sum of selected B_i's) and keep track of the minimum value encountered.

        }
        sort(a_b.begin(), a_b.end());
        
        priority_queue<int> max_heap;
        int sum_b = 0;
        int min_value = inf;
        
        for (int i = 0; i < n; ++i) {
            int current_a = a_b[i].first;
            int current_b = a_b[i].second;
            
            if (max_heap.size() < k - 1) {
                max_heap.push(current_b);
                sum_b += current_b;
            } else if (!max_heap.empty() && current_b < max_heap.top()) {
                sum_b -= max_heap.top();
                max_heap.pop();
                max_heap.push(current_b);
                sum_b += current_b;
            }
            
            if (max_heap.size() == k - 1) {
                int current_sum = sum_b + current_b;
                min_value = min(min_value, current_a * current_sum);

        }

    }

3. **Sum Calculation**: For each element, if the heap contains K-1 elements, the sum of these elements plus the current B_i gives the sum of B_i values for the subset where the current A_i is the maximum. The product of this sum and the current A_i is computed and compared to the current minimum value.
