To address the problem efficiently, we need to consider how to minimize the expression \((\max_{i \in S} A_i) \times (\sum_{i \in S} B_i)\). This involves selecting a subset \(S\) of size \(K\) such that the product of the maximum element in \(A\) and the sum of elements in \(B\) is minimized.

Here's a step-by-step breakdown of the solution:

1. **Sort Arrays**: We start by sorting the arrays \(A\) and \(B\) based on their values. This helps us easily select the largest and smallest elements.

2. **Sliding Window Technique**: To find the optimal subset, we can use a sliding window approach. This allows us to efficiently compute the sum of elements in \(B\) for different subsets of size \(K\).

3. **Maintain Priority Queue**: Using a priority queue (max-heap), we keep track of the largest elements in \(B\) within the current window. This helps us quickly adjust the sum when removing or adding elements to the subset.

4. **Calculate Minimum Product**: As we slide the window, we calculate the product of the maximum element in \(A\) and the sum of elements in \(B\) for each valid subset. We keep track of the minimum product encountered.

#include <iostream>
#include <vector>
#include <queue>
#include <algorithm>
using namespace std;

#define int long long
const int INF = 1e18;

int n, k, t;
pair<int, int> a[200005], b[200005];

signed main() {
    ios_base::sync_with_stdio(false);
    cin.tie(0);

    cin >> t;
    while (t--) {
        cin >> n >> k;
        
        for (int i = 1; i <= n; i++) {
            cin >> a[i].first;
            a[i].second = i;
        }
        
        for (int i = 1; i <= n; i++) {
            cin >> b[i].first;
            b[i].second = i;
        }
        
        sort(a + 1, a + n + 1);
        sort(b + 1, b + n + 1);
        
        priority_queue<int> pq;
        int sum = 0;
        for (int i = 1; i <= k; i++) {
            pq.push(b[i].first);
            sum += b[i].first;
        }
        
        int ans = INF;
        for (int i = k; i <= n; i++) {
            ans = min(ans, (long long)a[i].first * sum);
            
            sum += b[i].first - pq.top();
            pq.pop();
            pq.push(b[i].first);
        }
        
        cout << ans << '\n';
    }
    
    return 0;
}

2. **Sorting**: Both arrays are sorted based on their first elements.
3. **Sliding Window with Priority Queue**:
   - Initialize a priority queue to store the largest elements in `B` for the initial window of size `k`.
   - Calculate the sum of these elements.
   - Slide the window across the array, updating the sum and the priority queue as necessary.
4. **Minimum Product Calculation**: For each valid window, calculate the product of the maximum element in `A` and the sum of elements in `B`, keeping track of the minimum product found.
