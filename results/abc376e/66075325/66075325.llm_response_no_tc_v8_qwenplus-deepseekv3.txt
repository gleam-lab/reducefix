### Approaches
1. **Sorting**: Sort the elements based on \(A_i\) in ascending order. This allows us to consider subsets where the maximum \(A_i\) is the current element being processed.
2. **Prefix Sum and Priority Queue**: For each element \(A_i\) (from \(K\) to \(N\)), consider it as the maximum \(A_i\) in the subset. The corresponding \(B_i\) values should be the smallest \(K-1\) values from the first \(i-1\) elements to minimize the sum. This can be efficiently managed using a max-heap (priority queue) to keep track of the largest \(B_i\) values that should be excluded to get the smallest \(K-1\) values.
3. **Calculate Minimum Product**: For each \(i\) (from \(K\) to \(N\)), compute the product of \(A_i\) and the sum of the smallest \(K-1\) \(B_j\) values (from \(j < i\)) plus \(B_i\), and keep track of the minimum product encountered.

#include <iostream>

#include <queue>
#include <algorithm>
using namespace std;
#define int long long
const int N = 2e5 + 9;
const int inf = 1e18;

pair<int, int> a[N];

signed main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    int t;
    cin >> t;
    while (t--) {
        int n, k;
        cin >> n >> k;
        for (int i = 0; i < n; i++) {
            cin >> a[i].first;
            a[i].second = i;
        }
        for (int i = 0; i < n; i++) {
            cin >> b[i];
        }
        sort(a, a + n);
        vector<int> B(n);
        for (int i = 0; i < n; i++) {
            B[i] = b[a[i].second];
        }
        int ans = inf;
        priority_queue<int> pq;
        int sum = 0;
        for (int i = 0; i < n; i++) {
            int current_sum = sum + B[i];
            if (pq.size() == k - 1) {
                ans = min(ans, a[i].first * current_sum);
            }
            pq.push(B[i]);
            sum += B[i];
            if (pq.size() > k - 1) {
                sum -= pq.top();
                pq.pop();
            }
        }
        cout << ans << '\n';
    }
    return 0;
}

3. **Product Calculation**: For each element \(A_i\) (starting from the \(K\)-th element), the product of \(A_i\) and the sum of the smallest \(K-1\) \(B_j\) values (from previous elements) plus \(B_i\) is computed. The minimum of these products across all elements is the answer.
