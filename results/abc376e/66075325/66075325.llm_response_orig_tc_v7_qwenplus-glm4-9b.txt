
The given problem is to find the minimum possible value of the expression \((\max_{i \in S} A_i) \times (\sum_{i \in S} B_i)\) for a given sequence of integers \(A\) and \(B\) of length \(N\), where \(S\) is a subset of \(\{1, 2, \dots, N\}\) of size \(K\).

Let's break down the problem and understand the solution approach:

### Problem Breakdown:
1. **Expanding the Expression**: Expand the expression to \((A_i + A_j + \dots + A_k) \times (\sum_{i \in S} B_i)\) where \(i, j, \dots, k\) are elements of \(S\).
2. **Interpreting the Problem**: The problem can be interpreted as finding two sequences, one with the size of \(K\) from \(A\) and one with the size of \(N\) from \(B\), such that the product of the maximum value from the \(A\) sequence and the sum of the \(B\) sequence is minimized.

### Solution Approach:
1. **Sorting**: First, sort the array \(A\) in descending order. This allows us to consider the largest values of \(A\) first.
2. **Prefix Sum**: Compute the prefix sum of the sorted \(A\) sequence. This allows us to quickly calculate the sum of the first \(K\) elements.
3. **Iterate through B**: For each element \(B[i]\), calculate the sum of the first \(K\) elements from the sorted \(A\) sequence (using the prefix sum).
4. **Keep Track of Minimum**: Keep track of the minimum value of the product of the maximum value from the \(A\) sequence and the current sum of the \(B\) sequence.

### Code Explanation:
- **Sorting**: We sort the array \(A\) in descending order.
- **Prefix Sum**: We compute the prefix sum of the sorted \(A\) sequence.
- **Loop through B**: For each element \(B[i]\), we calculate the sum of the first \(K\) elements from the prefix sum and update the minimum value if the current product is smaller.
- **Output**: After processing all elements of \(B\), output the minimum value.

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int n, k;
vector<int> A, B;

int main() {
    cin >> n >> k;
    A.resize(n);
    B.resize(n);

    for (int i = 0; i < n; i++) {
        cin >> A[i];
    }
    for (int i = 0; i < n; i++) {
        cin >> B[i];
    }

    sort(A.begin(), A.end(), greater<int>());
    vector<int> prefix(A.size());
    prefix[0] = A[0];
    for (int i = 1; i < A.size(); i++) {
        prefix[i] = prefix[i - 1] + A[i];
    }

    int min_value = INT_MAX;
    for (int i = 0; i < n; i++) {
        int sum_A = (i + 1 >= k) ? prefix[i - k + 1] : 0;
        int product = (i + 1 >= k) ? sum_A * B[i] : 0;
        min_value = min(min_value, product);
    }

    cout << min_value << endl;
    return 0;
}
