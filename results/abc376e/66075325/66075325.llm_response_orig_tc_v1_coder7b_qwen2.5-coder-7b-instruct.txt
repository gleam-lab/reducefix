To solve this problem efficiently, we need to minimize the expression \((\max_{i \in S} A_i) \times (\sum_{i \in S} B_i)\) where \(S\) is a subset of \(\{1, 2, \dots, N\}\) of size \(K\). 

### Key Observations:
1. **Maximization and Minimization**: To minimize the product, we want the maximum element in \(A\) paired with the smallest possible sum of corresponding elements in \(B\).
2. **Sorting**: Sorting both arrays \(A\) and \(B\) can help us efficiently find the best subsets.
3. **Sliding Window**: Using a sliding window approach can help manage the sum of elements in \(B\) as we consider different windows of size \(K\) in the sorted array.

### Approach:
1. Sort array \(A\) and store their original indices.
2. Sort array \(B\) and store their original indices.
3. Use two pointers to maintain a window of size \(K\) in the sorted array \(B\), keeping track of the sum of the elements within this window.
4. For each position in the sorted array \(A\), calculate the product of the current maximum in \(A\) and the sum of the current window in \(B\).
5. Update the result with the minimum product found.

### Implementation:
```cpp
#include<iostream>
#include<vector>
#include<algorithm>
#include<queue>
using namespace std;

#define int long long

struct Element {
    int val, idx;
};

int main() {
    ios_base::sync_with_stdio(0); cin.tie(0); cout.tie(0);

    int T;
    cin >> T;
    while (T--) {
        int N, K;
        cin >> N >> K;
        
        vector<Element> A(N), B(N);
        for (int i = 0; i < N; ++i) {
            cin >> A[i].val;
            A[i].idx = i;
        }
        for (int i = 0; i < N; ++i) {
            cin >> B[i].val;
            B[i].idx = i;
        }

        sort(A.begin(), A.end());
        sort(B.begin(), B.end());

        vector<long long> prefix_sum(N + 1, 0);
        for (int i = 0; i < N; ++i) {
            prefix_sum[i + 1] = prefix_sum[i] + B[i].val;
        }

        long long ans = LLONG_MAX;
        for (int i = K - 1; i < N; ++i) {
            ans = min(ans, (long long)A[i].val * (prefix_sum[i + 1] - prefix_sum[i - K + 1]));
        }

        cout << ans << '\n';
    }
    
    return 0;
}
```

### Explanation:
1. **Struct Definition**: We define a `struct` to hold the value and index of each element in arrays \(A\) and \(B\).
2. **Input Reading**: Read the number of test cases and process each one.
3. **Sorting**: Sort both arrays based on their values.
4. **Prefix Sum Calculation**: Compute the prefix sum of the sorted \(B\) array to quickly calculate the sum of any subarray.
5. **Sliding Window**: Iterate through the sorted \(A\) array and use the prefix sum to compute the sum of the corresponding subarray in \(B\). Update the answer with the minimum product found.

This approach ensures that we efficiently find the minimum product for each test case with a time complexity of \(O(N \log N)\), which is suitable given the constraints.
