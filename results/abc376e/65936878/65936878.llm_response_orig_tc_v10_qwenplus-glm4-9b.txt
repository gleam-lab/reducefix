
The problem requires finding the minimum possible value of the expression:

\[
\left(\max_{i \in S} A_i\right) \times \left(\sum_{i \in S} B_i\right)
\]

where \( S \) is a subset of \(\{1, 2, \dots, N\}\) of size \( K \).

### Approach

1. **Sorting**: Start by sorting the arrays \( A \) and \( B \) in descending order. This allows us to consider the largest elements of \( A \) and smallest elements of \( B \) first, which might lead to a smaller result for the expression.

2. **Sliding Window**: We can use a sliding window approach to find the best combination of \( K \) elements from \( A \) and \( B \). The window slides over the sorted arrays, and for each window, we calculate the value of the expression.

3. **Two Pointers Technique**: We can use two pointers to maintain a window of size \( K \). One pointer will iterate over the sorted \( A \) array (from the beginning to the \( N-K+1 \)th element), and the other pointer will slide over the sorted \( B \) array (from the \( N \)th element to the \( N-K+1 \)th element). This way, we can efficiently calculate the sum of \( B \) for each window in \( A \).

4. **Calculation of Expression**: For each valid window, calculate the expression and keep track of the minimum value found.

### Algorithm

1. Sort arrays \( A \) and \( B \) in descending order.
2. Initialize a result variable to a large value.
3. Use a sliding window of size \( K \) over \( A \) and \( B \):
   - For each position of the window, calculate the maximum \( A_i \) in the window.
   - Calculate the sum of \( B \) for the current window.
   - Update the result with the minimum value of the expression for the current window.
4. Return the result.

### Python Code

```python
def min_expression_value(n, k, A, B):
    A.sort(reverse=True)
    B.sort(reverse=True)
    min_value = float('inf')
    
    for i in range(n - k + 1):
        max_A = A[i]
        sum_B = sum(B[i:i+k])
        min_value = min(min_value, max_A * sum_B)
    
    return min_value

# Input reading and processing
import sys
input = sys.stdin.read
data = input().split()

index = 0
T = int(data[index])
index += 1
results = []

for _ in range(T):
    n, k = int(data[index]), int(data[index+1])
    index += 2
    A = list(map(int, data[index:index+n]))
    index += n
    B = list(map(int, data[index:index+n]))
    index += n
    result = min_expression_value(n, k, A, B)
    results.append(result)

for result in results:
    print(result)
```

This solution sorts the arrays and then uses a sliding window approach to find the minimum value of the expression. It reads input in a single go and processes each test case efficiently.
