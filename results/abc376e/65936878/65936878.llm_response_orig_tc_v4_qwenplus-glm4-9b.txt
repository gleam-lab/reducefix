
The problem asks to minimize the expression: \(\left(\max_{i \in S} A_i\right) \times \left(\sum_{i \in S} B_i\right)\) for all subsets \(S\) of \(\{1, 2, \dots, N\}\) of size \(K\). Given the constraints, a brute-force approach of generating all subsets is infeasible. 

To solve the problem, we need to find an efficient strategy to select \(K\) indices from the sequences \(A\) and \(B\) that minimizes the given expression. Here are the steps you can follow:

1. **Initial Observation**: If we take the largest \(A_i\) from the subset, it should be paired with the smallest sum of \(B\) values that are not in the subset to minimize the product. To achieve this, you can sort the elements of \(A\) and \(B\) in descending and ascending order respectively.

2. **Use Priority Queue**: To efficiently keep track of the sum of the smallest values of \(B\) not in the subset, we can use a min-heap (priority queue). The min-heap will store the values of \(B\) as they appear in ascending order, and we can pop the smallest values when the subset size exceeds \(K\).

3. **Iterate Over Sorted \(A\)**: After sorting, iterate over the sorted \(A\) and calculate the product for the current element being considered as \(\max A_i\). Maintain the sum of \(K\) smallest \(B\) values not in the subset using the min-heap. 

4. **Calculate the Expression**: For each element in \(A\), calculate the product with the sum from the min-heap and update the minimum product seen so far.

5. **Output the Result**: After processing all elements in \(A\) for a test case, output the minimum product.

Here is the corrected code implementing the above strategy:

```cpp
#include <bits/stdc++.h>
#define int long long
using namespace std;

int solve() {
    int n, k;
    cin >> n >> k;
    vector<int> A(n), B(n);
    
    for (int i = 0; i < n; i++) {
        cin >> A[i];
    }
    for (int i = 0; i < n; i++) {
        cin >> B[i];
    }
    
    sort(A.begin(), A.end(), greater<int>()); // Descending
    sort(B.begin(), B.end()); // Ascending
    
    priority_queue<int, vector<int>, greater<int>> pq; // Min-heap
    long long sumB = 0;
    
    for (int i = 0; i < n; i++) {
        if (i >= k) {
            sumB += B[i]; // Add next k-th largest element from B
        }
        pq.push(B[i]); // Add to heap
        if (i >= k) {
            sumB -= pq.top(); // Remove the smallest element from heap
            pq.pop();
        }
        
        if (i >= k - 1) {
            // Calculate the current value of the expression
            long long current_val = (long long) A[i - k + 1] * sumB;
            if (current_val < (long long) res) {
                res = current_val;
            }
        }
    }
    
    return res;
}

int32_t main() {
    ios::sync_with_stdio(false);
    cin.tie(NULL);
    int t;
    cin >> t;
    
    while (t--) {
        res = LLONG_MAX;
        cout << solve() << '\n';
    }
    
    return 0;
}
```
