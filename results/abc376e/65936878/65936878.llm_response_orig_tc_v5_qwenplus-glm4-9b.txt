
### Problem Analysis

The problem requires finding the minimum possible value of a specific expression involving two sequences and a subset of their indices. The expression is:

\[ \left(\max_{i \in S} A_i\right) \times \left(\sum_{i \in S} B_i\right) \]

where \( S \) is a subset of \( \{1, 2, \dots, N\} \) of size \( K \).

### Solution Strategy

To solve this problem, we can leverage the following observations:

1. **Sorting**: By sorting the arrays \( A \) and \( B \) together, we can more easily find combinations of indices that potentially minimize the expression.
2. **Greedy Approach**: We can use a greedy approach to select the top \( K \) elements from \( A \) and \( B \) to minimize the product of their maximum and sum.
3. **Two-pointer Technique**: This technique can be applied to efficiently find the optimal subset size \( K \) by adjusting the sum of \( B \) values dynamically while considering the maximum of \( A \).

### Detailed Steps

1. **Input Reading**: Read the number of test cases \( T \). For each test case, read \( N \), \( K \), \( A \), and \( B \).
2. **Sorting**: Sort the arrays \( A \) and \( B \) together based on \( A \) values. This will help in pairing the maximum values of \( A \) with the suitable sum of \( B \).
3. **Finding Minimum**: Initialize pointers for arrays \( A \) and \( B \). Move the pointers to find the minimum value of the expression by adjusting the sum of \( B \) values and checking the product of the maximum element of \( A \) and the current sum of \( B \).
4. **Output**: For each test case, print the computed minimum value.

#include <iostream>

        int N, K;

            cin >> A[i];
        }
        for (int i = 0; i < N; ++i) {
            cin >> B[i];
        }
        
        // Pair elements from A and B and sort by A[i]
        vector<pair<int, int>>AB(N);
        for (int i = 0; i < N; ++i) {
            AB[i] = {A[i], B[i]};
        }
        sort(AB.begin(), AB.end());
        
        // Use a greedy approach with a two-pointer technique
        long long sumB = 0, maxA = AB[0].first;
        set<int> S;
        for (int i = 0; i < N; ++i) {
            sumB += AB[i].second;
            S.insert(AB[i].second);
            if (S.size() > K) {
                sumB -= *S.begin();
                S.erase(S.begin());
            }
        }
        
        cout << maxA * sumB << '\n';
    }
    return 0;
}
