```cpp
#include <bits/stdc++.h>

signed main() {
    ios_base::sync_with_stdio(false); cin.tie(NULL);
    
    int t; cin >> t;
    while(t--) {
        int n, k; cin >> n >> k;
        
        vector<int> A(n), B(n);
        for(int i = 0; i < n; ++i) cin >> A[i];
        for(int i = 0; i < n; ++i) cin >> B[i];
        
        sort(A.begin(), A.end());
        sort(B.begin(), B.end(), greater<int>());
        
        int max_A = A[k-1];
        int sum_B = 0;
        for(int i = 0; i < k; ++i) sum_B += B[i];
        
        cout << max_A * sum_B << '\n';
    }
    return 0;
}
```

### Explanation
1. **Input Handling**: We read the number of test cases `t` and then proceed to handle each test case individually.
2. **Sorting**: For each test case, we read the arrays `A` and `B`. We then sort array `A` in ascending order and array `B` in descending order. This ensures that when we take the maximum element from `A` and the smallest elements from `B`, we can achieve the minimum product.
3. **Calculation**: The maximum element in the top `K` elements of `A` is stored in `max_A`. The sum of the top `K` elements of `B` is stored in `sum_B`.
4. **Output**: The result is calculated as `max_A * sum_B` and printed.

This approach ensures that we efficiently compute the minimum possible value for the given expression within the constraints.
