The key insight is that to minimize this product, we should try to balance the maximum element in \(A\) with the smallest sum of corresponding elements in \(B\).

Here's a step-by-step approach to achieve this:

2. **Select Elements**: For the first \(K-1\) elements, select the largest elements from \(A\) and the smallest elements from \(B\). This ensures that the sum of \(B\) remains small while balancing the maximum value from \(A\).
3. **Handle the Last Element**: For the last element, select the largest remaining element from \(A\) and the smallest remaining element from \(B\). This minimizes the overall product.

This approach ensures that we are always trying to balance the maximum value in \(A\) with the smallest possible sum in \(B\), leading to the minimum product.
