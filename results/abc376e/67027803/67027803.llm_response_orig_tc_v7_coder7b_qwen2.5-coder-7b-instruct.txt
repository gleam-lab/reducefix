To solve this problem efficiently, we need to find the minimum possible value of the expression \((\max\_{i \in S} A\_i) \times (\sum\_{i \in S} B\_i)\) for a subset \(S\) of size \(K\). This involves selecting \(K\) elements from the arrays \(A\) and \(B\) such that the product of the maximum element from \(A\) and the sum of corresponding elements from \(B\) is minimized.

Here's a step-by-step approach to achieve this:

1. **Sort the Arrays**: Sort array \(A\) in descending order and array \(B\) in ascending order. This helps in pairing the largest element of \(A\) with the smallest element of \(B\) which minimizes the product.

2. **Select the First \(K\) Elements**: After sorting, select the first \(K\) elements from both arrays. The maximum element from these \(K\) elements in \(A\) will be the largest element among the selected elements, and the sum of these \(K\) elements in \(B\) will be the smallest sum possible.

3. **Calculate the Product**: Compute the product of the maximum element from \(A\) and the sum of elements from \(B\).

This approach ensures that we are always picking the best combination of elements from both arrays to minimize the product.

#include <bits/stdc++.h>
using namespace std;

using ll = long long;

void solve() {
    int n, k; cin >> n >> k;
    vector<pair<ll, ll>> ab(n);
    for (int i = 0; i < n; i++) {
        cin >> ab[i].first >> ab[i].second;
    }
    
    // Sort A in descending order and B in ascending order
    sort(ab.begin(), ab.end(), [&](const pair<ll, ll>& a, const pair<ll, ll>& b) {
        return a.first > b.first || (a.first == b.first && a.second < b.second);
    });
    
    ll max_A = ab[k-1].first;
    ll sum_B = 0;
    for (int i = 0; i < k; i++) {

    }
    
    ll ans = max_A * sum_B;

}

int main() {

    while (t--) solve();
