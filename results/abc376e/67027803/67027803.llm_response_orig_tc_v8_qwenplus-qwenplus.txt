To solve this problem efficiently, we need to compute the minimum value of the expression:

$$
\left(\max_{i \in S} A_i\right) \times \left(\sum_{i \in S} B_i\right)
$$

for a subset `S` of size `K` from two given arrays `A` and `B`.

---

### **Approach**

1. **Sort by A[i]**:
   - Sort the pairs `(A[i], B[i])` in increasing order of `A[i]`. This allows us to consider each element as a candidate for being the maximum in the subset.

2. **Use a Min-Heap**:
   - We maintain a min-heap (priority queue) to keep track of the top `K-1` smallest `B[i]` values among the first `i` elements.
   - As we iterate through the sorted list, for each possible candidate `i`, we try to construct a valid subset of size `K` where `A[i]` is the max element and the rest are selected from the previous `K-1` smallest `B[i]` values.

3. **Greedy Selection**:
   - Maintain a running sum of the `B[i]` values that form the subset with the smallest `B[i]` values.
   - When we encounter a new element `(A[i], B[i])`, check if replacing the largest element in the current heap with `B[i]` improves the total sum.

4. **Update Result**:
   - For each `i >= K-1`, calculate the value `A[i] * (sum_B + B[i])` and update the minimum result accordingly.

This approach ensures an efficient solution with time complexity **O(N log N)** per test case due to sorting and heap operations.

#include <bits/stdc++.h>
using namespace std;

using ll = long long;
const ll INF = 1e18;

void solve() {
    int n, k; cin >> n >> k;
    vector<pair<ll, ll>> ab(n);
    for (int i = 0; i < n; ++i) cin >> ab[i].first;
    for (int i = 0; i < n; ++i) cin >> ab[i].second;
    
    sort(ab.begin(), ab.end());
    
    if (k == 1) {
        ll ans = INF;
        for (auto [a, b] : ab) {
            ans = min(ans, a * b);
        }
        cout << ans << "\n";
        return;
    }

    priority_queue<ll, vector<ll>, greater<ll>> pq;
    ll sum_b = 0;
    for (int i = 0; i < k - 1; ++i) {
        sum_b += ab[i].second;
        pq.push(ab[i].second);
    }

    ll res = INF;
    for (int i = k - 1; i < n; ++i) {
        auto [a, b] = ab[i];
        res = min(res, a * (sum_b + b));
        if (!pq.empty() && pq.top() > b) {
            sum_b -= pq.top();
            pq.pop();
            sum_b += b;
            pq.push(b);
        }
    }

    cout << res << "\n";
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int t; cin >> t;
    while (t--) solve();
}

- We always compute the cost with the current `A[i]` and the accumulated `B[i]` values, ensuring correctness and efficiency.
