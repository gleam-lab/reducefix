
The problem requires you to minimize a specific expression involving two sequences A and B, given a subset of their indices. The expression is:

\[
\max_{i \in S} A_i \times \sum_{i \in S} B_i
\]

where S is a subset of \(\{1, 2, \dots, N\}\) of size K.

### Approach

1. **Sort and Pair**: Pair elements from A and B and sort them based on A in descending order. This ensures that when you pick the maximum A value, it comes from the largest available A value.

2. **Iterative Calculation**:
   - Initialize a priority queue (min-heap) to keep track of the sum of B values.
   - Start with the first K-1 elements as these will establish the initial sum for the heap.
   - For each subsequent element in sorted A, calculate the potential minimum value of the expression using the current heap sum and the B value of the current element.
   - If the current B value is greater than the top of the heap, replace the top of the heap with the current B value and adjust the sum accordingly.
   - Keep track of the minimum value encountered during these iterations.

3. **Output the Minimum Value**: After processing all elements, the minimum value tracked will be the answer for the current test case.

### Code Explanation

```cpp
#include <bits/stdc++.h>
using namespace std;

using ll = long long;
const int INF = 1e9 + 10;
const ll inf = 1LL << 60;

void solve() {
  int n, k; cin >> n >> k;
  vector<pair<ll, ll>> ab(n);
  for (int i = 0; i < n; i++) cin >> ab[i].first;
  for (int i = 0; i < n; i++) cin >> ab[i].second;
  
  // Sort pairs by A in descending order
  sort(ab.begin(), ab.end(), greater<pair<ll, ll>>());
  
  priority_queue<ll, vector<ll>, greater<ll>> pq; // Min-heap for sum of B values
  ll heapSum = 0;
  
  // Initialize the heap with the first K-1 elements
  for (int i = 0; i < k - 1; i++) {
    pq.push(ab[i].second);
    heapSum += ab[i].second;
  }
  
  ll minExpressionVal = inf;
  
  // Process the remaining elements
  for (int i = k - 1; i < n; i++) {
    auto [a, b] = ab[i];
    // Calculate potential value if this A is chosen as max in the subset
    ll currentExpressionVal = (heapSum + b) * a;
    minExpressionVal = min(minExpressionVal, currentExpressionVal);
    
    // If the current B is greater than the top of the heap, replace and adjust the sum
    if (pq.top() > b) {
      heapSum -= pq.top();
      pq.pop();
      pq.push(b);
    }
  }
  
  cout << minExpressionVal << '\n';
}

int main() {
  ios::sync_with_stdio(false);
  std::cin.tie(nullptr);
  int t; cin >> t;
  while (t--) solve();
}
```
