\[ \left(\max_{i \in S} A_i\right) \times \left(\sum_{i \in S} B_i\right) \]

where S is any subset of \(\{1, 2, \dots, N\}\) of size K.

### Key Observations:
1. **Maximization of A**: The first term of the expression is maximized by choosing the maximum value of A from the subset S.
2. **Summation of B**: The second term of the expression is the sum of the values from B in the subset S. 
3. **Subset Size K**: We need to ensure the subset S has exactly K elements.

### Approach:
To minimize the expression, we should aim to minimize the sum of the second term while keeping the maximum of the first term as small as possible. This can be done by considering the values of A in increasing order and adding corresponding values from B to the sum, ensuring the sum remains as small as possible.

1. Pair each element from A with its corresponding element from B.
2. Sort these pairs in increasing order of A.
3. Consider each element from the sorted pairs in order, maintaining a running sum of B values and a count of how many elements from B have been included.
4. Once the count reaches K, calculate the expression using the maximum value of A (current element) and the running sum of B.
5. Update the minimum result if the current result is smaller.
