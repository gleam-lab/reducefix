
The problem statement is asking to find the minimum possible value of the expression:

\[
\left(\max\_{i \in S} A\_i\right) \times \left(\sum\_{i \in S} B\_i\right)
\]

where \( S \) is a subset of \(\{1, 2, \dots, N\}\) with size \( K \). The array \( A \) is of size \( N \) and contains integer values, and the array \( B \) is also of size \( N \) and contains integer values.

### Key Observations:
1. The expression is minimized by minimizing the maximum value in \( A \) and maximizing the sum of \( B \) for the selected subset.
2. To minimize the maximum value of \( A \), we should consider the smallest \( K \) values of \( A \).
3. To maximize the sum of \( B \), we should consider the largest values of \( B \) corresponding to the smallest values of \( A \).

### Solution Strategy:
1. Sort the array \( A \) in ascending order.
2. For the smallest \( K \) values of \( A \), sort the corresponding values of \( B \) in descending order to get the maximum sum of \( B \).
3. Compute the product of the maximum value of \( A \) (which is \( A[K-1] \)) and the sum of the \( K \) largest values of \( B \).

### Implementation:
```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main() {
    int T;
    cin >> T;
    while (T--) {
        int N, K;
        cin >> N >> K;
        vector<int> A(N), B(N);
        
        for (int i = 0; i < N; i++) cin >> A[i];
        for (int i = 0; i < N; i++) cin >> B[i];
        
        sort(A.begin(), A.end());
        sort(B.begin(), B.end(), greater<int>());
        
        long long min_val = A[K-1] * accumulate(B.begin(), B.begin() + K, 0LL);
        cout << min_val << endl;
    }
    return 0;
}
```

This solution first sorts both arrays. For the array \( A \), it sorts in ascending order to consider the smallest \( K \) values. For the array \( B \), it sorts in descending order to maximize the sum of the \( K \) largest values. Then, it calculates the minimum value of the expression by considering the maximum value of \( A \) and the sum of the \( K \) largest values of \( B \).
