
The problem is asking to find the minimum possible value of the expression \((\max_{i \in S} A_i) \times (\sum_{i \in S} B_i)\) where \(S\) is a subset of \(\{1, 2, \dots, N\}\) of size \(K\). This means you need to select \(K\) elements from the array to minimize the given expression.

### Approach

1. **Sort Arrays**: Start by sorting both arrays A and B in non-decreasing order. This will help in efficiently finding the maximum value of \(A_i\) for the subset of size \(K\).

2. **Prefix Sums**: Compute the prefix sums for array B. This will help in quickly calculating the sum of the selected \(K\) elements from B.

3. **Two Pointers Technique**: Use a two-pointer approach to find the optimal \(K\) elements. One pointer will iterate over A from the end (to get the maximum value of \(A_i\)), and the other pointer will iterate over B (using the prefix sums) from the start (to get the sum of selected \(K\) elements).

4. **Calculate Minimum Expression**: For each selected set of \(K\) elements from A and B, calculate the expression and update the minimum value.

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

// Function to solve one test case
void solve() {
    int n, k;
    cin >> n >> k;
    vector<int> A(n), B(n);
    for (int i = 0; i < n; ++i) cin >> A[i];
    for (int i = 0; i < n; ++i) cin >> B[i];

    // Sort A and B
    sort(A.begin(), A.end());
    sort(B.begin(), B.end());

    // Compute prefix sums for B
    vector<long long> prefixSum(n + 1, 0);
    for (int i = 1; i <= n; ++i) {
        prefixSum[i] = prefixSum[i - 1] + B[i - 1];
    }

    long long minExpression = LLONG_MAX;
    // Use two pointers to find the optimal subset
    int start = 0;
    for (int end = n - k; end < n; ++end) {
        long long sum = prefixSum[n] - prefixSum[end + 1];
        minExpression = min(minExpression, A[end + k - 1] * sum);
    }

    // Output the result for the current test case
    cout << minExpression << endl;
}

int main() {
    int t;
    cin >> t;
    while (t--) solve();
    return 0;
}
```

This code first reads the number of test cases. For each test case, it reads \(N\) and \(K\), then arrays \(A\) and \(B\). It sorts \(A\) and \(B\), computes prefix sums for \(B\), and uses a two-pointer approach to find the minimum expression. Finally, it outputs the result for each test case.
