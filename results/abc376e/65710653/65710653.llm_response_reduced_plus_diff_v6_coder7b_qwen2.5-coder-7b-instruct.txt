To address the problem, we need to minimize the expression \((\max_{i \in S} A_i) \times (\sum_{i \in S} B_i)\) for a subset \(S\) of size \(K\) from the given sequences \(A\) and \(B\). The key insight here is that to minimize this product, we should ideally choose the largest \(K\) elements from \(A\) and their corresponding smallest \(K\) elements from \(B\).

Here's how we can approach the solution step-by-step:

1. **Sort the Arrays**: Sort array \(A\) in descending order and array \(B\) in ascending order.
2. **Select Elements**: Select the top \(K\) elements from \(A\) and the bottom \(K\) elements from \(B\).
3. **Calculate the Product**: Compute the product of the maximum element from the selected \(K\) elements of \(A\) and the sum of the selected \(K\) elements of \(B\).

This approach ensures that we are balancing the maximum value from \(A\) with the smallest values from \(B\), which minimizes the overall product.

#include <bits/stdc++.h>
using namespace std;

int main() {

    cin.tie(nullptr);

    int t;
    cin >> t;
    while (t--) {
        int n, k;
        cin >> n >> k;
        
        vector<int> a(n), b(n);
        for (int i = 0; i < n; ++i) cin >> a[i];
        for (int i = 0; i < n; ++i) cin >> b[i];
        
        sort(a.begin(), a.end(), greater<int>());
        sort(b.begin(), b.end());
        
        int max_a = a[k - 1];
        int sum_b = 0;
        for (int i = 0; i < k; ++i) sum_b += b[i];
        
        cout << max_a * sum_b << "\n";
    }

    return 0;
}
