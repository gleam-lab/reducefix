
This problem is about optimizing the expression \(\left(\max_{i \in S} A_i\right) \times \left(\sum_{i \in S} B_i\right)\) by choosing a subset \(S\) of size \(K\) from sequences \(A\) and \(B\).

### Key Observations:
1. The maximum value \(A_i\) should be maximized to get the maximum value of the product.
2. The sum \(\sum_{i \in S} B_i\) should be minimized if \(K\) is greater than the number of elements with maximum \(A_i\) (because including elements with smaller \(A_i\) will decrease the sum and thus the product value).

### Solution Strategy:
1. Sort the sequence \(A\) in descending order. This allows us to quickly find the \(K\) largest elements.
2. Calculate prefix sums of sequence \(B\) to quickly calculate the sum for any subset of \(B\).
3. Use a max-heap (priority queue) to select \(K\) largest elements from \(B\) while minimizing the sum.
4. Iterate over the indices of \(A\) and compute the product for each possible subset size up to \(K\). Keep track of the minimum product found.

### Optimized Solution:
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <queue>
using namespace std;

int main() {
    int T;
    cin >> T;
    while (T--) {
        int N, K;
        cin >> N >> K;
        vector<int> A(N), B(N);
        for (int i = 0; i < N; i++) cin >> A[i];
        for (int i = 0; i < N; i++) cin >> B[i];

        // Sort A in descending order
        sort(A.begin(), A.end(), greater<int>());

        // Priority queue to store the sums of the smallest K elements in B
        priority_queue<int, vector<int>, greater<int>> pq;

        // Calculate prefix sums of B
        vector<long long> prefixSum(N + 1, 0);
        for (int i = 1; i <= N; i++) {
            prefixSum[i] = prefixSum[i - 1] + B[i - 1];
        }

        // Initialize the answer with a large value
        long long ans = LLONG_MAX;

        // Try to form all subsets of size K
        for (int i = 0; i <= N - K; i++) {
            long long sumB = 0;
            // Add the largest K elements of B to sumB
            for (int j = 0; j < K; j++) {
                sumB += B[i + j];
            }
            // Calculate the product with the maximum element from A
            long long product = sumB * A[i + K - 1];
            // Update the answer if this product is smaller
            ans = min(ans, product);
        }

        cout << ans << '\n';
    }
    return 0;
}
```

This solution leverages sorting and prefix sums to efficiently compute the minimum product for each test case. The use of a priority queue is replaced with direct summation to avoid unnecessary complexity.
