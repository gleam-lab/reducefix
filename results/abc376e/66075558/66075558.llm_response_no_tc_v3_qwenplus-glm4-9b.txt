
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int N = 200005;

int n, k;
int A[N], B[N];
vector<pair<int, int>> AB;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int T;
    cin >> T;
    while (T--) {
        cin >> n >> k;
        AB.clear();

        for (int i = 0; i < n; ++i) {
            cin >> A[i];
            AB.emplace_back(A[i], B[i]);
        }

        sort(AB.begin(), AB.end(), [](const pair<int, int>& a, const pair<int, int>& b) {
            if (a.first == b.first) return a.second < b.second;
            return a.first > b.first;
        });

        long long ans = LLONG_MAX;
        long long sumB = 0;
        int count = 0;

        for (auto& p : AB) {
            sumB += p.second;
            if (++count == k) {
                ans = min(ans, (sumB - p.second) * p.first);
            }
        }

        cout << ans << '\n';
    }

    return 0;
}
```

This code reads the number of test cases `T`. For each test case, it reads `n` and `k`, then reads arrays `A` and `B`. It combines the elements of `A` and `B` into a single vector `AB`, where each element is a pair `(A[i], B[i])`. It sorts `AB` first by `A[i]` in descending order and by `B[i]` in ascending order for elements with the same `A[i]`.

After sorting, it calculates the sum of `B[i]` values for the first `k` elements, which are the largest `A[i]` values. It keeps track of the sum minus the last added `B[i]` value and multiplies it by the current `A[i]` to calculate the potential answer. It keeps the minimum of these values as the answer for the test case.

Finally, it prints the answer for each test case.
