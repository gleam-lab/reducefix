
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    int T;
    cin >> T;
    while (T--) {
        int N, K;
        cin >> N >> K;
        vector<int> A(N), B(N);
        for (int i = 0; i < N; ++i) {
            cin >> A[i];
        }
        for (int i = 0; i < N; ++i) {
            cin >> B[i];
        }

        vector<pair<int, int>>AB(N); // Pair of (A[i], B[i])
        for (int i = 0; i < N; ++i) {
            AB[i] = {A[i], B[i]};
        }
        sort(AB.begin(), AB.end(), [](const pair<int, int>& a, const pair<int, int>& b) {
            if (a.first == b.first) return a.second < b.second;
            return a.first > b.first;
        });

        long long min_value = LLONG_MAX;
        long long sum = 0;
        for (int i = 0; i < N; ++i) {
            sum += AB[i].second;
            if (i >= K - 1) { // We have selected K elements
                min_value = min(min_value, AB[i].first * sum);
            }
        }

        cout << min_value << endl;
    }
    return 0;
}
```

### Explanation:
1. **Input Reading**: We read the number of test cases `T` and for each test case, the values of `N` and `K`, followed by the sequences `A` and `B`.
2. **Data Preparation**: We pair each element of `A` with its corresponding element in `B` and store them in a vector `AB` of pairs.
3. **Sorting**: We sort `AB` first by `A[i]` in descending order and, in case of ties, by `B[i]` in ascending order. This ensures that the maximum value of `A[i]` in any subset of size `K` will be at the top, and elements of the same `A[i]` will be sorted by `B[i]`.
4. **Subset Calculation**: We iterate through the sorted pairs and calculate the sum of `B[i]` values as we go. We keep track of the sum of the last `K` elements. For each element in the sorted list, we calculate the product of the maximum `A[i]` in the subset (`AB[i].first`) and the sum of the last `K` `B[i]` values (`sum`). We update `min_value` with the minimum of these products.
5. **Output**: We print the minimum value for each test case.
