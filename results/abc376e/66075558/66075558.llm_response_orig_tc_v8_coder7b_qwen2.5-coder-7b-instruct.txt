To address the problem efficiently, we need to focus on finding the optimal subset \( S \) of size \( K \) that minimizes the expression \((\max_i A_i) \times (\sum_i B_i)\). Given the constraints, we can use a greedy approach combined with sorting to achieve this.

Here's a step-by-step breakdown of the solution:

1. **Sort the Arrays**: We will sort both arrays \( A \) and \( B \) based on their values. This allows us to easily select the largest elements from \( A \) and smallest elements from \( B \) to minimize the product.

2. **Select Elements**: We will iterate through the sorted arrays, selecting the top \( K \) elements from \( A \) and the corresponding elements from \( B \). We maintain a running sum of these selected elements from \( B \) and keep track of the maximum element from \( A \).

3. **Calculate the Result**: Finally, we calculate the result as the product of the maximum element from \( A \) and the sum of the selected elements from \( B \).

This approach ensures that we are always considering the best possible combination of elements from \( A \) and \( B \) to minimize the desired expression.

#include<iostream>
#include<vector>
#include<algorithm>
using namespace std;

typedef long long ll;

struct Element {
    ll val;
    int index;
};

bool cmp(const Element &a, const Element &b) {
    return a.val != b.val ? a.val < b.val : a.index < b.index;
}

ll solve(int n, int k, vector<ll>& A, vector<ll>& B) {
    vector<Element> a(n), b(n);
    
    // Populate elements with their values and indices
    for (int i = 0; i < n; ++i) {
        a[i] = {A[i], i};
        b[i] = {B[i], i};
    }
    
    // Sort elements based on their values
    sort(a.begin(), a.end(), cmp);
    sort(b.begin(), b.end(), cmp);
    
    // Find the maximum element in A
    ll maxA = a[k-1].val;
    
    // Calculate the sum of the first K elements in B
    ll sumB = 0;
    for (int i = 0; i < k; ++i) {
        sumB += b[i].val;
    }
    
    // Return the result
    return maxA * sumB;
}

int main() {
    int t;
    cin >> t;
    while (t--) {
        int n, k;
        cin >> n >> k;
        vector<ll> A(n), B(n);
        
        for (int i = 0; i < n; ++i) {
            cin >> A[i];
        }
        
        for (int i = 0; i < n; ++i) {
            cin >> B[i];
        }

    }

}

4. **Sum Calculation**: We calculate the sum of the first \( K \) elements of \( B \).
5. **Result Calculation**: The result is the product of the maximum element from \( A \) and the sum of the selected elements from \( B \).
