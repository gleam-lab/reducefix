```cpp
#include <bits/stdc++.h>
using namespace std;

#define LL long long

LL n, m;
set<LL> row, col, diag1, diag2;

int main() {
    cin >> n >> m;
    while (m--) {
        LL x, y; cin >> x >> y;
        row.insert(x); col.insert(y); diag1.insert(x + y); diag2.insert(x - y);
    }
    
    LL res = n * n - row.size() - col.size() - diag1.size() - diag2.size();
    for (auto& x : row) res -= diag1.count(x + y);
    for (auto& y : col) res -= diag2.count(x - y);
    cout << res / 2 << endl; // Each intersection is counted twice
    return 0;
}
```
This solution uses sets to store all unique row, column, diagonal1 (top-left to bottom-right), and diagonal2 (top-right to bottom-left) indices. The total number of empty squares is calculated by subtracting the sizes of these sets from the total number of squares. For each unique row and column index, we adjust the result to account for intersections between diagonals and the corresponding row or column. Finally, we divide the result by two since each intersection is counted twice in the initial calculation.
