
This problem involves placing a new piece on a grid such that it is not captured by any of the existing pieces. The grid is characterized by its size \(N \times N\) and \(M\) existing pieces placed at specific positions. The challenge is to determine how many empty squares are safe for placing the new piece.

### Key Concepts:
1. **Capture Rules**: A piece placed at \((i, j)\) can capture pieces that:
   - Are in the same row or column.
   - Are at any \((a, b)\) such that \(i+j = a+b\) or \(i-j = a-b\).

2. **Efficient Calculation**: Given the constraints, an efficient approach is necessary to determine the number of safe squares, especially since \(N\) can be as large as \(10^9\).

3. **Data Structures**: Utilize data structures to keep track of occupied rows, columns, and diagonals to quickly determine if a placement is safe.

### Approach:
1. **Initialize Data Structures**: Create arrays to track occupied rows, columns, and diagonals.
2. **Mark Occupied Positions**: For each piece, mark the corresponding row, column, and diagonals as occupied.
3. **Calculate Safe Squares**:
   - Subtract the total number of occupied rows, columns, and diagonals.
   - Add back the intersections of these occupied lines, as they were subtracted multiple times.
   - Adjust for the cases where two or more lines intersect the same square.

#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

const int MAX_N = 1009;

int N, M;
vector<pair<int, int>> pieces;

vector<int> occupied_rows, occupied_cols, occupied_diags_pos, occupied_diags_neg;

bool is_safe(int r, int c) {
    // Check if the row or column is occupied
    if (find(occupied_rows.begin(), occupied_rows.end(), r) != occupied_rows.end()) return false;
    if (find(occupied_cols.begin(), occupied_cols.end(), c) != occupied_cols.end()) return false;

    // Check if any diagonal is occupied
    int diag1 = r + c;
    int diag2 = r - c + N - 1;
    if (find(occupied_diags_pos.begin(), occupied_diags_pos.end(), diag1) != occupied_diags_pos.end()) return false;
    if (find(occupied_diags_neg.begin(), occupied_diags_neg.end(), diag2) != occupied_diags_neg.end()) return false;

    return true;
}

int main() {
    cin >> N >> M;
    for (int i = 0; i < M; i++) {
        int r, c;
        cin >> r >> c;
        r++, c++; // Convert to 1-based index
        pieces.push_back({r, c});

        // Mark row and column as occupied
        occupied_rows.push_back(r);
        occupied_cols.push_back(c);

        // Mark diagonals as occupied
        occupied_diags_pos.push_back(r + c);
        occupied_diags_neg.push_back(r - c + N - 1);

        // Sort and remove duplicates
        sort(occupied_rows.begin(), occupied_rows.end());
        sort(occupied_cols.begin(), occupied_cols.end());
        sort(occupied_diags_pos.begin(), occupied_diags_pos.end());
        sort(occupied_diags_neg.begin(), occupied_diags_neg.end());

        occupied_diags_neg.erase(unique(occupied_diags_neg.begin(), occupied_diags_neg.end()), occupied_diags_neg.end());
    }

    int safe_squares = 0;
    for (int r = 1; r <= N; r++) {
        for (int c = 1; c <= N; c++) {
            if (is_safe(r, c)) safe_squares++;
        }
    }
