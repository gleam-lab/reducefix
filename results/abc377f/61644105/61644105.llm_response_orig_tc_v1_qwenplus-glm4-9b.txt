
### Problem Analysis

The problem involves placing a piece on a grid such that it is not captured by any existing pieces. The grid is N x N, and each piece can capture other pieces based on its row, column, and diagonal positions. The task is to determine how many empty squares are available for placing a new piece.

### Solution Strategy

The solution involves calculating the number of safe squares where the new piece can be placed. This is done by:

1. **Identifying Captured Squares**: For each piece, determine which squares it can capture based on its position. This involves checking rows, columns, and diagonals.
2. **Counting Safe Squares**: Subtract the number of captured squares from the total number of squares to get the number of safe squares.

### Steps to Implement the Solution

1. **Input Reading**: Read the size of the grid (N) and the number of pieces (M). Then, read the positions of each piece.
2. **Initialization**: Initialize data structures to keep track of captured squares in rows, columns, and diagonals.
3. **Capture Calculation**: For each piece, mark the squares it can capture.
4. **Safe Square Calculation**: Calculate the total number of squares and subtract the number of captured squares to get the number of safe squares.

### Optimized Code Implementation

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <cmath>

using namespace std;

long long N, M;
vector<pair<int, int>> pieces;

// Directions for rows and columns
const vector<pair<int, int>> directions = {
    {0, 1}, {1, 0}, {1, 1}, {1, -1}
};

// Function to calculate the total number of safe squares
long long calculateSafeSquares() {
    long long totalSquares = N * N;
    vector<vector<bool>> captured(N + 1, vector<bool>(N + 1, false));
    
    // Mark captured squares for each piece
    for (auto p : pieces) {
        int x = p.first, y = p.second;
        for (auto dir : directions) {
            int nx = x + dir.first * (N + 1), ny = y + dir.second * (N + 1);
            captured[nx][ny] = true;
        }
    }
    
    // Calculate the number of safe squares
    long long safeSquares = 0;
    for (int i = 1; i <= N; ++i) {
        for (int j = 1; j <= N; ++j) {
            if (!captured[i][j]) {
                safeSquares++;
            }
        }
    }
    
    return safeSquares;
}

int main() {
    cin >> N >> M;
    pieces.resize(M);
    for (int i = 0; i < M; ++i) {
        cin >> pieces[i].first >> pieces[i].second;
    }
    
    cout << calculateSafeSquares() << endl;
    return 0;
}
```

### Explanation

1. **Input Reading**: The grid size `N` and the number of pieces `M` are read. The positions of each piece are stored in a vector.
2. **Capture Calculation**: Using the `directions` vector, each piece's position is used to mark the squares it can capture in a 2D vector `captured`.
3. **Safe Square Calculation**: Iterate over each square in the grid and count those that are not marked as captured.
4. **Output**: The number of safe squares is printed.

This approach ensures that we efficiently calculate the number of safe squares by directly marking captured squares and counting the unmarked ones.
