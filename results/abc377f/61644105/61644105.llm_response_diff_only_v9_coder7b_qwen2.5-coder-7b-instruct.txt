To solve this problem, we need to determine how many empty squares on an \(N \times N\) grid can have a new piece placed without being captured by any of the existing pieces. The capturing rules are based on rows, columns, and diagonals.

Here's a step-by-step approach to solving this problem:

1. **Input Parsing**: Read the grid size \(N\) and the number of existing pieces \(M\). Then, read the positions of these pieces.

2. **Data Structures**: Use sets to store the positions of pieces for quick lookup and to avoid duplicates.

3. **Capture Rules**: For each possible position \((i, j)\) on the grid, check if placing a piece there would result in it being captured. A piece at \((i, j)\) is captured if there is already a piece in the same row, column, or diagonal.

4. **Count Valid Positions**: Iterate through all possible positions on the grid and count those that do not violate the capture rules.

Here is the corrected and optimized C++ code:

```cpp
#include <iostream>
#include <set>
#include <algorithm>

using namespace std;

int main() {
    long long N, M;
    cin >> N >> M;

    set<long long> rows, cols, diags1, diags2;

    for (long long i = 0; i < M; ++i) {
        long long x, y;
        cin >> x >> y;
        rows.insert(x);
        cols.insert(y);
        diags1.insert(x + y);
        diags2.insert(x - y);
    }

    long long valid_positions = 0;

    for (long long i = 1; i <= N; ++i) {
        for (long long j = 1; j <= N; ++j) {
            if (rows.find(i) == rows.end() && 
                cols.find(j) == cols.end() && 
                diags1.find(i + j) == diags1.end() && 
                diags2.find(i - j) == diags2.end()) {
                ++valid_positions;
            }
        }
    }

    cout << valid_positions << endl;

    return 0;
}
```

### Explanation:
1. **Input Reading**: We read the grid size \(N\) and the number of pieces \(M\). Then, for each piece, we read its coordinates and add them to sets `rows`, `cols`, `diags1`, and `diags2` which represent the occupied rows, columns, and diagonals respectively.

2. **Valid Position Check**: For each position \((i, j)\) on the grid, we check if it does not belong to any of the sets `rows`, `cols`, `diags1`, or `diags2`. If it doesn't, then it is a valid position.

3. **Counting Valid Positions**: We count all valid positions and print the result.

This solution ensures that we efficiently check each position and count only those that meet the criteria. The use of sets allows for quick lookups and insertions, making the solution both correct and efficient within the given constraints.
