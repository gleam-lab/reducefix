The main problem is to count safe positions on an N x N grid where a new piece can be placed without being captured by any existing pieces. The existing pieces can capture along rows, columns, and both diagonals.

### Approaches
1. **Identify Threatened Positions**: For each existing piece, mark all positions it threatens in rows, columns, and both diagonals.
2. **Union of Threatened Positions**: The total number of unsafe positions is the union of all positions threatened by any existing piece.
3. **Inclusion-Exclusion Principle**: Calculate the total number of safe positions by subtracting the union of all threatened positions from the total possible positions (N²).

#include <bits/stdc++.h>

using namespace std;

int main() {
    long long N, M;
    cin >> N >> M;
    
    set<long long> rows, cols, diag1, diag2;
    
    for (int i = 0; i < M; ++i) {
        long long a, b;
        cin >> a >> b;
        rows.insert(a);
        cols.insert(b);
        diag1.insert(a + b);
        diag2.insert(a - b);
    }
    
    long long row_cnt = rows.size();
    long long col_cnt = cols.size();
    long long diag1_cnt = diag1.size();
    long long diag2_cnt = diag2.size();
    
    // Total unsafe positions: rows + cols + diag1 + diag2 intersections
    long long unsafe = row_cnt * N + col_cnt * N - row_cnt * col_cnt;
    
    // Additional for diagonals
    for (auto d : diag1) {
        long long max_a = min(d - 1, N);
        long long min_a = max(1LL, d - N);
        unsafe += (max_a - min_a + 1);
    }
    for (auto d : diag2) {
        long long max_a = min(d + N, N);
        long long min_a = max(1LL, d + 1);
        unsafe += (max_a - min_a + 1);
    }
    
    // Subtract intersections
    for (auto a : rows) {
        for (auto b : cols) {
            if (diag1.find(a + b) != diag1.end() && diag2.find(a - b) != diag2.end()) {
                unsafe -= 1;
            }
        }
    }
    
    for (auto d1 : diag1) {
        for (auto d2 : diag2) {
            long long a = (d1 + d2) / 2;
            long long b = (d1 - d2) / 2;
            if ((d1 + d2) % 2 == 0 && (d1 - d2) % 2 == 0 && a >= 1 && a <= N && b >= 1 && b <= N) {
                if (rows.find(a) != rows.end() && cols.find(b) != cols.end()) {
                    unsafe -= 1;
                }
            }
        }
    }
    
    long long total = N * N;
    long long safe = total - unsafe;
    
    cout << safe << endl;
    
    return 0;
}

### Explanation
1. **Initialization**: We read the size of the grid (N) and the number of existing pieces (M).
2. **Tracking Threatened Positions**: We use sets to keep track of unique rows, columns, and diagonals (sum and difference diagonals) that are threatened by existing pieces.
3. **Counting Threatened Positions**: We calculate the initial count of threatened positions by considering rows, columns, and diagonals separately.
4. **Inclusion-Exclusion Principle**: We adjust the count by subtracting the intersections of these sets to avoid double-counting positions that are threatened in multiple ways (e.g., a position in both a threatened row and column).
5. **Final Calculation**: The total number of safe positions is the total grid size (N²) minus the adjusted count of threatened positions.

This approach efficiently handles the large constraints (N up to 10^9 and M up to 10^3) by leveraging set operations and mathematical calculations to avoid brute-force checks.
