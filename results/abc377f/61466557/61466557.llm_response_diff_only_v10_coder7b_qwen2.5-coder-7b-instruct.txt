To address the problem of placing a new piece on an empty square such that it cannot be captured by any existing pieces, we need to consider the constraints and properties of the board and the positions of the existing pieces.

### Key Observations:
1. **Rows and Columns**: Each piece captures all pieces in its row and column.
2. **Diagonals**: Pieces also capture all pieces on two types of diagonals: \(i + j = \text{constant}\) and \(i - j = \text{constant}\).

Given these observations, we can deduce the following:
- A position \((i, j)\) is safe if no other piece is present in its row, column, or diagonals.

### Algorithm:
1. Identify all positions occupied by existing pieces.
2. For each possible position \((i, j)\), check if it is in any of the unsafe zones (rows, columns, or diagonals).
3. Count the number of safe positions.

### Implementation:
We will use sets to store the positions of pieces in rows, columns, and diagonals. This allows us to efficiently check if a position is unsafe.

Here is the corrected and optimized C++ code:

```cpp
#include <iostream>
#include <vector>
#include <set>

using namespace std;

const int MAXN = 1e9;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);

    long long n, m;
    cin >> n >> m;

    vector<long long> rows(MAXN + 1, 0);
    vector<long long> cols(MAXN + 1, 0);
    vector<long long> diags1(2 * MAXN + 1, 0);
    vector<long long> diags2(2 * MAXN + 1, 0);

    for (long long i = 0; i < m; ++i) {
        long long a, b;
        cin >> a >> b;
        rows[a]++;
        cols[b]++;
        diags1[a + b]++;
        diags2[a - b + MAXN]++;
    }

    long long safe_positions = 0;

    for (long long i = 1; i <= n; ++i) {
        for (long long j = 1; j <= n; ++j) {
            if (rows[i] == 0 && cols[j] == 0 && diags1[i + j] == 0 && diags2[i - j + MAXN] == 0) {
                safe_positions++;
            }
        }
    }

    cout << safe_positions << endl;

    return 0;
}
```

### Explanation:
1. **Initialization**: We initialize vectors to keep track of the number of pieces in each row, column, diagonal \(i + j\), and diagonal \(i - j\).
2. **Reading Input**: We read the number of pieces and their positions, updating the respective counters.
3. **Counting Safe Positions**: We iterate through all possible positions on the board and check if they are safe by verifying that no piece is in the same row, column, or diagonal.
4. **Output**: Finally, we print the count of safe positions.

This approach ensures that we efficiently determine the number of safe positions using linear time complexity relative to the number of pieces and the size of the board.
