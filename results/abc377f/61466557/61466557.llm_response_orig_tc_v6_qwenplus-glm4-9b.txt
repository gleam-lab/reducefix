
Given the problem, the goal is to find out how many squares on an \(N \times N\) grid are safe to place a piece on such that it can't be captured by any of the \(M\) pre-placed pieces. A piece placed on any square \((i, j)\) can capture pieces that are in the same row or column, or that are on any diagonal passing through \((i, j)\).

### Approach

1. **Identify captured rows and columns**: First, identify all the rows and columns that are already fully captured by existing pieces.

2. **Identify captured diagonals**: Next, identify all the diagonals that are fully captured. There are two sets of diagonals: one with the sum of the indices equal and the other with the difference of the indices equal.

3. **Count safe squares**: Subtract all the fully captured rows, columns, and diagonals from the total number of squares to get the number of safe squares.

### Detailed Steps

1. **Input handling**: Read the grid size \(N\) and the number of pieces \(M\). Then, read the positions of the \(M\) pieces.

2. **Identify captured rows and columns**: Create two sets, `captured_rows` and `captured_cols`, to store the indices of captured rows and columns.

3. **Identify captured diagonals**:
   - For diagonals with the sum of indices equal, use the form \(i + j = k\). Iterate over all possible row and column combinations to find all such diagonals that are fully captured.
   - For diagonals with the difference of indices equal, use the form \(i - j = k\). Similarly, iterate over all possible row and column combinations to find all such diagonals that are fully captured.

4. **Count safe squares**: Initialize `safe_count` to \(N^2\), the total number of squares. Subtract \(N\) for each captured row and column. Subtract the sizes of the sets of captured diagonals.

5. **Output the result**: Print the value of `safe_count`.

### C++ Code

```cpp
#include <iostream>
#include <unordered_set>
using namespace std;

int main() {
    int N, M;
    cin >> N >> M;

    unordered_set<int> captured_rows, captured_cols, captured_d1, captured_d2;

    for (int i = 0; i < M; i++) {
        int a, b;
        cin >> a >> b;
        a--; b--; // Convert to 0-based indexing
        captured_rows.insert(a);
        captured_cols.insert(b);
        captured_d1.insert(a + b);
        captured_d2.insert(a - b);
    }

    long long safe_count = N * N;

    // Subtract captured rows and columns
    for (int i = 0; i < N; i++) {
        if (captured_rows.count(i) || captured_cols.count(i)) {
            safe_count -= N;
        }
    }

    // Subtract captured diagonals
    safe_count -= captured_d1.size();
    safe_count -= captured_d2.size();

    cout << safe_count << endl;

    return 0;
}
```

This code efficiently calculates the number of safe squares by using sets to track captured rows, columns, and diagonals. It avoids the need for multiple checks and nested loops, making it suitable for large grid sizes and a moderate number of pieces.
