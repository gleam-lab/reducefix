To address the issue, we need to carefully count the number of valid positions where a new piece can be placed without being captured by any existing pieces. The key observation here is to use sets to track occupied rows, columns, diagonals, and anti-diagonals. This allows us to efficiently determine if a position is safe.

#include <bits/stdc++.h>
using namespace std;

#define ll long long

const int MAXN = 1e9 + 10;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    int n, m;
    cin >> n >> m;

    unordered_set<int> rows, cols, diags, anti_diags;
    for (int i = 0; i < m; ++i) {
        int a, b;
        cin >> a >> b;
        rows.insert(a);
        cols.insert(b);
        diags.insert(a + b);
        anti_diags.insert(a - b);
    }

    ll total_positions = n * n;
    ll occupied_positions = rows.size() * n + cols.size() * n - rows.size() * cols.size();

    // Diagonal captures
    for (int d = -n; d <= n; ++d) {
        if (diags.count(d)) {
            int count = min(rows.size(), cols.size());
            occupied_positions += count;
        }
    }

    // Anti-diagonal captures
    for (int d = -n; d <= n; ++d) {
        if (anti_diags.count(d)) {
            int count = min(rows.size(), cols.size());
            occupied_positions += count;
        }
    }

    ll safe_positions = total_positions - occupied_positions;
    cout << safe_positions << endl;

    return 0;
}

### Explanation:
1. **Initialization**: We initialize sets `rows`, `cols`, `diags`, and `anti_diags` to keep track of occupied positions in rows, columns, diagonals, and anti-diagonals respectively.
2. **Reading Input**: We read the input values and populate the sets with the positions of the existing pieces.
3. **Total Positions Calculation**: We calculate the total number of positions on the board (`total_positions = n * n`).
4. **Occupied Positions Calculation**:

   - We then add back the contributions from diagonal and anti-diagonal captures. Each diagonal or anti-diagonal captures all positions along its line, so we add the minimum of the sizes of the row and column sets to account for overlaps.
5. **Safe Positions Calculation**: Finally, we subtract the occupied positions from the total positions to get the number of safe positions.

This approach ensures that we correctly count all valid positions without being captured, handling all edge cases and constraints effectively.
