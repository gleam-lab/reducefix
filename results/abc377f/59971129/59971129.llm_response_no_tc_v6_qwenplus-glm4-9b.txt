
```cpp
#include<bits/stdc++.h>

using namespace std;

typedef long long ll;

#define N 1005

ll n, m;

int main() {
    scanf("%lld %lld", &n, &m);
    set<ll> rows, cols, diag1, diag2;
    for (int i = 0; i < m; ++i) {
        ll r, c;
        scanf("%lld %lld", &r, &c);
        rows.insert(r);
        cols.insert(c);
        diag1.insert(r + c);
        diag2.insert(abs(r - c));
    }

    // Total squares are n*n
    ll total_squares = n * n;

    // Subtract squares that are covered by rows and columns
    ll covered_by_rows = rows.size();
    ll covered_by_cols = cols.size();
    ll covered_by_diag1 = diag1.size();
    ll covered_by_diag2 = diag2.size();

    // Subtract overlaps
    set<ll> all_covered;
    all_covered.insert(rows.begin(), rows.end());
    all_covered.insert(cols.begin(), cols.end());
    all_covered.insert(diag1.begin(), diag1.end());
    all_covered.insert(diag2.begin(), diag2.end());

    ll overlaps = all_covered.size();

    // Calculate the number of safe squares
    ll safe_squares = total_squares - covered_by_rows - covered_by_cols -
                       covered_by_diag1 - covered_by_diag2 + overlaps;

    printf("%lld\n", safe_squares);
    return 0;
}
```

### Explanation:
1. **Input Reading**: The program starts by reading the total number of rows `n` and columns `m`, followed by the positions of the `M` pieces.
2. **Set of Covered Squares**: For each piece, it adds its row, column, and both diagonals to their respective sets. This helps in determining which rows, columns, and diagonals are covered by the pieces.
3. **Total and Covered Squares**: The total number of squares is `n*n`.
4. **Subtract Covered Squares**: It subtracts the number of squares covered by rows, columns, and both diagonals.
5. **Subtract Overlaps**: Since some squares might be counted multiple times (e.g., the intersection of a row and a column), it adds the size of the combined set of all covered squares to get the correct count.
6. **Output**: Finally, it prints the number of safe squares where a new piece can be placed without being captured by any existing pieces.
