3. **Calculate Safe Squares**: Subtract the number of threatened squares from the total number of squares (N^2), adjusting for overlaps and ensuring we only count empty squares.

        }

    for (auto it : pieces) {
        ll i = it.first;

        if (rows.count(i) && cols.count(j)) {
