
```cpp
#include <iostream>
#include <set>
#include <vector>

using namespace std;
using ll = long long;

int main() {
    ll n, m;
    cin >> n >> m;
    vector<ll> pieces_x(m), pieces_y(m);
    
    for (ll i = 0; i < m; ++i) {
        cin >> pieces_x[i] >> pieces_y[i];
    }

    set<ll> row, col, diag1, diag2;

    // Collecting rows, columns, and diagonals that are occupied by pieces
    for (ll i = 0; i < m; ++i) {
        row.insert(pieces_x[i]);
        col.insert(pieces_y[i]);
        diag1.insert(pieces_x[i] + pieces_y[i]);
        diag2.insert(pieces_x[i] - pieces_y[i]);
    }

    // Calculate the number of free cells in rows, columns, diagonals
    ll free_rows = n - row.size(), free_cols = n - col.size();
    ll free_diag1 = n - diag1.size(), free_diag2 = n - diag2.size();

    // Calculate the number of free cells in 2x2 blocks
    ll free_2x2_blocks = (n - 1) * (n - 1) - (free_diag1 + free_diag2 + free_rows + free_cols - 3 * n);

    // Output the number of free cells
    cout << free_2x2_blocks << endl;

    return 0;
}
```

### Explanation

1. **Input Reading**: The code starts by reading the size of the grid `n` and the number of pieces `m`. It then reads the positions of the pieces into vectors `pieces_x` and `pieces_y`.

2. **Collecting Pieces Information**: For each piece, its row, column, and both diagonals are added to their respective sets (`row`, `col`, `diag1`, `diag2`). Using sets ensures that duplicates are eliminated.

3. **Calculating Free Cells**: 
   - The number of free rows is the total number of rows `n` minus the number of occupied rows in the `row` set.
   - Similarly, free columns and diagonals are calculated.
   - The number of free 2x2 blocks is the total number of 2x2 blocks in the grid minus the number of these blocks that are occupied by pieces. This is done by subtracting the occupied blocks from each other and from the total number of rows and columns.

4. **Output**: The total number of free cells is printed. This is the sum of free rows, free columns, free diagonals, and free 2x2 blocks.

This solution is efficient and handles the constraints well, as it only requires linear time relative to the number of pieces `m` and constant time for each operation on the sets.
