
The problem involves finding a safe square on an \(N \times N\) grid where no existing pieces can capture your new piece. The challenge lies in the fact that a piece can capture others based on its row, column, or diagonal positions. Given the constraints, a brute-force approach to check each empty square would be computationally infeasible, especially when \(N\) can be as large as \(10^9\).

### Strategy

The solution involves identifying regions of the grid that are safe from capture by the existing pieces. This can be broken down into the following steps:

1. **Identify Capture Patterns**: Determine the capture patterns for each existing piece.
2. **Mark Capture Regions**: Mark regions of the grid that are unsafe due to these capture patterns.
3. **Count Safe Squares**: Calculate the number of squares that are not marked as unsafe.

### Detailed Steps

1. **Initialize Capture Sets**: Create sets to keep track of rows, columns, and diagonals that are under threat from the existing pieces.
2. **Locate Capture Centers**: Identify points on the grid that are critical in determining capture regions. These include intersection points of rows and columns, and points on diagonals.
3. **Sort Capture Centers**: Sort these critical points to facilitate efficient marking of unsafe regions.
4. **Mark Unsafe Regions**: Iterate over the sorted list of capture centers and mark the corresponding rows, columns, and diagonals as unsafe.
5. **Calculate Safe Region**: Use the marked information to calculate the total number of safe squares.

### Python Code Implementation

```python
def safe_squares(n, m, positions):
    # Initialize sets to track unsafe rows, columns, and diagonals
    unsafe_rows = set()
    unsafe_cols = set()
    unsafe_d1 = set()  # Diagonal from top-left to bottom-right
    unsafe_d2 = set()  # Diagonal from top-right to bottom-left

    # Mark the capture regions
    for x, y in positions:
        x, y = x + 1, y + 1  # Convert to 1-based index
        unsafe_rows.add(x)
        unsafe_cols.add(y)
        unsafe_d1.add(x + y)
        unsafe_d2.add(x - y)

    # Count the number of rows, columns, and diagonals that are unsafe
    unsafe_row_count = len(unsafe_rows)
    unsafe_col_count = len(unsafe_cols)
    unsafe_d1_count = len(unsafe_d1)
    unsafe_d2_count = len(unsafe_d2)

    # Calculate the total number of squares
    total_squares = n * n

    # Subtract the number of unsafe squares
    # Each row or column that is unsafe includes all its squares
    unsafe_squares = unsafe_row_count * n + unsafe_col_count * n
    # Each diagonal can capture up to n squares, but we need to subtract the overlap
    for d in range(1, n + 1):
        # Count the unique diagonals in both directions that are unsafe
        unique_d1 = unsafe_d1.difference(set(range(d, n + 1)))
        unique_d2 = unsafe_d2.difference(set(range(n - d + 1)))
        unsafe_squares += min(len(unique_d1), d) * n
        unsafe_squares += min(len(unique_d2), d) * n

    # The final number of safe squares
    return total_squares - unsafe_squares

# Example usage
n = 8
m = 6
positions = [(1, 4), (2, 1), (3, 8), (4, 5), (5, 2), (8, 3)]
print(safe_squares(n, m, positions))  # Output: 2
```

This implementation efficiently calculates the number of safe squares by leveraging set operations to mark capture regions and then calculating the total number of squares that are not marked as unsafe. This approach avoids the need to check each individual square on the grid, making it feasible even for large values of \(N\).
