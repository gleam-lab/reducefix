
The problem requires us to find the number of empty squares on an N x N grid where a new piece can be placed without being captured by any of the M existing pieces.

### Solution Explanation

The solution involves several steps:

1. **Identifying Captured Rows and Columns**:
    - First, we need to determine which rows and columns are fully occupied by the enemy pieces. A row or column is considered fully occupied if all its cells are under threat by any enemy piece.
    - This is done by iterating over each piece and checking which rows and columns it affects (based on its capture conditions).

2. **Handling Diagonals**:
    - Diagonals can either be fully occupied or threatened by multiple pieces. We need to handle this separately.
    - We will count the number of threats on each diagonal (both `i + j` and `i - j`) and determine if a diagonal is fully occupied or not.

3. **Calculating Safe Squares**:
    - After determining which rows, columns, and diagonals are under threat, we can calculate the number of safe squares.
    - We subtract the number of threatened rows, columns, and diagonals from the total number of squares (N^2).

    set<int> rows, cols, diag1, diag2, anti_diag1, anti_diag2;
    for (auto &piece : pieces) {
        int r = piece.first, c = piece.second;
        rows.insert(r);
        cols.insert(c);
        diag1.insert(r + c);
        diag2.insert(r - c);
        anti_diag1.insert(c - r + n);
        anti_diag2.insert(c + r);
    }

    // Total number of squares minus the number of threatened squares
    int total_squares = n * n;
    int threatened_rows = rows.size();
    int threatened_cols = cols.size();
    int threatened_diag1 = diag1.size();
    int threatened_diag2 = diag2.size();
    int threatened_anti_diag1 = anti_diag1.size();
    int threatened_anti_diag2 = anti_diag2.size();

    // Subtract threatened squares
    int safe_squares = total_squares - threatened_rows * n - threatened_cols * n;
    safe_squares += threatened_diag1 * (n - 1) + threatened_diag2 * (n - 1);
    safe_squares += threatened_anti_diag1 * (n - 1) + threatened_anti_diag2 * (n - 1);
