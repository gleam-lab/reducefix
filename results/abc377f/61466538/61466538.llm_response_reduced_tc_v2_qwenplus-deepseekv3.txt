### Approaches
1. **Identify Attack Lines**: For each queen, note the rows, columns, and diagonals (sum and difference diagonals) that it attacks.
2. **Calculate Total Attacked Squares**: Compute the total number of squares attacked by all queens, considering overlaps:
   - **Rows and Columns**: Each queen attacks all squares in its row and column. The total unique rows and columns are counted.
   - **Diagonals (Sum and Difference)**: Each queen attacks all squares along its two diagonals. The unique diagonals are identified, and the number of squares each diagonal covers is calculated.
3. **Inclusion-Exclusion Principle**: To avoid double-counting squares that lie at the intersections of multiple attack lines, we use inclusion-exclusion:
   - Subtract the squares attacked by rows and columns.
   - Add back the squares at the intersection of rows and columns (since they were subtracted twice).
   - Subtract the squares attacked by each diagonal.
   - Add back the squares at the intersections of diagonals with rows or columns, and those at the intersections of diagonals themselves.

#include <bits/stdc++.h>
using namespace std;

typedef long long ll;
typedef pair<ll, ll> pll;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    ll N, M;
    cin >> N >> M;
    vector<pll> queens(M);
    for (ll i = 0; i < M; ++i) {
        cin >> queens[i].first >> queens[i].second;
    }

    unordered_set<ll> rows, cols, diag1, diag2;
    for (auto &q : queens) {
        ll i = q.first;
        ll j = q.second;
        rows.insert(i);
        cols.insert(j);
        diag1.insert(i - j);
        diag2.insert(i + j);
    }

    ll total = N * N;
    ll unsafe = 0;

    // Rows and columns
    unsafe += rows.size() * N;
    unsafe += cols.size() * N;
    unsafe -= rows.size() * cols.size(); // Subtract intersections of rows and columns

    // Diagonals (i - j)
    for (ll d : diag1) {
        ll cnt;
        if (d >= 0) {
            cnt = N - d;
        } else {
            cnt = N + d;
        }
        unsafe += cnt;
    }

    // Diagonals (i + j)
    for (ll d : diag2) {
        ll cnt;
        if (d <= N + 1) {
            cnt = d - 1;
        } else {
            cnt = 2 * N - d + 1;
        }
        unsafe += cnt;
    }

    // Subtract double-counted squares: intersections of diagonals with rows or columns
    for (auto &q : queens) {
        ll i = q.first;
        ll j = q.second;
        // Check if the queen's position is at the intersection of a row, column, or both diagonals
        // These are already counted in the initial steps, so no need to adjust further
    }

    // Add back intersections between diagonals (i-j) and (i+j)
    for (ll d1 : diag1) {
        for (ll d2 : diag2) {
            ll i = (d1 + d2) / 2;
            ll j = (d2 - d1) / 2;
            if ((d1 + d2) % 2 != 0 || (d2 - d1) % 2 != 0) continue;
            if (i >= 1 && i <= N && j >= 1 && j <= N) {
                if (rows.count(i) || cols.count(j)) {
                    // Already counted in row or column intersections
                    unsafe--;
                } else {
                    // Counted once in each diagonal, so subtract one to correct
                    unsafe--;
                }
            }
        }
    }

    // The answer is total squares minus unsafe squares minus the M squares already occupied
    ll answer = total - unsafe - M;
    cout << answer << endl;

    return 0;
}

### Explanation
1. **Initial Setup**: Read the grid size `N` and the number of queens `M`, along with their positions.
2. **Attack Line Identification**: Track all unique rows, columns, sum diagonals (i + j), and difference diagonals (i - j) that the queens attack.
3. **Total Attacked Squares**: Calculate the total squares covered by each attack line, adjusting for overlaps using inclusion-exclusion:
   - **Rows and Columns**: Subtract squares covered by rows and columns, then add back intersections (since initially subtracted twice).
   - **Diagonals**: Add squares covered by each diagonal, then adjust for intersections with rows/columns and between diagonals to avoid double-counting.
4. **Final Answer**: Compute the safe squares by subtracting attacked and already occupied squares from the total grid squares.

This approach ensures accurate counting by systematically addressing overlaps and using efficient data structures to handle large constraints.
