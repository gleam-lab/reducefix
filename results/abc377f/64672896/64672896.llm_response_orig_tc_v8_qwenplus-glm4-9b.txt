
To solve this problem, we need to understand the capturing conditions. A piece placed at (i, j) can capture pieces in:

- Same row i
- Same column j
- Diagonal i+j
- Diagonal i-j

Given the constraints, especially with N up to 10^9, a brute-force checking every empty square for each capturing condition is infeasible. Instead, we need a more efficient approach.

### Strategy

1. **Count Captured Squares**: First, we count how many squares are captured by existing pieces in each row, column, and diagonal. This will help us remove these squares from the list of possible placements.

2. **Calculate Non-Captured Squares**: After identifying captured squares, we calculate the number of squares that are not captured by any of the conditions.

### Detailed Steps

1. **Initialize Counters**:
    - `rowCapt`: Array of size N to count captured squares in each row.
    - `colCapt`: Array of size N to count captured squares in each column.
    - `diaCapt`: Two arrays (`diaCapt1` and `diaCapt2`) of size 2N-1 to count captured squares on diagonals i+j and i-j respectively.

2. **Count Captured Squares**:
    - For each piece, increment the count in `rowCapt`, `colCapt`, `diaCapt1`, and `diaCapt2` based on its position.

3. **Calculate Non-Captured Squares**:
    - For each row, column, and diagonal, check how many squares are not captured. For diagonals, we need to handle the overlap carefully.

### Implementation

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

long long countNonCapturedSquares(int N, vector<pair<int, int>>& pieces) {
    vector<int> rowCapt(N + 1, 0);
    vector<int> colCapt(N + 1, 0);
    vector<int> diaCapt1(2 * N + 1, 0);
    vector<int> diaCapt2(2 * N + 1, 0);
    
    // Count captured squares
    for (auto& p : pieces) {
        int a = p.first;
        int b = p.second;
        rowCapt[a]++;
        colCapt[b]++;
        diaCapt1[a + b]++;
        diaCapt2[a - b]++;
    }
    
    long long nonCaptured = N * N;
    
    // Subtract captured squares
    for (int i = 1; i <= N; ++i) {
        nonCaptured -= rowCapt[i];
        nonCaptured -= colCapt[i];
        for (int j = 1; j <= 2 * N; ++j) {
            nonCaptured -= max(0LL, diaCapt1[j] - 1);
            nonCaptured -= max(0LL, diaCapt2[j] - 1);
        }
    }
    
    return nonCaptured;
}

int main() {
    int N, M;
    cin >> N >> M;
    vector<pair<int, int>> pieces(M);
    for (int i = 0; i < M; ++i) {
        cin >> pieces[i].first >> pieces[i].second;
    }
    
    cout << countNonCapturedSquares(N, pieces) << endl;
    return 0;
}
```

This solution efficiently calculates the number of non-captured squares by counting the captured squares and subtracting them from the total number of squares. The time complexity is O(N + M), which is efficient given the constraints.
