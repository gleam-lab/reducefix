To solve this problem efficiently, we need to determine how many empty squares on an \( N \times N \) grid can be occupied without being captured by any of the existing pieces. Each piece can capture pieces based on their position relative to the new piece's position. 

Given the constraints, brute-force checking every possible square is impractical due to the large size of \( N \). Instead, we can use a more analytical approach to count the valid positions.

Here’s a step-by-step breakdown of the solution:

1. **Identify Capture Zones**: For each piece, identify the rows, columns, diagonals, and anti-diagonals it can capture. These zones will help us determine which squares are unsafe for placing new pieces.

2. **Count Unsafe Squares**: Using the identified capture zones, count all the squares that fall within these zones.

3. **Calculate Valid Squares**: Subtract the count of unsafe squares from the total number of squares to get the number of valid squares.

Here is the corrected and optimized C++ code implementing the above logic:

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    long long N, M;
    cin >> N >> M;

    set<pair<long long, long long>> pieces;
    vector<long long> rows(N+1, 0), cols(N+1, 0), diags(N+1, 0), anti_diags(N+1, 0);

    for (long long i = 0; i < M; ++i) {
        long long a, b;
        cin >> a >> b;
        pieces.insert({a, b});
        rows[a]++;
        cols[b]++;
        diags[a + b]++;
        anti_diags[a - b]++;
    }

    long long total_unsafe = 0;

    for (const auto& [a, b] : pieces) {
        total_unsafe += rows[a];
        total_unsafe += cols[b];
        total_unsafe += diags[a + b];
        total_unsafe += anti_diags[a - b];
    }

    // Remove double-counted intersections
    for (const auto& [a, b] : pieces) {
        total_unsafe -= rows[a] > 1;
        total_unsafe -= cols[b] > 1;
        total_unsafe -= diags[a + b] > 1;
        total_unsafe -= anti_diags[a - b] > 1;
    }

    // Total valid squares
    long long valid_squares = N * N - total_unsafe;
    cout << valid_squares << endl;

    return 0;
}
```

### Explanation:
1. **Input Reading**: Read the dimensions \( N \) and the number of pieces \( M \).
2. **Data Structures**: Use sets and vectors to keep track of the positions of pieces and the counts of pieces in each row, column, diagonal, and anti-diagonal.
3. **Capture Zone Calculation**: For each piece, add the counts of pieces in its respective row, column, diagonal, and anti-diagonal.
4. **Double-Counting Correction**: Since pieces on the same row, column, diagonal, or anti-diagonal are counted multiple times, subtract the overcounted pieces.
5. **Result Calculation**: The total number of valid squares is the total number of squares minus the total number of unsafe squares.

This approach ensures that we efficiently calculate the number of valid squares without having to check each square individually.
