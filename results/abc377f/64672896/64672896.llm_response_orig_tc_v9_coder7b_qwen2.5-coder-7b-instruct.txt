#include<bits/stdc++.h>
using namespace std;
using ll = long long;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);

    ll N, M;
    cin >> N >> M;

    vector<pair<ll, ll>> pieces(M);
    set<ll> sums, diffs;
    for (ll i = 0; i < M; i++) {
        cin >> pieces[i].first >> pieces[i].second;
        sums.insert(pieces[i].first + pieces[i].second);
        diffs.insert(pieces[i].first - pieces[i].second);
    }

    ll total_empty_squares = N * N;
    for (const auto& [x, y] : pieces) {
        total_empty_squares -= (sums.count(x + y) > 0) + (diffs.count(x - y) > 0) + (x > 0) + (y > 0) + (x < N) + (y < N);
    }
    total_empty_squares += M;

    cout << total_empty_squares << '\n';

    return 0;
}

This solution works as follows:

1. We read the grid dimensions \(N\) and the number of pieces \(M\).
2. We store the positions of the pieces in a vector `pieces`.
3. We use sets `sums` and `diffs` to keep track of the lines where pieces are located. The lines are identified by their sums \(x + y\) and differences \(x - y\).
4. We initialize `total_empty_squares` to \(N \times N\).
5. For each piece, we subtract from `total_empty_squares` based on whether there are other pieces in the same row, column, diagonal (both directions).
6. Finally, we add back the number of pieces because they are not empty squares but occupy valid positions.
7. We output the result.

This approach ensures that we correctly count the number of empty squares where no piece can capture another piece.
