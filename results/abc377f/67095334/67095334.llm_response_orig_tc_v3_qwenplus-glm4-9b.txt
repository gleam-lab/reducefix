
The problem is asking to find the number of empty squares on an N x N grid where a new piece can be placed without being captured by existing pieces. Given the constraints, a direct simulation is impractical due to the large size of N (up to 10^9). Instead, we need to use a more efficient approach.

### Solution Overview:

1. **Count Empty Squares**: First, we calculate the total number of empty squares in the grid, which is N^2 minus the number of occupied squares.

2. **Determine Capture Range**: For each occupied square, determine which other squares it can capture. This involves checking the row, column, and two diagonals (i+j and i-j).

3. **Avoid Captured Squares**: For each capture range (row, column, diagonal), we need to subtract the number of squares that are already captured by existing pieces.

4. **Output the Result**: Finally, subtract the number of captured squares from the total empty squares to get the number of squares where the new piece can be placed safely.

### Detailed Steps:

1. **Initialize Variables**: We will use sets to store the occupied squares, rows, columns, and diagonals for efficient lookup and removal of elements.

2. **Count Occupied Squares**: For each occupied square, add it to the sets for its row, column, and both diagonals.

3. **Calculate Capture Range**: For each set (rows, columns, diagonals), count the number of occupied squares. For each empty square in the grid, check if it is in any of these sets. If it is, it means it is captured by existing pieces.

4. **Count Safe Squares**: Subtract the number of captured squares from the total number of empty squares.

5. **Output the Result**: Print the number of safe squares.

### Python Code:

```python
def main():
    import sys
    input = sys.stdin.read
    data = input().split()
    
    N = int(data[0])
    M = int(data[1])
    
    occupied = set()
    
    index = 2
    for _ in range(M):
        a = int(data[index]) - 1
        b = int(data[index + 1]) - 1
        occupied.add((a, b))
        index += 2
    
    total_empty_squares = N * N - M
    
    occupied_rows = set()
    occupied_cols = set()
    occupied_diag1 = set()
    occupied_diag2 = set()
    
    for r, c in occupied:
        occupied_rows.add(r)
        occupied_cols.add(c)
        occupied_diag1.add(r + c)
        occupied_diag2.add(r - c)
    
    # Calculate captured squares
    captured_squares = 0
    
    # Captured by rows and columns
    captured_squares += len(occupied_rows)
    captured_squares += len(occupied_cols)
    
    # Captured by diagonals
    captured_squares += len(occupied_diag1)
    captured_squares += len(occupied_diag2)
    
    # Each diagonal point is subtracted twice, so we add it once
    captured_squares += len(set((r + c for r, c in occupied)))
    
    # Subtract captured squares from total empty squares
    safe_squares = total_empty_squares - captured_squares
    
    print(safe_squares)

if __name__ == "__main__":
    main()
```

This solution is efficient and works within the constraints, as it uses sets for constant time lookups and avoids unnecessary computations.
