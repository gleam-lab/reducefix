### Approaches
1. **Rows and Columns**: First, count the squares that are not in any row or column occupied by existing pieces. This is done by subtracting the occupied rows and columns from the total grid size.
2. **Diagonals**: For each diagonal (both types: i + j = d and i - j = d), calculate the number of squares that are not already covered by rows or columns. 
3. **Overlap Handling**: Ensure that squares covered by both rows or columns and diagonals are not double-counted. This requires careful subtraction of overlaps.

### Solution Code
```cpp
#include<bits/stdc++.h>
#define int long long
using namespace std;

void solve() {
    int N, M;
    cin >> N >> M;
    set<int> rows, cols, diag1, diag2;
    for (int i = 0; i < M; ++i) {
        int a, b;
        cin >> a >> b;
        rows.insert(a);
        cols.insert(b);
        diag1.insert(a + b);
        diag2.insert(a - b);
    }
    
    int safe_rows = N - rows.size();
    int safe_cols = N - cols.size();
    int initial_safe = safe_rows * safe_cols;
    
    for (auto d : diag1) {
        int min_i = max(1LL, d - N);
        int max_i = min(N, d - 1);
        int diagonal_length = max_i - min_i + 1;
        
        set<int> covered_rows;
        for (auto r : rows) {
            if (1 <= d - r && d - r <= N) {
                covered_rows.insert(r);
            }
        }
        for (auto c : cols) {
            if (1 <= d - c && d - c <= N) {
                covered_rows.insert(d - c);
            }
        }
        
        int safe_in_diag = diagonal_length - covered_rows.size();
        initial_safe -= safe_in_diag;
    }
    
    for (auto d : diag2) {
        int min_i = max(1LL, 1 + d);
        int max_i = min(N, N + d);
        int diagonal_length = max_i - min_i + 1;
        
        set<int> covered_rows;
        for (auto r : rows) {
            if (1 <= r - d && r - d <= N) {
                covered_rows.insert(r);
            }
        }
        for (auto c : cols) {
            if (1 <= c + d && c + d <= N) {
                covered_rows.insert(c + d);
            }
        }
        
        int safe_in_diag = diagonal_length - covered_rows.size();
        initial_safe -= safe_in_diag;
    }
    
    cout << initial_safe << endl;
}

signed main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int _ = 1;
    while (_--) {
        solve();
    }
    
    return 0;
}
```

2. **Safe Rows and Columns**: The number of safe rows and columns is calculated by subtracting the occupied ones from the total grid size. The initial safe squares are the product of safe rows and columns.
3. **Diagonal Handling**: For each diagonal (i + j = d and i - j = d), the code calculates the length of the diagonal and checks for intersections with occupied rows and columns. The safe squares on each diagonal are those not covered by any row or column.
4. **Overlap Subtraction**: The safe squares on diagonals are subtracted from the initial safe count to ensure overlaps are not double-counted.
5. **Output**: The final count of safe squares is printed.

This approach efficiently handles the constraints and ensures correctness by carefully accounting for all possible overlaps and intersections.
