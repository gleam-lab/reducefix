
### Problem Analysis

The problem involves placing a piece on an empty square on an N x N grid such that it is not captured by any of the M existing pieces. The capturing rules are based on rows, columns, and two types of diagonals (sum and difference of row and column indices).

### Solution Strategy

1. **Identify Captured Lines**: First, identify which rows, columns, and diagonals are already captured by the given pieces.
2. **Count Captured Squares**: Calculate the number of squares captured by these lines.
3. **Calculate Remaining Squares**: Subtract the number of captured squares from the total number of squares (N^2) to get the number of squares where the new piece can be placed without being captured.

### Detailed Steps

1. **Input Parsing**: Read the grid size N and the number of pieces M. Then, read the positions of the M pieces.
2. **Capture Detection**:
   - Use sets to track captured rows, columns, and diagonals.
   - For each piece, update these sets.
3. **Counting Captured Squares**:
   - For rows and columns, count the number of consecutive captured squares.
   - For diagonals, consider both the sum and difference forms. Handle special cases where the diagonals might wrap around the grid edges.
4. **Calculate Result**:
   - Subtract the number of captured squares from N^2.

### Implementation

```python
def main():
    import sys
    input = sys.stdin.read
    data = input().split()
    
    n = int(data[0])
    m = int(data[1])
    
    index = 2
    rows = set()
    cols = set()
    diag1 = set()
    diag2 = set()
    
    for _ in range(m):
        x = int(data[index])
        y = int(data[index + 1])
        index += 2
        rows.add(x)
        cols.add(y)
        diag1.add(x + y)
        diag2.add(x - y)
    
    total_squares = n * n
    
    # Calculate captured squares by rows and columns
    row_captured = 0
    last_row = 0
    for r in sorted(rows):
        row_captured += r - last_row
        last_row = r
    
    col_captured = 0
    last_col = 0
    for c in sorted(cols):
        col_captured += c - last_col
        last_col = c
    
    # Calculate captured squares by diagonals
    diag1_captured = 0
    last_diag1 = 0
    for d in sorted(diag1):
        diag1_captured += d - last_diag1
        last_diag1 = d
    
    diag2_captured = 0
    last_diag2 = 0
    for d in sorted(diag2):
        diag2_captured += d - last_diag2
        last_diag2 = d
    
    # Subtract captured squares from total
    captured_squares = row_captured + col_captured + diag1_captured + diag2_captured
    
    # Output the result
    print(total_squares - captured_squares)

if __name__ == "__main__":
    main()
```

This implementation reads the input, calculates which rows, columns, and diagonals are captured, and then determines the number of squares that can be safely occupied by the new piece. The use of sets and sorting ensures that we only consider the first occurrence of each line and correctly handle overlapping or consecutive lines.
