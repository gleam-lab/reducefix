
The problem requires finding the number of empty squares in a grid where a new piece can be placed without being captured by any existing pieces. The grid is of size \( N \times N \) and there are \( M \) pieces placed on specific squares. A piece placed on a square can capture other pieces based on certain conditions involving the row, column, and diagonals.

### Analysis of the Problem

1. **Capture Conditions**: A piece at position \( (i, j) \) can capture other pieces based on the rows, columns, and diagonals passing through \( (i, j) \). This includes:
   - All pieces in the same row \( i \).
   - All pieces in the same column \( j \).
   - All pieces on the diagonal \( i + j \).
   - All pieces on the anti-diagonal \( i - j \).

2. **Counting Captured Squares**: For each square \( (i, j) \), we need to determine how many pieces can capture it. This depends on the positions of the existing pieces.

3. **Calculating Empty Squares**: The total number of squares is \( N^2 \). Subtract the number of squares that can be captured by existing pieces from \( N^2 \) to get the number of safe squares where a new piece can be placed.

### Optimized Solution Approach

1. **Calculate Capture Set for Each Square**: For each square \( (i, j) \), determine the set of pieces that can capture it. This requires checking the row, column, and both diagonals.

2. **Determine Safe Squares**: Once the capture sets are determined for all squares, iterate through the grid and count the number of squares that have an empty capture set.

3. **Edge Cases**: Handle grids with minimal size (e.g., \( N = 1 \)) and grids where all pieces are placed in a single row or column.

### Implementation Details

1. **Efficient Set Operations**: Use sets to keep track of captured squares for each row, column, and diagonal. This allows for efficient checks and updates.

2. **Handling Large Grids**: Given the constraints, it's crucial to avoid iterating over the entire grid explicitly. Use mathematical properties to reduce the problem size where possible.

3. **Output**: After calculating the number of safe squares, output the result.

```python
def main():
    import sys
    input = sys.stdin.read
    data = input().split()
    
    N = int(data[0])
    M = int(data[1])
    pieces = []
    index = 2
    for _ in range(M):
        x = int(data[index])
        y = int(data[index + 1])
        pieces.append((x, y))
        index += 2
    
    # Capture sets for rows, columns, diagonals, and anti-diagonals
    row_capture = [set() for _ in range(N)]
    col_capture = [set() for _ in range(N)]
    diag_capture = [set() for _ in range(2 * N)]
    anti_diag_capture = [set() for _ in range(2 * N)]
    
    # Populate capture sets
    for x, y in pieces:
        row_capture[x - 1].add((x, y))
        col_capture[y - 1].add((x, y))
        diag_capture[x + y - 1].add((x, y))
        anti_diag_capture[x - y + N - 1].add((x, y))
    
    # Calculate the number of safe squares
    safe_squares = 0
    for i in range(N):
        for j in range(N):
            if ((x, y) in row_capture[i] or 
                (x, y) in col_capture[j] or 
                (x, y) in diag_capture[i + j] or 
                (x, y) in anti_diag_capture[i - j + N - 1]):
                continue
            safe_squares += 1
    
    print(safe_squares)

if __name__ == "__main__":
    main()
```

This implementation reads input, calculates capture sets for rows, columns, diagonals, and anti-diagonals, and then counts the number of safe squares by checking if each square is not captured by any piece. This approach ensures that we efficiently handle the constraints provided.
