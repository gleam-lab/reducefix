
### Problem Analysis

The problem involves placing a piece on an empty square on an \(N \times N\) grid such that it is not captured by any of the \(M\) existing pieces. Each piece can potentially capture other pieces based on its position in rows, columns, and diagonals. The task is to determine how many empty squares are available for placing the new piece.

### Solution Strategy

1. **Initial Count of Empty Squares**: Start by calculating the total number of empty squares, which is \(N^2 - M\).

2. **Capture Analysis**: For each existing piece, determine the regions (rows, columns, diagonals) that it can capture. This involves:
   - Identifying which rows, columns, and diagonals are directly affected by the piece's placement.
   - Using sets to keep track of which rows, columns, and diagonals are already captured.

3. **Overlap Handling**: If multiple pieces can capture the same row, column, or diagonal, the overlap needs to be handled correctly. This involves ensuring that the count of affected squares is not double-counted or over-counted.

4. **Final Calculation**: Subtract the total number of affected squares from the initial count of empty squares to get the number of safe squares.

### Implementation Details

- **Data Structures**: Use sets to store unique rows, columns, and diagonals that are affected by the placement of each piece.
- **Efficiency Considerations**: Given the constraints, the solution must be efficient in both time and space. The approach should ideally be \(O(N + M)\) or better.
- **Edge Cases**: Handle cases where \(N\) is very large (up to \(10^9\)) and \(M\) is relatively small (up to \(10^3\)), ensuring that the operations are efficient even for large values of \(N\).

### Python Code Implementation

```python
def solve():
    import sys
    input = sys.stdin.read
    data = input().split()
    
    n = int(data[0])
    m = int(data[1])
    
    # Initialize sets for rows, columns, and diagonals
    rows = set()
    cols = set()
    diag1 = set()  # i+j
    diag2 = set()  # i-j
    
    index = 2
    for _ in range(m):
        a = int(data[index])
        b = int(data[index + 1])
        index += 2
        
        rows.add(a)
        cols.add(b)
        diag1.add(a + b)
        diag2.add(a - b)
    
    # Function to calculate the number of elements in the intersection of two sorted lists
    def intersection_size(x, y):
        i, j = 0, 0
        size = 0
        while i < len(x) and j < len(y):
            if x[i] < y[j]:
                i += 1
            elif x[i] > y[j]:
                j += 1
            else:
                size += 1
                i += 1
                j += 1
        return size
    
    # Total number of affected squares
    affected = 0
    
    # Rows and columns
    affected += len(rows) + len(cols) - 1
    
    # Diagonals i+j
    diag1_sorted = sorted(diag1)
    affected += intersection_size(diag1_sorted, list(range(2, 2 * n + 1))) - 2
    
    # Diagonals i-j
    diag2_sorted = sorted(diag2)
    affected += intersection_size(diag2_sorted, list(range(-n + 1, n)))
    
    # Subtract the double counted intersections
    affected -= intersection_size(rows, cols)  # Overlap between rows and columns
    
    # Subtract the double counted intersections with diagonals
    affected -= intersection_size(diag1_sorted, cols)
    affected -= intersection_size(diag2_sorted, rows)
    
    # Calculate the number of safe squares
    safe_squares = n * n - affected
    
    print(safe_squares)

```
