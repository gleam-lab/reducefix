
To solve the problem, we need to consider the following:

1. **Grid Dimensions**: We have an \(N \times N\) grid.
2. **Existing Pieces**: We have \(M\) pieces placed on this grid, each at a specific coordinate.
3. **Capture Mechanism**: A piece can capture other pieces based on its row, column, and two diagonals.

The task is to count the number of empty squares where a new piece can be placed without being captured by any existing pieces.

### Key Observations:
- If we place a piece in any empty square, it can potentially capture pieces in its row, column, and two diagonals.
- We can use symmetry to simplify the problem; we don't need to consider both diagonals independently.

### Solution Strategy:
1. **Count Captured Squares**: For each piece, we calculate how many squares in its row, column, and two diagonals are no longer available.
2. **Calculate Safe Squares**: Subtract the number of captured squares from the total number of squares \(N^2\) to get the number of safe squares.

### Implementation Steps:
1. **Initialize Data Structures**: Use sets to store the rows, columns, and diagonals that are already occupied by pieces.
2. **For Each Piece**:
   - Add its row, column, and diagonals to the respective sets.
   - Calculate the number of squares in these sets and subtract from the total number of squares.
3. **Output the Result**: Print the number of safe squares.

### Python Code:
```python
def main():
    import sys
    input = sys.stdin.read
    data = input().split()
    
    n = int(data[0])
    m = int(data[1])
    
    rows = set()
    cols = set()
    d1 = set()  # i + j
    d2 = set()  # i - j
    
    index = 2
    for _ in range(m):
        a = int(data[index])
        b = int(data[index + 1])
        index += 2
        
        rows.add(a)
        cols.add(b)
        d1.add(a + b)
        d2.add(a - b)
    
    total_squares = n * n
    captured_squares = 0
    
    # Calculate captured squares in rows
    for r in rows:
        captured_squares += (n - len(cols))  # All columns except the ones in cols
    
    # Calculate captured squares in columns
    for c in cols:
        captured_squares += (n - len(rows))  # All rows except the ones in rows
    
    # Calculate captured squares in diagonals
    for d in d1:
        if 1 <= d <= 2 * n:  # Only consider valid diagonals
            left = max(1, 1 - d)
            right = min(n, n - d)
            captured_squares += (right - left + 1) * (n - len({d - x for x in rows if 1 <= x <= n}))
    
    for d in d2:
        if -2 * n <= d <= 2 * n:  # Only consider valid diagonals
            left = max(1, 1 + d)
            right = min(n, n + d)
            captured_squares += (right - left + 1) * (n - len({d + x for x in cols if 1 <= x <= n}))
    
    safe_squares = total_squares - captured_squares
    print(safe_squares)

if __name__ == "__main__":
    main()
```

This solution efficiently calculates the number of safe squares by considering the capture patterns of each piece and adjusting the count accordingly.
