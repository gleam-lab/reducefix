
The problem is asking to find the number of empty squares on a grid where you can place a piece without being captured by existing pieces. The capture rules are based on rows, columns, and diagonals.

### Problem Analysis
Given a grid of size \(N \times N\), we need to determine how many empty cells are safe from existing pieces. Each piece can capture based on its row, column, and two specific diagonals. The task is to compute the number of cells that are not in the capture range of any existing piece.

### Solution Strategy
1. **Capture Representation**: We will represent the capture ranges of the existing pieces in terms of rows, columns, and diagonals.
2. **Efficient Checking**: Instead of checking each empty cell individually, we will use a set-based approach to quickly check if a cell is safe.
3. **Counting Safe Cells**: For each cell, we will check if it is in any of the capture ranges using the precomputed sets.

### Steps to Solve
1. Parse the input to get the grid size \(N\) and the number of pieces \(M\).
2. Read the positions of the pieces and create sets for:
   - Rows occupied by pieces
   - Columns occupied by pieces
   - Diagonals \(i+j\) occupied by pieces
   - Diagonals \(i-j\) occupied by pieces
3. Calculate the total number of empty cells.
4. Subtract the number of cells that are in the capture range of any piece from the total number of cells.

### Implementation
```cpp
#include <iostream>
#include <vector>
#include <unordered_set>
using namespace std;

int main() {
    long long N, M;
    cin >> N >> M;
    
    unordered_set<long long> rows, cols, diag1, diag2;
    
    for (long long i = 0; i < M; ++i) {
        long long a, b;
        cin >> a >> b;
        rows.insert(a);
        cols.insert(b);
        diag1.insert(a + b);
        diag2.insert(a - b);
    }
    
    long long total_empty_cells = N * N;
    long long captured_cells = 0;
    
    // Count captured cells in rows
    for (long long i = 1; i <= N; ++i) {
        captured_cells += rows.count(i);
    }
    
    // Count captured cells in columns
    for (long long j = 1; j <= N; ++j) {
        captured_cells += cols.count(j);
    }
    
    // Count captured cells in diagonals i+j
    for (long long d = 2; d <= 2 * N; ++d) {
        captured_cells += diag1.count(d);
    }
    
    // Count captured cells in diagonals i-j
    for (long long d = -N + 1; d <= N - 1; ++d) {
        captured_cells += diag2.count(d);
    }
    
    // Subtract captured cells from total cells
    long long safe_cells = total_empty_cells - captured_cells;
    
    cout << safe_cells << endl;
    
    return 0;
}
```

### Explanation
- We use `unordered_set` for efficient lookups.
- We calculate the number of captured cells in each row, column, and diagonal separately.
- We subtract the total captured cells from the total number of cells to get the number of safe cells.
