
### Problem Analysis

The problem involves placing a piece on an empty square in an \(N \times N\) grid such that it is not captured by any of the \(M\) existing pieces. The capture conditions are based on rows, columns, and two types of diagonals (sum and difference of row and column indices).

### Solution Strategy

1. **Identify Captured Lines**: For each piece, determine which rows, columns, and diagonals are captured.
2. **Count Free Squares**: Subtract the number of squares on captured lines from the total number of squares in the grid.
3. **Handle Large Grids**: Given the constraints, directly simulating the grid is infeasible. Instead, use mathematical properties to calculate the impact of captured lines on free squares.

### Detailed Steps

1. **Initialization**:
   - Use sets to track captured rows (`h`), columns (`v`), and diagonals (`d1` and `d2`).
   - Calculate the total number of squares, \(\text{count\_total}\), as \(N^2\).

2. **Capture Analysis**:
   - For each piece, update the sets for captured rows, columns, and diagonals.
   - For each captured row, column, and diagonal, count how many times it is captured.

3. **Calculating Free Squares**:
   - Use the inclusion-exclusion principle to calculate the number of squares captured by multiple lines.
   - Subtract the total number of captured squares from \(\text{count\_total}\) to get the number of free squares.

4. **Output**:
   - Print the number of free squares.

### Optimizations

- **Efficient Counting**: Use a class `Counter` to efficiently count occurrences and intersections of captured lines.
- **Handling Large Diagonal Captures**: For large diagonals that span multiple rows and columns, calculate the intersection of captured segments to determine the total impact on free squares.

### Implementation

```cpp
#include <iostream>
#include <vector>
#include <set>
#include <algorithm>
using namespace std;

const int MOD = 1e9 + 7;

class Counter {
public:
    set<int> rows, cols, diag1, diag2;
    int total, row_capture, col_capture, diag1_capture, diag2_capture;

    void add_row(int r) { rows.insert(r); }
    void add_col(int c) { cols.insert(c); }
    void add_diag1(int d) { diag1.insert(d); }
    void add_diag2(int d) { diag2.insert(d); }

    void count() {
        row_capture = rows.size();
        col_capture = cols.size();
        diag1_capture = diag1.size();
        diag2_capture = diag2.size();

        // Calculate the total number of captured squares
        total = row_capture * N + col_capture * N + diag1_capture + diag2_capture;

        // Subtract double-counted intersections
        for (int r : rows) {
            if (diag1.count(r)) total--;
            if (diag2.count(r)) total--;
        }
        for (int c : cols) {
            if (diag1.count(N + c)) total--;
            if (diag2.count(c)) total--;
        }
    }
};

int main() {
    int N, M;
    cin >> N >> M;

    Counter counter;
    for (int i = 0; i < M; ++i) {
        int a, b;
        cin >> a >> b;
        counter.add_row(a);
        counter.add_col(b);
        counter.add_diag1(a + b);
        counter.add_diag2(a - b);
    }

    counter.count();

    // Total squares minus captured squares
    int free_squares = (long long)N * N - counter.total;
    cout << free_squares << endl;

    return 0;
}
```

This refined solution introduces a `Counter` class to manage and calculate the impact of captured lines efficiently. It avoids the need for complex intersection calculations by leveraging the properties of sets and the inclusion-exclusion principle.
