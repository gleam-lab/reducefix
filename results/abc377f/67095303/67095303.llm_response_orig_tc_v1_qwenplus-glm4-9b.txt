
This problem requires finding the number of empty squares on an N x N grid where a new piece can be placed without being captured by any existing pieces.

### Solution Strategy

The key to solving this problem is to understand the captured regions by existing pieces and then determine the number of squares that are not in these regions.

1. **Captured Regions**:
   - Each piece captures:
     - All cells in its row.
     - All cells in its column.
     - All cells on the two diagonals passing through it.
   
2. **Count Non-Captured Cells**:
   - For each cell that is not captured, count it towards the total.
   - Subtract cells that are on the boundaries of captured regions, as they are partially captured.

### Implementation Steps

1. **Input Parsing**:
   - Read grid size `N` and number of pieces `M`.
   - Read the positions of the pieces.

2. **Initialization**:
   - Create a matrix ` captures ` to mark captured cells.

3. **Mark Captured Cells**:
   - For each piece, mark its row, column, and both diagonals as captured.

4. **Count Non-Captured Cells**:
   - Iterate over all cells, and if a cell is not captured, increment the count.

5. **Output the Result**:
   - Print the total count of non-captured cells.

### Optimizations

- Using a boolean matrix to mark captured cells is not feasible due to memory constraints when N is large.
- Instead, use arrays to mark captured rows, columns, and diagonals.

Here's a C++ implementation that follows the above strategy:

```cpp
#include <iostream>
#include <vector>
#include <set>
using namespace std;

const int MOD = 1e9 + 7;

int main() {
    long long N, M;
    cin >> N >> M;
    
    vector<vector<bool>> rowCaptured(N+1, vector<bool>(N+1, false));
    vector<vector<bool>> colCaptured(N+1, vector<bool>(N+1, false));
    vector<pair<long long, long long>> pieces(M);
    
    for (int i = 0; i < M; ++i) {
        long long a, b;
        cin >> a >> b;
        pieces[i] = {a, b};
    }
    
    // Mark captured cells
    for (auto [a, b] : pieces) {
        rowCaptured[a][b] = true;
        colCaptured[b][a] = true;
        
        long long d1 = a + b;
        long long d2 = a - b;
        
        for (long long i = min(d1, N); i >= max(0LL, d1 - N + 1); --i) {
            rowCaptured[i][d1 - i] = colCaptured[d1 - i][i] = true;
        }
        
        for (long long i = min(d2, N); i >= max(0LL, d2 - N + 1); --i) {
            rowCaptured[i][d2 + i] = colCaptured[d2 + i][i] = true;
        }
    }
    
    // Count non-captured cells
    long long result = 0;
    for (long long i = 1; i <= N; ++i) {
        for (long long j = 1; j <= N; ++j) {
            if (!rowCaptured[i][j] && !colCaptured[j][i]) {
                result++;
            }
        }
    }
    
    cout << result << endl;
    
    return 0;
}
```

This implementation efficiently marks captured cells and counts non-captured cells without using excessive memory or computation.
