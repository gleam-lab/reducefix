
```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

int main() {
    int n;
    cin >> n;
    vector<ll> h(n);
    for (int i = 0; i < n; ++i) {
        cin >> h[i];
    }

    ll t = 0;
    for (ll h : h) {
        if (h % 3 == 0) {
            t += (h / 3);
        } else {
            t += (h / 3) + 2;  // When health % 3 != 0, it will need two attacks
        }
    }

    cout << t << endl;

    return 0;
}
```

### Explanation:
1. The code is simplified by removing the unnecessary `num_attack` function, as the pattern of the attack becomes clear when analyzing the problem statement.
2. For each enemy, if its health is a multiple of 3, it takes `h / 3` attacks to defeat it.
3. If the health is not a multiple of 3, it still takes `h / 3` full attacks, but one additional attack to take care of the remainder.
4. The total number of attacks `t` is accumulated during the loop for each enemy.
5. This approach ensures that the solution runs in O(n) time complexity, which is efficient given the constraints.
