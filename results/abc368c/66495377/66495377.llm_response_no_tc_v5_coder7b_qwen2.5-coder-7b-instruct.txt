```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

// Function to calculate the number of attacks needed for an enemy with health h
ll num_attack(ll h) {
    ll cnt = 0;
    while (h > 0) {
        cnt++;
        if (cnt % 3 == 0) {
            h -= 3;
        } else {
            h--;
        }
    }
    return cnt;
}

int main() {
    int n;
    cin >> n;
    ll total_attacks = 0;
    for (int i = 0; i < n; ++i) {
        ll h;
        cin >> h;
        total_attacks += num_attack(h);
    }

    cout << total_attacks << endl;
}
```

### Explanation of Changes
1. **Removed the `num_attack` function parameter `t`**: The original code was trying to use `t` as a counter that increased each time an attack was made. However, this approach was flawed because `t` was not being reset correctly between different enemies. Instead, we can simply count the number of attacks needed for each enemy independently.

2. **Simplified the `num_attack` function**: The function now directly calculates the number of attacks needed to reduce the enemy's health to zero or below without considering any external counter like `t`. This makes the function simpler and easier to understand.

3. **Corrected the main loop logic**: In the main loop, we now call `num_attack` once for each enemy and add the result to `total_attacks`, which keeps track of the total number of attacks across all enemies. Finally, we print `total_attacks`.
