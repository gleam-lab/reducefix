
```cpp
#include <iostream>
using namespace std;
using ll = long long;

int main() {
    ll n;
    cin >> n;
    vector<ll> H(n);
    for (ll i = 0; i < n; ++i) {
        cin >> H[i];
    }
    
    ll T = 0;
    for (ll health : H) {
        if (health == 0) continue;
        
        if (T % 3 == 0) {
            T += (health + 2) / 3;
        } else {
            T += health;
        }
    }
    
    cout << T << endl;
    return 0;
}
```

This code calculates the total number of attacks `T` needed to defeat all enemies. The main logic is as follows:

- If `T % 3 == 0`, the next attack will be a triple damage attack, so we need to increase `T` by `(health + 2) / 3` to ensure we can perform a triple damage attack after the current one.
- If `T % 3 != 0`, we can only make a single damage attack, so we increase `T` by the current enemy's health `health`.

This avoids the need to iterate through each attack, making the code more efficient.
