```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

ll num_attack(ll t, ll h) {
    ll cnt = 0;
    cnt += 3 * (h / 3); // 3 is the damage per attack
    int remind = h % 3;
    while (remind > 0) {
        t++;
        if (t % 3 == 0) {
            remind -= 3;
        } else {
            remind--;
        }
        cnt++;
    }
    return cnt;
}

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    int n;
    cin >> n;
    ll t = 0;
    for (int i = 0; i < n; ++i) {
        ll h;
        cin >> h;
        t += num_attack(t, h);
    }

    cout << t << endl;

    return 0;
}
```

### Explanation
1. **Function `num_attack`**: This function calculates the number of attacks needed to reduce an enemy's health to 0 or less. It divides the health by 3 and adds the result to the count. The remainder is then processed in a loop where each iteration increases the attack count and reduces the remaining health accordingly.
2. **Main Function**:
   - **Fast I/O Optimization**: `ios_base::sync_with_stdio(false); cin.tie(NULL);` is used to optimize the input/output operations.
   - Reading Input: The number of enemies `n` is read first. For each enemy, its health `h` is read, and the total attack count `t` is updated by calling `num_attack`.
   - Printing Result: Finally, the total attack count `t` is printed.

This solution ensures that the code is efficient and handles the constraints properly, avoiding any hard-coded values or special cases.
