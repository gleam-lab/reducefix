```cpp
#include<bits/stdc++.h>
using namespace std;
using ll = long long;

int main() {
    ll n, ans=0, tri=1;
    cin>>n;
    vector<ll> enemies(n);
    for(int i=0;i<n;i++) {
        cin>>enemies[i];
    }

    sort(enemies.begin(), enemies.end());

    for(auto &enemy : enemies) {
        while(enemy > 0) {
            if(tri == 3) {
                enemy -= 3;
                ans++;
                tri = 1;
            } else {
                enemy -= 1;
                ans++;
                tri++;
            }
        }
    }

    cout << ans;
    return 0;
}
```

In this solution, we first read the number of enemies `n` and their respective healths into a vector called `enemies`. We then sort this vector so that we can process the enemies from the one with the least health to the one with the most health. This way, we minimize the total number of attacks needed to reduce each enemy's health to zero or below. For each enemy, we repeatedly decrease its health based on the current value of `tri`, which alternates between 1 and 3, as described in the problem statement. The total number of attacks performed is stored in the variable `ans`, which is printed at the end of the program.
