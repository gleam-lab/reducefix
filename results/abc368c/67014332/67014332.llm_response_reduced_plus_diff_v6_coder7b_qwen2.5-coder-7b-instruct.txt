#include <bits/stdc++.h>

    long long ans = 0;

                } else {
                    h[i]--;
                }
                
                ans++;
                attacked = true;
                
                if(h[i] <= 0) {
                    h[i] = 0;
                }
            }
            
            if(attacked && h[i] == 0) {
                break;
            }
        }
        
        if(!attacked) {
            break;
        }
        
        tri++;
    }
    
    cout << ans << endl;
    return 0;
}

- **Main Loop**: We use an infinite loop (`while(true)`) to keep attacking enemies until no more attacks can be made.
- **Attacking Process**:
  - We iterate through each enemy.
  - If an enemy's health is greater than 0, we decrement its health based on whether `T` is a multiple of 3.
  - We increment the counter `ans` for each attack.

  - If an enemy's health reaches 0 after being attacked, we set its health to 0.
- **Break Condition**: If no enemies are attacked in a full iteration, it means all enemies' health have reached 0, so we break out of the loop.
