```cpp
#include <iostream>
#include <vector>
#include <cmath>

using namespace std;
using ll = long long;

int main() {
    int N;
    cin >> N;
    vector<ll> H(N);
    for (int i = 0; i < N; i++) {
        cin >> H[i];
    }
    
    ll T = 0;
    for (ll i = 0; i < N; i++) {
        while (H[i] > 0) {
            if (T % 3 == 0) {
                H[i] -= 3;
            } else {
                H[i]--;
            }
            T++;
            if (H[i] <= 0) {
                break;
            }
        }
    }
    cout << T << endl;
    return 0;
}
```

This code iterates over each enemy and, for each enemy, it continuously attacks until the enemy's health is 0 or less. The attack is determined by the current value of `T`. If `T` is a multiple of 3, it attacks with an attack that reduces the enemy's health by 3. Otherwise, it simply reduces the enemy's health by 1. The loop breaks early if the enemy's health becomes 0 or less during an attack. The total number of attacks is accumulated in `T` and printed out after all enemies have been processed.
