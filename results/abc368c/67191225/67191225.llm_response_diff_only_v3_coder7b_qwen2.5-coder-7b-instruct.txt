```cpp
#include<iostream>
#include<vector>
using namespace std;

int main(){
    int n;
    cin>>n;
    vector<int> h(n);
    for(int i=0;i<n;++i){
        cin>>h[i];
    }
    
    long long t = 0;
    while(true){
        bool done = true;
        for(int i=0;i<n;++i){
            if(h[i]>0){
                --h[i];
                ++t;
                done = false;
                if(t%3==0 && h[i]==0){
                    ++h[i];
                }
            }
        }
        if(done){
            break;
        }
    }

    cout<<t<<endl;
    return 0;
}
```

This solution works as follows:
- It reads the number of enemies `n` and their initial healths into a vector `h`.
- It initializes a counter `t` to 0.
- It enters a loop that continues until all enemies' healths are 0 or less.
- In each iteration of the loop, it checks each enemy from the front. If an enemy has health greater than 0, it decreases its health by 1, increments `t`, and marks the flag `done` as false.
- If the enemy's health becomes 0 after being attacked and `t` is a multiple of 3, it restores the enemy's health back to 1.
- After processing all enemies, if `done` remains true, it means no enemy had any health left, so it breaks out of the loop.
- Finally, it prints the total time taken `t`.
