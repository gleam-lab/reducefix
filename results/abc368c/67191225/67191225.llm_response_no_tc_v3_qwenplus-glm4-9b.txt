
```cpp
#include <iostream>
using namespace std;

int main() {
    long long N;
    cin >> N;
    vector<long long> H(N);
    for (int i = 0; i < N; ++i) {
        cin >> H[i];
    }
    
    long long T = 0;
    for (int i = 0; i < N; ++i) {
        // Process the health of the current enemy
        while (H[i] > 0) {
            if (T % 3 == 0) {
                H[i] -= 3;
            } else if (T % 3 == 1) {
                if (H[i] == 1) {
                    break; // If health is 1, just one attack is sufficient
                }
                H[i] -= 1;
            } else { // T % 3 == 2
                if (H[i] == 2 || H[i] == 3) {
                    break; // If health is 2 or 3, just one attack is sufficient
                }
                H[i] -= 1;
            }
            T++;
        }
    }
    
    cout << T << endl;
    return 0;
}
```

### Explanation:

1. The code reads the number of enemies and their respective healths.
2. It initializes `T` to 0, which represents the total number of attacks.
3. It loops through each enemy:
   - For each enemy, it continuously decreases its health until it becomes 0 or less.
   - Depending on the value of `T % 3`, it decides how much health to subtract:
     - If `T % 3 == 0`, subtract 3 health.
     - If `T % 3 == 1`, subtract 1 health unless the health is exactly 1, in which case only one attack is needed.
     - If `T % 3 == 2`, subtract 1 health unless the health is 2 or 3, in which case only one attack is needed.
   - Increment `T` after each attack.
4. Finally, it prints the value of `T`, which is the total number of attacks needed to defeat all enemies.
