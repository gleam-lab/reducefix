
```cpp
#include <iostream>
#include <vector>

using namespace std;

int main() {
    int N;
    cin >> N;
    vector<int> health(N);
    long long T = 0;
    
    for (int i = 0; i < N; ++i) {
        cin >> health[i];
        // Calculate the minimum number of attacks needed for this enemy
        T += (health[i] + 2) / 3; // This is equivalent to ceil(health[i] / 3)
    }
    
    cout << T << endl;
    
    return 0;
}
```

### Explanation:
1. The input is read and stored in a vector `health`.
2. We initialize `T` to 0, which represents the total number of attacks.
3. For each enemy, we calculate the minimum number of attacks needed by using the ceiling function of `health[i] / 3`. Since the ceiling function is not available in C++, we add 2 to `health[i]` before dividing by 3, effectively rounding up the division.
4. We sum up the number of attacks for all enemies and print the result.
