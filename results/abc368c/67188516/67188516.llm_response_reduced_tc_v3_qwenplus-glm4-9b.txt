
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    int N;
    cin >> N;
    vector<long long> H(N);
    for (int i = 0; i < N; ++i) {
        cin >> H[i];
    }
    
    long long T = 0;
    for (int i = 0; i < N; ++i) {
        if (H[i] % 3 == 0) {
            T += H[i] / 3;
        } else {
            T += (H[i] / 3) + 1;
        }
        T += (H[i] % 3) * (i + 1) / 3;
    }
    
    cout << T << endl;
    return 0;
}
```

This code calculates the total number of attacks `T` needed to defeat all enemies. The loop iterates through each enemy, and for each enemy, it calculates the total number of attacks needed to reduce its health to zero or less. If the health of an enemy is divisible by 3, it directly adds the quotient to `T`. Otherwise, it adds the total number of attacks based on the quotient and the remainder. The remainder is also multiplied by `(i + 1) / 3` because when the attack count is a multiple of 3, the decrement is 3, affecting the remaining enemies.
