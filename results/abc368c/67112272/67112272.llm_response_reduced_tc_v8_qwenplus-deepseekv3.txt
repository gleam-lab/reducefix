2. **Optimal Calculation**: For each enemy, we can determine the number of turns required to defeat it by considering the sequence of attacks. The key observation is that every 3 turns, the enemy takes 3 damage, and in the other turns, it takes 1 damage. This means that in every 3-turn cycle, the total damage dealt is 5 (1 + 1 + 3).

3. **Calculating Turns for Each Enemy**: For each enemy with health H_i:

   - The remaining health after these cycles is `remaining = H_i % 5`.
   - The remaining health can be dealt with in at most 2 turns (since 3 turns would form another full cycle).

4. **Determining the Exact Turns for Remaining Health**: 
   - If `remaining == 0`, no additional turns are needed.
   - If `remaining <= 2`, it takes `remaining` turns.
   - If `remaining == 3`, it takes 3 turns (two 1-damage turns and one 3-damage turn).

### Solution Code
```cpp
#include<bits/stdc++.h>
using namespace std;

#define rep(i,n) for (ll i=0;i < (n); i++)
using ll = long long;

int main()
{
    int n; cin >> n;
    ll ans = 0;
    vector<ll> h(n);
    rep(i, n) {
        cin >> h[i];
        ll cycles = h[i] / 5;
        ll remaining = h[i] % 5;
        
        ans += cycles * 3;
        
        if (remaining == 0) {
            continue;
        } else if (remaining <= 2) {
            ans += remaining;
        } else {
            ans += 3;
        }
    }
    cout << ans << endl;
}
```

### Explanation
- **Initialization**: The code reads the number of enemies `n` and their respective health values into a vector `h`.
- **Loop Through Enemies**: For each enemy, the code calculates the number of full 5-damage cycles (`cycles`) and the remaining health (`remaining`).
- **Total Turns Calculation**: 
  - The total turns from full cycles is `cycles * 3` because each cycle consists of 3 turns (1 + 1 + 3 damage).
  - For the remaining health:
    - If it's 0, no additional turns are needed.
    - If it's 1 or 2, it takes exactly that many turns (each dealing 1 damage).
    - If it's 3 or 4, it takes 3 turns (two 1-damage turns and one 3-damage turn).
- **Output**: The total turns `ans` is printed after processing all enemies.
