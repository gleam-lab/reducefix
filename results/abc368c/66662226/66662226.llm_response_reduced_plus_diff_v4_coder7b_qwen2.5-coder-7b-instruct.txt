```cpp
#include <bits/stdc++.h>
using namespace std;
#define rep(i, n) for (int i = 0; i < (n); ++i)

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;
    vector<long long> h(n);
    rep(i, n) cin >> h[i];

    long long t = 0;
    while (true) {
        bool all_zero = true;
        rep(i, n) {
            if (h[i] > 0) {
                all_zero = false;
                if ((t + 1) % 3 == 0) {
                    h[i] -= 3;
                } else {
                    h[i]--;
                }
            }
        }
        if (all_zero) break;
        t++;
    }

    cout << t << '\n';

    return 0;
}
```

### Explanation
1. **Initialization**: The program reads the number of enemies `n` and their respective health values into a vector `h`.

2. **Loop Until All Enemies Are Defeated**:
   - A boolean flag `all_zero` is used to check if all enemies' healths are zero.
   - For each enemy, if its health is greater than zero, it is attacked. 
     - If `(t + 1)` is divisible by 3, the enemy's health decreases by 3.
     - Otherwise, it decreases by 1.
   - After processing all enemies, if no enemy has health greater than zero, the loop breaks.

3. **Output**: Finally, the total number of attacks (`t`) is printed.

This approach ensures that we correctly simulate the attacks and count them until all enemies are defeated. The use of a simple loop and conditionals makes the solution both efficient and easy to understand.
