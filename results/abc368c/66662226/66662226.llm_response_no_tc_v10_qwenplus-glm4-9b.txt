#include <iostream>
using namespace std;
using ll = long long;

int main() {
    int n;
    cin >> n;
    vector<ll> h(n);
    ll ans = 0;

    for (int i = 0; i < n; ++i) {
        cin >> h[i];
        // Calculate the number of full attacks (T) needed to reduce the enemy's health to 0 or less
        ans += (h[i] + 2) / 3; // (h[i] + 2) / 3 is equivalent to ceil(h[i] / 3)
    }

    cout << ans << endl;
    return 0;
}

1. It no longer tries to handle the health decrement in a loop for each enemy, which would be inefficient especially for large values of `H_i`.
2. Instead, it directly calculates the total number of attacks required to bring all enemies' health down to zero or less. This is done by taking the ceiling of `h[i] / 3` for each enemy, which is equivalent to `(h[i] + 2) / 3`.
3. This approach ensures that the algorithm is efficient and works within the given constraints.
