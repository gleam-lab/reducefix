```cpp
#include<bits/stdc++.h>
using namespace std;

int main() {
    int N;
    cin >> N;
    vector<long long> H(N + 1);
    for (int i = 1; i <= N; i++) cin >> H[i];
    long long T = 0;
    while (true) {
        bool done = true;
        for (int i = 1; i <= N; i++) {
            if (H[i] > 0) {
                done = false;
                if (T % 3 == 0) H[i] -= 3;
                else H[i]--;
                T++;
            }
        }
        if (done) break;
    }
    cout << T << endl;
    return 0;
}
```

### Explanation:
1. **Initialization**: The code initializes `N` and reads the health values into a vector `H`. It also initializes `T` to 0.
2. **Main Loop**: The main loop continues until no enemy has any health left (`done` becomes true).
3. **Enemy Attack**: Inside the loop, for each enemy, if its health is greater than 0, it checks whether `T` is a multiple of 3. If so, it reduces the health by 3; otherwise, it reduces the health by 1. It then increments `T`.
4. **Check Completion**: After processing all enemies in one round, if no enemy has health left, `done` is set to true, and the loop breaks.
5. **Output**: Finally, it prints the total number of attacks `T`.

This approach ensures that the game ends correctly when all enemies' health reaches 0 or less, adhering to the constraints and requirements provided.
