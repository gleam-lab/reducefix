```cpp
#include<bits/stdc++.h>
using namespace std;

int main() {
    int N;
    cin >> N;
    vector<long long> H(N + 1); // Changed size to N+1 for convenience
    for(int i = 1; i <= N; ++i)
        cin >> H[i];

    long long T = 0;
    for(int i = 1; i <= N; ++i) {
        while(H[i] > 0) {
            if(T % 3 == 0) 
                H[i] -= 3;
            else 
                --H[i];
            ++T;
        }
    }

    cout << T << endl;
    return 0;
}
```

### Explanation of Changes:
1. **Vector Size Adjustment**: Changed `vector<int> H(N + 9)` to `vector<long long> H(N + 1)` for better handling of large numbers and to avoid unnecessary padding.
2. **Loop Simplification**: The inner loop directly decrements the health based on whether `T` is divisible by 3 or not, simplifying the logic.
3. **Removed Unnecessary Variable Calculation**: Removed the calculation `long long a = H[i] / 5;` as it was not used in the subsequent operations.
