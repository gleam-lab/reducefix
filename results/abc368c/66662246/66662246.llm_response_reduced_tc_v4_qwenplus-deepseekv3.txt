### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;
#define rep(i, n) for (int i = 0; i < (n); ++i)
using ll = long long;

int main() {
    int n;
    cin >> n;
    vector<ll> H(n);
    rep(i, n) cin >> H[i];
    
    ll T = 0;
    rep(i, n) {
        ll h = H[i];
        // The number of full cycles (3 turns) that deal 4 damage (1+1+3)
        ll cycles = h / 4;
        T += cycles * 3;
        h -= cycles * 4;
        
        // Handle the remaining health
        while (h > 0) {
            T++;
            if (T % 3 == 0) {
                h -= 3;
            } else {
                h -= 1;
            }
        }
    }
    cout << T << endl;
    return 0;
}
```

### Explanation
1. **Cycles Calculation**: For each enemy, calculate the number of full cycles (each consisting of 3 turns) that can be applied. Each cycle deals a total of 4 damage (1 + 1 + 3).
2. **Remaining Health**: After applying the full cycles, handle the remaining health by simulating each turn individually, checking if the current turn is a multiple of 3 to apply 3 damage or 1 damage otherwise.
3. **Turn Accumulation**: Accumulate the total turns T as each enemy is processed, ensuring that the damage pattern is correctly followed for each turn.

This approach efficiently calculates the total turns required by leveraging cycles for bulk damage and then fine-tuning with individual turns for the remaining health, ensuring correctness for all test cases.
