```cpp
#include <bits/stdc++.h>
using namespace std;
#define rep(i, n) for(int i = 0; i < (n); ++i)
using ll = long long;

int main(){
    ios::sync_with_stdio(false);
    cin.tie(0);
    
    int n;
    cin >> n;
    vector<ll> h(n);
    rep(i,n) cin >> h[i];

    ll ans = 0;
    while(true){
        bool all_zero = true;
        rep(i,n){
            if(h[i] > 0){
                all_zero = false;
                if((ans + 1) % 3 == 0) h[i] -= 3;
                else h[i] -= 1;
            }
        }
        if(all_zero) break;
        ans++;
    }

    cout << ans << endl;
    return 0;
}
```

This solution uses a loop that continues until all enemies' healths have been reduced to zero or below. Within each iteration of the loop, it checks each enemy's health. If an enemy has positive health, it checks whether the current turn number (starting at 0) plus one is divisible by three. If so, it reduces the enemy's health by three; otherwise, it reduces it by one. After updating all enemies' healths, it increments the turn count. The loop breaks when no enemies have positive health left, ensuring all enemies are defeated before stopping. This approach guarantees the minimum number of turns needed to defeat all enemies under the given conditions.
