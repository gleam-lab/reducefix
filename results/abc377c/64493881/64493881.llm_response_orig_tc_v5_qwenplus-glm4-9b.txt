
The provided problem involves a grid of size \(N \times N\) with certain pieces placed on it. The task is to determine how many empty squares can be occupied by a new piece without being captured by any of the existing pieces.

The existing pieces can capture other pieces in an "L" shape pattern centered around them. Given the constraints, especially \(N\) can be as large as \(10^9\), a brute-force approach is infeasible. Instead, we need to leverage efficient data structures and algorithms to handle the problem within time limits.

Here are some key observations and a refined approach:

1. **Piece Capture Pattern**: A piece placed at \((i, j)\) can capture pieces in an "L" shape centered at the 8 possible positions around it, including itself. This needs to be accounted for when determining the safety of an empty square.

2. **Efficient Data Structures**: Given the large size of \(N\), it is impractical to maintain a full grid to track capture patterns. Instead, we can use a set or map to store the coordinates of pieces and their capture ranges.

3. **Capture Range Calculation**: For each piece, calculate the "L" shaped capture range and store it in a set or map. This way, we can quickly check if an empty square is in the capture range of any existing piece.

4. **Counting Safe Squares**: Iterate over all possible squares and check if they are in the capture range of any existing piece using the precomputed capture ranges. Subtract this count from the total number of squares (\(N^2\)) to get the number of safe squares.

#include <iostream>
#include <vector>
#include <set>
#include <cstring>
using namespace std;

const int MAX_N = 1e9;
const int MAX_M = 2e5;

int main() {
    int N, M;
    cin >> N >> M;

    set<pair<int, int>> pieces;
    for (int i = 0; i < M; ++i) {
        int a, b;
        cin >> a >> b;
        pieces.insert({a, b});

        // Check all 8 possible directions
        for (int dx = -2; dx <= 2; ++dx) {
            for (int dy = -2; dy <= 2; ++dy) {
                if (dx == 0 && dy == 0) continue;
                int nx = a + dx;
                int ny = b + dy;
                if (1 <= nx && nx <= N && 1 <= ny && ny <= N) {
                    pieces.insert({nx, ny});
                }
            }
        }
    }

    int count = 0;
    for (int i = 1; i <= N; ++i) {
        for (int j = 1; j <= N; ++j) {
            bool is_safe = true;
            for (const auto& p : pieces) {
                if (p.first == i || p.first == i + 2 || p.first == i - 2) {
                    if (abs(p.second - j) <= 2) {
                        is_safe = false;
                        break;
                    }
                }
                if (p.second == j || p.second == j + 2 || p.second == j - 2) {
                    if (abs(p.first - i) <= 2) {
                        is_safe = false;
                        break;
                    }
                }
            }
            if (is_safe) count++;
        }
    }

    cout << count << endl;
    return 0;
}
