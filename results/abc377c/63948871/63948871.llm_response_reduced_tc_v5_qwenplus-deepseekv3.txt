2. **Tracking Affected Squares**: For each existing piece, mark all squares that can be captured by it (i.e., the 8 possible knight moves). Use a `set` to avoid duplicates and efficiently track these squares.
3. **Counting Safe Squares**: The total safe squares are calculated as the total squares \( N^2 \) minus the number of squares occupied by existing pieces and the squares that can be captured by any existing piece.
4. **Efficiency**: Given the constraints, ensure that operations are efficient, leveraging the properties of `set` for insertion and uniqueness checks.

### Solution Code
```cpp
#include <iostream>
#include <vector>
#include <set>
#include <utility>

using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    long long N, M;
    cin >> N >> M;

    set<pair<long long, long long>> unsafe;
    set<pair<long long, long long>> pieces;

    for (long long i = 0; i < M; ++i) {
        long long a, b;
        cin >> a >> b;
        pieces.insert({a, b});
        unsafe.insert({a, b});
    }

    for (const auto& p : pieces) {
        long long a = p.first;
        long long b = p.second;

        // Check and add all 8 possible knight moves
        if (a + 2 <= N && b + 1 <= N) {
            unsafe.insert({a + 2, b + 1});
        }
        if (a + 1 <= N && b + 2 <= N) {
            unsafe.insert({a + 1, b + 2});
        }
        if (a - 1 >= 1 && b + 2 <= N) {
            unsafe.insert({a - 1, b + 2});
        }
        if (a - 2 >= 1 && b + 1 <= N) {
            unsafe.insert({a - 2, b + 1});
        }
        if (a - 2 >= 1 && b - 1 >= 1) {
            unsafe.insert({a - 2, b - 1});
        }
        if (a - 1 >= 1 && b - 2 >= 1) {
            unsafe.insert({a - 1, b - 2});
        }
        if (a + 1 <= N && b - 2 >= 1) {
            unsafe.insert({a + 1, b - 2});
        }
        if (a + 2 <= N && b - 1 >= 1) {
            unsafe.insert({a + 2, b - 1});
        }
    }

    long long total_squares = N * N;
    long long unsafe_squares = unsafe.size();
    long long safe_squares = total_squares - unsafe_squares;

    cout << safe_squares << endl;

    return 0;
}
```

### Explanation
1. **Input Handling**: The code reads \( N \) (grid size) and \( M \) (number of existing pieces). It initializes two sets: `unsafe` to store all squares that are either occupied or can be captured by existing pieces, and `pieces` to store the positions of existing pieces.
2. **Tracking Affected Squares**: For each existing piece, the code checks all 8 possible knight moves. For each valid move (within the grid boundaries), it adds the resulting square to the `unsafe` set, ensuring no duplicates.
3. **Counting Safe Squares**: The total number of squares is \( N^2 \). The number of unsafe squares is the size of the `unsafe` set. The safe squares are calculated by subtracting the unsafe squares from the total squares.
4. **Output**: The result, which is the count of safe squares, is printed.

This approach efficiently processes the input and ensures accurate counting of safe squares by leveraging set operations to handle uniqueness and boundary checks. The complexity is \( O(M) \) for processing each piece and \( O(1) \) for each set insertion, making it feasible for large \( N \) and \( M \).
