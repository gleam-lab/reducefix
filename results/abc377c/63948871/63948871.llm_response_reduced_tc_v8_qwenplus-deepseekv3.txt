### Analysis of the Original Code

1. Read the grid size \( N \) and the number of existing pieces \( M \).
2. For each existing piece, mark all positions that it can capture (8 possible positions) as unsafe.
3. Calculate the total number of safe squares as \( N^2 - \text{number of unsafe squares} \).

### Solution Approach

2. **Handling Large \( N \)**: Since \( N \) can be as large as \( 10^9 \), we must avoid storing all positions explicitly. Instead, we use a set to keep track of unique positions that are either existing pieces or their attackable squares.
3. **Correct Calculation**: After collecting all unique unsafe positions, the answer is \( N^2 - \text{size of the unsafe set} \). However, we must ensure that the calculation is done using 64-bit integers to avoid overflow.
