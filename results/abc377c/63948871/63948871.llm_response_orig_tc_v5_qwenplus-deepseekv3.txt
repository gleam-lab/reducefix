### Approaches
1. **Problem Analysis**: The problem requires us to count the number of squares in an N x N grid that are not under attack by any of the M knight pieces placed on the grid. A knight can attack squares that are two squares in one direction and one square in the perpendicular direction (like a chess knight's move).

2. **Key Insight**: For each knight placed at `(a_k, b_k)`, there are at most 8 squares it can attack. However, these squares must be within the grid bounds (1 ≤ i ≤ N, 1 ≤ j ≤ N).

3. **Algorithm Selection**: 
   - Use a set to keep track of all squares that are either occupied by a knight or under attack by any knight. This ensures each square is only counted once, even if it's under attack by multiple knights.
   - After processing all knights, the total number of safe squares is the total grid squares (N²) minus the size of the set.

4. **Optimization**: Given N can be as large as 10^9 and M up to 2×10^5, we must ensure that the solution is efficient. The set operations (insert and find) are O(log M) per operation, leading to an overall complexity of O(M log M), which is acceptable for the given constraints.

### Solution Code
```cpp
#include <iostream>
#include <vector>
#include <set>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    long long N, M;
    cin >> N >> M;

    set<pair<long long, long long>> attacked;
    vector<pair<long long, long long>> knights(M);

    for (long long i = 0; i < M; ++i) {
        long long a, b;
        cin >> a >> b;
        knights[i] = {a, b};
        attacked.insert({a, b});
    }

    for (const auto& knight : knights) {
        long long a = knight.first;
        long long b = knight.second;

        const long long dx[] = {2, 1, -1, -2, -2, -1, 1, 2};
        const long long dy[] = {1, 2, 2, 1, -1, -2, -2, -1};

        for (long long k = 0; k < 8; ++k) {
            long long x = a + dx[k];
            long long y = b + dy[k];
            if (x >= 1 && x <= N && y >= 1 && y <= N) {
                attacked.insert({x, y});
            }
        }
    }

    long long total_squares = N * N;
    long long safe_squares = total_squares - attacked.size();

    cout << safe_squares << '\n';

    return 0;
}
```

### Explanation
1. **Input Handling**: The code reads the grid size `N` and the number of knights `M`. It then reads the positions of the M knights and stores them in a vector.
2. **Tracking Attacked Squares**: For each knight, the code checks all 8 possible squares the knight can attack. If these squares are within the grid bounds, they are added to a set to ensure uniqueness.
3. **Counting Safe Squares**: The total safe squares are calculated by subtracting the number of attacked squares (size of the set) from the total grid squares (N²).
4. **Output**: The result is printed, which is the count of safe squares.

This approach efficiently handles the constraints by using a set to track attacked squares, ensuring optimal performance and correctness. The solution avoids double-counting squares and correctly processes all edge cases, including very large values of N.
