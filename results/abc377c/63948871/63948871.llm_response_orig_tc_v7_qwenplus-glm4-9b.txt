
### Problem Analysis
The problem involves a grid of size \(N \times N\) where certain positions are occupied by pieces. Each piece can potentially capture other pieces based on specific movement rules resembling a knight in chess but generalized for an \(N \times N\) grid. The goal is to determine the number of empty squares where a new piece can be placed without being captured by any existing pieces.

### Solution Strategy
The solution involves the following steps:
1. **Representation of Capturing Scope**: Each piece defines a capturing scope based on its position. This scope must be efficiently represented and checked for overlaps with other pieces' scopes.
2. **Overlap Detection**: For each piece, determine if its capturing scope overlaps with the capturing scope of other pieces. This helps in identifying regions of the grid that are unsafe for the new piece.
3. **Counting Safe Squares**: By subtracting the number of unsafe squares (those that fall within the capturing scope of any piece) from the total number of squares, we can determine the number of safe squares.

#include <iostream>
#include <vector>
#include <set>
#include <algorithm>

using namespace std;

const int dx[8] = {2, 2, 1, 1, -1, -1, -2, -2};
const int dy[8] = {1, -1, 2, -2, 2, -2, 1, -1};

int N, M;
vector<pair<int, int>> pieces;

bool in_bounds(int x, int y) {
    return 1 <= x && x <= N && 1 <= y && y <= N;
}

bool is_safe(int x, int y) {
    for (auto& p : pieces) {
        int px = p.first;
        int py = p.second;
        for (int i = 0; i < 8; ++i) {
            int nx = x + dx[i];
            int ny = y + dy[i];
            if (in_bounds(nx, ny) && (nx == px || ny == py)) {
                return false;
            }
        }
    }
    return true;
}

int main() {
    cin >> N >> M;
    for (int i = 0; i < M; ++i) {
        int x, y;
        cin >> x >> y;
        pieces.push_back({x, y});
    }

    int safe_count = 0;
    for (int x = 1; x <= N; ++x) {
        for (int y = 1; y <= N; ++y) {
            if (is_safe(x, y)) {
                safe_count++;
            }
        }
    }

    cout << safe_count << endl;

    return 0;
}

This code calculates the number of safe squares by iterating over each square in the grid and checking if it is safe using the `is_safe` function, which checks for any capturing piece.
