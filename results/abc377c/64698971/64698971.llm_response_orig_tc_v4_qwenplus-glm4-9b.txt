
The problem is asking us to find out how many empty squares in a grid of size \( N \times N \) we can place a new piece on without being captured by existing pieces. The existing pieces can capture pieces in a certain pattern which involves neighboring squares up to 3 steps away.

**Key Observations:**
1. For \( N \geq 5 \), we can always place the new piece in an empty square not captured by the existing pieces.
2. For \( N = 3 \), there are only 2 squares where a piece can be placed without being captured.
3. For \( N = 4 \), the answer depends on the positions of the existing pieces.

Given these observations, the solution can be outlined as follows:

1. If \( N \geq 5 \), the answer is \( N^2 - M \).
2. If \( N = 3 \), the answer is 2.
3. If \( N = 4 \), we need to check if the existing pieces have any of the configurations that make it impossible to place a new piece without being captured. If not, the answer is \( N^2 - 2M \).

**Implementation:**
```cpp
#include <iostream>
#include <vector>

using namespace std;

int main() {
    int N, M;
    cin >> N >> M;
    vector<pair<int, int>> pieces(M);

    for (int i = 0; i < M; ++i) {
        cin >> pieces[i].first >> pieces[i].second;
    }

    if (N == 3) {
        cout << 2 << endl;
    } else if (N == 4) {
        int ans = N * N - 2 * M;
        // Check if any of the squares (1,1), (1,2), (1,3), (2,1), (2,2), (3,1) is occupied
        if (pieces.end() != find(ALL(pieces), make_pair(1, 1)) || 
            pieces.end() != find(ALL(pieces), make_pair(1, 2)) || 
            pieces.end() != find(ALL(pieces), make_pair(1, 3)) || 
            pieces.end() != find(ALL(pieces), make_pair(2, 1)) || 
            pieces.end() != find(ALL(pieces), make_pair(2, 2)) || 
            pieces.end() != find(ALL(pieces), make_pair(3, 1))) {
            ans = 0;
        }
        cout << ans << endl;
    } else {
        cout << N * N - M << endl;
    }

    return 0;
}
```
