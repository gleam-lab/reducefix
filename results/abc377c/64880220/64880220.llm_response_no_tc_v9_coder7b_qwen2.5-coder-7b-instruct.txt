```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    unordered_set<long long> ex;
    long long n, m;
    cin >> n >> m;
    long long ans = n * n;
    vector<int> dx = {1, 1, -1, -1, 2, 2, -2, -2};
    vector<int> dy = {2, -2, 2, -2, 1, -1, 1, -1};

    for (int i = 0; i < m; i++) {
        long long a, b;
        cin >> a >> b;
        long long key = (a - 1) * n + b;
        if (!ex.count(key)) {
            ex.insert(key);
            ans--;
        }

        for (int d = 0; d < 8; d++) {
            long long x = a + dx[d];
            long long y = b + dy[d];
            long long newKey = (x - 1) * n + y;
            if (x > 0 && x <= n && y > 0 && y <= n && !ex.count(newKey)) {
                ex.insert(newKey);
                ans--;
            }
        }
    }

    cout << ans << endl;
    return 0;
}
```
Explanation:
1. **Data Structure**: Changed `map` to `unordered_set` for better average-case time complexity of insertions and lookups.
2. **Key Calculation**: Used a unique key to represent each cell `(a, b)` as `(a-1)*n + b` to store in the set efficiently.
3. **Initialization**: Initialized `ans` with `n * n` which is the total number of cells.
4. **Loop through Pieces**: For each piece, decrement `ans` immediately after adding it to the set to avoid double counting.
5. **Neighbor Checks**: For each neighbor, check if it exists within bounds and hasn't been added before, then decrement `ans`.
6. **Final Output**: The final value of `ans` represents the count of empty cells that can be safely occupied by a piece without being captured.
