### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    long long N, M;
    cin >> N >> M;
    set<pair<long long, long long>> unsafe;
    int dx[] = {2, 2, -2, -2, 1, 1, -1, -1};
    int dy[] = {1, -1, 1, -1, 2, -2, 2, -2};
    
    for (int i = 0; i < M; ++i) {
        long long a, b;
        cin >> a >> b;
        unsafe.insert({a, b});
        for (int j = 0; j < 8; ++j) {
            long long x = a + dx[j];
            long long y = b + dy[j];
            if (x >= 1 && x <= N && y >= 1 && y <= N) {
                unsafe.insert({x, y});
            }
        }
    }
    
    cout << N * N - unsafe.size() << endl;
    
    return 0;
}
```

### Explanation
1. **Input Reading**: The code reads the grid size `N` and the number of pieces `M`.
2. **Tracking Unsafe Squares**: A set `unsafe` is used to store all squares that are either occupied by existing pieces or can be captured by them. This ensures each square is only counted once, even if it is capturable by multiple pieces.
3. **Boundary Checks**: For each piece, the code checks all 8 possible knight moves. For each move, if the resulting position is within the grid bounds (1 ≤ x, y ≤ N), it is added to the `unsafe` set.
4. **Result Calculation**: The number of safe squares is calculated by subtracting the size of the `unsafe` set from the total number of squares `N*N`, giving the correct count of squares where the new piece can be safely placed.

This approach efficiently handles the constraints and ensures correctness by leveraging set operations to avoid duplicates and boundary checks to prevent invalid positions.
