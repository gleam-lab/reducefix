
The problem requires us to determine the number of empty squares in a grid where we can place our own piece without it being captured by any existing pieces. The grid is `N x N` and contains `M` existing pieces, each with specific capture rules.

### Analysis:
1. **Grid Representation**: The grid is `N x N` and the total number of cells is `N^2`.
2. **Existing Pieces**: There are `M` pieces placed on the grid, each in a specific cell `(a_k, b_k)`.
3. **Capture Rules**: Each piece can capture other pieces based on its position. For instance, a piece at `(i, j)` can capture pieces in certain cells around it as described in the problem statement.
4. **Counting Safe Empty Squares**: To count the number of squares where we can safely place a piece, we need to subtract the number of squares that can be captured by existing pieces from the total number of squares.

### Solution Strategy:
1. **Marking Captured Squares**: We first need to mark the squares that can be captured by each existing piece.
2. **Calculating Safe Squares**: Once we know which squares are captured, we subtract the number of captured squares from the total squares to get the number of safe squares.

### Implementation Details:
1. **Data Structures**: We will use a 2D vector or a data structure to mark captured squares.
2. **Iterate Over Existing Pieces**: For each piece, calculate the squares it can capture and mark them.
3. **Counting and Output**: Count the remaining unmarked squares to output the answer.

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    long long n, m;
    cin >> n >> m;
    vector<vector<int>> grid(n+1, vector<int>(n+1, 0));

    // Mark positions of all existing pieces
    for (long long i = 0; i < m; ++i) {
        long long x, y;
        cin >> x >> y;
        grid[x][y] = -1; // -1 to indicate a piece is placed here
    }

    long long safe_squares = 0;

    // Count all empty squares that are not captured by any piece
    for (long long i = 1; i <= n; ++i) {
        for (long long j = 1; j <= n; ++j) {
            if (grid[i][j] == 0) { // 0 indicates an empty square
                safe_squares++;
                // Check if it can be captured by a piece
                bool is_captured = false;
                vector<pair<long long, long long>> directions = {
                    {2, 1}, {1, 2}, {-1, 2}, {-2, 1}, {-2, -1}, {-1, -2}, {1, -2}, {2, -1}
                };
                for (auto dir : directions) {
                    long long nx = i + dir.first;
                    long long ny = j + dir.second;
                    if (nx >= 1 && nx <= n && ny >= 1 && ny <= n && grid[nx][ny] == -1) {
                        is_captured = true;
                        break;
                    }
                }
                if (!is_captured) {
                    safe_squares++; // Safe square
                }
            }
        }
    }

    cout << safe_squares << endl;
    return 0;
}

### Explanation:
1. **Input and Grid Initialization**: We read the grid size `n` and the number of pieces `m`. Initialize a grid with dimensions `n+1 x n+1` to easily handle the edges.
2. **Marking Pieces**: For each piece, mark its position on the grid with `-1`.
3. **Counting Safe Squares**: Iterate over all grid cells. If a cell is empty (indicated by `0`), check if it can be captured by any piece. If it cannot be captured, it is a safe square.
4. **Output**: Print the number of safe squares.
