To solve this problem, we need to determine how many empty squares can be placed without being captured by any existing pieces. Each piece can capture up to 8 other squares based on its position. Therefore, we need to mark all these positions as unavailable for placing new pieces.

Here's the step-by-step approach:

1. Parse the input to get the dimensions \(N\) and the list of positions of existing pieces.
2. Create a grid to represent the board and initialize it to zero.
3. For each existing piece, mark the squares it can capture as unavailable.
4. Count the number of available squares (those marked as zero in the grid).
5. Subtract the number of existing pieces from the total number of squares to get the result.

Let's implement this in C++:

```cpp
#include <iostream>
#include <vector>
#include <unordered_set>
#include <utility>

using namespace std;

int main() {
    long long n, m;
    cin >> n >> m;

    // Directions a piece can move to capture another piece
    vector<pair<int, int>> directions = {
        {2, 1}, {1, 2}, {-1, 2}, {-2, 1},
        {-2, -1}, {-1, -2}, {1, -2}, {2, -1}
    };

    // Set to keep track of all positions that cannot be placed upon
    unordered_set<long long> capturedPositions;

    // Process each piece
    for (long long i = 0; i < m; ++i) {
        long long a, b;
        cin >> a >> b;
        
        // Convert 1-based index to 0-based index
        --a, --b;
        
        // Mark the positions that can be captured by this piece
        for (const auto& dir : directions) {
            long long x = a + dir.first;
            long long y = b + dir.second;
            
            // Check if the position is within bounds
            if (x >= 0 && x < n && y >= 0 && y < n) {
                capturedPositions.insert(x * n + y);
            }
        }
    }

    // Calculate the number of available positions
    long long totalPositions = n * n;
    long long occupiedPositions = capturedPositions.size();
    
    // The number of empty squares is the total positions minus the occupied positions
    cout << totalPositions - occupiedPositions << endl;

    return 0;
}
```

### Explanation:
1. **Input Parsing**: We read \(N\) and \(M\) from the input. Then, for each of the \(M\) pieces, we read their coordinates and convert them to 0-based indexing.
2. **Direction Vectors**: We define the eight possible directions a piece can move to capture another piece.
3. **Captured Positions**: We use an `unordered_set` to store the positions that are captured by any piece. This ensures that each position is counted only once even if multiple pieces can capture it.
4. **Available Positions Calculation**: The total number of positions is \(N \times N\). We subtract the size of the `capturedPositions` set from this total to get the number of available positions.
5. **Output**: Finally, we print the number of available positions.

This solution handles all the given constraints and examples efficiently. It avoids hard-coded values and special cases, ensuring robustness and correctness.
