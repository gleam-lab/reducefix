### Approaches

1. **Problem Analysis**: The task is to count the number of empty squares on an \( N \times N \) grid where a new piece can be placed such that none of the existing \( M \) pieces can capture it. The existing pieces can capture pieces in specific knight move positions.

2. **Key Insight**: For each existing piece, we need to mark all positions it can capture. These positions are determined by the knight's move pattern (eight possible positions around each piece). 

3. **Algorithm Selection**: 
   - Use a hash set to store all positions that are either occupied by existing pieces or can be captured by them.
   - For each existing piece, generate all eight possible capture positions and add them to the set if they are within the grid boundaries.
   - The total safe positions is the total grid squares \( N^2 \) minus the size of this set.

4. **Optimization**: Since \( N \) can be as large as \( 10^9 \), we need an efficient way to handle large grids without storing all positions explicitly. The approach here efficiently tracks only the necessary positions using a set.

### Solution Code

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    long long N, M;
    cin >> N >> M;
    
    unordered_set<long long> unsafe;
    vector<pair<long long, long long>> pieces(M);
    
    for (int i = 0; i < M; ++i) {
        long long a, b;
        cin >> a >> b;
        pieces[i] = {a, b};
        unsafe.insert((a - 1) * N + b); // Using a unique identifier for each coordinate
    }
    
    vector<int> dx = {2, 2, -2, -2, 1, 1, -1, -1};
    vector<int> dy = {1, -1, 1, -1, 2, -2, 2, -2};
    
    for (auto &piece : pieces) {
        long long a = piece.first;
        long long b = piece.second;
        for (int k = 0; k < 8; ++k) {
            long long x = a + dx[k];
            long long y = b + dy[k];
            if (x >= 1 && x <= N && y >= 1 && y <= N) {
                unsafe.insert((x - 1) * N + y);
            }
        }
    }
    
    long long safe = N * N - unsafe.size();
    cout << safe << endl;
    
    return 0;
}
```

### Explanation

1. **Initialization**: The code reads the grid size \( N \) and the number of pieces \( M \). It initializes an unordered set `unsafe` to store positions that are either occupied or can be captured by existing pieces.

2. **Input Handling**: For each piece, its coordinates are read and stored in a vector. The piece's position is marked as unsafe by adding it to the set.

3. **Marking Unsafe Positions**: For each existing piece, the code checks all eight possible knight move positions around it. If these positions are within the grid boundaries, they are added to the `unsafe` set.

4. **Calculating Safe Positions**: The total number of safe positions is computed by subtracting the size of the `unsafe` set from the total grid squares \( N^2 \).

5. **Output**: The result is printed, which gives the number of safe positions where the new piece can be placed without being captured.

This approach efficiently handles the constraints by leveraging a hash set to track unsafe positions, ensuring optimal performance for large grids.
