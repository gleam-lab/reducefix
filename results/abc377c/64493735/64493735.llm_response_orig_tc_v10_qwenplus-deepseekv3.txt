```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    long long N, M;
    cin >> N >> M;

    set<pair<long long, long long>> occupied;
    set<pair<long long, long long>> attacked;

    for (int i = 0; i < M; ++i) {
        long long a, b;
        cin >> a >> b;
        occupied.insert({a, b});

        // All 8 possible knight moves
        vector<pair<long long, long long>> moves = {
            {a + 2, b + 1}, {a + 1, b + 2},
            {a - 1, b + 2}, {a - 2, b + 1},
            {a - 2, b - 1}, {a - 1, b - 2},
            {a + 1, b - 2}, {a + 2, b - 1}
        };

        for (const auto &move : moves) {
            long long x = move.first;
            long long y = move.second;
            if (x >= 1 && x <= N && y >= 1 && y <= N) {
                attacked.insert({x, y});
            }
        }
    }

    // The safe squares are total squares minus occupied squares minus attacked squares (but not occupied)
    long long safe = N * N - occupied.size();
    for (const auto &pos : attacked) {
        if (occupied.find(pos) == occupied.end()) {
            safe--;
        }
    }

    cout << safe << '\n';

    return 0;
}
```

### Explanation:
1. **Input Handling**: We read the size of the board `N` and the number of pieces `M`.
2. **Tracking Occupied and Attacked Squares**: 
   - `occupied` set keeps track of squares with existing pieces.
   - `attacked` set keeps track of squares that can be captured by existing pieces (knight moves).
3. **Knight Moves Calculation**: For each existing piece, we calculate all 8 possible knight moves and add valid positions (within the board bounds) to the `attacked` set.
4. **Safe Squares Calculation**: The number of safe squares is the total squares minus the occupied squares minus the attacked squares that are not already occupied.
5. **Output**: Print the number of safe squares.

This approach ensures that we efficiently compute the safe squares by leveraging set operations to avoid duplicates and unnecessary checks, resulting in an optimal solution.
