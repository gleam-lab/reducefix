3. **Efficiency**: Use a set to store the positions of existing pieces and another set to store all squares that are under attack. This ensures that we manage large grids efficiently.
4. **Counting Safe Squares**: The total safe squares is calculated as `N*N` minus the number of squares that are either occupied or under attack.

### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    long long N, M;
    cin >> N >> M;

    set<pair<long long, long long>> pieces;
    set<pair<long long, long long>> unsafe;

    for (int i = 0; i < M; ++i) {
        long long a, b;
        cin >> a >> b;
        pieces.insert({a, b});
        unsafe.insert({a, b});

        const int dx[] = {2, 1, -1, -2, -2, -1, 1, 2};
        const int dy[] = {1, 2, 2, 1, -1, -2, -2, -1};

        for (int k = 0; k < 8; ++k) {
            long long x = a + dx[k];
            long long y = b + dy[k];
            if (1 <= x && x <= N && 1 <= y && y <= N) {
                unsafe.insert({x, y});
            }
        }
    }

    long long total = N * N;
    long long unsafe_count = unsafe.size();
    long long safe = total - unsafe_count;

    cout << safe << '\n';

    return 0;
}
```

### Explanation
1. **Input Reading**: The code reads `N` (grid size) and `M` (number of pieces) from the input.
2. **Sets for Tracking**: `pieces` set stores the positions of existing pieces, and `unsafe` set stores all squares that are either occupied or can be attacked by any piece.
3. **Marking Unsafe Squares**: For each piece, the code checks all 8 possible knight moves (as a knight in chess) and marks those squares as unsafe if they are within the grid bounds.
4. **Counting Safe Squares**: The total safe squares is calculated by subtracting the number of unsafe squares from the total squares (`N*N`). This result is printed.

This approach efficiently handles the constraints by leveraging sets to avoid duplicate checks and ensures optimal performance even for large grid sizes.
