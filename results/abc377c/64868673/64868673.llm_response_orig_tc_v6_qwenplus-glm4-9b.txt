
This problem is about placing a piece on a grid in such a way that it cannot be captured by any of the existing pieces. The grid is N x N, and there are M pieces already placed on the grid. Each piece can capture pieces on certain adjacent squares, including diagonals. The task is to calculate the number of empty squares where a new piece can be placed without the risk of being captured.

### Key Observations:
1. Each piece can capture pieces in a diamond shape centered on itself, ranging from 1 to 2 rows away horizontally and vertically.
2. The capture range for each piece is symmetric about the horizontal and vertical axes.
3. The total number of possible positions on the grid is \(N^2\).
4. We need to subtract the number of positions that are in the capture range of existing pieces from the total number of positions.

### Solution Strategy:
1. For each existing piece, calculate the positions it can capture.
2. Use a set or a map to keep track of all captured positions.
3. Subtract the size of this set from \(N^2\) to get the number of safe positions.

### Optimized Solution:
```python
def main():
    import sys
    input = sys.stdin.read
    data = input().split()
    
    N = int(data[0])
    M = int(data[1])
    
    # Initialize the grid size and captured positions
    total_positions = N ** 2
    captured_positions = set()
    
    index = 2
    for _ in range(M):
        a = int(data[index])
        b = int(data[index + 1])
        index += 2
        
        # Calculate the range of rows and columns affected by this piece
        rows = set(range(max(1, a - 2), min(N + 1, a + 3)))
        cols = set(range(max(1, b - 2), min(N + 1, b + 3)))
        
        # Add all positions in the range to the captured set
        for r in rows:
            for c in cols:
                captured_positions.add((r, c))
    
    # Calculate the number of safe positions
    safe_positions = total_positions - len(captured_positions)
    print(safe_positions)

if __name__ == "__main__":
    main()
```

### Explanation:
1. We read the input and parse the grid size \(N\) and the number of pieces \(M\).
2. We initialize the total number of positions and a set to keep track of captured positions.
3. For each piece, we calculate the affected rows and columns, considering the capture range.
4. We add all positions in this range to the captured positions set.
5. Finally, we subtract the number of captured positions from the total to get the number of safe positions and print it.

This solution efficiently calculates the number of safe positions by using sets to avoid counting duplicates and by carefully calculating the range of affected positions for each piece.
