To solve this problem, we need to determine how many empty squares on an \(N \times N\) grid can be safely occupied by a piece without being captured by any existing pieces. The key observation here is that a piece at position \((i, j)\) can capture pieces at positions \((i+2, j+1)\), \((i+1, j+2)\), \((i-1, j+2)\), \((i-2, j+1)\), \((i-2, j-1)\), \((i-1, j-2)\), \((i+1, j-2)\), and \((i+2, j-1)\). We need to count all empty squares that do not lie within the capturing range of any existing piece.

Here's the step-by-step approach:

1. Parse the input to get the dimensions \(N\) and the positions of the existing pieces.
2. Create a set to store the positions of the existing pieces.
3. Iterate over all possible positions on the grid and check if they are within the capturing range of any existing piece.
4. Count the number of empty squares that are not within the capturing range of any existing piece.

Let's implement this in C++:

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 1e9 + 10;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(0);

    long long N, M;
    cin >> N >> M;

    unordered_set<long long> pieces;
    for (long long i = 0; i < M; ++i) {
        long long a, b;
        cin >> a >> b;
        pieces.insert(((a - 1) * N + b));
    }

    long long count = 0;
    for (long long i = 1; i <= N; ++i) {
        for (long long j = 1; j <= N; ++j) {
            bool safe = true;
            for (int di = -2; di <= 2; ++di) {
                for (int dj = -2; dj <= 2; ++dj) {
                    if (abs(di) + abs(dj) == 3 && di != 0 && dj != 0) {
                        long long ni = i + di, nj = j + dj;
                        if (ni >= 1 && ni <= N && nj >= 1 && nj <= N) {
                            if (pieces.count(((ni - 1) * N + nj))) {
                                safe = false;
                                break;
                            }
                        }
                    }
                }
                if (!safe) break;
            }
            if (safe) count++;
        }
    }

    cout << count << endl;

    return 0;
}
```

### Explanation:
1. **Input Parsing**: We read the dimensions \(N\) and the number of pieces \(M\). Then, we read the positions of each piece and store them in an unordered set for quick lookup.
2. **Iterate Over Grid**: We iterate over all possible positions \((i, j)\) on the grid.
3. **Check Capture Range**: For each position, we check if it lies within the capturing range of any existing piece. If it does, we mark it as unsafe.
4. **Count Safe Positions**: We count the number of positions that are not marked as unsafe and output this count.

This solution ensures that we correctly identify and count all empty squares that can be safely occupied without being captured by any existing pieces.
