
### Problem Analysis

The problem involves placing a piece on an empty grid such that it is not captured by any existing pieces. The grid is \(N \times N\) and there are \(M\) pieces already placed on it. Each piece can capture other pieces based on a specific set of moves in the grid. The task is to determine how many empty squares are available for placing a new piece without it being captured by any existing pieces.

### Solution Strategy

The solution involves the following steps:

1. **Identify Captured Squares**: For each existing piece, determine all the squares it can capture based on the given moves. This involves checking the grid boundaries and ensuring the moves do not go out of bounds.

2. **Mark Captured Squares**: Use a data structure to mark all the squares that are either occupied by a piece or are captured by one or more pieces.

3. **Count Empty Squares**: Subtract the number of marked squares from the total number of squares in the grid to get the number of squares available for placing a new piece.

### Implementation

```cpp
#include <iostream>
#include <vector>
using namespace std;

const vector<pair<int, int>> moves = {
    {2, 1}, {1, 2}, {-1, 2}, {-2, 1},
    {-2, -1}, {-1, -2}, {1, -2}, {2, -1}
};

bool isWithinBounds(int x, int y, int n) {
    return x > 0 && x <= n && y > 0 && y <= n;
}

int main() {
    int n, m;
    cin >> n >> m;

    vector<vector<bool>> captured(n + 1, vector<bool>(n + 1, false));

    for (int i = 0; i < m; ++i) {
        int x, y;
        cin >> x >> y;
        int nx, ny;

        // Mark the piece and its capturing area
        captured[x][y] = true;
        for (auto& move : moves) {
            nx = x + move.first;
            ny = y + move.second;
            if (isWithinBounds(nx, ny, n)) {
                captured[nx][ny] = true;
            }
        }
    }

    int count = 0;
    for (int i = 1; i <= n; ++i) {
        for (int j = 1; j <= n; ++j) {
            if (!captured[i][j]) {
                count++;
            }
        }
    }

    cout << count << endl;

    return 0;
}
```

### Explanation

1. **Initialization**: We initialize a `captured` matrix to keep track of all the squares that are either occupied by a piece or captured by any piece.

2. **Input Reading**: For each existing piece, we read its coordinates and update the `captured` matrix accordingly.

3. **Marking Captured Squares**: For each piece, we apply all possible moves to determine which squares it can capture. We ensure that these moves do not go out of bounds using the `isWithinBounds` function.

4. **Counting Empty Squares**: Finally, we iterate over the entire grid and count all the squares that are not marked as captured.

This solution efficiently handles the problem within the constraints, ensuring that we accurately determine the number of available squares for placing a new piece.
