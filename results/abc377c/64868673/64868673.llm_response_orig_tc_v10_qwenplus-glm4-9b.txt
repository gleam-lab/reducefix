The task is to determine how many empty squares can be occupied without being captured by the existing pieces.

The problem can be approached using a Disjoint Set Union (DSU) to efficiently manage the regions controlled by each piece. The steps are:

1. **Initialize DSU**: Initialize a DSU to keep track of which pieces control which regions.
2. **Union Operations**: For each piece, perform union operations to group all reachable pieces, forming a connected component.
3. **Count Regions**: The number of regions without any pieces can be determined by counting the connected components in the DSU that have not been occupied by any piece.
4. **Calculate Answer**: The answer will be the total number of squares (\(N^2\)) minus the number of occupied squares in the regions controlled by the existing pieces.
