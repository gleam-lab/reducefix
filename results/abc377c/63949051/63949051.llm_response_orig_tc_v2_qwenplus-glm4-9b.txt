
The problem requires us to determine the number of empty squares on a grid where a new piece can be placed without being captured by existing pieces according to specific movement rules. The grid is `N x N` and there are `M` pieces placed on it. The movement rules allow a piece to capture other pieces within certain distances in eight directions.

### Analysis
1. **Capture Pattern**: Each piece can capture in an L-shape pattern with a distance of 2 squares in one axis and 1 square in the other axis (both positive and negative directions).
2. **Initial Observations**:
   - If a piece is placed at `(i, j)`, it can potentially block up to 9 squares on the grid, including the squares it is directly adjacent to.
   - We need to subtract these blocked squares from the total number of squares to find the number of available squares.
3. **Efficient Calculation**:
   - Instead of checking each square, we can mark the squares that are blocked by the existing pieces. Then, we use bitwise operations to efficiently calculate the number of available squares.
   - We use a 2D array `A` where `A[i][j]` represents if the square at `(i, j)` is blocked (`1`) or not (`0`).
   - If a square is blocking one or more squares, we mark it as `1`.
   - Finally, we calculate the number of squares that are not marked as blocked.

### Solution
```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int N, M;
    cin >> N >> M;
    vector<vector<int>> A(N + 1, vector<int>(N + 1, 0));

    for (int i = 0; i < M; ++i) {
        int x, y;
        cin >> x >> y;
        // Mark the squares that are blocked by the piece at (x, y)
        A[x][y] = 1;
        if (x + 2 <= N && y + 1 <= N) A[x + 2][y + 1] = 1;
        if (x + 1 <= N && y + 2 <= N) A[x + 1][y + 2] = 1;
        if (x - 1 >= 1 && y + 2 <= N) A[x - 1][y + 2] = 1;
        if (x - 2 >= 1 && y + 1 <= N) A[x - 2][y + 1] = 1;
        if (x - 2 >= 1 && y - 1 >= 1) A[x - 2][y - 1] = 1;
        if (x - 1 >= 1 && y - 2 >= 1) A[x - 1][y - 2] = 1;
        if (x + 1 <= N && y - 2 >= 1) A[x + 1][y - 2] = 1;
        if (x + 2 <= N && y - 1 >= 1) A[x + 2][y - 1] = 1;
    }

    // Calculate the number of available squares
    int count = 0;
    for (int i = 1; i <= N; ++i) {
        for (int j = 1; j <= N; ++j) {
            if (A[i][j] == 0) count++;
        }
    }

    cout << count << endl;
    return 0;
}
```

This solution initializes a grid `A` to represent the blocked squares and then iterates over the existing pieces to mark the blocked squares. It finally counts the number of squares that are not blocked to determine the number of available squares for placing a new piece.
