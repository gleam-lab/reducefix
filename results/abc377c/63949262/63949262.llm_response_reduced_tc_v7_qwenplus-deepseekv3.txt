The problem involves calculating the number of safe squares where a new piece can be placed without being captured by any existing pieces. The incorrect output suggests there might be an issue with the way the total number of squares is being calculated or how the unsafe squares are being counted.

### Approaches
1. **Understanding the Problem**: The problem requires us to count the number of empty squares in an N x N grid where a new piece can be placed such that it is not under attack by any existing pieces. The existing pieces can attack in 8 knight-like moves.
2. **Constraints**: Given the constraints (N can be up to 10^9 and M up to 2 x 10^5), we need an efficient approach that doesn't involve checking every square individually.
3. **Key Insight**: Each existing piece marks its own position and the 8 possible attack positions as unsafe. The total unsafe squares is the union of all these positions minus any overlaps.
4. **Efficient Calculation**: We can use a set to keep track of all unique unsafe squares (including the positions of existing pieces). The total safe squares would then be N^2 minus the size of this set.

### Solution Code
```cpp
#include <iostream>
#include <vector>
#include <set>
#include <algorithm>

using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    long long N, M;
    cin >> N >> M;

    set<pair<long long, long long>> unsafe_squares;

    for (long long i = 0; i < M; ++i) {
        long long a, b;
        cin >> a >> b;
        unsafe_squares.insert({a, b});

        // Check all 8 possible attack positions
        if (a + 2 <= N && b + 1 <= N) unsafe_squares.insert({a + 2, b + 1});
        if (a + 1 <= N && b + 2 <= N) unsafe_squares.insert({a + 1, b + 2});
        if (a - 1 >= 1 && b + 2 <= N) unsafe_squares.insert({a - 1, b + 2});
        if (a - 2 >= 1 && b + 1 <= N) unsafe_squares.insert({a - 2, b + 1});
        if (a - 2 >= 1 && b - 1 >= 1) unsafe_squares.insert({a - 2, b - 1});
        if (a - 1 >= 1 && b - 2 >= 1) unsafe_squares.insert({a - 1, b - 2});
        if (a + 1 <= N && b - 2 >= 1) unsafe_squares.insert({a + 1, b - 2});
        if (a + 2 <= N && b - 1 >= 1) unsafe_squares.insert({a + 2, b - 1});
    }

    long long total_squares = N * N;
    long long unsafe_count = unsafe_squares.size();
    long long safe_squares = total_squares - unsafe_count;

    cout << safe_squares << '\n';

    return 0;
}
```

### Explanation
1. **Input Handling**: The code reads the grid size `N` and the number of existing pieces `M`.
2. **Tracking Unsafe Squares**: A set `unsafe_squares` is used to store all squares that are either occupied by an existing piece or can be attacked by any existing piece. This ensures that each square is only counted once, even if it is marked as unsafe by multiple pieces.
3. **Checking Attack Positions**: For each existing piece at `(a, b)`, the code checks all 8 possible knight moves from `(a, b)` and adds those positions to the `unsafe_squares` set if they are within the grid bounds.
4. **Calculating Safe Squares**: The total number of safe squares is computed by subtracting the number of unsafe squares from the total number of squares in the grid (`N^2`).
5. **Output**: The result is printed, which is the count of safe squares.

This approach efficiently handles the large grid size by leveraging set operations to avoid duplicates and ensures optimal performance even with the upper constraint limits.
