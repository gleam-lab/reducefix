The problem statement provides a grid of `N^2` squares, and each square can either be empty or occupied by a piece. The existing pieces can capture pieces based on specific conditions involving the surrounding squares.

The key challenge is to efficiently determine the number of safe empty squares, especially given the constraints where `N` can be as large as `10^9` and `M` can be up to `200,000`.

2. **Correct Capture Logic**: To correctly determine which squares can be captured by each piece, iterate over the capture patterns defined in the problem statement and mark the affected squares in the `vector<bool>`.

3. **Counting Safe Squares**: After marking the squares that can be captured by existing pieces, iterate over the grid and count the squares that are still marked as empty.

#include <iostream>

using namespace std;

        int x = a, y = b;
        
        // Mark all squares that can be captured by the piece at (a, b)
        grid[x] = false;
        grid[y] = false;
        grid[x + 2] = false;
        grid[y + 1] = false;
        grid[x + 1] = false;
        grid[y + 2] = false;
        grid[x - 1] = false;
        grid[x - 2] = false;
        grid[y - 1] = false;
        grid[x - 2] = false;
        grid[y - 2] = false;
        grid[x + 2] = false;
        grid[y - 1] = false;
    }
    
    // Count the number of empty squares
    long long emptySquares = 0;
    for (int i = 1; i <= N; ++i) {

    }
