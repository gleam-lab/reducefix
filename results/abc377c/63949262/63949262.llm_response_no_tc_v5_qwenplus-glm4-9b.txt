
```cpp
#include <iostream>
#include <vector>
#include <set>
#include <algorithm>

using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    long long N, M;
    cin >> N >> M;

    set<pair<int, int>> occupied;
    for (long long i = 0; i < M; ++i) {
        int a, b;
        cin >> a >> b;
        occupied.insert({a, b});
    }

    long long count = 0;
    for (int i = 1; i <= N; ++i) {
        for (int j = 1; j <= N; ++j) {
            bool can_place = true;
            for (int x = -2; x <= 2; ++x) {
                for (int y = -2; y <= 2; ++y) {
                    if (abs(x) + abs(y) == 2) {
                        int dx = x + i, dy = y + j;
                        if (dx >= 1 && dx <= N && dy >= 1 && dy <= N && occupied.count({dx, dy})) {
                            can_place = false;
                            break;
                        }
                    }
                }
                if (!can_place) break;
            }
            if (can_place) {
                count++;
            }
        }
    }

    cout << count << '\n';

    return 0;
}
```

### Explanation

1. **Input Handling**: We read the grid size `N` and the number of occupied pieces `M`. We then read the positions of the occupied pieces and store them in a set for quick lookup.

2. **Grid Traversal**: We iterate over all possible positions `(i, j)` in the grid, from `(1, 1)` to `(N, N)`.

3. **Capture Check**: For each position `(i, j)`, we check if placing a piece there would allow it to capture any other piece. We do this by checking all positions `(i+x, j+y)` where `x` and `y` are within the range of the capture pattern (i.e., `-2 <= x, y <= 2` and `abs(x) + abs(y) == 2`). If any such position is occupied, the current position `(i, j)` cannot be a safe placement.

4. **Counting Safe Positions**: If a position `(i, j)` is found to be safe (i.e., no occupied piece can capture it), we increment the count of safe positions.

5. **Output**: Finally, we output the count of safe positions.

This solution efficiently checks each position in the grid and determines if it's safe to place a piece there, considering the constraints and the capture pattern.
