To solve this problem efficiently, we need to determine how many empty squares on the board cannot be captured by any of the existing pieces. The key observation here is that each piece can capture up to 8 surrounding squares. Therefore, we can mark all these squares as "unsafe" and then count the number of empty squares that remain.

Here's a step-by-step approach:

1. **Mark Unsafe Squares**: For each piece, mark the 8 adjacent squares (if they exist) as unsafe.
2. **Count Empty Squares**: Count the number of empty squares that are not marked as unsafe.

This approach ensures that we correctly identify all squares that cannot be placed on without being captured.

Here's the corrected C++ code implementing this logic:

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    long long n, m;
    cin >> n >> m;
    
    // Set to store all positions of existing pieces
    set<pair<long long, long long>> pieces;
    
    // Directions for capturing squares
    vector<pair<int, int>> directions = {
        {2, 1}, {2, -1}, {1, 2}, {1, -2},
        {-1, 2}, {-1, -2}, {-2, 1}, {-2, -1}
    };
    
    // Mark all unsafe squares
    for (long long i = 0; i < m; ++i) {
        long long x, y;
        cin >> x >> y;
        --x; --y; // Convert 1-based to 0-based index
        
        pieces.insert({x, y});
        
        for (auto& dir : directions) {
            long long nx = x + dir.first;
            long long ny = y + dir.second;
            
            if (nx >= 0 && nx < n && ny >= 0 && ny < n) {
                pieces.insert({nx, ny});
            }
        }
    }
    
    // Count empty squares
    long long empty_squares = n * n - pieces.size();
    
    cout << empty_squares << endl;
    
    return 0;
}
```

### Explanation:
1. **Reading Input**: We read the dimensions `N` and `M`, followed by the positions of `M` pieces.
2. **Marking Unsafe Squares**:
   - We use a set to store the positions of all pieces.
   - For each piece, we mark its 8 adjacent squares as unsafe by checking their bounds and inserting them into the set.
3. **Counting Empty Squares**: 
   - The total number of empty squares is calculated as `N * N` minus the number of unsafe squares stored in the set.
4. **Output**: Finally, we print the number of empty squares that cannot be captured.

This solution handles all edge cases and ensures efficient computation even for large values of `N` and `M`. It leverages the properties of sets to maintain unique positions and quickly check membership.
