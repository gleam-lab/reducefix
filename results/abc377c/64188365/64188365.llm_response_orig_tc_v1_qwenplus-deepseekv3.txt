```cpp
#include <bits/stdc++.h>
using namespace std;
#define rep(i, n) for(int i = 0; i < (n); ++i)
using ll = long long;

int main() {
    ll n, m;
    cin >> n >> m;
    set<pair<ll, ll>> occupied;
    set<pair<ll, ll>> unsafe;

    rep(i, m) {
        ll a, b;
        cin >> a >> b;
        occupied.insert({a, b});
        // Mark all 8 possible knight moves as unsafe
        for (ll da : {-2, -1, 1, 2}) {
            for (ll db : {-2, -1, 1, 2}) {
                if (abs(da) == abs(db)) continue; // ensures L-shaped move
                ll ni = a + da;
                ll nj = b + db;
                if (ni >= 1 && nj >= 1 && ni <= n && nj <= n) {
                    unsafe.insert({ni, nj});
                }
            }
        }
    }

    // Total unsafe squares are those in 'unsafe' plus the squares with pieces
    // But since occupied squares are included in 'unsafe' only if they can be captured by another piece,
    // we need to ensure that all occupied squares are considered unsafe for placing new pieces.
    // So the total unsafe squares are the union of 'unsafe' and 'occupied'.

    for (auto p : occupied) {
        unsafe.insert(p);
    }

    ll safe_squares = n * n - unsafe.size();
    cout << safe_squares << endl;
}
```

### Explanation
1. **Input Reading**: The code reads the grid size `n` and the number of pieces `m`.
2. **Marking Occupied and Unsafe Squares**: For each piece at (a, b), it marks all 8 possible knight moves (L-shaped) as unsafe if they are within the grid bounds. This ensures all squares that can be captured by any piece are marked.
3. **Union of Unsafe and Occupied Squares**: The total unsafe squares are the union of squares marked as unsafe and squares already occupied by pieces, as new pieces cannot be placed on these squares.
4. **Calculating Safe Squares**: The number of safe squares is the total grid squares minus the number of unsafe squares, which is printed as the result.

This approach efficiently marks all unsafe squares and correctly calculates the number of safe squares using set operations, ensuring no duplicates and accurate counting.
