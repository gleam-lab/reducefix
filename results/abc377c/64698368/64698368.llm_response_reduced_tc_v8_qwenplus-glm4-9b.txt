        }
    }
}

        set_insert(grid, a, b);
    }

    ll count = 0;
    for (auto& cell : grid) {

We use a single-dimensional array to represent the grid since the grid's size can potentially be very large. The cell index is calculated using `i * N + j`, where `i` and `j` are the row and column indices of the cell, respectively. The final count of empty cells is then printed.
