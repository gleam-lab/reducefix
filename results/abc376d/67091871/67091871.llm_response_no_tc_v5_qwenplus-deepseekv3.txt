### Fixed Code

```cpp
#include<bits/stdc++.h>
#define ll long long
using namespace std;

int main() {
    ll n, m;
    cin >> n >> m;
    vector<vector<int>> adj(n + 1);
    for (int i = 0; i < m; i++) {
        int u, v;
        cin >> u >> v;
        adj[u].push_back(v);
    }

    queue<pair<int, int>> q;
    vector<int> dist(n + 1, -1);
    q.push({1, 1});
    dist[1] = 0;
    int min_cycle = -1;

    while (!q.empty()) {
        auto [u, parent] = q.front();
        q.pop();
        for (int v : adj[u]) {
            if (v == 1 && u != parent) {
                if (min_cycle == -1 || dist[u] + 1 < min_cycle) {
                    min_cycle = dist[u] + 1;
                }
            }
            if (dist[v] == -1) {
                dist[v] = dist[u] + 1;
                q.push({v, u});
            }
        }
    }

    cout << min_cycle << endl;
    return 0;
}
```

### Explanation of Fixes
1. **BFS for Shortest Path**: BFS is used to explore the graph level by level, ensuring that the first time node 1 is revisited, the path length is the shortest possible.
2. **Distance Tracking**: The `dist` array keeps track of the distance from node 1 to each node, initialized to -1 (unvisited). When node 1 is revisited from a different parent, a cycle is detected, and the cycle length is updated if it's shorter than the current minimum.
3. **Cycle Detection**: The condition `if (v == 1 && u != parent)` ensures that we only consider cycles where node 1 is revisited from a different node, avoiding trivial cycles like 1 -> 1.
4. **Efficiency**: BFS ensures that each node is visited only once, making the algorithm efficient with a time complexity of O(N + M), where N is the number of nodes and M is the number of edges.
