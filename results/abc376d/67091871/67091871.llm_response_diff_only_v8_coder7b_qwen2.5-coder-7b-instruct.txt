```cpp
#include<bits/stdc++.h>
#define ll long long
using namespace std;

const int MAXN = 200005;

int n, m, mn = INT_MAX, vis[MAXN];
vector<int> adj[MAXN];

void dfs(int node, int depth) {
    vis[node] = 1;
    for (int neighbor : adj[node]) {
        if (!vis[neighbor] || neighbor == 1) {
            if (neighbor == 1) {
                mn = min(mn, depth + 1);
            } else {
                dfs(neighbor, depth + 1);
            }
        }
    }
}

int main() {
    cin >> n >> m;
    for (int i = 1; i <= m; ++i) {
        int u, v;
        cin >> u >> v;
        adj[u].push_back(v);
        adj[v].push_back(u); // Assuming undirected graph
    }

    dfs(1, 0);

    if (mn == INT_MAX) {
        cout << -1;
    } else {
        cout << mn;
    }

    return 0;
}
```

### Explanation of Fixes:
1. **Initialization of `mn`**: Changed from a large hardcoded value to `INT_MAX` for better clarity and to handle larger values of `n`.
2. **Graph Representation**: Added an assumption that the graph is undirected, so both `(u, v)` and `(v, u)` edges are added.
3. **Edge Case Handling**: Ensured that the DFS function correctly handles nodes connected directly to node 1.
4. **Output**: Correctly outputs `-1` if no path is found or the minimum distance otherwise.
