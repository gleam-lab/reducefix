To solve this problem correctly, we need to find the shortest cycle that starts and ends at node 1 in a **directed** graph.

### Issues in Your Code:
- The DFS approach can miss the shortest path because DFS does not guarantee the shortest path in unweighted graphs.
- You are using a `vis` array but not tracking distances or properly handling revisits.
- You're returning early on hitting node 1 again, which might miss shorter paths due to lack of priority for minimal steps.

---

### Correct Approach:
Use **Breadth-First Search (BFS)** from node 1 to find the shortest cycle that returns to node 1. BFS is ideal for finding shortest paths in unweighted graphs.

### Key Points:
- Start BFS from node 1 with distance 0.
- Use a `distance[]` array to track the shortest number of steps to each node.
- When traversing neighbors, if you reach node 1 again and it's not the starting point (i.e., `current_node != 1`), record the total steps taken (`distance[current] + 1`).
- Return the minimum such cycle length found, or -1 if no cycle exists.

Here is the corrected code:

```cpp
#include <bits/stdc++.h>
#define ll long long
using namespace std;

const int MAX = 200005;
vector<int> adj[MAX];
int dist[MAX];

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);

    ll n, m;
    cin >> n >> m;

    for (ll i = 0; i < m; ++i) {
        int u, v;
        cin >> u >> v;
        adj[u].push_back(v);
    }

    // Initialize distance array
    fill(dist, dist + n + 1, -1);

    queue<int> q;
    q.push(1);
    dist[1] = 0;

    int min_cycle = -1;

    while (!q.empty()) {
        int u = q.front();
        q.pop();

        for (int v : adj[u]) {
            if (dist[v] == -1) {
                dist[v] = dist[u] + 1;
                q.push(v);
            } else if (v == 1 && dist[v] != 0) {
                // Found a cycle back to node 1
                int cycle_len = dist[u] + 1;
                if (min_cycle == -1 || cycle_len < min_cycle) {
                    min_cycle = cycle_len;
                }
            }
        }
    }

    cout << min_cycle << endl;

    return 0;
}
```
