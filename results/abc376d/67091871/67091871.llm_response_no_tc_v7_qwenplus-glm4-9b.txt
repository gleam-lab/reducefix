
```cpp
#include<bits/stdc++.h>
#define ll long long
using namespace std;

ll n, m, mn = 200005, vis[200005];
vector<int> b[200005];

void dfs(int x, int cnt) {
    vis[x] = 1;
    ll ans = cnt + 1;
    for (int i = 0; i < b[x].size(); i++) {
        if (!vis[b[x][i]]) {
            dfs(b[x][i], ans);
            ans--;  // Decrement ans because we cannot count the same path tree
        }
    }
    mn = min(mn, ans);
}

int main() {
    cin >> n >> m;
    for (int i = 1; i <= m; i++) {
        int u, v;
        cin >> u >> v;
        b[u].push_back(v);
        b[v].push_back(u);  // Assuming the graph is undirected
    }
    dfs(1, 0);
    if (mn == 200005) {
        cout << -1;
    } else {
        cout << mn;
    }
    return 0;
}
```

### Explanation:
1. **Undirected Graph**: The code now adds both `u` and `v` to the adjacency list to ensure the graph is undirected.
2. **Global Minimum**: The variable `mn` is initialized to a large number and is only updated inside `dfs` if the current node's depth (`ans`) is less than the current `mn`.
3. **Depth Calculation**: The depth is calculated by incrementing `cnt` only when visiting nodes for the first time. Inside the recursive call, `ans` is decremented because we consider the path tree once.
4. **Output**: If `mn` remains unchanged after the DFS, it means no valid path was found, and `-1` is printed. Otherwise, the minimum depth `mn` is printed.
