To fix this code, we need to ensure that the BFS correctly identifies the shortest cycle in the graph. The original code was not correctly handling the detection of cycles, especially in multiple edge cases. The revised approach will use BFS to explore the graph and detect cycles by checking for already visited nodes while traversing.

### Approaches
1. **Graph Representation**: The graph is represented using adjacency lists.
2. **BFS for Cycle Detection**: We perform BFS from each node to explore the graph level by level. During BFS, if we encounter a node that has already been visited and it is not the parent of the current node, we have found a cycle. The length of the cycle can be determined by summing the depths of the current node and the encountered node plus one.
3. **Shortest Cycle Tracking**: Keep track of the shortest cycle found during the BFS traversals.

### Solution Code
```cpp
#include <bits/stdc++.h>

using namespace std;
using ll = long long;
using pii = pair<int, int>;

const int N = 2e5 + 5;
vector<int> g[N];
int n, m;
int parent[N];
int dist[N];
int min_cycle = INT_MAX;

void bfs(int start) {
    queue<int> q;
    q.push(start);
    dist[start] = 0;
    parent[start] = -1;

    while (!q.empty()) {
        int u = q.front();
        q.pop();

        for (int v : g[u]) {
            if (v == parent[u]) continue;
            if (dist[v] == -1) {
                dist[v] = dist[u] + 1;
                parent[v] = u;
                q.push(v);
            } else {
                min_cycle = min(min_cycle, dist[u] + dist[v] + 1);
            }
        }
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);

    cin >> n >> m;
    for (int i = 1; i <= m; i++) {
        int u, v;
        cin >> u >> v;
        g[u].push_back(v);
        g[v].push_back(u);
    }

    for (int i = 1; i <= n; i++) {
        fill(dist, dist + n + 1, -1);
        fill(parent, parent + n + 1, -1);
        bfs(i);
    }

    if (min_cycle == INT_MAX) {
        cout << -1 << '\n';
    } else {
        cout << min_cycle << '\n';
    }

    return 0;
}
```

### Explanation
1. **Initialization**: The graph is read from input, and each node's adjacency list is populated.
2. **BFS Initialization**: For each node in the graph, BFS is initialized with the node as the starting point. The `dist` array keeps track of the distance from the starting node, and the `parent` array keeps track of the parent of each node in the BFS tree.
3. **Cycle Detection**: During BFS, if a node is encountered that has already been visited and is not the parent of the current node, a cycle is detected. The length of the cycle is calculated as the sum of the distances from the starting node to the current node and the encountered node plus one.
4. **Result Output**: The shortest cycle length found is printed. If no cycle is found, -1 is printed.

This approach ensures that we efficiently detect cycles in the graph and correctly identify the shortest cycle length by leveraging BFS traversal.
