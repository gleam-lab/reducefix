### Approaches
1. **BFS for Shortest Path**: We perform a BFS starting from node 0 to compute the shortest distances from node 0 to all other nodes.
2. **Check for Back Edge**: For each node `u` that has an edge to node 0, we calculate the potential cycle length as `dist[u] + 1` and keep track of the minimum such value.
3. **Edge Case Handling**: If no such cycle is found, we output -1.
