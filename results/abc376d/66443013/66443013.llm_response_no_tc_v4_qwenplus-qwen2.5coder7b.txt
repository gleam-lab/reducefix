```cpp
#include<bits/stdc++.h>
using namespace std;
using ll = long long;
map<ll, vector<ll>> graph;
map<ll, bool> seen;
const ll inf = 1LL << 60;
ll ans = inf;

void dfs(ll node, ll steps) {
    if (steps >= ans) return; // Pruning to avoid unnecessary work
    seen[node] = true;
    for (auto neighbor : graph[node]) {
        if (neighbor == 1) {
            ans = min(ans, steps + 1);
            continue;
        }
        if (!seen[neighbor]) {
            dfs(neighbor, steps + 1);
        }
    }
    seen[node] = false; // Backtrack
}

int main() {
    ll n, m;
    cin >> n >> m;
    for (int i = 0; i < m; i++) {
        ll a, b;
        cin >> a >> b;
        graph[a].push_back(b);
        graph[b].push_back(a); // Assuming undirected graph
    }
    dfs(1, 0);
    if (ans != inf) cout << ans << endl;
    else cout << "-1" << endl;
    return 0;
}
```

### Explanation of Fixes
1. **Pruning**: Added a check `if (steps >= ans) return;` in the DFS function to stop exploring paths that cannot lead to a shorter answer, thus improving efficiency.
2. **Backtracking**: Ensured that `seen[node] = false;` is called after the recursive call to allow other paths through this node to be explored.
3. **Graph Representation**: Added `graph[b].push_back(a);` assuming the graph is undirected, which was missing in the original code.
4. **Edge Case Handling**: The code now correctly handles the case where there is no path to node 1 by checking if `ans` remains `inf`.
