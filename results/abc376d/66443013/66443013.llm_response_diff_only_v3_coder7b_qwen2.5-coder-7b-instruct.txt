```cpp
#include<bits/stdc++.h>
using namespace std;
using ll = long long;
map<ll, vector<ll>> graph;
map<ll, bool> seen;
const ll inf = 1LL << 60;
ll ans = inf;

void dfs(ll n, ll hen) {
    seen[n] = true;
    for (auto x : graph[n]) {
        if (x == 1 && hen != 1) {
            ans = min(ans, hen + 1);
            continue;
        }
        if (!seen[x]) dfs(x, hen + 1);
    }
}

int main() {
    ll n, m;
    cin >> n >> m;
    for (int i = 0; i < m; i++) {
        ll a, b;
        cin >> a >> b;
        graph[a].push_back(b);
        graph[b].push_back(a); // Ensure undirected graph
    }
    dfs(1, 0);
    if (ans != inf) cout << ans << endl;
    else cout << "-1" << endl;
}
```

### Explanation of Changes:
1. **Graph Representation**: The original code assumed an undirected graph but only added edges from `a` to `b`. This was corrected to ensure both directions are added (`graph[a].push_back(b)` and `graph[b].push_back(a)`).
2. **Edge Case Handling**: The condition to check if `x == 1` and `hen != 1` is correctly placed inside the loop to avoid unnecessary checks after all nodes have been visited.
3. **Seen Array Initialization**: The initialization of the `seen` array can be removed as it is already done within the `dfs` function.
4. **Output Formatting**: The output format remains unchanged as per the problem statement.
