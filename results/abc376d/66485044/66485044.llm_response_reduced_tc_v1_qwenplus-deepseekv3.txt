To fix this code, we need to correctly identify the shortest cycle in the directed graph that includes node 1. The original code attempts to find such a cycle using a BFS approach, but it fails to correctly track the shortest path back to node 1 from other nodes.

### Approaches
1. **BFS for Shortest Cycle Detection**: The idea is to perform a BFS starting from node 1 and keep track of the steps taken to reach each node. If we encounter node 1 again during the BFS (indicating a cycle), we compare the steps taken to see if it's the shortest cycle found so far.
2. **Handling Direct Edges to Node 1**: If there's a direct edge from node 1 back to itself, the shortest cycle length is 1.
3. **Initialization and Visited Tracking**: Properly initialize the step counts and visited status to ensure accurate tracking of the shortest paths.

### Solution Code
```cpp
#include<bits/stdc++.h>
#define INF 0x3f3f3f3f3f3f3f3f
using namespace std;
typedef long long ll;
const int MAXN=2e5+10;
vector<ll> vec[MAXN];
queue<ll> q;
ll step[MAXN];
bool vis[MAXN];
int main(){
    ll n, m, u, v;
    scanf("%lld %lld", &n, &m);
    for(ll i=1; i<=m; i++){
        scanf("%lld %lld", &u, &v);
        vec[u].push_back(v);
    }
    
    // Check for direct self-loop
    for(ll neighbor : vec[1]){
        if(neighbor == 1){
            printf("1");
            return 0;
        }
    }
    
    memset(step, -1, sizeof(step));
    q.push(1);
    step[1] = 0;
    vis[1] = true;
    
    ll min_cycle = INF;
    
    while(!q.empty()){
        ll top = q.front();
        q.pop();
        for(ll i=0; i<vec[top].size(); i++){
            ll to = vec[top][i];
            if(to == 1 && top != 1){
                if(step[top] + 1 < min_cycle){
                    min_cycle = step[top] + 1;
                }
            }
            if(!vis[to]){
                step[to] = step[top] + 1;
                vis[to] = true;
                q.push(to);
            }
        }
    }
    
    if(min_cycle != INF){
        printf("%lld", min_cycle);
    } else {
        printf("-1");
    }
    return 0;
}
```

### Explanation
1. **Initialization and Input Handling**: The code starts by reading the number of nodes `n` and edges `m`. It then reads each directed edge and stores them in an adjacency list `vec`.
2. **Direct Self-loop Check**: Before proceeding with BFS, the code checks if there's a direct edge from node 1 back to itself. If such an edge exists, it immediately prints `1` as the shortest cycle length.
3. **BFS Setup**: The BFS is initialized with node 1, marking its step count as 0 and setting it as visited.
4. **BFS Execution**: During the BFS, for each node `top`, the code examines its neighbors. If a neighbor `to` is node 1 (and `top` is not node 1, to avoid counting the self-loop again), it updates the minimum cycle length found so far. Other neighbors are processed normally, updating their step counts and marking them as visited.
5. **Result Output**: After BFS completes, the code checks if a valid cycle was found (i.e., `min_cycle` is not `INF`). If so, it prints the shortest cycle length; otherwise, it prints `-1`.

This approach ensures that the shortest cycle involving node 1 is found efficiently, adhering to the problem constraints and requirements.
