### Analysis
The original code uses a BFS approach starting from node 1 and checks for cycles by looking for edges that lead back to node 1. However, the BFS only records the first time it visits a node, which might not account for the shortest path in all cases, especially when multiple paths exist to the same node.

### Solution
We need to modify the BFS to keep track of the shortest path from node 1 to every other node and then check for edges that lead back to node 1. The shortest cycle involving node 1 will be the shortest path from node 1 to any node that has a direct edge back to node 1, plus 1 for the edge back to node 1.

#define INF 0x3f3f3f3f3f3f3f3f
using namespace std;
typedef long long ll;
const int MAXN=2e5+10;
vector<ll> vec[MAXN];
queue<ll> q;
ll step[MAXN];
bool vis[MAXN];
int main(){
    ll n, m, u, v;
    scanf("%lld %lld", &n, &m);
    for(ll i=1; i<=m; i++){
        scanf("%lld %lld", &u, &v);
        vec[u].push_back(v);
    }
    fill(step, step + MAXN, INF);
    q.push(1);
    step[1] = 0;
    vis[1] = true;
    while(!q.empty()){
        ll top = q.front();
        q.pop();
        for(ll i=0; i<vec[top].size(); i++){
            ll to = vec[top][i];
            if(to == 1){
                if(step[top] + 1 < step[1]){
                    step[1] = step[top] + 1;
                }
            }
            if(!vis[to]){
                step[to] = step[top] + 1;
                vis[to] = true;
                q.push(to);
            }
        }
    }
    if(step[1] != 0 && step[1] != INF){
        printf("%lld", step[1]);
    } else if(step[1] == 0){
        printf("0");
    } else {
        printf("-1");
    }
    return 0;
}

2. **BFS Traversal**: We perform a BFS from node 1, updating the shortest path (`step`) to each node as we visit them.
3. **Cycle Detection**: During the BFS, if we encounter an edge leading back to node 1, we update the shortest cycle length if the current path plus 1 is shorter than the previously recorded value.
