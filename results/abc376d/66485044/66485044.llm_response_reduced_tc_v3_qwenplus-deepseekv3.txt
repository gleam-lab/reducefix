### Approaches
1. **BFS Initialization**: Start BFS from node 1 to explore all reachable nodes.
2. **Cycle Detection**: During BFS, if we encounter node 1 again (other than the starting node), it means we've found a cycle. The shortest such cycle length is the answer.
3. **Visited Tracking**: Mark nodes as visited when they are first encountered to avoid reprocessing and ensure the shortest path is found.
4. **Termination**: If node 1 is found during the traversal (indicating a cycle), return the current step count plus one. If no cycle is found after BFS completes, return -1.

```cpp
#include<bits/stdc++.h>

using namespace std;
typedef long long ll;
const int MAXN=2e5+10;
vector<ll> vec[MAXN];
queue<ll> q;
ll step[MAXN];
bool vis[MAXN];
int main(){
    ll n, m, u, v;
    scanf("%lld %lld", &n, &m);
    for(ll i=1; i<=m; i++){
        scanf("%lld %lld", &u, &v);
        vec[u].push_back(v);
    }
    q.push(1);
    vis[1] = true;
    step[1] = 0;
    while(!q.empty()){
        ll top = q.front();
        q.pop();
        for(ll i=0; i<vec[top].size(); i++){
            ll to = vec[top][i];
            if(to == 1 && step[top] + 1 > 0){
                printf("%lld", step[top] + 1);
                return 0;
            }
            if(!vis[to]){
                vis[to] = true;
                step[to] = step[top] + 1;
                q.push(to);
            }
        }
    }
    printf("-1");
    return 0;
}
```

### Explanation
1. **Initialization**: The BFS starts at node 1, marking it as visited with a step count of 0.
2. **Traversal**: For each node dequeued, we check all its neighbors. If a neighbor is node 1 (indicating a cycle back to the start), we immediately return the step count (current node's steps + 1).
3. **Visited Nodes**: Nodes are marked as visited when first encountered to ensure each node is processed only once, guaranteeing the shortest path.
4. **Termination Conditions**: If the queue empties without finding node 1 again, it means no cycle exists, and we return -1.

This approach ensures efficient and correct detection of the shortest cycle involving node 1, adhering to BFS principles for shortest path in unweighted graphs.
