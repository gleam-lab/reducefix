The problem seems to be related to finding the shortest path from node 1 to any other node in a graph using Depth-First Search (DFS). However, DFS is not suitable for finding the shortest path because it does not consider the weights of the edges.

We should use Breadth-First Search (BFS) instead, which is more appropriate for unweighted graphs and can find the shortest path efficiently.

Here's the corrected code:

```cpp
#include <bits/stdc++.h>
#define int long long
using namespace std;

const int N = 2e5 + 5;

bool vis[N];
vector<int> g[N];
int n, m, ans = 1e18;

void bfs() {
    queue<int> q;
    vector<int> dist(n + 1, INT_MAX);
    dist[1] = 0;
    q.push(1);

    while (!q.empty()) {
        int u = q.front();
        q.pop();

        if (vis[u]) continue;
        vis[u] = true;

        for (int v : g[u]) {
            if (dist[u] + 1 < dist[v]) {
                dist[v] = dist[u] + 1;
                q.push(v);
            }
        }
    }

    for (int i = 1; i <= n; ++i) {
        if (dist[i] != INT_MAX) {
            ans = min(ans, dist[i]);
        }
    }
}

signed main() {
    cin >> n >> m;
    for (int i = 1; i <= m; i++) {
        int u, v;
        cin >> u >> v;
        g[u].push_back(v);
        g[v].push_back(u); // Assuming undirected graph
    }
    bfs();
    if (ans != INT_MAX) cout << ans << endl;
    else cout << -1 << endl;
    return 0;
}
```

### Explanation:
1. **Graph Representation**: We use an adjacency list `g` to represent the graph.
2. **Distance Array**: We maintain a distance array `dist` where `dist[i]` stores the shortest distance from node 1 to node `i`.
3. **BFS Initialization**: We initialize the distance of node 1 to 0 and push it into the queue.
4. **BFS Loop**: We process each node in the queue, update the distances of its neighbors, and push them into the queue if they haven't been visited yet.
5. **Result Calculation**: After processing all nodes, we find the minimum distance from node 1 to any other node. If no valid path exists, we print `-1`.

This approach ensures that we find the shortest path correctly and efficiently.
