To solve this problem, we need to find the shortest path from node 1 to any other node in a directed graph. The given problem is essentially asking for the minimum distance from node 1 to all other nodes using BFS (Breadth-First Search). BFS is suitable here because it guarantees finding the shortest path in an unweighted graph.

Here's the corrected code:

```cpp
#include <bits/stdc++.h>
#define int long long
using namespace std;

const int N = 2e5 + 5;

vector<int> g[N];
int dist[N];

void bfs(int start) {
    queue<int> q;
    q.push(start);
    dist[start] = 0;

    while (!q.empty()) {
        int u = q.front();
        q.pop();

        for (int v : g[u]) {
            if (dist[v] == -1) { // If not visited
                dist[v] = dist[u] + 1;
                q.push(v);
            }
        }
    }
}

signed main() {
    cin >> n >> m;
    memset(dist, -1, sizeof(dist));

    for (int i = 1; i <= m; i++) {
        int u, v;
        cin >> u >> v;
        g[u].push_back(v);
    }

    bfs(1);

    int ans = 1e18;
    for (int i = 1; i <= n; i++) {
        if (dist[i] != -1) {
            ans = min(ans, dist[i]);
        }
    }

    if (ans == 1e18) cout << -1 << endl;
    else cout << ans << endl;

    return 0;
}
```

### Explanation of Changes:
1. **Initialization**: We initialize `dist` array with `-1` to indicate that no node has been visited yet.
2. **BFS Function**: We perform BFS starting from node 1. For each node `u`, we check its neighbors `v`. If `v` hasn't been visited (`dist[v] == -1`), we mark it as visited and set its distance to be `dist[u] + 1`.
3. **Result Calculation**: After the BFS completes, we iterate through the `dist` array to find the minimum distance from node 1 to any other node. If no node is reachable (i.e., all distances remain `-1`), we output `-1`.

This approach ensures that we correctly compute the shortest path from node 1 to all other nodes using BFS, which has a time complexity of \(O(n + m)\), making it efficient enough for large graphs.
