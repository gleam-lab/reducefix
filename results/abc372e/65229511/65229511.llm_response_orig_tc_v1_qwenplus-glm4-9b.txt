1. Add an edge between two vertices `u` and `v`.
2. Find the `k`-th largest vertex connected to a vertex `v`.

Union-Find is used to keep track of connected components in the graph. When an edge is added, the components of the vertices `u` and `v` are merged if they are not already connected. When querying for the `k`-th largest vertex connected to `v`, we need to find the component of `v` and then retrieve the `k`-th largest element from the sorted list of vertices in that component.

The error in your code is that after merging two components, you are not updating the `cnt` array correctly. The `cnt` array is used to store the set of vertices in each component. When you merge two components, you should not clear the `cnt` array of the smaller component because there might be vertices in that set which are the `k`-th largest in the merged component.
