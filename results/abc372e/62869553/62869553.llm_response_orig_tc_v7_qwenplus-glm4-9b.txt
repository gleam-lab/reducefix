
The problem involves managing an undirected graph with dynamic connectivity queries. The main task is to efficiently add edges between vertices and then find the k-th largest connected vertex for a given vertex v.

To solve this problem, we can use the Union-Find data structure (also known as Disjoint Set Union, DSU) with path compression and union by rank. This will allow us to efficiently merge sets and find roots of vertices.

### Key Concepts:
1. **Union-Find Data Structure**: This is used for efficiently merging two sets and finding the representative (or root) of a set.
2. **Path Compression**: This technique helps flattening the structure of the tree whenever find operation is performed. It ensures that all nodes directly point to the root node, making future operations faster.
3. **Union by Rank**: This technique helps in keeping the tree balanced by attaching the smaller tree under the root of the larger tree.

### Steps:
1. Initialize the Union-Find structure with `n` nodes.
2. For each query:
   - If it's an edge addition query (`type 1`), merge the sets containing the two vertices.
   - If it's a connectivity query (`type 2`), find the root of the given vertex, gather the connected vertices, sort them, and return the k-th largest.

### Code Explanation:
```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int parent[200005];
int rank[200005];

int find(int v) {
    if (parent[v] != v) {
        parent[v] = find(parent[v]);
    }
    return parent[v];
}

void merge(int u, int v) {
    int root_u = find(u);
    int root_v = find(v);
    if (root_u != root_v) {
        if (rank[root_u] > rank[root_v]) {
            parent[root_v] = root_u;
        } else if (rank[root_u] < rank[root_v]) {
            parent[root_u] = root_v;
        } else {
            parent[root_v] = root_u;
            rank[root_u] += 1;
        }
    }
}

int main() {
    int n, q;
    cin >> n >> q;
    for (int i = 1; i <= n; i++) {
        parent[i] = i;
        rank[i] = 0;
    }
    while (q--) {
        int type, u, v, k;
        cin >> type >> u >> v;
        if (type == 1) {
            merge(u, v);
        } else {
            int root = find(u);
            vector<int> connected;
            for (int i = 1; i <= n; i++) {
                if (find(i) == root) {
                    connected.push_back(i);
                }
            }
            sort(connected.begin(), connected.end(), greater<int>());
            if (k > connected.size()) {
                cout << "-1\n";
            } else {
                cout << connected[k - 1] << "\n";
            }
        }
    }
    return 0;
}
```

This code sets up the Union-Find structure, processes each query, and uses the `find` function to determine the root of each vertex. For type 2 queries, it collects all vertices connected to the root, sorts them, and outputs the k-th largest. Note that this code does not handle the optimization for k < 10, which could be implemented by using a segment tree or binary indexed tree for efficient k-th largest queries.
