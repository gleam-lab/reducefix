
The problem is about processing an undirected graph with `N` vertices and `0` edges. We need to handle two types of queries on this graph:

1. Add an edge between two vertices `u` and `v`.
2. Find the `k`-th largest vertex connected to a given vertex `v`.

To address this problem, we can use the Union-Find data structure along with a union by size heuristic. Additionally, we'll maintain a list of vertices for each component to efficiently find the `k`-th largest vertex for queries of type 2.

### Key Observations:
1. **Union-Find Data Structure**: Helps in efficiently finding the root of the set containing a particular vertex and in merging two sets.
2. **Union by Size**: During a union operation, the smaller tree is always attached to the root of the larger tree, which ensures that the tree remains flat and operations are efficient.
3. **Path Compression**: This technique helps in flattening the structure of the tree whenever `find` is called so that all nodes directly point to the root.

### Steps:
1. **Initialization**:
   - Create a Union-Find structure with `n` elements.
   - Maintain a list `adj` of size `n` to store adjacency lists for each vertex.
   - Maintain a list `size` of size `n` to store the size of each component.
   
2. **Processing Queries**:
   - For an edge addition query (`1 u v`), use the union operation to merge the sets containing `u` and `v`. Update the adjacency lists and component sizes as necessary.
   - For a query to find the `k`-th largest vertex connected to `v` (`2 v k`), retrieve the adjacency list of `v`. If the size of the list is less than `k`, print `-1`. Otherwise, sort the list in descending order and print the `k`-th largest vertex.

### Implementation:

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

const int MAX_N = 200005;

int parent[MAX_N];
int size[MAX_N];
vector<int> adj[MAX_N];

void initialize(int n) {
    fill(parent, parent + n, -1);
    fill(size, size + n, 1);
}

int find(int x) {
    if (parent[x] == -1) return x;
    parent[x] = find(parent[x]);
    return parent[x];
}

void union_sets(int x, int y) {
    x = find(x);
    y = find(y);
    if (x != y) {
        if (size[x] < size[y]) {
            swap(x, y);
        }
        parent[y] = x;
        size[x] += size[y];
    }
}

int main() {
    int n, q;
    cin >> n >> q;
    initialize(n);

    while (q--) {
        int type, u, v, k;
        cin >> type >> u >> v;

        if (type == 1) {
            union_sets(u, v);
        } else if (type == 2) {
            int root = find(v);
            sort(adj[root].rbegin(), adj[root].rend());
            if (k > adj[root].size()) {
                cout << -1 << '\n';
            } else {
                cout << adj[root][k - 1] << '\n';
            }
        }
    }

    return 0;
}
```

This code initializes the Union-Find structure, processes the queries, and maintains adjacency lists for each component. For queries of type 2, it finds the root of the component containing `v`, sorts the vertices in the adjacency list in descending order, and then prints the `k`-th largest vertex. If `k` exceeds the number of vertices in the adjacency list, it prints `-1`.
