To solve this problem, we need to efficiently handle both adding edges and querying the k-th largest vertex connected to a specific vertex. Here's how we can approach it:

1. **Union-Find Data Structure**: We will use a Union-Find (Disjoint Set Union, DSU) data structure to manage the connected components of the graph. This allows us to quickly merge sets and find the representative of each set.

2. **Sorted Sets for Connected Components**: For each component managed by the Union-Find structure, we will maintain a sorted set of vertices that belong to that component. This will allow us to efficiently find the k-th largest vertex.

3. **Handling Queries**:
   - For type 1 queries (adding an edge), we will merge the sets containing the two vertices and update the sorted sets accordingly.
   - For type 2 queries (finding the k-th largest vertex), we will simply access the k-th element from the sorted set of the corresponding component.

Here's the corrected C++ code implementing the above approach:

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 2 * 100005;

struct DSU {
    vector<int> parent, rank;
    vector<multiset<int>> sets;

    DSU(int n) {
        parent.resize(n + 1);
        rank.resize(n + 1, 1);
        sets.resize(n + 1);

        for (int i = 1; i <= n; ++i) {
            parent[i] = i;
            sets[i].insert(i);
        }
    }

    int find(int u) {
        if (parent[u] != u) {
            parent[u] = find(parent[u]);
        }
        return parent[u];
    }

    void unionSets(int u, int v) {
        int pu = find(u), pv = find(v);
        if (pu == pv) return;

        if (rank[pu] > rank[pv]) {
            swap(pu, pv);
        }

        parent[pu] = pv;
        rank[pv] += rank[pu];
        sets[pv].insert(sets[pu].begin(), sets[pu].end());
        sets[pu].clear();
    }

    int kthLargest(int u, int k) {
        int pu = find(u);
        auto &s = sets[pu];
        if (s.size() < k) return -1;
        auto it = s.rbegin();
        advance(it, k - 1);
        return *it;
    }
};

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, q;
    cin >> n >> q;

    DSU dsu(n);

    while (q--) {
        int op, u, v, k;
        cin >> op >> u >> v;
        if (op == 1) {
            dsu.unionSets(u, v);
        } else {
            cout << dsu.kthLargest(u, k) << "\n";
        }
    }

    return 0;
}
```

### Explanation:
1. **DSU Class**: 
   - `find`: Path compression to find the root of the set.
   - `unionSets`: Merges two sets based on their ranks and updates the sorted sets.
   - `kthLargest`: Returns the k-th largest element in the set of the given vertex.

2. **Main Function**:
   - Reads the input and processes each query using the DSU class methods.

This solution ensures efficient handling of both operations (add edge and query k-th largest vertex) within the constraints.
