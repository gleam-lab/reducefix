To solve this problem efficiently, we need to support two operations on a dynamic graph:

1. **Union of two vertices (Type 1 query)**: Add an edge between two vertices.
2. **Query for the k-th largest vertex in the connected component (Type 2 query)**.

### Key Observations:
- This is a classic **Disjoint Set Union (DSU/Union-Find)** problem with additional requirements: maintaining the **top-k largest elements** in each set.
- For efficient access to top-k values, we can maintain a sorted list (in descending order) of vertices in each disjoint set.
- The maximum value of `k` in queries is 10, so we only need to store the **top 10 largest vertices** in each set.

---

### Efficient Strategy:
- Use **path compression** and **union by size/rank** in DSU for fast union-find operations.
- Maintain a vector (`top_nodes`) for each root node that stores the **top 10 largest vertices** in the component.
- When merging two sets:
  - Merge their `top_nodes`, sort them in descending order, and keep only the top 10.

---

### Time Complexity:
- Each union or find operation: **O(α(N))**, where α is the inverse Ackermann function (very slow-growing).
- Merging top 10 elements: Constant time.
- Querying k-th largest: Constant time lookup.

Here's the corrected and optimized implementation:

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 2e5 + 5;

int parent[MAXN];
int sz[MAXN];
vector<int> top_nodes[MAXN];

int find(int x) {
    if (parent[x] != x)
        parent[x] = find(parent[x]);
    return parent[x];
}

void unite(int x, int y) {
    x = find(x);
    y = find(y);
    if (x == y) return;

    if (sz[x] < sz[y]) swap(x, y);

    // Merge smaller set into larger one
    for (int val : top_nodes[y])
        top_nodes[x].push_back(val);

    // Sort and keep top 10 largest nodes
    sort(top_nodes[x].begin(), top_nodes[x].end(), greater<int>());
    if (top_nodes[x].size() > 10)
        top_nodes[x].resize(10);

    parent[y] = x;
    sz[x] += sz[y];
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int N, Q;
    cin >> N >> Q;

    for (int i = 1; i <= N; ++i) {
        parent[i] = i;
        sz[i] = 1;
        top_nodes[i].push_back(i);
    }

    while (Q--) {
        int type;
        cin >> type;
        if (type == 1) {
            int u, v;
            cin >> u >> v;
            unite(u, v);
        } else {
            int v, k;
            cin >> v >> k;
            int root = find(v);
            if (top_nodes[root].size() < k)
                cout << "-1\n";
            else
                cout << top_nodes[root][k - 1] << "\n";
        }
    }

    return 0;
}
```

- **Initialization**: Correctly initialize each node as its own parent and insert itself into `top_nodes`.
- **Merge Logic**: Properly merge the lists and retain only the top 10 largest values.
- **Output**: Ensure correct output for Type 2 queries with bounds checking.
