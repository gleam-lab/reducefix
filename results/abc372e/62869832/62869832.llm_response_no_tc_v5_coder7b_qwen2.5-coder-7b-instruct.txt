```cpp
#include<bits/stdc++.h>
using namespace std;

vector<int> adj[200005];

void addEdge(int u, int v) {
    adj[u].push_back(v);
    adj[v].push_back(u);
}

int findKthLargest(vector<int>& arr, int k) {
    sort(arr.begin(), arr.end());
    if (arr.size() < k) return -1;
    return arr[arr.size() - k];
}

int main() {
    int n, q;
    cin >> n >> q;
    
    for (int i = 0; i < q; ++i) {
        int type, u, v;
        cin >> type >> u >> v;
        
        if (type == 1) {
            addEdge(u, v);
        } else {
            vector<int> connectedVertices = adj[u];
            cout << findKthLargest(connectedVertices, v) << endl;
        }
    }
    
    return 0;
}
```

### Explanation:
1. **Graph Representation**: We use an adjacency list (`adj`) to represent the graph.
2. **Adding Edges**: The function `addEdge` adds an undirected edge between vertices `u` and `v`.
3. **Finding Kth Largest Element**: The function `findKthLargest` sorts the list of connected vertices and returns the k-th largest element. If there aren't enough elements, it returns `-1`.
4. **Main Loop**: In the main loop, we read each query and process it accordingly. For type 1 queries, we add the edge. For type 2 queries, we find the k-th largest connected vertex.

This approach ensures that the solution is efficient and handles all constraints effectively.
