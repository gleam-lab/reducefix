### Analysis
The problem can be reduced to simulating the process of water flooding the island and calculating the area of the remaining land at each year.

1. **Initialization**: Create a 2D array to represent the island, where `1` is land and `0` is water.
2. **Simulation**: For each year, starting from the sea level at 1 and incrementing by 1 each year:
   - **Propagate the flood**: For each land cell (1), if it is not on the boundary (or already flooded), flood it and set its value to 0 (water).
   - **Count remaining land**: After each year's flood, count the number of `1`s in the array.
3. **Output**: Print the remaining land area for each year.

### Approach
- Use a 2D array `island` to represent the island, where `0` is water and `1` is land.
- Use a queue to propagate the flood.
- For each year, process each cell in the queue, flood adjacent cells if they are land, and add them to the queue.
- After processing all cells in the queue for a year, count the number of land cells and store it for the result.
- Repeat the process for each year and print the results.

Let's implement this approach.

```cpp
#include <bits/stdc++.h>
using namespace std;

#define H 1000
#define W 1000

int island[H][W];
int h, w, y;

void flood_fill(int x, int y) {
    queue<pair<int, int>> Q;
    Q.push({x, y});
    island[x][y] = 0;
    while (!Q.empty()) {
        auto [i, j] = Q.front();
        Q.pop();
        for (int dx : {-1, 0, 1}) {
            int nx = i + dx;
            for (int dy : {-1, 0, 1}) {
                int ny = j + dy;
                if (nx >= 0 && nx < h && ny >= 0 && ny < w && island[nx][ny] == 1) {
                    island[nx][ny] = 0;
                    Q.push({nx, ny});
                }
            }
        }
    }
}

int main() {
    cin >> h >> w >> y;
    for (int i = 0; i < h; i++) {
        for (int j = 0; j < w; j++) {
            cin >> island[i][j];
        }
    }
    
    int year;
    for (year = 1; year <= y; year++) {
        int remaining_area = 0;
        for (int i = 0; i < h; i++) {
            for (int j = 0; j < w; j++) {
                if (island[i][j] == 1) {
                    flood_fill(i, j);
                    remaining_area++;
                }
            }
        }
        cout << remaining_area << endl;
    }
    
    return 0;
}
```
