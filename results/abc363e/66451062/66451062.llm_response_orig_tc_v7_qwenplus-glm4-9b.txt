
The problem is essentially about simulating the rising sea levels on an island of given elevation grid. Each year, the sea level rises by 1 unit, and any cell whose elevation is less than or equal to the current sea level will sink. Adjacent cells to a sinking cell will also sink if their elevation is less than or equal to the sea level. The task is to determine the area of the island that remains above sea level after Y years.

The provided code has several issues:

1. The code is using a priority queue to simulate the process, but it does not properly handle the adjacency of sinking cells. Once a cell sinks, it should cause its adjacent cells to also potentially sink if their elevation is less than or equal to the current sea level.
2. The code is designed to handle only the first multiple of 100 (i.e., 100, 200, 300 ... years), but since Y can be as high as 100,000, this is insufficient. 
3. The code is also not handling the case where multiple cells have the same elevation and can potentially sink at the same time correctly.

To solve this problem, we can use a more efficient approach:

### Approach

1. Start by marking all cells that are immediately at or below the current sea level (0 initially) as "sunk".
2. For each subsequent year, increase the sea level by 1 and mark all adjacent cells to any "sunk" cells that are at or below the new sea level as "sunk".
3. Calculate the area of the island that remains above sea level each year by subtracting the area of the "sunk" cells from the total area of the island.

We can use a flood fill algorithm to mark the sinking cells. Flood fill will help us efficiently find and mark all adjacent cells that should sink at the same time.

Here's a conceptual explanation of how to implement the solution:

1. Create a 2D array `sunk` to keep track of the cells that are sunk (initially all are false).
2. For each year:
   - Create a queue to perform a flood fill starting from all sunk cells (from last year's simulation).
   - For each cell in the queue, mark it as sunk (`sunk[x][y] = true`) and add its adjacent cells to the queue if they haven't been sunk yet and are at or below the current sea level.
   - Calculate the area of the island that remains above sea level by subtracting the number of sunk cells from the total area of the island.
3. Repeat the above steps for Y years and print the result for each year.

This approach ensures that we only simulate the sinking process in the grid for Y years, making it feasible even for large values of Y.
