2. **BFS Propagation:** The BFS should continue to propagate until all cells that will be flooded this year are processed. The current code does not handle this correctly, as it processes all cells that will be flooded in one go, which is not accurate.

int H, W, Y;

    int dy[4] = {0, 1, 0, -1};

            int ny = y + dy[k];
            if (nx >= 0 && nx < H && ny >= 0 && ny < W && A[nx][ny] > startElev && F[nx][ny]) {

                Q[A[nx][ny]].push({nx, ny});
