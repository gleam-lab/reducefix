
The task is to compute the area of the island that remains above sea level `i` years from now. The island is represented as a grid of `H x W` cells with specified elevations. Each year, the sea level rises by 1 unit, and a cell will sink if its elevation is less than or equal to the sea level, and it is adjacent to the sea or another recently sunk cell.

### The Approach
The problem can be solved using a breadth-first search (BFS) approach to simulate the process of the sea-level rise and the sinking of cells. Here's a step-by-step breakdown:

1. **Initialization**:
   - Use a 2D array `visited` to keep track of cells that have already been visited or sunk.
   - Use a queue to perform the BFS.

2. **BFS Simulation**:
   - For each cell that is initially adjacent to the sea (or outside the grid), mark it as sunk and add it to the BFS queue.
   - For each cell processed in BFS, check its neighboring cells. If they haven't been visited and their elevation is less than or equal to the current sea level, mark them as sunk and add them to the BFS queue.

3. **Calculating the Area**:
   - After simulating the sinking for `i` years, count the number of cells that have not been marked as sunk.

4. **Output the Result**:
   - For each year from 1 to `Y`, compute and print the area of the island that remains above sea level.

    cin >> h >> w >> y;

            // If the cell is on the boundary or adjacent to an already sunk cell
            if (i == 0 || i == h - 1 || j == 0 || j == w - 1 || (i > 0 && grid[i - 1][j] == -1) || (i < h - 1 && grid[i + 1][j] == -1) ||
                (j > 0 && grid[i][j - 1] == -1) || (j < w - 1 && grid[i][j + 1] == -1)) {
                visited[i][j] = true;
                q.push({i, j});
            }
        }
    }

    // BFS to simulate the sinking
    while (!q.empty()) {
        auto [r, c] = q.front();
        q.pop();

        for (int dr = -1; dr <= 1; dr++) {
            for (int dc = -1; dc <= 1; dc++) {
                int nr = r + dr, nc = c + dc;
                if (nr < 0 || nr >= h || nc < 0 || nc >= w || visited[nr][nc]) continue;
                if (grid[nr][nc] <= grid[r][c]) {
                    visited[nr][nc] = true;
                    q.push({nr, nc});
                }
            }
        }
    }

    // Calculate the area of the island remaining above sea level
    int area = 0;
    for (int i = 0; i < h; i++) {
        for (int j = 0; j < w; j++) {
            if (!visited[i][j]) {
                area++;
            }
        }
    }

    // Output the result for each year
    for (int i = 0; i < y; i++) {
        cout << area << '\n';
    }

    return 0;
}

This code initializes the grid with the given elevations and identifies cells that are initially adjacent to the sea or the grid boundary. It then performs a BFS to simulate the sinking of cells each year. Finally, it counts the number of cells that remain above sea level and outputs this count for each year.
