### Explanation:
1. **Breadth-First Search (BFS)**: The function `bfs` is used to explore the island starting from cells that are above the current sea level. It counts the number of cells that remain above the sea level for each year.
2. **Visited Array**: The `visited` array ensures that each cell is only processed once during BFS.
3. **Result Calculation**: For each year, the total number of cells above the sea level is calculated as the total area minus the count of cells that have sunk.
4. **Efficiency**: The BFS approach ensures that we efficiently count the cells above the sea level without redundant calculations.
