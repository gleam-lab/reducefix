### Approaches
1. **Priority Queue Initialization**: We start by initializing a priority queue (min-heap) with the boundary sections of the island since these are the first to be considered for sinking as the sea level rises.
2. **Dijkstra-like Processing**: For each section, we determine the earliest time (sea level) at which it sinks. This is done by processing sections in order of their elevation, propagating the sinking condition to adjacent sections if they meet the criteria (elevation <= current sea level).
3. **Result Calculation**: We maintain a frequency array that counts how many sections sink at each sea level. Using a prefix sum array, we then compute the remaining area of the island for each year from 1 to Y.

### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;
#define all(x) (x).begin(), (x).end()

int main() {
    ios_base::sync_with_stdio(0);
    cin.tie(0);
    int h, w, y;
    cin >> h >> w >> y;
    vector<vector<int>> height(h, vector<int>(w));
    vector<vector<int>> sink_time(h, vector<int>(w, INT_MAX));
    priority_queue<pair<int, pair<int, int>>, vector<pair<int, pair<int, int>>>, greater<pair<int, pair<int, int>>>> pq;
    int di[4][2] = {{-1, 0}, {1, 0}, {0, -1}, {0, 1}};

    for (int i = 0; i < h; ++i) {
        for (int j = 0; j < w; ++j) {
            cin >> height[i][j];
            if (i == 0 || i == h - 1 || j == 0 || j == w - 1) {
                sink_time[i][j] = height[i][j];
                pq.push({height[i][j], {i, j}});
            }
        }
    }

    while (!pq.empty()) {
        auto current = pq.top();
        pq.pop();
        int current_time = current.first;
        int r = current.second.first;
        int c = current.second.second;
        if (current_time > sink_time[r][c]) continue;
        for (int k = 0; k < 4; ++k) {
            int nr = r + di[k][0];
            int nc = c + di[k][1];
            if (nr < 0 || nr >= h || nc < 0 || nc >= w) continue;
            int new_time = max(current_time, height[nr][nc]);
            if (new_time < sink_time[nr][nc]) {
                sink_time[nr][nc] = new_time;
                pq.push({new_time, {nr, nc}});
            }
        }
    }

    vector<int> freq(y + 2, 0);
    for (int i = 0; i < h; ++i) {
        for (int j = 0; j < w; ++j) {
            if (sink_time[i][j] <= y) {
                freq[sink_time[i][j]]++;
            }
        }
    }

    int total_sunk = 0;
    for (int i = 1; i <= y; ++i) {
        total_sunk += freq[i];
        cout << h * w - total_sunk << '\n';
    }

    return 0;
}
```

### Explanation
1. **Initialization**: The `height` matrix stores the elevation of each section. The `sink_time` matrix is initialized to `INT_MAX` except for the boundary sections, which are initialized to their respective elevations and added to the priority queue.
2. **Priority Queue Processing**: The priority queue processes sections by their elevation (sea level at which they sink). For each section, adjacent sections are checked. If an adjacent section's elevation is less than or equal to the current sea level, its sink time is updated, and it is added to the queue for further processing.
3. **Frequency Array**: The `freq` array counts how many sections sink at each sea level. This is populated by iterating over the `sink_time` matrix.
4. **Result Calculation**: The prefix sum of the `freq` array gives the total number of sections sunk up to each year. The remaining area is then calculated by subtracting this sum from the total area of the island (h * w), and the results are printed for each year from 1 to Y.

This approach ensures that we efficiently determine the sinking times of all sections and accurately compute the remaining island area for each year, adhering to the constraints and requirements.
