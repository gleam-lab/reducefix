To fix this code, we need to address the issue where the output for a single cell island with a high elevation was incorrect. The original code fails to handle cases where the elevation of the single cell is higher than the sea level for all years, resulting in incorrect outputs. The solution involves properly initializing the priority queue with all border cells and correctly processing them as the sea level rises each year.

### Approaches
1. **Initialization**: Start by reading the grid dimensions and the maximum number of years.
2. **Border Processing**: Initialize a priority queue with all border cells of the grid. These cells are the first to be checked for sinking as the sea level rises.
3. **Priority Queue Handling**: Use a min-heap to process cells in order of their elevation. This allows us to efficiently determine which cells sink as the sea level increases each year.
4. **BFS for Adjacent Cells**: For each cell that sinks, check its adjacent cells. If they are within bounds and not already processed, add them to the priority queue if their elevation is less than or equal to the current sea level.
5. **Output**: For each year, output the remaining area of the island.

### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;
#define int long long
#define close ios::sync_with_stdio(false); cin.tie(0); cout.tie(0);
#define MAXN 1000
#define LIMIT 22
#define inf 0x3f3f3f3f3f3f
#define endl '\n'
const double eps = 1e-7;
int dx[] = {0, 0, 1, -1};
int dy[] = {1, -1, 0, 0};

struct Node {
    int x, y, val;
    bool operator<(const Node &a) const {
        return val > a.val; // Min-heap based on val
    }
};

signed main() {
    close;
    int h, w; cin >> h >> w;
    int Y; cin >> Y;
    vector<vector<int>> grid(h, vector<int>(w));
    for (int i = 0; i < h; i++) {
        for (int j = 0; j < w; j++) {
            cin >> grid[i][j];
        }
    }

    vector<vector<bool>> vis(h, vector<bool>(w, false));
    priority_queue<Node> q;
    
    // Push all border cells into the priority queue
    for (int i = 0; i < h; i++) {
        for (int j = 0; j < w; j++) {
            if (i == 0 || i == h - 1 || j == 0 || j == w - 1) {
                q.push({i, j, grid[i][j]});
                vis[i][j] = true;
            }
        }
    }

    int res = h * w;
    vector<int> ans(Y + 1, res); // ans[0] is initial state

    for (int level = 1; level <= Y; level++) {
        while (!q.empty() && q.top().val <= level) {
            Node cur = q.top();
            q.pop();
            res--;
            for (int i = 0; i < 4; i++) {
                int nx = cur.x + dx[i];
                int ny = cur.y + dy[i];
                if (nx >= 0 && nx < h && ny >= 0 && ny < w && !vis[nx][ny]) {
                    vis[nx][ny] = true;
                    q.push({nx, ny, grid[nx][ny]});
                }
            }
        }
        ans[level] = res;
    }

    for (int i = 1; i <= Y; i++) {
        cout << ans[i] << endl;
    }
    
    return 0;
}
```

### Explanation
1. **Initialization**: The grid dimensions and the number of years are read. The grid is then populated with elevation values.
2. **Border Processing**: All cells on the borders of the grid are added to a priority queue (min-heap) based on their elevation. These cells are marked as visited to avoid reprocessing.
3. **Priority Queue Handling**: For each year, the priority queue processes cells in order of increasing elevation. If a cell's elevation is less than or equal to the current sea level, it is removed from the island (res decremented).
4. **BFS for Adjacent Cells**: For each processed cell, adjacent cells are checked. If they are within bounds and not visited, they are added to the priority queue.
5. **Output**: The remaining island area for each year is stored in an array and printed sequentially.

This approach ensures that the solution efficiently processes the grid and correctly updates the remaining island area for each year, handling all edge cases including single-cell islands with high elevations.
