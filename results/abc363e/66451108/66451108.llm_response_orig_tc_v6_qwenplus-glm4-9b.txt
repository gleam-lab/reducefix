
### Problem Analysis

The problem involves simulating the process of erosion of an island as the sea level rises over multiple years. The island is represented as a grid where each cell has a specified elevation. The erosion process affects cells adjacent to the sea or to other cells that have already sunk due to their elevation being less than or equal to the rising sea level. The task is to compute the area of the island that remains above the sea level after a specified number of years.

### Solution Strategy

The solution involves the following steps:

1. **Initialization**: Read the input values and initialize the grid and necessary data structures.
2. **Bone Fracture**: Determine which cells are guaranteed to remain above the sea level without being affected by the erosion, even if the sea level rises. This can be thought of as identifying the "bones" of the island that will not erode.
3. **Sea Level Simulation**: For each year, simulate the rising sea level and check which cells will sink based on their elevation relative to the current sea level.
4. **Area Calculation**: After determining the cells that have sunk, calculate the remaining area of the island that is still above the sea level.
5. **Output**: For each year, output the remaining area of the island.

### Detailed Steps and Code Explanation

1. **Initialization**:
   - Read the dimensions of the grid and the number of years.
   - Read the grid values which represent the elevation of each cell.
   - Initialize a visited matrix to keep track of which cells have been processed or are guaranteed not to sink.

2. **Bone Fracture**:
   - Use a breadth-first search (BFS) to identify cells that are connected (either directly or through other cells) to the boundary of the grid or to cells with higher elevation. These cells are marked as not sinking.
   - This step ensures that we only simulate the sinking of cells that are directly or indirectly connected to the sea or cells that have already sunk.

3. **Sea Level Simulation**:
   - For each year, simulate the rising sea level by checking each cell's elevation against the current sea level.
   - Use a queue to process cells that are guaranteed not to sink and mark their neighbors that are about to sink.

4. **Area Calculation**:
   - After processing all the sinking cells for a year, calculate the area of the island that remains above the sea level by subtracting the number of sinking cells from the total area of the island.

5. **Output**:
   - For each year, output the remaining area of the island.

#include <bits/stdc++.h>
using namespace std;
#define MAXN 1005
#define MOD 1000000007
int h, w, y;
int grid[MAXN][MAXN];
bool visited[MAXN][MAXN];
int dx[4] = {-1, 1, 0, 0};
int dy[4] = {0, 0, -1, 1};

void solve() {
    queue<pair<int, int>> q;
    for(int i = 0; i < w; i++) {
        if(grid[0][i] > 0) q.push({0, i});
        if(grid[h - 1][i] > 0) q.push({h - 1, i});
        visited[0][i] = visited[h - 1][i] = true;
    }
    for(int i = 1; i < h - 1; i++) {
        if(grid[i][0] > 0) q.push({i, 0});
        if(grid[i][w - 1] > 0) q.push({i, w - 1});
        visited[i][0] = visited[i][w - 1] = true;
    }

    while(!q.empty()) {
        auto p = q.front(); q.pop();
        int x = p.first, y = p.second;
        for(int i = 0; i < 4; i++) {
            int nx = x + dx[i], ny = y + dy[i];
            if(nx >= 0 && nx < h && ny >= 0 && ny < w && !visited[nx][ny] && grid[nx][ny] > 0) {
                visited[nx][ny] = true;
                q.push({nx, ny});
            }
        }
    }

    int remaining_area = 0;
    for(int i = 0; i < h; i++) {
        for(int j = 0; j < w; j++) {
            if(!visited[i][j]) remaining_area++;
        }
    }

    for(int year = 1; year <= y; year++) {
        int sinking_area = 0;
        for(int i = 0; i < h; i++) {
            for(int j = 0; j < w; j++) {
                if(visited[i][j] && grid[i][j] <= year) {
                    sinking_area++;
                    visited[i][j] = false;
                    for(int k = 0; k < 4; k++) {
                        int nx = i + dx[k], ny = j + dy[k];
                        if(nx >= 0 && nx < h && ny >= 0 && ny < w && !visited[nx][ny] && grid[nx][ny] > year) {
                            q.push({nx, ny});
                            visited[nx][ny] = true;
                        }
                    }
                }
            }
        }
        cout << remaining_area - sinking_area << '\n';
    }
}

int main() {

    for(int i = 0; i < h; i++) {
        for(int j = 0; j < w; j++) {
            cin >> grid[i][j];
        }
    }
    solve();
    return 0;
}
