### Approaches
1. **Priority Queue Initialization**: Start by adding all the border sections of the island to a priority queue, as these are the first to be affected by the rising sea level.
2. **Process Sections in Elevation Order**: Use a min-heap (priority queue) to process sections in order from lowest to highest elevation. For each year, process all sections with elevation less than or equal to the current sea level.
3. **Propagate Sinking**: For each section that sinks, check its adjacent sections (up, down, left, right). If an adjacent section hasn't been processed yet and its elevation is less than or equal to the current sea level, add it to the priority queue.
4. **Track Remaining Area**: Keep a count of the remaining sections above sea level and update this count as sections sink each year.

```cpp
#include <bits/stdc++.h>
using namespace std;
#define int long long

struct Node {
    int x, y, val;
    bool operator<(const Node &a) const {
        return val > a.val;
    }
};

signed main() {
    ios::sync_with_stdio(false);
    cin.tie(0);
    cout.tie(0);

    int h, w, y;
    cin >> h >> w >> y;
    vector<vector<int>> grid(h, vector<int>(w));
    for (int i = 0; i < h; ++i) {
        for (int j = 0; j < w; ++j) {
            cin >> grid[i][j];
        }
    }

    vector<vector<bool>> vis(h, vector<bool>(w, false));
    priority_queue<Node> q;

    // Initialize the priority queue with border cells
    for (int i = 0; i < h; ++i) {
        if (!vis[i][0]) {
            vis[i][0] = true;
            q.push({i, 0, grid[i][0]});
        }
        if (!vis[i][w - 1]) {
            vis[i][w - 1] = true;
            q.push({i, w - 1, grid[i][w - 1]});
        }
    }
    for (int j = 1; j < w - 1; ++j) {
        if (!vis[0][j]) {
            vis[0][j] = true;
            q.push({0, j, grid[0][j]});
        }
        if (!vis[h - 1][j]) {
            vis[h - 1][j] = true;
            q.push({h - 1, j, grid[h - 1][j]});
        }
    }

    int res = h * w;
    for (int level = 1; level <= y; ++level) {
        while (!q.empty() && q.top().val <= level) {
            Node cur = q.top();
            q.pop();
            res--;
            int dx[] = {0, 0, 1, -1};
            int dy[] = {1, -1, 0, 0};
            for (int k = 0; k < 4; ++k) {
                int nx = cur.x + dx[k];
                int ny = cur.y + dy[k];
                if (nx >= 0 && nx < h && ny >= 0 && ny < w && !vis[nx][ny]) {
                    vis[nx][ny] = true;
                    q.push({nx, ny, grid[nx][ny]});
                }
            }
        }
        cout << res << '\n';
    }

    return 0;
}
```

### Explanation
1. **Initialization**: The border sections of the island are added to the priority queue because they are adjacent to the sea and will be the first to sink as the sea level rises.
2. **Processing Elevations**: The priority queue processes sections in order of their elevation. For each year (level), all sections with elevation less than or equal to the current level are processed, and their adjacent sections are checked.
3. **Sinking Propagation**: Each time a section is processed (sinks), its neighbors are checked. If they meet the sinking criteria (elevation <= current level and not already processed), they are added to the queue to be processed in subsequent iterations.
4. **Result Calculation**: The remaining area is initialized to the total area of the island. As sections sink, the remaining area is decremented, and the result for each year is printed.

This approach efficiently handles the constraints by leveraging a priority queue to process sections in order and ensures that each section is processed only once, making it optimal for large input sizes.
