To solve this problem, we need to simulate the rising sea levels and determine the area of the island that remains above water each year. The key steps involve:

1. Initializing the grid with the given elevations.
2. Using a priority queue to efficiently manage the cells that will sink first based on their elevation.
3. Iterating through each year, simulating the sinking of cells and updating the grid accordingly.
4. Calculating the remaining area above water after each year.

Here's the corrected and optimized C++ code:

```cpp
#include <bits/stdc++.h>
using namespace std;

const int dx[] = {0, 1, 0, -1};
const int dy[] = {1, 0, -1, 0};

struct Cell {
    int x, y, height;
    Cell(int x, int y, int height) : x(x), y(y), height(height) {}
    bool operator>(const Cell& other) const {
        return height > other.height;
    }
};

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int h, w, y;
    cin >> h >> w >> y;

    vector<vector<int>> grid(h, vector<int>(w));
    vector<vector<bool>> visited(h, vector<bool>(w, false));

    priority_queue<Cell, vector<Cell>, greater<Cell>> pq;

    for (int i = 0; i < h; ++i) {
        for (int j = 0; j < w; ++j) {
            cin >> grid[i][j];
            if (i == 0 || i == h - 1 || j == 0 || j == w - 1) {
                pq.push(Cell(i, j, grid[i][j]));
                visited[i][j] = true;
            }
        }
    }

    while (!pq.empty()) {
        auto [x, y, height] = pq.top();
        pq.pop();

        for (int k = 0; k < 4; ++k) {
            int nx = x + dx[k], ny = y + dy[k];
            if (nx >= 0 && nx < h && ny >= 0 && ny < w && !visited[nx][ny] && grid[nx][ny] <= height) {
                pq.push(Cell(nx, ny, grid[nx][ny]));
                visited[nx][ny] = true;
            }
        }
    }

    int total_area = h * w;

    for (int i = 0; i < y; ++i) {
        total_area -= pq.size();
        while (!pq.empty() && pq.top().height <= i + 1) {
            pq.pop();
        }
        cout << total_area << endl;
    }

    return 0;
}
```

### Explanation:
1. **Initialization**: We read the grid dimensions and elevations. We use a priority queue to keep track of the cells that will sink first based on their height. Initially, all boundary cells are added to the priority queue.
2. **Flood Fill Simulation**: We perform a flood fill starting from the boundary cells. For each cell, we mark it as visited and add its neighbors to the priority queue if they haven't been visited yet and have a height less than or equal to the current cell's height.
3. **Area Calculation**: After processing all cells up to the current sea level, we calculate the remaining area above water by subtracting the number of cells still in the priority queue (which represent the submerged cells).
4. **Output**: For each year, we print the remaining area above water.

This approach ensures that we efficiently simulate the rising sea levels and accurately compute the remaining land area.
