### Approaches

2. **Priority Queue Setup**: The priority queue is initialized with the boundary sections of the island, as these are the first to be potentially submerged when the sea level rises.
3. **Processing Each Year**: For each year, we process all sections in the priority queue that have an elevation not greater than the current sea level. Each time a section is submerged, we check its adjacent sections and add them to the queue if they haven't been processed yet.
4. **Output**: After processing each year, the remaining area of the island is printed.

#include <bits/stdc++.h>
using namespace std;
#define ll long long

ll dx[4] = {0, 1, 0, -1}, dy[4] = {1, 0, -1, 0};

int main() {
    ll h, w, y;
    cin >> h >> w >> y;
    ll ans = h * w;
    vector<vector<ll>> room(h, vector<ll>(w));
    vector<vector<bool>> used(h, vector<bool>(w, false));
    priority_queue<tuple<ll, ll, ll>, vector<tuple<ll, ll, ll>>, greater<tuple<ll, ll, ll>>> pq;

    for (ll i = 0; i < h; ++i) {
        for (ll j = 0; j < w; ++j) {
            cin >> room[i][j];
        }
    }

    // Push boundary cells into the priority queue
    for (ll i = 0; i < h; ++i) {
        if (!used[i][0]) {
            pq.push({room[i][0], i, 0});
            used[i][0] = true;
        }
        if (!used[i][w - 1]) {
            pq.push({room[i][w - 1], i, w - 1});
            used[i][w - 1] = true;
        }
    }
    for (ll j = 1; j < w - 1; ++j) {
        if (!used[0][j]) {
            pq.push({room[0][j], 0, j});
            used[0][j] = true;
        }
        if (!used[h - 1][j]) {
            pq.push({room[h - 1][j], h - 1, j});
            used[h - 1][j] = true;
        }
    }

    for (ll current_year = 1; current_year <= y; ++current_year) {
        while (!pq.empty() && get<0>(pq.top()) <= current_year) {
            auto [elev, i, j] = pq.top();
            pq.pop();
            ans--;
            for (ll k = 0; k < 4; ++k) {
                ll ni = i + dx[k], nj = j + dy[k];
                if (ni >= 0 && ni < h && nj >= 0 && nj < w && !used[ni][nj]) {
                    pq.push({room[ni][nj], ni, nj});
                    used[ni][nj] = true;
                }
            }
        }
        cout << ans << endl;
    }
    return 0;
}

### Explanation
1. **Initialization**: The code reads the dimensions of the island (H, W), the number of years (Y), and the elevation data for each section.
2. **Priority Queue Setup**: The boundary sections of the island are added to the priority queue. These sections are prioritized because they are adjacent to the sea and will be the first to submerge as the sea level rises.
3. **Processing Each Year**: For each year, the code checks all sections in the priority queue with elevations not greater than the current sea level. These sections are removed from the island (decreasing the remaining area), and their adjacent sections are evaluated and added to the queue if they haven't been processed yet.
4. **Output**: After processing each year, the remaining area of the island is printed. This continues for each year up to Y, providing the required output for each year's sea level rise.

This approach efficiently tracks the sinking sections using a priority queue, ensuring that the solution is both correct and optimal for large input sizes.
