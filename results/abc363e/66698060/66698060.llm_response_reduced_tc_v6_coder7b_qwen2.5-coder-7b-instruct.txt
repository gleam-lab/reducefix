To solve this problem, we need to simulate the rising sea levels and determine how much of the island remains above water after each year. The key steps involve:

1. **Initialize the Island**: Represent the island as a grid where each cell contains its elevation.
2. **Simulate Sea Level Rise**: For each year, increment the sea level and mark all cells that are submerged (i.e., their elevation is less than or equal to the current sea level).
3. **Breadth-First Search (BFS)**: Use BFS to spread the sinking effect from submerged cells to adjacent cells that are also submerged.
4. **Count Remaining Land**: After processing each year, count the number of cells that remain above water.

Here's the corrected and optimized C++ code:

```cpp
#include <bits/stdc++.h>
using namespace std;

#define ll long long
const int MAXN = 1010;

vector<vector<int>> grid(MAXN, vector<int>(MAXN));
queue<pair<int, int>> q;
bool visited[MAXN][MAXN];

void bfs(int n, int m, int sea_level) {
    while (!q.empty()) {
        auto [x, y] = q.front();
        q.pop();

        for (int dx = -1; dx <= 1; ++dx) {
            for (int dy = -1; dy <= 1; ++dy) {
                if (abs(dx) + abs(dy) == 1) {
                    int nx = x + dx, ny = y + dy;
                    if (nx >= 0 && nx < n && ny >= 0 && ny < m && !visited[nx][ny] && grid[nx][ny] <= sea_level) {
                        visited[nx][ny] = true;
                        q.emplace(nx, ny);
                    }
                }
            }
        }
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, m, y;
    cin >> n >> m >> y;

    for (int i = 0; i < n; ++i) {
        for (int j = 0; j < m; ++j) {
            cin >> grid[i][j];
        }
    }

    for (int year = 1; year <= y; ++year) {
        memset(visited, false, sizeof(visited));
        queue<pair<int, int>> initial_q;

        for (int i = 0; i < n; ++i) {
            for (int j = 0; j < m; ++j) {
                if (grid[i][j] <= year) {
                    visited[i][j] = true;
                    initial_q.emplace(i, j);
                }
            }
        }

        bfs(n, m, year);

        int remaining_area = 0;
        for (int i = 0; i < n; ++i) {
            for (int j = 0; j < m; ++j) {
                if (!visited[i][j]) {
                    ++remaining_area;
                }
            }
        }

        cout << remaining_area << '\n';
    }

    return 0;
}
```

### Explanation:
1. **Initialization**: We read the dimensions of the island (`n`, `m`) and the number of years (`y`). We then populate the grid with the elevations of each cell.
2. **BFS Setup**: For each year, we initialize a queue with all cells that are submerged (elevation ≤ current sea level). We use a boolean array `visited` to keep track of which cells have been processed.
3. **BFS Execution**: We perform BFS starting from all submerged cells. During BFS, we mark all connected submerged cells as visited.
4. **Remaining Area Calculation**: After BFS, we count the number of cells that remain unvisited (above water) and print it.

This approach ensures that we correctly simulate the rising sea levels and efficiently calculate the remaining land using BFS.
