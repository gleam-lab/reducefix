        queue<pair<ll, ll>> q;
        rep(i, h) rep(j, w) {
            if (room[i][j] > level) {
                q.push({i, j});
                seen[i][j] = true;
            }
        }

        ll count = 0;

    }

This solution uses a breadth-first search (BFS) approach to determine the number of cells above a given sea level. The BFS starts from all cells that are initially above the current sea level and expands outward, marking all reachable cells as seen. This ensures that we correctly account for all connected components of cells above the sea level. The time complexity is efficient due to the use of BFS, which processes each cell at most once.
