To solve this problem, we need to simulate the rising sea levels and determine the remaining land area each year. The key steps involve:

1. Parsing the input grid.
2. Simulating the sinking of land as the sea level rises.
3. Efficiently checking and updating the state of the grid.
4. Outputting the remaining land area for each year.

We can use a breadth-first search (BFS) approach to efficiently handle the spreading of water. Here's the corrected and optimized C++ code:

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 1010;
vector<vector<int>> A(MAXN, vector<int>(MAXN));
vector<vector<bool>> visited(MAXN, vector<bool>(MAXN, false));
queue<pair<int, int>> q;

void bfs(int H, int W, int sea_level) {
    while (!q.empty()) {
        auto [i, j] = q.front(); q.pop();
        visited[i][j] = true;
        
        for (int k = 0; k < 4; ++k) {
            int ni = i + dx[k];
            int nj = j + dy[k];
            if (ni >= 0 && ni < H && nj >= 0 && nj < W && !visited[ni][nj] && A[ni][nj] <= sea_level) {
                q.push({ni, nj});
                visited[ni][nj] = true;
            }
        }
    }
}

int main() {
    ios::sync_with_stdio(false); cin.tie(0); cout.tie(0);
    
    int H, W, Y;
    cin >> H >> W >> Y;
    
    for (int i = 0; i < H; ++i) {
        for (int j = 0; j < W; ++j) {
            cin >> A[i][j];
            if (A[i][j] > 0) {
                q.push({i, j});
                visited[i][j] = true;
            }
        }
    }
    
    int ans = H * W;
    for (int y = 1; y <= Y; ++y) {
        bfs(H, W, y);
        for (int i = 0; i < H; ++i) {
            for (int j = 0; j < W; ++j) {
                if (A[i][j] > y && visited[i][j]) {
                    --ans;
                }
            }
        }
        fill(visited.begin(), visited.end(), false);
        queue<pair<int, int>>().swap(q);
        for (int i = 0; i < H; ++i) {
            for (int j = 0; j < W; ++j) {
                if (A[i][j] > y && !visited[i][j]) {
                    q.push({i, j});
                    visited[i][j] = true;
                }
            }
        }
        cout << ans << '\n';
    }
    
    return 0;
}
```

### Explanation:
1. **Initialization**: We read the dimensions \( H \) and \( W \), and the number of years \( Y \). We then parse the grid into matrix `A` and initialize a BFS queue `q` with cells having elevations greater than zero.

2. **BFS Function**: The `bfs` function simulates the sinking of land by processing cells level by level. It marks cells as visited and adds their neighbors to the queue if they have not been visited and their elevation is less than or equal to the current sea level.

3. **Main Loop**: For each year, we perform a BFS to mark all reachable land as visited. We then count the remaining unvisited land cells. After counting, we reset the visited array and reinitialize the queue for the next iteration.

This approach ensures that we correctly simulate the rising sea levels and efficiently update the land areas using BFS. The time complexity is approximately \( O(Y \cdot (H \cdot W)) \), which is efficient given the problem constraints.
