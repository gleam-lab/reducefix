
### Problem Analysis

The problem involves simulating the process of sea level rise on an island represented as a 2D grid. Each cell in the grid has an elevation, and the sea level starts at 0. Each year, the sea level rises by 1 unit, causing any land cell that is at or below the current sea level to sink into the sea. This sinking process is contagious, affecting adjacent cells that are at or below the new sea level. The task is to compute the area of the island that remains above sea level for each year up to Y years.

### Solution Strategy

The solution uses a Breadth-First Search (BFS) approach to simulate the process of sinking. Here's the step-by-step strategy:

1. **Initialization**:
   - Read input values for H (height), W (width), and Y (number of years).
   - Initialize the grid `a` to store elevations.
   - Initialize a grid `f` of the same dimensions to mark land cells that will sink.

2. **Edge Land Sink**:
   - Mark all land cells on the edges of the grid (first and last rows and columns) as sinking immediately since they are adjacent to the sea.

3. **BFS setup**:
   - Create a list of queues `q` to manage the BFS process for different potential sea levels. Each queue `q[i]` will store cells whose elevation is `i` or more.
   - Populate the initial queue `q[0]` with the positions of all edge cells marked as sinking.

4. **Simulation**:
   - For each year up to Y:
     - Process all cells in the current queue `q[i]` (sorted by elevation). For each cell, mark it as sinking (in `f`).
     - For each cell, check its neighbors. If a neighbor is at or above the current sea level and has not been marked as sinking, add it to the queue `q[i+1]`.
     - After processing all cells in `q[i]`, calculate the remaining area of the island by subtracting the number of sinking cells from the total area.

5. **Output**:
   - For each year, output the remaining area of the island.

#include <bits/stdc++.h>
using namespace std;

int main() {
    int h, w, y;
    cin >> h >> w >> y;
    vector<vector<int>> a(h, vector<int>(w));
    vector<vector<bool>> f(h, vector<bool>(w, true));
    cin >> &a[0][0];

    // Mark border cells as sinking
    for (int i = 0; i < h; i++) {
        f[i][0] = f[i][w - 1] = true;
    }
    for (int j = 0; j < w; j++) {
        f[0][j] = f[h - 1][j] = true;
    }

    // Prepare BFS
    vector<queue<pair<int, int>>> q(h * w + 1);
    for (int i = 0; i < h; i++) {
        for (int j = 0; j < w; j++) {
            if (i == 0 || i == h - 1 || j == 0 || j == w - 1) {
                q[a[i][j]].push({i, j});
                f[i][j] = false;
            }
        }
    }

    // Process each year
    for (int y = 1; y <= y; y++) {
        int remaining_area = h * w;
        for (auto& p : q[y]) {
            int i = p.first, j = p.second;
            remaining_area -= f[i][j]; // reduce the remaining area by the number of sinking cells

                if (ni >= 0 && ni < h && nj >= 0 && nj < w) {
                    if (f[ni][nj] && a[ni][nj] >= y) { // only sink if the cell has not sunk yet and is above current sea level
                        f[ni][nj] = false;
                        q[a[ni][nj]].push({ni, nj});
                    }
                }
            }
        }
        cout << remaining_area << endl;
    }

    return 0;
}
