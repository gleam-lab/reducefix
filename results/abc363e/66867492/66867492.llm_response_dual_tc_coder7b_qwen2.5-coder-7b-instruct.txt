To solve this problem correctly, we need to simulate the rising sea levels step-by-step and keep track of the areas that remain above water. Here's how you can do it:

1. **Initialize**: Read the grid dimensions \(H\) and \(W\), the number of years \(Y\), and the initial heights of the grid cells.
2. **Priority Queue**: Use a min-priority queue to always process the lowest height first. This helps in simulating the rising sea levels efficiently.
3. **Tracking Used Cells**: Maintain a `used` array to keep track of which cells have already been processed.
4. **Simulation Loop**: For each year up to \(Y\):
   - Process all cells with heights less than or equal to the current year.
   - Decrease the count of remaining land area accordingly.
   - Add neighboring cells (up, down, left, right) to the priority queue if they haven't been used yet and their height is greater than the current year.
5. **Output**: Print the remaining land area after each year.

Here is the corrected and optimized code:

```cpp
#include <bits/stdc++.h>
using namespace std;

const int INF = 1e9 + 10;
const int MAXN = 1005;

int main() {
    int H, W, Y;
    cin >> H >> W >> Y;
    vector<vector<int>> A(H, vector<int>(W));
    cin >> A;

    priority_queue<pair<int, pair<int, int>>, vector<pair<int, pair<int, int>>>, greater<>> pq;
    vector<vector<bool>> used(H, vector<bool>(W, false));

    // Initialize the priority queue with boundary cells
    for (int i = 0; i < H; ++i) {
        pq.emplace(A[i][0], {i, 0});
        pq.emplace(A[i][W - 1], {i, W - 1});
        used[i][0] = true;
        used[i][W - 1] = true;
    }

    for (int i = 1; i < W - 1; ++i) {
        pq.emplace(A[0][i], {0, i});
        pq.emplace(A[H - 1][i], {H - 1, i});
        used[0][i] = true;
        used[H - 1][i] = true;
    }

    int ans = H * W;
    vector<int> dx = {1, 0, -1, 0}, dy = {0, 1, 0, -1};

    for (int y = 1; y <= Y; ++y) {
        while (!pq.empty() && pq.top().first <= y) {
            auto [height, pos] = pq.top();
            pq.pop();
            --ans;

            for (int d = 0; d < 4; ++d) {
                int ny = pos.first + dy[d], nx = pos.second + dx[d];
                if (ny < 0 || nx < 0 || ny >= H || nx >= W) continue;
                if (!used[ny][nx] && A[ny][nx] > y) {
                    used[ny][nx] = true;
                    pq.emplace(A[ny][nx], {ny, nx});
                }
            }
        }
        cout << ans << endl;
    }

    return 0;
}
```

### Explanation:
- **Initialization**: We initialize the priority queue with the boundary cells and mark them as used.
- **Processing**: For each year, we process all cells whose height is less than or equal to the current year. We decrease the count of remaining land and add unvisited neighboring cells to the priority queue.
- **Output**: After processing each year, we output the remaining land area.

This approach ensures that we correctly simulate the rising sea levels and handle the constraints efficiently.
