To fix this code, we need to ensure that when sections sink into the sea, all adjacent sections that meet the criteria are processed correctly, and the count of remaining sections is accurately updated. The initial code had some issues with how the priority queue was being used, particularly in how sections were being marked as used and how adjacent sections were being checked.

### Approaches
1. **Priority Queue Initialization**: We start by pushing all the border sections into a priority queue (min-heap) because these sections are adjacent to the sea and will be the first to sink as the sea level rises.
2. **Processing Sinking Sections**: For each year, we process all sections in the priority queue whose elevation is less than or equal to the current sea level. For each such section, we decrement the remaining area count and check its adjacent sections. If an adjacent section hasn't been processed yet and its elevation is less than or equal to the current sea level, we add it to the priority queue.
3. **Tracking Processed Sections**: We use a 2D array to keep track of which sections have been processed to avoid revisiting and reprocessing the same sections, which could lead to incorrect counts.

### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int H, W, Y;
    cin >> H >> W >> Y;
    vector<vector<int>> A(H, vector<int>(W));
    for (int i = 0; i < H; ++i) {
        for (int j = 0; j < W; ++j) {
            cin >> A[i][j];
        }
    }
    
    vector<vector<bool>> used(H, vector<bool>(W, false));
    priority_queue<tuple<int, int, int>, vector<tuple<int, int, int>>, greater<tuple<int, int, int>>> pq;
    
    // Push border cells into the priority queue
    for (int i = 0; i < H; ++i) {
        if (!used[i][0]) {
            pq.emplace(A[i][0], i, 0);
            used[i][0] = true;
        }
        if (!used[i][W-1]) {
            pq.emplace(A[i][W-1], i, W-1);
            used[i][W-1] = true;
        }
    }
    for (int j = 1; j < W-1; ++j) {
        if (!used[0][j]) {
            pq.emplace(A[0][j], 0, j);
            used[0][j] = true;
        }
        if (!used[H-1][j]) {
            pq.emplace(A[H-1][j], H-1, j);
            used[H-1][j] = true;
        }
    }
    
    int ans = H * W;
    vector<int> res(Y + 1, ans); // res[0] is initial state
    
    const int dx[] = {1, 0, -1, 0};
    const int dy[] = {0, 1, 0, -1};
    
    for (int year = 1; year <= Y; ++year) {
        while (!pq.empty() && get<0>(pq.top()) <= year) {
            auto [elev, i, j] = pq.top();
            pq.pop();
            --ans;
            for (int d = 0; d < 4; ++d) {
                int ni = i + dy[d];
                int nj = j + dx[d];
                if (ni >= 0 && ni < H && nj >= 0 && nj < W && !used[ni][nj]) {
                    used[ni][nj] = true;
                    pq.emplace(A[ni][nj], ni, nj);
                }
            }
        }
        res[year] = ans;
    }
    
    for (int i = 1; i <= Y; ++i) {
        cout << res[i] << '\n';
    }
    
    return 0;
}
```

### Explanation
1. **Initialization**: We read the input values for the island dimensions, the number of years, and the elevation matrix. We initialize a priority queue to process sections in order of their elevation and a 2D array to track processed sections.
2. **Border Processing**: We push all border sections into the priority queue because these sections are adjacent to the sea and will sink first as the sea level rises.
3. **Yearly Processing**: For each year, we process all sections in the queue with elevations less than or equal to the current year's sea level. For each processed section, we decrement the remaining area and check its adjacent sections. If an adjacent section hasn't been processed and meets the elevation criteria, it is added to the queue.
4. **Output**: After processing each year, we store the remaining area in a result array, which is printed at the end for each year from 1 to Y.

This approach ensures that we efficiently and accurately track which sections sink each year, providing the correct remaining area for each year's sea level.
