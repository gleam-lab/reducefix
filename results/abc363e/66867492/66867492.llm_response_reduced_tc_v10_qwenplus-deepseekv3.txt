The problem involves simulating the sinking of sections of an island as the sea level rises each year. Sections sink if they are adjacent to the sea or a sunk section and their elevation is not greater than the current sea level. The goal is to compute the remaining area of the island for each year from 1 to Y.

### Approaches
1. **Priority Queue (Min-Heap):** We use a priority queue to efficiently process the sections in order of their elevation. This allows us to handle sections as the sea level rises.
2. **Breadth-First Search (BFS):** For each section that sinks, we check its adjacent sections to see if they should also sink. This is done using BFS to ensure all connected sections that meet the criteria are processed.
3. **Visited Tracking:** We maintain a visited matrix to keep track of sections that have already been sunk to avoid reprocessing them.

### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;

struct Init {
    Init() {
        ios::sync_with_stdio(false);
        cin.tie(nullptr);
        cout << fixed << setprecision(12);
    }
} init;

int main() {
    int H, W, Y;
    cin >> H >> W >> Y;
    vector<vector<int>> A(H, vector<int>(W));
    for (int i = 0; i < H; ++i) {
        for (int j = 0; j < W; ++j) {
            cin >> A[i][j];
        }
    }

    vector<vector<bool>> visited(H, vector<bool>(W, false));
    priority_queue<pair<int, pair<int, int>>, vector<pair<int, pair<int, int>>>, greater<pair<int, pair<int, int>>>> pq;

    // Push border cells into the priority queue
    for (int i = 0; i < H; ++i) {
        pq.push({A[i][0], {i, 0}});
        visited[i][0] = true;
        if (W > 1) {
            pq.push({A[i][W-1], {i, W-1}});
            visited[i][W-1] = true;
        }
    }
    for (int j = 1; j < W - 1; ++j) {
        pq.push({A[0][j], {0, j}});
        visited[0][j] = true;
        if (H > 1) {
            pq.push({A[H-1][j], {H-1, j}});
            visited[H-1][j] = true;
        }
    }

    int ans = H * W;
    const int dx[] = {1, 0, -1, 0};
    const int dy[] = {0, 1, 0, -1};

    for (int year = 1; year <= Y; ++year) {
        while (!pq.empty() && pq.top().first <= year) {
            auto current = pq.top();
            pq.pop();
            int x = current.second.first;
            int y = current.second.second;
            --ans;
            for (int d = 0; d < 4; ++d) {
                int nx = x + dx[d];
                int ny = y + dy[d];
                if (nx >= 0 && nx < H && ny >= 0 && ny < W && !visited[nx][ny]) {
                    visited[nx][ny] = true;
                    pq.push({A[nx][ny], {nx, ny}});
                }
            }
        }
        cout << ans << '\n';
    }

    return 0;
}
```

### Explanation
1. **Initialization:** We initialize the priority queue to process sections in increasing order of their elevation. The border sections are added to the queue first since they are adjacent to the sea.
2. **Processing:** For each year, we process all sections in the queue that have an elevation less than or equal to the current sea level. For each such section, we mark it as sunk and check its adjacent sections. If an adjacent section hasn't been processed yet, it is added to the queue.
3. **Output:** After processing all relevant sections for the current year, the remaining area of the island is printed. This continues for each year up to Y.

This approach ensures that we efficiently process the sections in the correct order and handle the sinking process accurately for each year. The use of a priority queue and BFS ensures that the solution is both correct and efficient.
