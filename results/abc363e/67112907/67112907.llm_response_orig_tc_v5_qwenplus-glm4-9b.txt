### Problem Understanding:
- The island's elevation grid is given.
- Each cell has a certain elevation above sea level.
- The sea level rises by 1 unit each year.
- Cells adjacent (horizontally or vertically) to the sea or any new cells that sink (elevation <= sea level) will also sink.
- We need to find the area above the sea level for each year from 1 to Y.

### Solution Approach:
- Use a priority queue to simulate the process of cells sinking into the sea.
- Start by marking the cells adjacent to the sea as already "sunk."
- Each year, pop the cells with the lowest elevation (greater than the sea level) from the queue and mark them as sunk.
- Check the neighbors of these cells and mark them if their elevation is less than or equal to the new sea level.
- Update the total area of the island each year after marking the new cells.

    int total_area = H * W;

    // Initialize the priority queue with all cells that are adjacent to the sea
    for (int i = 0; i < H; ++i) {
        for (int j = 0; j < W; ++j) {
            if ((i == 0 || i == H - 1) || (j == 0 || j == W - 1)) {
                pq.emplace(A[i][j], {i, j});
                vis[i][j] = 1;
                total_area--;
            }
        }
    }

    // Process the sinking each year
    for (int i = 0; i < Y; ++i) {

}
