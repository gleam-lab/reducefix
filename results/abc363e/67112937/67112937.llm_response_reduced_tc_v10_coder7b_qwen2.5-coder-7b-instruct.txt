To solve this problem, we need to simulate the rising sea levels and determine how much of the island remains above water each year. The key steps are as follows:

1. **Initialize the Grid**: Read the grid of elevations.
2. **Simulate Sea Level Rise**: For each year, increment the sea level and check which sections sink.
3. **Check Adjacent Sections**: When a section sinks, mark all adjacent sections that are below the new sea level as sinking.
4. **Count Remaining Area**: After each year, count the number of sections that remain above the sea level.

Here's the corrected C++ code:

```cpp
#include <bits/stdc++.h>

using namespace std;

const int MAXN = 1010;

int A[MAXN][MAXN];
bool visited[MAXN][MAXN];
int H, W, Y;
int d[4][2] = {{0, 1}, {0, -1}, {1, 0}, {-1, 0}};

void bfs(int start_i, int start_j, int sea_level) {
    queue<pair<int, int>> q;
    q.push({start_i, start_j});
    visited[start_i][start_j] = true;

    while (!q.empty()) {
        auto [x, y] = q.front();
        q.pop();

        for (int k = 0; k < 4; ++k) {
            int nx = x + d[k][0];
            int ny = y + d[k][1];

            if (nx >= 0 && nx < H && ny >= 0 && ny < W && !visited[nx][ny] && A[nx][ny] <= sea_level) {
                visited[nx][ny] = true;
                q.push({nx, ny});
            }
        }
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    cin >> H >> W >> Y;

    for (int i = 0; i < H; ++i) {
        for (int j = 0; j < W; ++j) {
            cin >> A[i][j];
        }
    }

    for (int year = 1; year <= Y; ++year) {
        memset(visited, false, sizeof(visited));

        // Mark sections that sink
        for (int i = 0; i < H; ++i) {
            for (int j = 0; j < W; ++j) {
                if (A[i][j] <= year) {
                    bfs(i, j, year);
                }
            }
        }

        // Count remaining sections
        int remaining_area = 0;
        for (int i = 0; i < H; ++i) {
            for (int j = 0; j < W; ++j) {
                if (!visited[i][j]) {
                    remaining_area++;
                }
            }
        }

        cout << remaining_area << '\n';
    }

    return 0;
}
```

### Explanation:
1. **Initialization**: We read the dimensions `H` and `W`, and the number of years `Y`. We then read the elevation matrix `A`.
2. **BFS Function**: This function marks all sections that will sink due to the current sea level using BFS.
3. **Main Loop**: For each year, we reset the `visited` array and use BFS to mark all sinking sections. Then, we count the number of sections that remain above the sea level and print it.

This approach ensures that we correctly simulate the rising sea levels and accurately count the remaining land area each year. The complexity is manageable given the constraints.
