
### Problem Analysis

The problem involves simulating the process of sea level rise on an island represented as a grid. Each cell in the grid has an elevation, and the sea is represented as a boundary around the island. Each year, the sea level rises by 1 unit, and cells that are adjacent to the sea or are at or below the current sea level sink.

The challenge is to determine the area of the island that remains above the sea level after a given number of years, for each year in that span.

### Solution Strategy

The solution involves several key steps:

1. **Initialization**: Read the input values and initialize the grid.

2. **Propagation of Sinking**: For each year, simulate the sinking of cells that are at or below the sea level. This involves checking adjacent cells and marking them as sinking if their elevation is at or below the current sea level.

3. **Queue Management**: Use a priority queue to manage the cells that are most likely to sink first, based on their elevation and proximity to the sea.

4. **Boundary Conditions**: Handle the boundary conditions, particularly the edges of the grid and the corners.

5. **Output**: After simulating the sinking for each year, output the remaining area of the island.

#include <iostream>
#include <vector>
#include <queue>

using namespace std;

struct Point {
    int x, y, level;
    bool operator>(const Point& other) const {
        return level > other.level;
    }
};

int main() {
    int H, W, Y;
    cin >> H >> W >> Y;
    
    vector<vector<int>> grid(H, vector<int>(W));
    for (int i = 0; i < H; ++i) {
        for (int j = 0; j < W; ++j) {
            cin >> grid[i][j];
        }
    }
    
    priority_queue<Point, vector<Point>, greater<Point>> pq;
    vector<vector<bool>> visited(H, vector<bool>(W, false));
    vector<vector<int>> directions = {{0, 1}, {0, -1}, {-1, 0}, {1, 0}};
    
    // Initialize the queue with boundary points
    for (int i = 0; i < H; ++i) {
        for (int j = 0; j < W; ++j) {
            if (i == 0 || j == 0 || i == H - 1 || j == W - 1) {
                pq.push({i, j, grid[i][j]});
                visited[i][j] = true;
            }
        }
    }
    
    // Process each year
    while (Y--) {
        int remainingArea = H * W;
        while (!pq.empty()) {
            Point current = pq.top();
            pq.pop();
            remainingArea -= 1; // Current cell sinks
            
            for (auto& dir : directions) {
                int ni = current.x + dir[0], nj = current.y + dir[1];
                if (ni < 0 || ni >= H || nj < 0 || nj >= W || visited[ni][nj]) continue;
                if (grid[ni][nj] <= current.level) {
                    pq.push({ni, nj, grid[ni][nj]});
                    visited[ni][nj] = true;
                }
            }
        }
        cout << remainingArea << "\n";
    }
    
    return 0;
}

1. **Initialization**: We read the grid and initialize a priority queue with the boundary cells (those adjacent to the sea).

2. **Boundary Conditions**: We mark all boundary cells as visited and push them into the queue.

3. **Propagation**: For each year, we process the cells in the queue. If a cell's elevation is less than or equal to the current sea level, it sinks. We then check its neighbors and add them to the queue if they are also eligible to sink.

4. **Output**: After processing each year's sinking, we output the remaining area of the island.
