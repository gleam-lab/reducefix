
### Problem Analysis

The problem involves simulating the process of rising sea levels on an island represented by a grid. The grid cell values represent the elevation of the island above the sea level. Each year, the sea level rises by one unit, and cells that are adjacent to the sea or submerged cells (cells with elevation not greater than the sea level) will sink.

The challenge is to compute the area of the island that remains above the sea level after Y years of rising sea levels.

### Solution Approach

1. **Initialization**: Read the grid dimensions and the number of years. Initialize the grid and a priority queue to keep track of the highest elevations.

2. **Boundary Initialization**: Populate the priority queue with the elevations of the boundary cells (edges of the grid) since these are automatically submerged once the sea level rises to their elevation.

3. **Processing Each Year**:
   - For each year, simulate the sinking of cells up to the current sea level.
   - Extract the highest elevation from the priority queue until it's less than the current sea level.
   - For each submerged cell, check its neighbors. If a neighbor is not yet submerged and has an elevation less than or equal to the current sea level, add it to the priority queue.
   - Update the total area of the island by subtracting the number of submerged cells.

4. **Output**: After processing each year, output the remaining area of the island.

```cpp
#include <bits/stdc++.h>
using namespace std;

const int dx[4] = {-1, 1, 0, 0};
const int dy[4] = {0, 0, -1, 1};

int H, W, Y;
int grid[1001][1001];

int main() {
    cin >> H >> W >> Y;
    for (int i = 0; i < H; ++i) {
        for (int j = 0; j < W; ++j) {
            cin >> grid[i][j];
        }
    }

    priority_queue<pair<int, int>, vector<pair<int, int>>, greater<pair<int, int>>> pq;
    vector<bool> visited(H * W, false);

    // Initialize the priority queue with boundary cells
    for (int i = 0; i < H; ++i) {
        pq.push({grid[i][0], i * W});
        pq.push({grid[i][W - 1], i * W + W - 1});
        visited[i * W] = visited[i * W + W - 1] = true;
    }
    for (int j = 0; j < W; ++j) {
        pq.push({grid[0][j], j});
        pq.push({grid[H - 1][j], (H - 1) * W + j});
        visited[j] = visited[(H - 1) * W + j] = true;
    }

    int remaining_area = H * W;

    while (Y--) {
        int current_level = pq.top().first;
        pq.pop();

        while (!pq.empty() && pq.top().first <= current_level) {
            pq.pop();
        }

        for (int i = 0; i < 4; ++i) {
            int x = pq.top().second / W;
            int y = pq.top().second % W;
            int nx = x + dx[i], ny = y + dy[i];

            if (nx >= 0 && nx < H && ny >= 0 && ny < W && !visited[nx * W + ny]) {
                if (grid[nx][ny] <= current_level) {
                    pq.push({grid[nx][ny], nx * W + ny});
                    visited[nx * W + ny] = true;
                }
            }
        }

        remaining_area -= pq.size();
        cout << remaining_area << endl;
    }

    return 0;
}
```

### Explanation

- **Priority Queue**: The priority queue is used to efficiently find and process the highest elevated cell that will sink each year.
- **Visited Array**: This array keeps track of which cells have already sunk.
- **Boundary Initialization**: Boundary cells are added to the priority queue initially as they will automatically sink when the sea level reaches their elevation.
- **Processing Each Year**: For each year, cells are submerged up to the current sea level, and their neighbors are checked for potential sinking.

This solution efficiently simulates the process of rising sea levels and calculates the remaining area of the island for each year.
