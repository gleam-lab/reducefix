    }

        pq.emplace(A[0][j], 0, j);
        pq.emplace(A[H-1][j], H-1, j);
    }

### Explanation:
1. **Initialization**: We read the dimensions `H` and `W`, and the number of years `Y`. We also read the initial elevations into the array `A`.
2. **Priority Queue**: We use a min-priority queue to always process the cell with the lowest elevation first. This helps us simulate the rising sea levels correctly.
3. **Boundary Initialization**: We push all boundary cells (cells touching the edges of the grid) into the priority queue since they will be the first to get submerged.
4. **Processing Cells**: While there are cells in the priority queue, we pop the cell with the lowest elevation. If it hasn't been visited before, we mark it as visited and decrease the total land area `ans`. Then, we push its neighboring cells into the priority queue.
5. **Output**: After processing all cells up to the point where no more land remains (`ans == 0`), we output the remaining land area for each year from 0 to `Y-1`.
