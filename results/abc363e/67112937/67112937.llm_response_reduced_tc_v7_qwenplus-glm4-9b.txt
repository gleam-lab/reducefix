4. For each year, remove cells from the priority queue that are currently below sea level and calculate the area they cover.
5. For each removed cell, check its neighbors and if their elevation is less than or equal to the sea level, add them to the priority queue.

bool operator>(const Cell& a, const Cell& b) {
    return a.h < b.h; // We use '<' as the priority queue, so the smallest value is at the top.

            else {
                pq.push({grid[i][j], j, i});

            if (c.h <= current_sea_level) {
