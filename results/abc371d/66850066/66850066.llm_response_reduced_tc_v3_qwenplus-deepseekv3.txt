### Approaches

1. **Input Reading**: Read the number of elements `N`, the array `X`, and the prefix sums array `P`.
2. **Mapping Indices**: Create a map that maps each element in `X` to its index. This helps in quickly finding the positions of elements within a range.
3. **Query Processing**: For each query, determine the indices of the elements that are just greater than or equal to `L` (lower bound) and just greater than `R` (upper bound). The sum of elements between these indices is then calculated using the prefix sums array `P`.
