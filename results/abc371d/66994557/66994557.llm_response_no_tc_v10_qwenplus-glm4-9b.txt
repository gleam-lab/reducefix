```cpp
#include<bits/stdc++.h>
using namespace std;
int n, q;
int x[200005], p[200005];
int sum[200005];

int main() {
    cin >> n;
    for (int i = 1; i <= n; i++) {
        cin >> x[i];
    }
    for (int i = 1; i <= n; i++) {
        cin >> p[i];
        sum[i] = sum[i - 1] + p[i];
    }
    cin >> q;
    while (q--) {
        int L, R;
        cin >> L >> R;
        int l = lower_bound(x + 1, x + n + 1, L) - x;
        int r = upper_bound(x + 1, x + n + 1, R) - x;
        if (l == 0 || r > n + 1) {
            cout << "0" << endl;
        } else {
            cout << sum[r - 1] - sum[l - 1] << endl;
        }
    }
    return 0;
}
```

### Explanation:
1. **Array Initialization:** The arrays `x`, `p`, and `sum` are initialized with size `200005` to accommodate all possible values including zero-based indexing.
2. **Input Reading:** The values of `n` and `x` are read initially. The values of `p` are also read and used to populate the `sum` array which holds the prefix sums.
3. **Prefix Sum Calculation:** `sum[i]` is calculated using the formula `sum[i] = sum[i - 1] + p[i]`, which builds up the cumulative sum from the first element to the `i`-th element.
4. **Query Handling:** For each query, the values `L` and `R` are read. Using `lower_bound` and `upper_bound`, the positions `l` and `r` are determined. These positions are used to find the range of interest in the `sum` array. If the range is outside the bounds of the array (i.e., `l` is 0 or `r` is greater than `n + 1`), the output is `0`. Otherwise, the result is calculated as `sum[r - 1] - sum[l - 1]` to get the sum of the subarray from index `l` to `r - 1`.
5. **Edge Cases:** The code checks if the calculated indices `l` and `r` are within the valid range to handle edge cases where `L` or `R` are at the boundaries of the array.
