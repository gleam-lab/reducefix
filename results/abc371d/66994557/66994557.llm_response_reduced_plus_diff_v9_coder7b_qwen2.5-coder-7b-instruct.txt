```cpp
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(0);

    int n, q;
    cin >> n;
    vector<int> x(n + 1), p(n + 1);
    
    // Read input values
    for(int i = 1; i <= n; ++i) {
        cin >> x[i];
    }
    for(int i = 1; i <= n; ++i) {
        cin >> p[i];
    }

    // Compute prefix sums
    vector<long long> sum(n + 1);
    for(int i = 1; i <= n; ++i) {
        sum[i] = sum[i - 1] + p[i];
    }

    cin >> q;
    while(q--) {
        int L, R;
        cin >> L >> R;
        
        // Adjust indices because arrays are 1-based
        --L;
        --R;
        
        // Calculate the sum in the range [L, R]
        long long result = sum[R] - sum[L];
        cout << result << '\n';
    }

    return 0;
}
```

### Explanation:

1. **Data Types**:
   - All arrays (`x`, `p`, `sum`) are now of type `vector<long long>` to handle large numbers without overflow.

2. **Prefix Sum Calculation**:
   - We compute the prefix sum array `sum` such that `sum[i]` contains the cumulative sum of elements from the start up to `x[i]`. This allows us to quickly calculate the sum of any subarray in constant time.

3. **Query Processing**:
   - For each query, we adjust the indices to be zero-based since the input is one-based.
   - We then use the prefix sum array to compute the sum of the desired subarray `[L, R]` as `sum[R] - sum[L]`.

This approach ensures efficient handling of large inputs and avoids overflow issues, making it suitable for competitive programming and real-world applications.
