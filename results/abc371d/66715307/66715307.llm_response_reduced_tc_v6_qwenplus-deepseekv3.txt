### Approaches
1. **Binary Search for Lower Bound (`l1`)**:
   - Find the first index where `x[mid] >= l`. If no such index exists, `l1` should be set to `n + 1`, indicating that all elements are less than `l`.

2. **Binary Search for Upper Bound (`l2`)**:
   - Find the last index where `x[mid] <= r`. If no such index exists, `l2` should be set to `0`, indicating that all elements are greater than `r`.

3. **Check Validity of Range**:
   - After determining `l1` and `l2`, check if `l1 <= l2`. If not, the sum should be zero because there are no elements in the range `[l, r]`.

#include <bits/stdc++.h>
using namespace std;
int n, x[200050], p[200050], q, a[200050];

int main() {
    cin >> n;
    for (int i = 1; i <= n; i++) cin >> x[i];
    for (int i = 1; i <= n; i++) {
        cin >> p[i];
        a[i] = a[i - 1] + p[i];
    }
    cin >> q;
    for (int i = 1; i <= q; i++) {
        int l, r;
        cin >> l >> r;
        int l1 = 1, r1 = n, ans1 = n + 1;
        while (l1 <= r1) {
            int mid = (l1 + r1) / 2;
            if (x[mid] >= l) {
                ans1 = mid;
                r1 = mid - 1;
            } else {
                l1 = mid + 1;
            }
        }
        int l2 = 1, r2 = n, ans2 = 0;
        while (l2 <= r2) {
            int mid = (l2 + r2) / 2;
            if (x[mid] <= r) {
                ans2 = mid;
                l2 = mid + 1;
            } else {
                r2 = mid - 1;
            }
        }
        if (ans1 <= ans2) {
            cout << a[ans2] - a[ans1 - 1] << endl;
        } else {
            cout << 0 << endl;
        }
    }
    return 0;
}

### Explanation
1. **Binary Search for Lower Bound (`ans1`)**:

2. **Binary Search for Upper Bound (`ans2`)**:
   - This search finds the largest index where `x[mid] <= r`. If no such index is found, `ans2` remains `0`, indicating all elements are greater than `r`.

3. **Range Check and Sum Calculation**:

This approach ensures correct handling of all edge cases, including when the query range does not overlap with any elements in the array. The binary searches efficiently narrow down the valid range, and the sum is computed only when valid indices are found.
