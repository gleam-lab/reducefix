```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(0);

    int n, q;
    cin >> n;
    vector<int> x(n), p(n + 1), prefix_sum(n + 1);

    for (int i = 0; i < n; ++i) {
        cin >> x[i];
    }
    for (int i = 0; i < n; ++i) {
        cin >> p[i];
        prefix_sum[i + 1] = prefix_sum[i] + p[i];
    }

    cin >> q;
    while (q--) {
        int l, r;
        cin >> l >> r;
        --l; // Adjusting for 0-based index
        cout << prefix_sum[r] - prefix_sum[l] << '\n';
    }

    return 0;
}
```

### Explanation of Changes
1. **Vector Usage**: Replaced arrays with `std::vector` for better memory management and easier handling of dynamic sizes.
2. **Prefix Sum Calculation**: Moved the prefix sum calculation inside the loop where `p` is read, ensuring it's done correctly.
3. **Index Adjustment**: Adjusted indices to be zero-based since C++ uses zero-based indexing.
4. **Efficiency**: The solution remains efficient with a time complexity of \(O(Q \log N)\) due to binary search operations, which is necessary for finding the correct positions in sorted arrays.
