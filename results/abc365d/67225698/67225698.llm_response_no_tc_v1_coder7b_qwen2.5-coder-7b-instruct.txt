```cpp
#include <bits/stdc++.h>
using namespace std;

void solve() {
    int n;
    cin >> n;
    string s;
    cin >> s;
    
    int win = 0;
    char last_played = '?';
    
    for (char c : s) {
        if (c != last_played) {
            ++win;
            last_played = c;
        }
    }
    
    cout << win << '\n';
}

int main() {
    ios::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);

    int t = 1;
    cin >> t;
    while (t--) {
        solve();
    }

    return 0;
}
```

This solution iterates through the string `s` and counts the number of distinct moves taken by Takahashi. Since Takahashi must not lose to Aoki and cannot repeat the same move consecutively, each unique move represents a potential win. The final count gives the maximum number of games Takahashi could have won. This approach ensures both correctness and efficiency within the constraints provided.
