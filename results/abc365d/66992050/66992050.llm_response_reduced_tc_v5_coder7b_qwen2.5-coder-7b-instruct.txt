```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int N;
    string S;
    cin >> N >> S;
    
    int ans = 0;
    char past = 'X'; // Initialize past move with a value that doesn't affect the first move
    
    for (char c : S) {
        if (c == 'P') {
            if (past != 'R') { // Takahashi should play Rock if Aoki plays Paper
                ans++;
                past = 'R';
            } else { // Otherwise, play Scissors
                past = 'S';
            }
        } else if (c == 'R') {
            if (past != 'S') { // Takahashi should play Scissors if Aoki plays Rock
                ans++;
                past = 'S';
            } else { // Otherwise, play Paper
                past = 'P';
            }
        } else { // c == 'S'
            if (past != 'P') { // Takahashi should play Paper if Aoki plays Scissors
                ans++;
                past = 'P';
            } else { // Otherwise, play Rock
                past = 'R';
            }
        }
    }
    
    cout << ans << endl;
    return 0;
}
```

### Explanation:
1. **Initialization**: We initialize `past` with a value that does not interfere with the first move (e.g., 'X').
2. **Iteration**: We iterate through each character in the string `S`.
3. **Conditional Logic**:
   - If Aoki plays `P`, Takahashi plays `R` if `past` is not `R`; otherwise, he plays `S`.
   - If Aoki plays `R`, Takahashi plays `S` if `past` is not `S`; otherwise, he plays `P`.
   - If Aoki plays `S`, Takahashi plays `P` if `past` is not `P`; otherwise, he plays `R`.
4. **Count Wins**: Each time Takahashi makes a winning move, we increment the `ans`.

This approach ensures that Takahashi always has a valid move that wins against Aoki and alternates correctly, thus maximizing the number of wins. The time complexity is \(O(N)\), which is efficient for the given constraints.
