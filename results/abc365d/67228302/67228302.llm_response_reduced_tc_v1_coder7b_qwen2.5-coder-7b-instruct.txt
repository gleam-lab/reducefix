To solve this problem, we need to determine the maximum number of games Takahashi could have won while satisfying two conditions:
1. Takahashi never loses to Aoki.
2. Takahashi's moves alternate between each other.

Here's how we can approach the solution:

1. **Understand the Moves**:
   - Rock (`R`) beats Scissors (`S`)
   - Scissors (`S`) beat Paper (`P`)
   - Paper (`P`) beats Rock (`R`)

2. **Constraints**:
   - Takahashi must win as many games as possible without losing to Aoki.
   - Takahashi's moves must alternate.

3. **Strategy**:
   - If Aoki plays Rock (`R`), Takahashi should play Paper (`P`).
   - If Aoki plays Paper (`P`), Takahashi should play Scissors (`S`).
   - If Aoki plays Scissors (`S`), Takahashi should play Rock (`R`).

4. **Implementation**:
   - We will iterate through Aoki's moves and determine the corresponding winning move for Takahashi.
   - We will count the number of wins by checking if the current move in the loop matches the expected winning move for Takahashi.

Here is the corrected C++ code:

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(0);

    map<char, char> beats = {{'R', 'P'}, {'P', 'S'}, {'S', 'R'}};
    int n;
    cin >> n;
    string s;
    cin >> s;
    int total_wins = 0;
    
    // Determine Takahashi's move based on Aoki's first move
    char takahashi_first_move = beats[s[0]];
    
    // Initialize the previous move to be the first move of Takahashi
    char prev_takahashi_move = takahashi_first_move;
    
    for (int i = 0; i < n; i++) {
        char aoki_move = s[i];
        
        // Determine the expected move for Takahashi to win against Aoki's move
        char expected_takahashi_move = beats[aoki_move];
        
        // Check if Takahashi's move alternates correctly
        if (prev_takahashi_move != expected_takahashi_move) {
            total_wins++;
            prev_takahashi_move = expected_takahashi_move;
        } else {
            // If not alternating, take the next move that beats Aoki's move
            prev_takahashi_move = expected_takahashi_move;
        }
    }
    
    cout << total_wins << endl;
    
    return 0;
}
```

### Explanation:
- We use a map `beats` to store which move beats another.
- We initialize `total_wins` to count the number of games Takahashi wins.
- We determine Takahashi's initial move based on Aoki's first move.
- We iterate through Aoki's moves, determining the expected move for Takahashi to win.
- We check if Takahashi's move alternates correctly; if not, we adjust it to ensure it does.
- Finally, we print the total number of wins.

This code ensures that Takahashi's moves alternate and maximizes the number of wins against Aoki's moves.
