### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    int N;
    string S;
    cin >> N;
    cin >> S;
    int wins = 0;
    char prev = '\0';
    for (char c : S){
        char winmove;
        if (c == 'R'){
            winmove = 'P';
        } else if (c == 'P'){
            winmove = 'S';
        } else if (c == 'S'){
            winmove = 'R';
        }
        if (winmove != prev){
            wins++;
            prev = winmove;
        }
    }   
    cout << wins;
    return 0;
}
```

1. **Initialization**: We read the input values for the number of characters `N` and the string `S`.
2. **Loop Through Characters**: For each character in `S`, we determine the winning move:
   - 'R' is beaten by 'P'.
   - 'P' is beaten by 'S'.
   - 'S' is beaten by 'R'.
3. **Count Wins**: If the current winning move is different from the previous move, we increment the win count and update the previous move to the current winning move. This ensures that we do not count consecutive duplicate winning moves.
4. **Output Result**: Finally, we print the total number of wins.

This approach efficiently counts the maximum number of distinct winning moves in sequence, addressing the issue in the initial solution where consecutive identical moves were incorrectly handled.
