```cpp
    int N;
    string S;
    cin >> N >> S;
    int wins = 0;
    for (char c : S) {
        if (c == 'R') {
            wins++;  // Play 'P' to win
        } else if (c == 'P') {
            wins++;  // Play 'S' to win
        } else if (c == 'S') {
            wins++;  // Play 'R' to win
        }
    }
    cout << wins;
}
```

2. **Counting Wins:** For each move in `S`, the code increments the `wins` counter because there exists a move (Rock->Paper, Paper->Scissors, Scissors->Rock) that can beat it.
3. **Output:** The total number of wins is printed.

This approach ensures correctness by always counting a win for each opponent move, as there's no restriction on repeating the same winning move consecutively. The solution is efficient with a time complexity of O(N), where N is the length of the string `S`.
