```cpp
#include <iostream>
#include <string>

using namespace std;

int main() {
    int n;
    string s;
    cin >> n >> s;

    int wins = 0;
    char prev = 'X';

    for (int i = 0; i < n; ++i) {
        if (prev != s[i]) {
            ++wins;
            prev = s[i];
        }
    }

    cout << wins << endl;
    return 0;
}
```

This solution simply counts the number of transitions where Takahashi plays differently from Aoki, which directly corresponds to the maximum number of games Takahashi can win while satisfying the given conditions. The time complexity is O(N), making it efficient enough for the given constraint.
