```cpp
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

char win(char x) {
    if (x == 'R') return 'P';
    if (x == 'S') return 'R';
    if (x == 'P') return 'S';
    return 'X'; // dummy
}

int main() {
    int n;
    string s;
    cin >> n >> s;
    
    if (n == 0) {
        cout << 0 << endl;
        return 0;
    }
    
    int dp[3] = {0}; // 0: R, 1: P, 2: S
    for (char c : s) {
        int new_dp[3] = {0};
        char w = win(c);
        for (int last = 0; last < 3; ++last) {
            for (int current = 0; current < 3; ++current) {
                char current_move;
                if (current == 0) current_move = 'R';
                else if (current == 1) current_move = 'P';
                else current_move = 'S';
                
                int add = (current_move == w) ? 1 : 0;
                new_dp[current] = max(new_dp[current], dp[last] + add);
            }
        }
        for (int i = 0; i < 3; ++i) {
            dp[i] = new_dp[i];
        }
    }
    
    cout << *max_element(dp, dp + 3) << endl;
    return 0;
}
```

### Explanation:
1. **Initialization**: The `dp` array tracks the maximum wins achievable ending with each possible move (R, P, S).
2. **Processing Each Character**: For each opponent's move, compute the best move to maximize wins:
   - Determine the winning move against the opponent's current move.
   - Update the DP state by considering transitions from the last move to the current move.
3. **Result Extraction**: The result is the maximum value in the DP array after processing all moves, representing the highest number of wins achievable.

This approach ensures optimality by systematically exploring all possible move sequences and selecting the one that maximizes wins.
