3. **Optimal Strategy**: To maximize wins, we should:

### Solution Code
```cpp
#include <iostream>
#include <string>
#include <algorithm>

using namespace std;

char win(char x) {
    if (x == 'R') return 'P';
    if (x == 'S') return 'R';
    if (x == 'P') return 'S';
    return 'X';
}

int main() {
    int n;
    string s;
    cin >> n >> s;

    int ans = 0;
    char prev = '\0';
    for (int i = 0; i < n; ++i) {
        char current = win(s[i]);
        if (current != prev) {
            ans++;
            prev = current;
        }
    }

    int sub = 0;
    prev = '\0';
    for (int i = n - 1; i >= 0; --i) {
        char current = win(s[i]);
        if (current != prev) {
            sub++;
            prev = current;
        }
    }

    cout << max(ans, sub) << endl;
    return 0;
}
```

### Explanation
1. **Function `win(char x)`**: This helper function returns the move that beats the input move `x` (Rock-Paper-Scissors rules).
2. **Main Function**: 
   - **Input Reading**: The code reads the length of the sequence `n` and the sequence of moves `s`.
   - **Forward Pass**: The code iterates through the sequence from left to right, counting the number of wins by choosing the optimal move that beats the current opponent's move, ensuring no consecutive identical moves unless necessary.
   - **Backward Pass**: Similarly, the code iterates from right to left to count the number of wins, again avoiding consecutive identical moves unless beneficial.
   - **Result Calculation**: The maximum of the forward and backward pass counts is printed as the result, ensuring the optimal number of wins is achieved.

This approach efficiently maximizes the number of wins by considering both directions of traversal, ensuring no unnecessary move repetitions and capturing the optimal sequence of moves.
