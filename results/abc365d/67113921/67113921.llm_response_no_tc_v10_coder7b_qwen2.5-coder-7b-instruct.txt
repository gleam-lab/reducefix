```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    int N;
    string S;
    cin >> N;
    cin >> S;
    int wins = 0;
    char prev = 'A'; // Initialize prev to a character that is not 'R', 'P', or 'S'
    for (char c : S) {
        char winmove;
        if (c == 'R') {
            winmove = 'P';
        } else if (c == 'P') {
            winmove = 'S';
        } else if (c == 'S') {
            winmove = 'R';
        }
        if (winmove != prev) {
            wins++;
            prev = winmove;
        }
    }
    cout << wins;

    return 0;
}
```

### Explanation of Changes
1. **Initialization of `prev`**: Changed the initialization of `prev` from `'a'` to `'A'`. This ensures that `prev` does not match any possible move (`'R'`, `'P'`, or `'S'`) initially, allowing the first comparison to always count as a win.
2. **Edge Case Handling**: The initial check for `prev` ensures that the first move is always counted as a win, which handles the edge case where the sequence starts with a winning move.
