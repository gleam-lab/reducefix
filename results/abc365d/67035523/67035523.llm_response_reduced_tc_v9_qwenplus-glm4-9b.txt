
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    int n;
    string s;
    cin >> n >> s;

    int consecutiveRocks = 0;
    bool lastWasP = false;

    for (char c : s) {
        if (c == 'R') {
            if (lastWasP) {
                consecutiveRocks++;
                lastWasP = false;
            }
        } else if (c == 'P') {
            lastWasP = true;
        }
    }

    cout << consecutiveRocks + 1;
    return 0;
}
```

Explanation:
- The code iterates over the string `s` and keeps track of consecutive 'R' characters following 'P' characters.
- When an 'R' is found after a 'P', it increments the `consecutiveRocks` counter.
- After every 'R' or 'P', it resets `lastWasP` to keep track of whether the last character was 'P'.
- Finally, it outputs `consecutiveRocks + 1` because the last 'R' or 'P' sequence is not counted until after the loop ends.
