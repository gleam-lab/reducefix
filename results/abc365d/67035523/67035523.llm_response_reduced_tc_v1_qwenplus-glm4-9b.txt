
```cpp
#include <iostream>
#include <string>

using namespace std;

int main() {
    int n;
    string s;
    cin >> n >> s;
    
    int count = 0;
    char prev = 'X'; // Set to 'X' to handle the case when the first character is 'R', 'S', or 'P'.
    
    for (char c : s) {
        if (c != prev) {
            count++;
            prev = c;
        }
    }
    
    cout << count;
    return 0;
}
```

### Explanation:
1. **Initialization**: Start with `count` set to 0 to keep track of the number of transitions. `prev` is initialized to 'X' to ensure the first character is also considered part of the pattern.
2. **Loop through string**: Iterate through each character in the string `s`.
3. **Transition Check**: If the current character `c` is different from `prev`, it indicates a transition which increments the `count`. Update `prev` to the current character.
4. **Output**: After processing the entire string, `count` will have the number of transitions. Print it.

This solution correctly counts the number of transitions without relying on an array or specific conditions for characters 'R', 'S', and 'P'. It handles all characters correctly, including the first character in the string.
