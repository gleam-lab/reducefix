```cpp
using namespace std;

int main() {
    int n;
    string s;
    cin >> n >> s;
    
    // Initialize variables
    int wins = 0;
    char last_move = 'X'; // Initial value should not be R, P, or S
    
    // Iterate through each move in the game
    for (int i = 0; i < n; ++i) {
        char current_move = s[i];
        
        // Check if the current move beats the previous move
        if ((current_move == 'P' && last_move == 'R') ||
            (current_move == 'R' && last_move == 'S') ||
            (current_move == 'S' && last_move == 'P')) {
            ++wins;
        }
        
        // Update the last move to the current move
        last_move = current_move;
    }
    
    // Output the maximum number of games Takahashi could have won
    cout << wins << endl;
    
    return 0;
}
```

### Explanation:
1. **Initialization**: The variable `wins` is initialized to 0 to keep track of the number of games Takahashi wins. The variable `last_move` is initialized to 'X', which is not any of the valid moves ('R', 'P', 'S'), ensuring that the first comparison will always count as a win.
2. **Loop Through Moves**: The loop iterates through each move in the string `s`. For each move, it checks if the current move can beat the previous move based on the rules of rock-paper-scissors.
3. **Update Wins and Last Move**: If the current move beats the previous move, `wins` is incremented. Then, `last_move` is updated to the current move.
4. **Output Result**: Finally, the total number of wins is printed.

This solution ensures that Takahashi wins as many games as possible while adhering to the constraints provided.
