### Solution Code
```cpp
#include <iostream>
#include <string>
#include <vector>
#include <algorithm>
using namespace std;

char getWinningMove(char c) {
    if (c == 'R') return 'P';
    if (c == 'P') return 'S';
    if (c == 'S') return 'R';
    return ' '; // Should not happen
}

int main() {
    int n;
    string s;
    cin >> n >> s;
    
    if (n == 0) {
        cout << 0 << endl;
        return 0;
    }
    
    int changes = 0;
    char prev = getWinningMove(s[0]);
    
    for (int i = 1; i < n; ++i) {
        char current = getWinningMove(s[i]);
        if (current != prev) {
            changes++;
            prev = current;
        }
    }
    
    cout << changes + 1 << endl;
    return 0;
}
```

### Explanation
1. **Function `getWinningMove`**: This helper function returns the move that beats the input move (e.g., 'P' beats 'R').
2. **Initial Check**: Handles the edge case where the input string is empty.
3. **Main Loop**: Iterates through the string starting from the second character. For each character, it determines the winning move and checks if it differs from the previous move. If it does, it increments the change count and updates the previous move.
4. **Output**: The total number of changes is printed, plus one to account for the initial move.

This approach ensures that we minimize the number of changes while always choosing the optimal move to counter the next move in the sequence. The complexity is O(N), which is efficient for the given constraints.
