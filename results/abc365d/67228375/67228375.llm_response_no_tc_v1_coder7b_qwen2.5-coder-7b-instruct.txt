```cpp
#include <bits/stdc++.h>
using namespace std;
#define endl "\n"
#define MOD 1000000007

int main() {

    ios_base::sync_with_stdio(false);
    cin.tie(0);

    map<char, char> beats = {{'R', 'P'}, {'P', 'S'}, {'S', 'R'}};
    int n;
    cin >> n;
    string s;
    cin >> s;
    
    // Initialize variables to count wins and track the last move
    int wins = 0;
    char last_move = s[0];
    bool first_turn = true;
    
    for (int i = 0; i < n; i++) {
        if (first_turn || s[i] != last_move) {
            wins++;
            last_move = beats[s[i]];
            first_turn = false;
        } else {
            last_move = s[i];
        }
    }

    cout << wins << endl;

    return 0;
}
```

### Explanation of Changes:
1. **Initialization**: We initialize `wins` to count the number of games Takahashi has won, `last_move` to keep track of the last move made, and `first_turn` to handle the special case of the first turn without comparison.
2. **Loop Logic**: The loop iterates through each move of Aoki. If it's the first turn or the current move is not the same as the last move, Takahashi wins that round. We then update `last_move` to be the move that beats the current move of Aoki and increment the win counter. If the current move is the same as the last move, we simply update `last_move` to the current move.
3. **Output**: Finally, we output the total number of wins.

This approach ensures that we correctly count the maximum number of games Takahashi could have won while adhering to the constraints and conditions provided.
