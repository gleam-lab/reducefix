#include <bits/stdc++.h>
using namespace std;
#define endl "\n"

int main() {

    map<char, char> beats = {{'R', 'P'}, {'P', 'S'}, {'S', 'R'}};
    int n;
    cin >> n;
    string s;
    cin >> s;

    // Initialize counters for wins
    int wins = 0;

    // Track the previous move
    char prev_move = '\0';

    for (int i = 0; i < n; ++i) {
        char current_move = s[i];

        // Check if Takahashi can win this round
        if (beats[current_move] != prev_move) {
            ++wins;
        }

        // Update the previous move
        prev_move = current_move;
    }

    cout << wins << endl;

    return 0;
}

### Explanation of Changes:
1. **Initialization**: We initialize `wins` to 0 to keep track of the number of games Takahashi wins.
2. **Previous Move Tracking**: We maintain `prev_move` to store the last move made by Takahashi.
3. **Winning Condition**: For each move, we check if the move Takahashi makes beats the previous move. If it does, we increment the `wins` counter.
4. **Updating Previous Move**: After checking the condition, we update `prev_move` with the current move.

This approach ensures that we only count the games where Takahashi can potentially win based on the rules provided. It runs in O(N) time complexity, making it efficient even for large inputs.
