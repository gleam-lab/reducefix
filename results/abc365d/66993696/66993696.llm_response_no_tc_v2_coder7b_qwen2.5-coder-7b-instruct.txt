```cpp
#include<bits/stdc++.h>
using namespace std;

string s, c;
int n;

bool canWin(char x, char y) {
    return (x == 'R' && y == 'S') || (x == 'S' && y == 'P') || (x == 'P' && y == 'R');
}

int main() {
    cin >> n >> s;
    c = s; // Copy Aoki's moves to Takahashi's initial guess

    // Adjust Takahashi's moves to ensure he doesn't lose and alternates moves
    for (int i = 1; i < n; ++i) {
        if (!canWin(c[i], c[i - 1])) {
            if (s[i] == 'R') c[i] = 'P';
            else if (s[i] == 'P') c[i] = 'S';
            else c[i] = 'R';
        }
    }

    int wins = 0;
    for (int i = 0; i < n; ++i) {
        if (canWin(c[i], s[i])) ++wins;
    }

    cout << wins << endl;
    return 0;
}
```

### Explanation
1. **Initialization**: We read the input values for `n` and `s`. We then create a copy of `s` into `c` as our initial guess for Takahashi's moves.
2. **Adjusting Moves**:
   - We iterate through each move in Takahashi's sequence (`c`) starting from the second move.
   - If Takahashi's move at position `i` loses against Aoki's move at position `i-1` (checked using the `canWin` function), we adjust Takahashi's move at position `i` to ensure he does not lose and alternates moves.
3. **Counting Wins**:
   - We count how many games Takahashi wins by comparing his moves with Aoki's moves using the `canWin` function.
4. **Output**: Finally, we print the number of games Takahashi could have won.

This solution ensures that Takahashi never loses and alternates his moves, maximizing the number of games he can win while adhering to the constraints.
