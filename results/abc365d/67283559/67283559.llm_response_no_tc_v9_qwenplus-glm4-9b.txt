
```cpp
#include <iostream>
#include <string>
#include <vector>
using namespace std;

int main() {
    int N;
    cin >> N;
    string S;
    cin >> S;

    string hand = "RP";
    int totalScore = 0;

    for (int i = 0; i < N; ++i) {
        int score = 0;
        char optimalMove = ' ';

        // Determine the optimal move for the current player
        if (S[i] == 'R') {
            optimalMove = 'P';
        } else if (S[i] == 'P') {
            optimalMove = 'S';
        } else if (S[i] == 'S') {
            optimalMove = 'R';
        }

        // Simulate the game
        for (char move : hand) {
            if (move == optimalMove) {
                ++score;
            }
        }
        totalScore += score;
        hand = string(1, optimalMove); // Update the hand to only include the next move
    }

    cout << totalScore;
    return 0;
}
```

### Explanation
1. The code starts by reading the number of rounds `N` and the sequence `S` of moves.
2. It initializes a string `hand` containing the possible moves 'R', 'P', 'S'.
3. The `totalScore` is initialized to 0 to keep track of the total score.
4. A loop runs for `N` rounds. In each round:
   - It determines the optimal move for the next player based on the current player's move `S[i]`.
   - It then simulates the game for that round by checking each possible move against the optimal move.
   - It updates the `totalScore` based on the number of optimal moves made.
   - It then updates the `hand` to only include the optimal move for the next round.
5. After all rounds, it prints the total score.
