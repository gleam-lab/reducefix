```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int N;
    cin >> N;
    string S;
    cin >> S;
    
    int total = 0;
    char prev = '\0';
    
    for (int i = 0; i < N; i++) {
        char current = S[i];
        if (prev != current) {
            if (current == 'R') {
                total += (S[i+1] == 'S');
            } else if (current == 'P') {
                total += (S[i+1] == 'R');
            } else if (current == 'S') {
                total += (S[i+1] == 'P');
            }
            prev = current;
        } else {
            prev = '\0';
        }
    }
    
    cout << total << endl;
    return 0;
}
```

1. We only need to consider consecutive moves in the sequence to determine when we can score points.
2. When we choose a move different from our previous move, we can potentially score against the next opponent's move.
3. The optimal strategy is to switch moves whenever we can score against the next opponent's move, and reset our move otherwise.

This approach correctly calculates the maximum score by:
- Iterating through the sequence
- Keeping track of our previous move
- Scoring when our current move beats the next opponent's move
- Resetting our move after consecutive moves to maximize scoring opportunities
