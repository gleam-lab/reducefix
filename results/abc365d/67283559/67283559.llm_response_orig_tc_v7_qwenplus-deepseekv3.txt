The problem involves a game where the player and opponent choose moves (Rock, Paper, Scissors), and the player's goal is to maximize their score by selecting the optimal move at each step based on certain constraints.

### Analysis
1. **Problem Understanding**: The player can choose a move that beats the opponent's move (Rock beats Scissors, Paper beats Rock, Scissors beats Paper). The player's move in the next step must be different from the current move. The objective is to maximize the total score over all moves.

2. **Initial Approach**: The initial code attempts to maximize the score by greedily choosing the best move at each step, alternating between the remaining moves. However, it fails to account for the optimal sequence of moves that maximizes the total score over the entire string, leading to a suboptimal solution.

3. **Optimal Strategy**: To maximize the score, the player should choose the best possible move at each step, considering that the next move must be different. This can be approached by considering all possible sequences of moves and selecting the one that maximizes the score, which can be efficiently computed using dynamic programming.

### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int N;
    cin >> N;
    string S;
    cin >> S;
    
    // The three possible moves: Rock, Paper, Scissors
    vector<char> moves = {'R', 'P', 'S'};
    
    // DP table: dp[i][j] represents the maximum score up to position i with last move j
    vector<vector<int>> dp(N + 1, vector<int>(3, 0));
    
    // Initialize the first move
    for (int j = 0; j < 3; ++j) {
        char move = moves[j];
        if ((move == 'R' && S[0] == 'S') || (move == 'P' && S[0] == 'R') || (move == 'S' && S[0] == 'P')) {
            dp[1][j] = 1;
        } else {
            dp[1][j] = 0;
        }
    }
    
    // Fill the DP table
    for (int i = 2; i <= N; ++i) {
        for (int j = 0; j < 3; ++j) {
            char current_move = moves[j];
            int current_score = 0;
            if ((current_move == 'R' && S[i-1] == 'S') || (current_move == 'P' && S[i-1] == 'R') || (current_move == 'S' && S[i-1] == 'P')) {
                current_score = 1;
            }
            // The previous move must be different from the current move
            for (int k = 0; k < 3; ++k) {
                if (k != j) {
                    dp[i][j] = max(dp[i][j], dp[i-1][k] + current_score);
                }
            }
        }
    }
    
    // The answer is the maximum score after N moves
    int max_score = max({dp[N][0], dp[N][1], dp[N][2]});
    cout << max_score << endl;
    
    return 0;
}
```

### Explanation
1. **Initialization**: The DP table `dp` is initialized where `dp[i][j]` keeps track of the maximum score achievable up to the `i-th` move, ending with the move `j` (Rock, Paper, or Scissors).

2. **Base Case**: For the first move, the score is initialized based on whether the chosen move beats the opponent's move at the first position.

3. **DP Table Filling**: For each subsequent move, the score is computed by considering all possible previous moves (ensuring they are different from the current move) and updating the current DP state with the maximum achievable score.

4. **Result Extraction**: The maximum score after `N` moves is found by taking the maximum value from the last row of the DP table.

This approach ensures that the solution is optimal and efficient, with a time complexity of O(N), which is suitable for large inputs up to the constraint limits.
