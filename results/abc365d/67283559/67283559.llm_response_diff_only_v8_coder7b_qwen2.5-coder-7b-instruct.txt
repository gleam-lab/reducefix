```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int N;
    cin >> N;
    string S;
    cin >> S;
    
    // Define possible moves
    const vector<char> hand = {'R', 'P', 'S'};
    int total = 0;
    
    // Iterate through each character in the string
    for (int i = 0; i < N; ++i) {
        int score = 0;
        char use = S[i]; // Default to using the current move
        
        // Check all possible moves against the current move
        for (char n : hand) {
            if ((n == 'R' && S[i] == 'S') || (n == 'P' && S[i] == 'R') || (n == 'S' && S[i] == 'P')) {
                score++;
                use = n;
                break;
            }
        }
        
        // Add the score to the total
        total += score;
        
        // Update the list of available moves for the next iteration
        vector<char> next(hand.begin(), hand.end());
        next.erase(remove(next.begin(), next.end(), use), next.end());
    }
    
    cout << total << endl;
    return 0;
}
```

### Explanation
1. **Initialization**: We initialize `total` to store the total score and define the possible moves (`hand`) as `{'R', 'P', 'S'}`.
2. **Main Loop**: For each character in the string `S`, we check which move would result in the highest score by comparing it with the current move `S[i]`.
3. **Score Calculation**: If a move results in a win, we increment the score and update the `use` variable to that move.
4. **Update Total Score**: We add the score to `total`.
5. **Update Available Moves**: After determining the best move for the current character, we remove that move from the list of available moves for the next iteration.
6. **Output**: Finally, we print the total score.
