```cpp
#include <bits/stdc++.h>
using namespace std;

const int N = 2e5 + 5;
int n, m, k, sum, cnt, a[N];

// Function to calculate the k-th closest distance
long long findKthClosest(int x, int y, int k) {
    vector<long long> distances;
    for (int i = 1; i <= n; ++i) {
        distances.push_back(abs(x - a[i]));
    }
    sort(distances.begin(), distances.end());
    return distances[k - 1];
}

int main() {
    ios::sync_with_stdio(0); cin.tie(0); cout.tie(0);
    
    // Read input
    cin >> n >> m;
    for (int i = 1; i <= n; ++i) {
        cin >> a[i];
    }
    
    // Process each query
    while (m--) {
        int x, y;
        cin >> x >> y;
        long long result = findKthClosest(x, y, y);
        cout << result << "\n";
    }
    
    return 0;
}
```

### Explanation of Changes
1. **Function `findKthClosest`**: This function calculates the k-th smallest distance from a given point `x` to any point in array `a`. It uses a vector to store all distances, sorts them, and then returns the k-th element (adjusting for zero-based indexing).
   
2. **Main Loop**: The main loop reads the queries one by one and calls the `findKthClosest` function with the appropriate parameters. The result is printed immediately after processing each query.

This approach ensures that we correctly handle the problem requirements within the given constraints and edge cases.
