```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int N, Q;
    cin >> N >> Q;
    vector<ll> A(N);
    for (int i = 0; i < N; ++i) {
        cin >> A[i];
    }
    sort(A.begin(), A.end());

    while (Q--) {
        ll b, k;
        cin >> b >> k;
        vector<ll> distances(N);
        for (int i = 0; i < N; ++i) {
            distances[i] = abs(A[i] - b);
        }
        nth_element(distances.begin(), distances.begin() + (k - 1), distances.end());
        cout << distances[k - 1] << '\n';
    }

    return 0;
}
```

2. **Query Processing**: For each query, given a point `B_j` and an integer `k_j`, the code computes the distances from `B_j` to all points in `A`. The distances are stored in a vector.

3. **Finding the k-th Closest Distance**: Using the `nth_element` algorithm, the code efficiently finds the k-th smallest distance without fully sorting the distances vector. This reduces the time complexity from O(N log N) per query to O(N) on average per query.

4. **Output**: The k-th smallest distance is printed for each query.
