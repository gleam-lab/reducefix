```cpp
#include <vector>
#include <iostream>
#include <algorithm>
#include <cmath>

using m_int = long long;
using std::cin;
using std::cout;
using std::vector;

int main() {
    m_int n, q;
    cin >> n >> q;
    
    vector<m_int> a(n);
    for (m_int i = 0; i < n; ++i) {
        cin >> a[i];
    }
    
    while (q--) {
        m_int b, k;
        cin >> b >> k;
        
        // Calculate distances from point b to all points a_i
        vector<m_int> distances;
        for (const auto& ai : a) {
            distances.push_back(std::abs(b - ai));
        }
        
        // Sort the distances
        std::sort(distances.begin(), distances.end());
        
        // Output the k-th smallest distance
        cout << distances[k - 1] << "\n";
    }
    
    return 0;
}
```

### Explanation:
1. **Distance Calculation**: For each query, we calculate the absolute difference between `b` and each point `a_i`. This gives us a list of distances.
2. **Sorting Distances**: We sort this list of distances in ascending order.
3. **Output the Result**: The k-th smallest distance corresponds to the element at index `k-1` in the sorted list (since indices start at 0).

This approach ensures that we correctly find the k-th smallest distance for each query, and it handles the constraints efficiently.
