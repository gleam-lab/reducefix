void solve() {
    int n,q; cin>>n>>q;
    rep(i,0,n) {
        int ai; cin>>ai;
        a[ai]++;
    }
    rep(i,1,MX) a[i]+=a[i-1];

    rep(i,0,q) {
        int bi,kj; cin>>bi>>kj;
        int l=0,r=MX;
        while(l<r) {
            int m=(l+r)/2;
            if(a[m]-a[max(0,bi-m)]>=kj) r=m;
            else l=m+1;
        }
        cout<<l-bi<<endl;
    }
}

This solution restructures the problem into finding the cumulative frequency of points up to each position, which allows for efficient querying of how many points fall within a certain range around any given point. The binary search then finds the smallest point that satisfies the condition of being at least `kj`th closest to the target point `B_j`. This approach ensures the solution handles large inputs efficiently.
