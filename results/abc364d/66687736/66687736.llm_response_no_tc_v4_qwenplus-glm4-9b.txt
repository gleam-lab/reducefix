
```cpp
#include <bits/stdc++.h>
using namespace std;
#define ll long long
#define rep(i, n) for (ll i = 0; i < (ll)n; i++)
#define vi vector<int>

int main() {
    ll n, q;
    cin >> n >> q;
    vi a(n);
    rep(i, n) {
        cin >> a[i];
    }

    multiset<pair<int, int>> sorted_points;
    rep(i, n) {
        sorted_points.insert({a[i], i});
    }

    rep(i, q) {
        ll b, k;
        cin >> b >> k;
        auto it = sorted_points.lower_bound({b - k, numeric_limits<int>::min()});
        if (it == sorted_points.begin()) {
            cout << 2 * k << endl;
        } else {
            auto prev_it = prev(it);
            cout << abs(a[prev_it->second] - b) << endl;
        }
    }

    return 0;
}
```

### Explanation:
1. We use a `multiset` to store pairs of points and their original indices. This allows us to efficiently find the k-th closest point.
2. For each query, we find the k-th closest point to each B\_j by using the `lower_bound` function on the sorted points. We then compute the distance to B\_j.
3. The `lower_bound` function ensures that we are looking for the smallest point that is still greater than or equal to B\_j - k. We then check if we are at the beginning of the set; if so, the answer is 2 * k.
4. Otherwise, we compute the distance to the previous point and print it.
