### Approaches
1. **Sort and Binary Search**: 
   - First, sort the array `a` to facilitate efficient binary search operations.
   - For each query point `b_j`, the distances from all points in `a` to `b_j` can be considered as the absolute differences `|a_i - b_j|`.
   - To find the `k_j`-th smallest distance, we can use binary search to determine the smallest distance `d` such that there are at least `k_j` points in `a` within the range `[b_j - d, b_j + d]`.

2. **Binary Search on Distance**:
   - For each query, perform binary search on the possible distance values to find the smallest `d` where the count of points within `[b_j - d, b_j + d]` is at least `k_j`.
   - The count of points within the range can be found using the upper and lower bound functions on the sorted array `a`.

### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int N, Q;
    cin >> N >> Q;
    vector<int> a(N);
    for (int i = 0; i < N; ++i) {
        cin >> a[i];
    }
    sort(a.begin(), a.end());
    
    while (Q--) {
        int b, k;
        cin >> b >> k;
        int low = 0, high = 2e8;
        int answer = 0;
        while (low <= high) {
            int mid = low + (high - low) / 2;
            auto left = lower_bound(a.begin(), a.end(), b - mid);
            auto right = upper_bound(a.begin(), a.end(), b + mid);
            int count = right - left;
            if (count >= k) {
                answer = mid;
                high = mid - 1;
            } else {
                low = mid + 1;
            }
        }
        cout << answer << '\n';
    }
    
    return 0;
}
```

1. **Initial Setup**:

2. **Processing Queries**:

   - The binary search checks if there are at least `k_j` points in `a` within the range `[b_j - d, b_j + d]` using `lower_bound` and `upper_bound` to efficiently count these points.
   - The smallest `d` that satisfies the condition is stored as the answer for the query.

3. **Output**:
   - The answer for each query is printed, which is the `k_j`-th smallest distance from `b_j` to any point in `a`.
