
The problem is asking you to find the k-th closest distance from a given set of points \(B_j\) to another set of points \(A_i\). You have already a function to calculate the k-th closest distance using a binary search approach. However, your code has a few issues which are causing it to fail. Let's identify and fix these issues step by step.

### Errors in Your Code

1. **Incorrect Binary Search Logic**: You are using a binary search to find the smallest distance \(d\) such that there are \(k\) or more distances less than \(d\). However, your binary search is not correctly implemented. The binary search should be searching for the point \(X\) such that there are \(k\) points in \(A\) closer to \(B_j\) than \(X\).

2. **Incorrect Calculation of Distance**: You are calculating the distance as \(b-j\), but it should be \(|b-j|\) since distances are always non-negative.

3. **Off-by-One Error**: In your binary search, you are comparing \(get(mp, b+m)-get(mp, b-m-1)\) with \(k\). However, since \(k\) is 0-based and your function \(get(mp, i)\) returns the count of points in \(A\) which are less than or equal to \(i\), you should compare it with \(k+1\).

4. **Memory Leaks**: You are using a map but not clearing it after use. This can lead to memory leaks.

### Corrected Solution

Let's correct the logic for your binary search. We will modify the binary search to find the smallest distance \(d\) such that there are \(k\) points in \(A\) closer to \(B_j\) than \(d\).

```cpp
#include <bits/stdc++.h>
using namespace std;
#define ll long long
#define rep(i,n) for(int i = 0; i < (int)n; i++)
#define vi vector<int>

int main() {
    int n, q; cin >> n >> q;
    vi a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    
    while (q--) {
        int b, k; cin >> b >> k;
        int lo = -1e9, hi = 1e9;
        
        // Binary search to find the k-th closest point
        while (lo + 1 < hi) {
            int mid = lo + (hi - lo) / 2;
            int count = 0;
            
            // Count how many points in A are closer to B than mid
            for (int i = 0; i < n; i++) {
                if (abs(a[i] - b) < mid) count++;
            }
            
            if (count >= k) hi = mid;
            else lo = mid;
        }
        
        cout << abs(b - lo) << endl; // The distance between X and B_j
    }
    
    return 0;
}
```

### Explanation

1. **Input Handling**: We read the number of points \(N\) and \(Q\). Then we read the list of points \(A\).

2. **Binary Search**: We perform a binary search on the distance range \([-10^9, 10^9]\). For each midpoint \(mid\), we count how many points in \(A\) are closer to \(B_j\) than \(mid\). If the count is at least \(k\), we update \(hi\) to \(mid\). Otherwise, we update \(lo\) to \(mid\).

3. **Output**: After the binary search, we output the absolute difference between \(B_j\) and the distance \(lo\), which is the k-th closest distance to \(B_j\).
