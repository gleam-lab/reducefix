```cpp
        int x; cin >> x;
        a[i] = abs(x);
    }
    sort(a, a + n);  // Sort the absolute values of a_i in ascending order

    for(int i = 0; i < q; i++) {
        int x, k; cin >> x >> k;
        int index = upper_bound(a, a + n, abs(x)) - a - 1;  // Find the index of the k-th closest point
        if (index < n) {
            if (k == 1) {
                index = 0;
            }
            index -= (k - 1);
            if (index < 0) index = 0;
        }
        cout << a[index] << endl;  // Output the distance to the k-th closest point
    }
    
    return 0;
}
```

### Explanation:
1. **Sorting**: The absolute values of `a_i` are sorted in ascending order to simplify the distance calculations.
2. **Upper_bound**: We use `upper_bound` to find the smallest element in `a` that is greater than `abs(x)`. This gives us the position of the closest point to `x`.
3. **Index Adjustment**: We calculate the index of the k-th closest point by subtracting `(k - 1)` from the position obtained from `upper_bound`. We ensure that we don't go below 0 by using a conditional check.
4. **Output**: The distance to the k-th closest point is directly obtained from the sorted array `a`.

This solution ensures that the algorithm adheres to the problem constraints and efficiently handles the given input size.
