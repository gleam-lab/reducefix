### Approaches
1. **Sorting the Array**: First, sort the array of points `a` to allow efficient binary search operations.
2. **Binary Search for Closest Points**: For each query point `b_j`, use binary search to find the position where `b_j` would be inserted in the sorted array. This helps in identifying the closest points around `b_j`.
3. **Two Pointers Technique**: Use two pointers to expand outward from the found position to collect the k_j closest points. This involves comparing the distances of points on both sides of `b_j` and selecting the closest ones.
4. **Distance Calculation**: For each of the closest points, calculate the absolute distance from `b_j` and then sort these distances to find the k_j-th smallest distance.

#include <bits/stdc++.h>
using namespace std;

typedef long long ll;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int N, Q;
    cin >> N >> Q;
    vector<ll> a(N);
    for (int i = 0; i < N; ++i) {
        cin >> a[i];
    }
    
    sort(a.begin(), a.end());
    
    while (Q--) {
        ll b;
        int k;
        cin >> b >> k;
        
        int pos = lower_bound(a.begin(), a.end(), b) - a.begin();
        
        int left = pos - 1;
        int right = pos;
        vector<ll> distances;
        
        while (distances.size() < k && (left >= 0 || right < N)) {
            if (left >= 0 && right < N) {
                if (abs(a[left] - b) <= abs(a[right] - b)) {
                    distances.push_back(abs(a[left] - b));
                    left--;
                } else {
                    distances.push_back(abs(a[right] - b));
                    right++;
                }
            } else if (left >= 0) {
                distances.push_back(abs(a[left] - b));
                left--;
            } else {
                distances.push_back(abs(a[right] - b));
                right++;
            }
        }
        
        nth_element(distances.begin(), distances.begin() + k - 1, distances.end());
        cout << distances[k - 1] << '\n';
    }
    
    return 0;
}

### Explanation
1. **Sorting the Array**: The array `a` is sorted to facilitate binary search operations, which helps in quickly locating the position of the query point `b_j`.
2. **Binary Search**: The `lower_bound` function is used to find the first position in the sorted array `a` where `b_j` could be inserted without disrupting the order. This position helps in starting the search for the closest points.
3. **Two Pointers Technique**: Two pointers, `left` and `right`, are initialized around the found position. These pointers expand outward to include points from both sides of `b_j` in the sorted array, ensuring that we consider the closest points first.
4. **Distance Collection**: Distances from `b_j` to the points around it are collected into a vector. The distances are collected in order of increasing distance from `b_j` by always picking the closer of the next left or right points.
5. **Finding the k-th Distance**: The `nth_element` function is used to efficiently find the k-th smallest distance in the collected distances vector, which is then printed as the result for the query.

This approach ensures that we efficiently find the k-th closest point for each query in logarithmic time relative to the number of points, making it suitable for large input sizes as specified in the problem constraints.
