
The problem involves finding the k-th closest point from a list of points \( A_1, A_2, \dots, A_N \) to a given point \( B_j \) on a number line. To solve this efficiently given the constraints, we can use binary search.

### Explanation of the Solution

1. **Input Reading**: We first read the number of points \( N \) and the number of queries \( Q \). We then read the coordinates of the \( N \) points \( A_i \), followed by \( Q \) pairs of values \( b_j \) and \( k_j \), where \( b_j \) is the coordinate of point \( B_j \) and \( k_j \) is the order of proximity we are interested in for each \( B_j \).

2. **Preprocessing**:
   - **Coordinate Conversion**: To handle the range of coordinates efficiently, we offset each coordinate \( a_i \) by \( 100,000,000 \). This allows us to use a simpler array for counting occurrences in the range.
   - **Counting Occurrences**: We use a counting sort approach to count occurrences of each coordinate within the offset range. This is done using the array `a`, where `a[i]` is the number of points \( a_1, a_2, \dots, a_N \) that fall within the range \([i, i + 1)\).

3. **Binary Search**:
   - For each query, we perform a binary search to find the smallest distance `mid` for which there are at least \( k_j \) points in the range \([x - mid, x + mid]\). `x` is the coordinate \( b_j \).
   - The binary search works on the distance `mid` and is determined by the condition that there are at least \( k_j \) points within the distance `mid` from \( x \).

4. **Output**: For each query, we output the computed distance `mid`.

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

const int MAX_A = 200000005; // Maximum value offseted by 100000000

int main() {
    int n, q;
    cin >> n >> q;
    vector<int> a(MAX_A, 0);
    for (int i = 0; i < n; ++i) {
        int x;
        cin >> x;
        x += 100000000; // Offset the coordinate
        a[x]++;
    }
    
    for (int i = 1; i < MAX_A; ++i) {
        a[i] += a[i - 1];
    }
    
    while (q--) {
        int x, kj;
        cin >> x >> kj;
        x += 100000000; // Offset the coordinate
        int l = 0, r = 1e8;
        
        while (l < r) {
            int mid = l + r + 1;
            if (a[mid + x] - a[x] < kj) {
                l = mid;
            } else {
                r = mid - 1;
            }
        }
        
        cout << l << endl;
    }
    
    return 0;
}
