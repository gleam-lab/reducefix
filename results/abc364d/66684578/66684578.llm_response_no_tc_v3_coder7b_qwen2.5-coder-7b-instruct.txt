// Function to find the k-th closest point
int findKthClosest(vector<int>& A, int B, int k) {
    vector<int> distances(A.size());
    rep(i, A.size()) {
        distances[i] = abs(A[i] - B);
    }
    sort(distances.begin(), distances.end());
    return distances[k - 1];
}

int main() {
    int n, q;
    cin >> n >> q;
    vi A(n);
    rep(i, n) {
        cin >> A[i];
    }

    rep(i, q) {
        int B, k;
        cin >> B >> k;
        int result = findKthClosest(A, B, k);
        cout << result << endl;
    }
    return 0;
}

### Explanation of Fixes:
1. **Functionization**: The logic for finding the k-th closest point was moved into its own function `findKthClosest`. This makes the code cleaner and more modular.
2. **Direct Calculation**: Instead of using a map and binary search, which can be inefficient for large inputs, the direct calculation of distances and sorting provides a simpler and more efficient approach.
3. **Efficiency**: Sorting the distances array takes O(N log N) time, which is efficient given the constraints. Accessing elements after sorting is then O(1).
4. **Readability**: By breaking down the problem into smaller functions, the code becomes easier to read and understand.
