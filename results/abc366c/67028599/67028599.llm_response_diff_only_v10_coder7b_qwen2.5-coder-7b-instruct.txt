```cpp
#include<bits/stdc++.h>

using namespace std;

int main() {

	int q;
	cin >> q;

	set<int> h;
	int type, x;
	while( q-- ) {
		cin >> type;
		if ( type == 1 ) {
			cin >> x;
			h.insert(x);
		} else if ( type == 2 ) {
			cin >> x;
			h.erase(h.find(x)); // Corrected erase method
		} else {
			cout << h.size() << endl;
		}
	}

	return 0;
}
```

**Explanation of Fixes:**
- In the original code, the erase operation was incorrectly using `h.erase(x);`. This would not work as expected because `x` is just the value, but `erase` requires an iterator or a range. The correct way to erase an element based on its value is to find the element first using `h.find(x)` and then erase the iterator returned by `find`. This ensures that the correct element is removed from the set.
