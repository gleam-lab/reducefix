Instead of erasing an element, we can use a map to keep track of the count of each element. #include <bits/stdc++.h>
```cpp
using namespace std;

int main() {
    int q;
    cin >> q;

    unordered_map<int, int> countMap;
    set<int> distinctElements;

    while (q--) {
        int type;
        cin >> type;
        if (type == 1) {
            int x;
            cin >> x;
            if (countMap[x] == 0) {
                distinctElements.insert(x);
            }
            countMap[x]++;
        } else if (type == 2) {
            int x;
            cin >> x;
            if (countMap[x] > 0) {
                countMap[x]--;
                if (countMap[x] == 0) {
                    distinctElements.erase(x);
                }
            }
        } else {
            cout << distinctElements.size() << endl;
        }
    }

    return 0;
}
```

   - `unordered_map<int, int> countMap`: Stores the count of each element.
   - `set<int> distinctElements`: Stores the distinct elements.

2. **Operations**:
   - For type 1 (`1 x`): If the element `x` is not already in `distinctElements`, insert it. Then increment its count in `countMap`.
   - For type 2 (`2 x`): If the element `x` exists in `countMap` and its count is greater than zero, decrement its count. If its count becomes zero, remove it from `distinctElements`.
   - For type 3 (`3`): Print the size of `distinctElements`, which represents the number of distinct elements.
