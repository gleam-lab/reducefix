```cpp
#include <bits/stdc++.h>

using namespace std;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int q;
    cin >> q;

    unordered_map<int, int> count;
    int distinct = 0;
    
    while (q--) {
        int type;
        cin >> type;
        
        if (type == 1) {
            int x;
            cin >> x;
            if (count[x] == 0) distinct++;
            count[x]++;
        } else if (type == 2) {
            int x;
            cin >> x;
            if (count[x] > 0) {
                count[x]--;
                if (count[x] == 0) distinct--;
            }
        } else {
            cout << distinct << '\n';
        }
    }

    return 0;
}
```

1. Uses `unordered_map` to track counts of each number for O(1) operations
2. Maintains a `distinct` counter to track unique numbers for O(1) query responses
3. Adds proper IO optimizations (`ios_base::sync_with_stdio(false)` and `cin.tie(nullptr)`) for handling large input
4. Uses `'\n'` instead of `endl` for faster output
5. Handles all query types efficiently in O(1) average case time per operation

The logic remains the same but is now more robust for different input formats and more efficient for large inputs. The use of a hash map to track counts ensures we can properly handle duplicate values while maintaining an accurate count of distinct numbers.
