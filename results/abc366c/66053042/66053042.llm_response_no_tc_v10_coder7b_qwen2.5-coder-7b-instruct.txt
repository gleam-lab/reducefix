```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    unordered_set<int> s;

    int Q;
    cin >> Q;
    while (Q--) {
        int type;
        cin >> type;

        if (type == 1) {
            int x;
            cin >> x;
            s.insert(x);
        } else if (type == 2) {
            int x;
            cin >> x;
            s.erase(x);
        } else if (type == 3) {
            cout << s.size() << endl;
        }
    }

    return 0;
}
```

### Explanation of Changes
1. **Data Structure Change**: The original solution used a `map<int, int>` where keys were numbers and values were their counts. This was inefficient for counting distinct elements because we had to keep track of the count and update it. The new solution uses an `unordered_set<int>`, which automatically handles duplicates and provides efficient insertions, deletions, and size checks.
   
2. **Code Simplification**: 
   - For type 1 queries (`1 x`), instead of checking if the element exists and then incrementing its count or adding it to the map, the set simply inserts the element.
   - For type 2 queries (`2 x`), the set directly erases the element, which removes it from the collection if present.
   - For type 3 queries (`3`), the size of the set is printed directly, which gives the number of unique elements.

This approach ensures the code is more efficient and easier to understand while adhering to the problem constraints.
