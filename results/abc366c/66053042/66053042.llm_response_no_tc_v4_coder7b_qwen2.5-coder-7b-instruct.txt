```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
  unordered_set<int> s;

  int Q;
  cin >> Q;
  for (int i = 0; i < Q; i++)
  {
    int type;
    cin >> type;

    if (type == 1)
    {
      int x;
      cin >> x;

      s.insert(x);
    }
    else if (type == 2)
    {
      int x;
      cin >> x;
      s.erase(x);
    }
    else if (type == 3)
    {
      cout << s.size() << endl;
    }
  }
}
```

### Explanation of Changes:
1. **Data Structure Change**: Changed from `map<int, int>` to `unordered_set<int>`. This change simplifies the implementation as we don't need to keep track of the count of each element since `unordered_set` inherently handles duplicates efficiently.
2. **Insertion and Deletion**: The `insert` method adds elements to the set, and the `erase` method removes them. Both operations maintain the uniqueness of elements.
3. **Size Query**: The size of the set directly gives us the number of unique elements, eliminating the need for maintaining a separate counter (`n`).
4. **Efficiency**: Using `unordered_set` provides average-case O(1) time complexity for insertions, deletions, and lookups, making the solution efficient even for large inputs up to the constraint limit.
