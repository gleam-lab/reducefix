```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    set<int> s;

    int Q;
    cin >> Q;
    for (int i = 0; i < Q; i++) {
        int type;
        cin >> type;

        if (type == 1) {
            int x;
            cin >> x;
            s.insert(x);
        } else if (type == 2) {
            int x;
            cin >> x;
            s.erase(s.find(x));
        } else if (type == 3) {
            cout << s.size() << endl;
        }
    }

    return 0;
}
```

This solution uses a `set` to store the unique elements efficiently. The `insert` operation adds an element to the set, ensuring uniqueness. The `erase` operation removes an element from the set. The `size` function of the set gives the number of unique elements currently stored, which directly answers the third type of query. This approach ensures both correctness and efficiency within the given constraints.
