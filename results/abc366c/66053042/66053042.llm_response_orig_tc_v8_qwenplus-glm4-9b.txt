
### Problem Analysis

The problem requires us to manage a collection of integers using a bag, which supports three types of operations:

1. Adding an integer to the bag.
2. Removing an integer from the bag.
3. Counting the number of unique integers in the bag.

Given the constraints, the solution must efficiently handle up to 200,000 operations, with each integer ranging up to 1,000,000.

### Solution Strategy

The solution involves using a set to keep track of unique integers. This is because sets in C++ (implemented as hash tables) provide average constant time complexity for insertions, deletions, and lookups, which is suitable given the problem constraints.

- **Insert Operation (1 x)**: If the integer `x` is not in the set, we add it. This increases the count of unique integers.
- **Remove Operation (2 x)**: If the integer `x` is in the set, we remove it. This decreases the count of unique integers.
- **Count Operation (3)**: We simply return the size of the set, which represents the number of unique integers.

### Corrected Code

```cpp
#include <iostream>
#include <set>
using namespace std;

int main() {
    set<int> bag;
    int Q;
    cin >> Q;
    
    for (int i = 0; i < Q; ++i) {
        int type;
        cin >> type;
        
        if (type == 1) {
            int x;
            cin >> x;
            bag.insert(x); // Insert x into the set
        } else if (type == 2) {
            int x;
            cin >> x;
            bag.erase(x); // Remove x from the set
        } else if (type == 3) {
            cout << bag.size() << endl; // Output the size of the set
        }
    }
    
    return 0;
}
```

### Explanation of the Corrected Code

1. **Data Structure**: A `set` from the `<set>` library is used to store the integers. The set automatically handles uniqueness and provides efficient average-time complexity for insertions and deletions.
2. **Input Handling**: The number of queries `Q` is read first. For each query, the type is determined, and the appropriate action is taken using conditional statements.
3. **Insertion and Deletion**: For type `1`, the integer is inserted into the set if it is not already present. For type `2`, the integer is removed from the set if it is present.
4. **Counting Unique Integers**: For type `3`, the size of the set is printed, which gives the count of unique integers.

This solution is efficient and correctly implements the required operations as specified in the problem statement.
