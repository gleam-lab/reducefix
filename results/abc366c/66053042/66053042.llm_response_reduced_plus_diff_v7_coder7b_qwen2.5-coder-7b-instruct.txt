```cpp
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(0);

    unordered_set<int> s;
    int Q;
    cin >> Q;

    while (Q--) {
        int type;
        cin >> type;

        if (type == 1) {
            int x;
            cin >> x;
            s.insert(x);
        } else if (type == 2) {
            int x;
            cin >> x;
            s.erase(x);
        } else if (type == 3) {
            cout << s.size() << endl;
        }
    }

    return 0;
}
```

1. **Data Structure**: We use an `unordered_set<int>` to store the unique numbers in the bag. This allows us to perform insertions, deletions, and size checks in average O(1) time.
  
2. **Operations**:

   - For `type 2`, we remove the number from the set.
   - For `type 3`, we output the size of the set, which represents the number of unique integers in the bag.

This approach ensures that the operations are efficient and meet the problem requirements. The use of `unordered_set` guarantees that each operation (insert, delete, and find size) runs in constant average time, making the solution suitable for large inputs up to the maximum constraints provided.
