```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    unordered_set<int> s;

    int Q;
    cin >> Q;
    for (int i = 0; i < Q; i++) {
        int type;
        cin >> type;

        if (type == 1) {
            int x;
            cin >> x;
            s.insert(x);
        } else if (type == 2) {
            int x;
            cin >> x;
            s.erase(s.find(x));
        } else if (type == 3) {
            cout << s.size() << endl;
        }
    }

    return 0;
}
```

### Explanation of Changes:
1. **Data Structure Change**: Instead of using a `map`, I've used an `unordered_set`. This change ensures that each operation (insert, erase, size) runs in average O(1) time, making the solution more efficient for large inputs.

2. **Simplified Logic**:
   - For query type `1`: Insert the element into the set.
   - For query type `2`: Erase the element from the set.
   - For query type `3`: Print the size of the set, which represents the number of unique elements.

This approach ensures that the code handles all edge cases efficiently and correctly implements the problem requirements.
