```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  unordered_set<int> s;

  int Q;
  cin >> Q;
  for (int i = 0; i < Q; i++) {
    int type;
    cin >> type;

    if (type == 1) {
      int x;
      cin >> x;
      s.insert(x);
    } else if (type == 2) {
      int x;
      cin >> x;
      s.erase(x);
    } else if (type == 3) {
      cout << s.size() << endl;
    }
  }

  return 0;
}
```

This solution uses an unordered set instead of a map to store the unique elements in the bag. The insert operation adds a new element to the set, and the erase operation removes an element from the set. The size of the set represents the number of different integers in the bag, which allows us to directly answer the third type of query without needing to count the elements manually. This approach is more efficient than using a map, especially for large inputs, as insertions and deletions in an unordered set average O(1).
