1. **Data Structure**: We use a `set<int>` to store the elements because sets inherently keep their elements sorted and do not allow duplicates. This helps in efficiently managing the unique elements.

2. **Insertion (`type == 1`)**: When inserting an element, we simply call `s.insert(x)` which adds the element to the set if it does not already exist.

3. **Deletion (`type == 2`)**: For deleting an element, we first find the iterator pointing to the element using `s.find(x)` and then erase it using `s.erase()`.

4. **Counting Unique Elements (`type == 3`)**: The size of the set gives us the count of unique elements directly using `s.size()`.

This approach ensures that all operations are performed efficiently, maintaining the required complexity.
