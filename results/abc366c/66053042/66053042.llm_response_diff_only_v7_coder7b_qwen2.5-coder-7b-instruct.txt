```cpp
#include <bits/stdc++.h>
using namespace std;

int main()
{
  unordered_set<int> s;

  int Q;
  cin >> Q;
  for (int i = 0; i < Q; i++)
  {
    int type;
    cin >> type;

    if (type == 1)
    {
      int x;
      cin >> x;

      s.insert(x); // Insert the element into the set
    }
    else if (type == 2)
    {
      int x;
      cin >> x;
      s.erase(x); // Remove the element from the set
    }
    else if (type == 3)
    {
      cout << s.size() << endl; // Output the size of the set
    }
  }

  return 0;
}
```

### Explanation of Changes:
1. **Data Structure Change**: Replaced `map<int, int>` with `unordered_set<int>`. This data structure inherently stores unique elements and allows for fast insertion, deletion, and lookup operations.
2. **Insertion**: For type 1 queries, use `s.insert(x)` instead of incrementing a count.
3. **Deletion**: For type 2 queries, use `s.erase(x)` to remove the element from the set.
4. **Count Unique Elements**: For type 3 queries, simply output `s.size()` which gives the number of unique elements in the set.

This solution ensures that the code remains efficient and correctly handles the constraints provided.
