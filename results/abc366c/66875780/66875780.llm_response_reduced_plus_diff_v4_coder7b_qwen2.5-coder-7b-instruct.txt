    bag.insert(x);

void remove(int x) {
    bag.erase(bag.find(x));

This solution uses an unordered_set to store unique numbers in the bag. The insert operation adds a number to the set, the remove operation removes a number from the set if it exists, and the countDistinctNumbers operation outputs the size of the set, which represents the number of distinct integers in the bag. This approach ensures efficient operations even for large inputs up to the constraint limits.
