
The problem involves managing a collection of integers in a bag and responding to three types of queries. The main goal is to efficiently handle the insertion, deletion, and counting of unique elements. Let's address the problem step by step.

### Problem Analysis
1. **Insertion (`1 x`)**: Add an integer `x` to the bag. If `x` is not already present, increment the count of unique integers.
2. **Deletion (`2 x`)**: Remove an integer `x` from the bag. It is guaranteed that `x` exists in the bag before this operation.
3. **Count (`3`)**: Output the number of unique integers currently in the bag.

### Solution Strategy
Given the constraints and requirements:
- Use a `std::set<int>` to store the integers. This data structure supports average O(log n) time complexity for insertion, deletion, and lookup operations.
- Maintain a count of the unique integers for efficient count queries.

### Code Explanation
Here's the corrected code with detailed comments:

```cpp
#include <iostream>
#include <set>

using namespace std;

void solve() {
    int q;
    cin >> q;
    
    set<int> st; // Use a set to store unique integers
    int unique_count = 0; // Variable to keep track of the number of unique integers
    
    for (int i = 0; i < q; i++) {
        int query_type;
        cin >> query_type;
        
        if (query_type == 1) { // Insertion query
            int x;
            cin >> x;
            st.insert(x); // Insert x into the set
            if (st.size() > unique_count) { // Check if the unique count has increased
                unique_count++;
            }
        } else if (query_type == 2) { // Deletion query
            int x;
            cin >> x;
            st.erase(x); // Remove x from the set
            if (st.size() < unique_count) { // Check if the unique count has decreased
                unique_count--;
            }
        } else if (query_type == 3) { // Count query
            cout << unique_count << endl; // Output the current count of unique integers
        }
    }
}

int main() {
    int t;
    cin >> t;
    while (t--) {
        solve();
    }
    return 0;
}
```

### Key Points
- The `std::set` automatically handles duplicate values, ensuring that each integer is stored only once.
- The `unique_count` variable is updated only when needed (on insertion if the set size increases, and on deletion if the set size decreases).
- The `solve` function processes each query and directly modifies the set or the `unique_count` accordingly, ensuring efficient handling of each type of query.
