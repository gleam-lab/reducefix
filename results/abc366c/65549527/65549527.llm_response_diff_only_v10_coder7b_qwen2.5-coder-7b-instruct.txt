		{

			st.insert(x);
		}
		else if (a == 2)

			st.erase(st.find(y));

		}

		}
	}
}

### Explanation of Fixes
1. **Use of `unordered_set`**: The original solution used `set`, which provides logarithmic time complexity for insertions and deletions. However, for this problem, we need constant average-time complexity for these operations. Therefore, we replaced `set` with `unordered_set`.
2. **Corrected `erase` Operation**: In the original code, `st.erase(y)` was incorrect because `y` might not be present in the set, causing undefined behavior. Instead, `st.erase(st.find(y))` is used, which first finds the element in the set before erasing it.
3. **Code Readability**: Improved readability by adding spaces around operators and after commas in function calls and conditions.
