Here's the fixed code that uses a combination of a set (to track unique integers) and a map (to track counts of each integer):

```cpp
#include <bits/stdc++.h>

using ll = long long;

    cin.tie(nullptr);
    
    int q;
    cin >> q;
    unordered_map<int, int> count;
    set<int> present;
    
    rep(i, q) {
        int type, x;

        if (type == 1) {
            cin >> x;
            count[x]++;
            if (count[x] == 1) {
                present.insert(x);
            }
        } else if (type == 2) {
            cin >> x;
            count[x]--;
            if (count[x] == 0) {
                present.erase(x);

        }
    }
}
```

Key changes made:
1. Added `unordered_map<int, int>` to track the count of each integer in the bag
2. Use `set` only to track which integers are present (count > 0)
3. When adding a ball (type 1), increment its count and add to set if it's newly present
4. When removing a ball (type 2), decrement its count and remove from set if count reaches 0
5. For type 3 queries, simply output the size of the set (number of unique integers)
