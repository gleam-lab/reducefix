The initial code provided uses a `set` to keep track of the unique integers in the bag, which correctly handles the count of different integers for query type 3. However, it fails to account for multiple occurrences of the same integer. For example, if we insert `x` twice and then remove one `x`, the set will remove `x` entirely, which is incorrect because there might still be another `x` in the bag.

To fix this, we need to maintain a frequency count of each integer in the bag. Only when the frequency of an integer drops to zero should we remove it from the set of unique integers.

### Approaches
1. **Data Structures**: 
   - Use a `std::unordered_map` (or `std::map`) to keep track of the frequency of each integer in the bag.
   - Use a `std::set` to keep track of the unique integers currently in the bag.

2. **Operations**:
   - **Insert (type 1)**:
     - Increment the frequency count of `x`.
     - If the frequency becomes 1, add `x` to the set.
   - **Remove (type 2)**:
     - Decrement the frequency count of `x`.
     - If the frequency drops to 0, remove `x` from the set.
   - **Query (type 3)**:
     - The size of the set gives the number of different integers in the bag.

### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;
#define rep(i, n) for (int i = 0; i < (n); ++i)
using ll = long long;

int main()
{
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    int q;
    cin >> q;
    unordered_map<int, int> freq;
    set<int> st;
    rep(i, q)
    {
        int type, x;
        cin >> type;
        if (type == 1)
        {
            cin >> x;
            freq[x]++;
            if (freq[x] == 1)
                st.insert(x);
        }
        else if (type == 2)
        {
            cin >> x;
            freq[x]--;
            if (freq[x] == 0)
                st.erase(x);
        }
        else if (type == 3)
        {
            cout << st.size() << '\n';
        }
    }
    return 0;
}
```

### Explanation
- **Frequency Map (`freq`)**: This map keeps count of how many times each integer `x` is present in the bag. For example, if `x` is inserted twice, `freq[x]` will be 2.
- **Set (`st`)**: This set contains only those integers `x` for which `freq[x]` is at least 1. This ensures that the size of the set accurately reflects the number of distinct integers in the bag at any time.
- **Operations**:
  - **Insert (type 1)**: The frequency of `x` is increased. If this operation makes `freq[x]` equal to 1, `x` is added to the set.
  - **Remove (type 2)**: The frequency of `x` is decreased. If this operation makes `freq[x]` drop to 0, `x` is removed from the set.
  - **Query (type 3)**: The size of the set is printed, which is the number of distinct integers in the bag.

This solution efficiently handles the constraints and ensures correct results for all operations.
