```cpp
#include <bits/stdc++.h>
using namespace std;
#define rep(i, n) for (int i = 0; i < (n); ++i)
using ll = long long;

int main()
{
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);

    int q;
    cin >> q;
    set<int> st;
    rep(i, q)
    {
        int type, x;
        cin >> type;
        if (type == 1)
        {
            cin >> x;
            st.insert(x);
        }
        else if (type == 2)
        {
            cin >> x;
            st.erase(st.find(x));
        }
        else if (type == 3)
        {
            cout << st.size() << endl;
        }
    }

    return 0;
}
```

**Explanation of Fixes:**
1. **Efficiency Improvement:** In the original code, when querying for the count of unique elements (`type == 3`), we were directly calling `st.size()` which is efficient. However, for removing an element (`type == 2`), the original code used `st.erase(x)` which would require traversing the set to find the element. This can be inefficient for large sets. The corrected version uses `st.erase(st.find(x))`, which is more efficient because it directly finds and removes the element using its iterator.
2. **Optimization for I/O Operations:** Added `ios_base::sync_with_stdio(false);` and `cin.tie(nullptr);` to optimize input/output operations in C++, which can significantly improve performance in competitive programming or large input scenarios.
