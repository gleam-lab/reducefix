```cpp
    int q;

    while(q--){
        int n;
        cin >> n;
        if(n==1){
            int x;
            cin >> x;
            s.insert(x);
        }else if(n==2){
            int x;
            cin >> x;

        }else{
            cout << s.size() << endl;
        }
    }

}
```
This solution uses a set to store the numbers as they are inserted. The insert operation adds a number to the set, and the erase operation removes a number from the set. The size of the set represents the number of unique numbers in the bag. This solution ensures all operations are performed in logarithmic time due to the properties of sets. It also handles all edge cases such as empty input, single elements, and max constraints effectively.
