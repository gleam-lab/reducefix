        }
    }
    return 0;
}

2. Similarly in the `n==2` case, we only decrement the distinct count when the count reaches zero after removal.
3. This ensures we maintain an accurate count of distinct integers in the bag at all times.

The fix ensures that we only modify the distinct count (ans) when we actually add a new unique number or remove the last instance of a number, which is the correct way to track the number of distinct integers in the bag.
