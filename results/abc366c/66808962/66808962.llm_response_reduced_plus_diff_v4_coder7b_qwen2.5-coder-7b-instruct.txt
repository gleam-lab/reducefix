To solve the problem, we need to maintain a set of unique integers and efficiently handle three types of queries:
1. Adding a new integer to the set.
2. Removing an existing integer from the set.
3. Printing the size of the set.

Given these requirements, using a `std::set` is ideal because it automatically maintains the uniqueness of its elements and provides efficient insertion, deletion, and size operations. Here's how you can implement this:

```cpp
#include <iostream>
#include <set>

int main() {
    std::set<int> s;
    int q;
    std::cin >> q;

    while (q--) {
        int n;
        std::cin >> n;
        if (n == 1) {
            int x;
            std::cin >> x;
            s.insert(x);
        } else if (n == 2) {
            int x;
            std::cin >> x;
            s.erase(x);
        } else {
            std::cout << s.size() << '\n';
        }
    }

    return 0;
}
```

### Explanation:
- **Initialization**: We initialize an empty set `s`.
- **Reading Queries**: We read the number of queries `q`.
- **Processing Queries**:
  - If the query type is `1`, we read an integer `x` and insert it into the set `s`. The `insert` operation ensures that the element is added only if it doesn't already exist.
  - If the query type is `2`, we read an integer `x` and remove it from the set `s`. The `erase` operation removes the element if it exists.
  - If the query type is `3`, we simply output the size of the set `s`.

This solution ensures that all operations are performed efficiently, maintaining the desired properties of the set.
