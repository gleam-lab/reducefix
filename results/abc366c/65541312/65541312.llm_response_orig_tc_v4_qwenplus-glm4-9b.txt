
The problem description is about managing a bag that can contain balls with integers written on them. We are provided with a series of queries that either add a ball to the bag, remove a ball from the bag, or ask for the number of unique balls currently in the bag. The goal is to process these queries efficiently, especially given the constraints where the number of queries can be as large as 200,000.

The provided code uses a `set` data structure to keep track of unique integers. It correctly handles the insertion (`1 x`) and removal (`2 x`) of balls with integers. However, there's an error in the removal logic. When a ball is removed, the integer is checked for existence in the set before actually performing the removal. This is unnecessary because the problem guarantees that the ball with integer `x` will always exist in the bag when a removal operation is attempted.

Here's the corrected code:

```cpp
#include<bits/stdc++.h>
using namespace std;

void solve() {
    int q;
    cin >> q;
    set<int> st;  // Using set container to store unique integers
    for (int i = 0; i < q; i++) {
        int a;
        cin >> a;
        if (a == 1) {
            int x;
            cin >> x;
            st.insert(x);  // Insert x into the set
        } else if (a == 2) {
            int y;
            cin >> y;
            st.erase(y);  // Remove y from the set
        } else if (a == 3) {
            cout << st.size() << endl;  // Output the size of the set
        }
    }
}

int main() {
```

### Explanation:
1. **Data Structure**: `set<int> st;` is used to store unique integers. The `set` automatically takes care of removing duplicates.
2. **Query Handling**: The loop processes each query one by one.
   - **Insertion (`1 x`)**: It reads an integer `x` and inserts it into the set `st`.
   - **Removal (`2 x`)**: It reads an integer `y` and removes it. Since the problem statement guarantees that `y` will be present in `st`, this operation is safe.
   - **Size Query (`3`)**: Outputs the size of the set `st`, which is the count of unique integers.
