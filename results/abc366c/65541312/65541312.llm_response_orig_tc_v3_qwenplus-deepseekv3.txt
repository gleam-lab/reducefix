}

### Explanation:
1. **Data Structures**:
   - `unordered_map<int, int> freq`: This keeps track of the count of each integer in the bag.
   - `set<int> distinct`: This keeps track of the distinct integers currently in the bag.

2. **Operations**:
   - **Insert (1 x)**:
     - Increment the count of `x` in the frequency map.
     - If the count becomes 1 (meaning it wasn't present before), add `x` to the set of distinct integers.
