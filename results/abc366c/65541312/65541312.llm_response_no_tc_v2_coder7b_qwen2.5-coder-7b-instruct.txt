void solve()
{
    int q;
    cin >> q;
    set<int> st;
    for (int i = 0; i < q; i++) {
        int a;
        cin >> a;
        if (a == 1) {
            int x;
            cin >> x;
            st.insert(x);
        }
        else if (a == 2) {
            int y;
            cin >> y;
            auto it = st.find(y);
            if(it != st.end()) st.erase(it);
        }
        else if (a == 3) {
            cout << st.size() << endl;
        }
    }
}

int main()
{

**Explanation:**
- The original code was almost correct but had a minor issue in handling the second type of query (`2 x`). Specifically, it did not check whether the element `x` exists in the set before attempting to erase it, which could lead to undefined behavior.
- The corrected version now includes a check using `auto it = st.find(y);` to ensure that the element exists in the set before erasing it. If the element is found, it proceeds to erase it. This ensures that the program behaves correctly even if a query tries to remove an element that does not exist in the set.
