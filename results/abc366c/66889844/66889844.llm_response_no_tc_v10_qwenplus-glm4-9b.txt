
```cpp
#include <iostream>
#include <set>
using namespace std;

int main() {
    int Q;
    cin >> Q;
    
    set<int> balls;
    
    while (Q--) {
        int query_type;
        cin >> query_type;
        
        if (query_type == 1) {
            int ball_number;
            cin >> ball_number;
            balls.insert(ball_number);
        } else if (query_type == 2) {
            int ball_number;
            cin >> ball_number;
            balls.erase(ball_number);
        } else if (query_type == 3) {
            cout << balls.size() << endl;
        }
    }
    
    return 0;
}
```

### Explanation:
1. The code initializes `Q` to read the number of queries and a `set` called `balls` to store the unique integers.
2. A while loop runs for the number of queries, processing each query in sequence.
3. Inside the loop, the type of query is read.
4. If the query type is `1`, the integer `ball_number` is read and added to the set `balls` using `insert`.
5. If the query type is `2`, the integer `ball_number` is read and removed from the set `balls` using `erase`.
6. If the query type is `3`, the size of the set `balls` is printed using `size()`.
7. The program continues to process queries until all are completed.
