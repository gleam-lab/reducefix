```cpp
  map<int, int> mp;

  while(q--) {
    int type; cin >> type;
    if(type == 1) {
      int x; cin >> x;
      mp[x]++;
    } else if(type == 2) {
      int x; cin >> x;
      mp[x]--;
      if(mp[x] == 0) mp.erase(x);
    } else if(type == 3) {
      cout << mp.size() << endl;
    }
  }

  return 0;
}
```

### Explanation of Changes
- **Use of Map Instead of Set**: Since we need to count occurrences of each number, using a map where the key is the number and the value is its count allows us to efficiently track duplicates.
- **Efficient Erase Operation**: After decrementing the count of a number, if it becomes zero, we remove it from the map to ensure that only distinct numbers remain.
- **Input Optimization**: Added `ios::sync_with_stdio(false)` and `cin.tie(0)` to optimize input operations, which can significantly improve performance for large inputs.
