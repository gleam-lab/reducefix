To solve the problem efficiently, we need to use a combination of a set and a map. The set will help us maintain the unique elements in the bag, while the map will keep track of the frequency of each element.

Here's the corrected code:

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios::sync_with_stdio(false);
  cin.tie(0);

  int n;
  cin >> n;
  set<int> bag;
  map<int, int> freq;

  for(int i = 0; i < n; ++i) {
    int op;
    cin >> op;
    if(op == 1) {
      int x;
      cin >> x;
      if(freq[x] == 0) {
        bag.insert(x);
      }
      ++freq[x];
    } else if(op == 2) {
      int x;
      cin >> x;
      --freq[x];
      if(freq[x] == 0) {
        bag.erase(x);
      }
    } else if(op == 3) {
      cout << bag.size() << '\n';
    }
  }

  return 0;
}
```

### Explanation:
1. **Set (`bag`)**: This will store the unique elements in the bag.
2. **Map (`freq`)**: This will store the frequency of each element in the bag.

### Operations:
- **Query 1 (`op == 1`)**: Add an element `x` to the bag. If `x` is not already in the bag, insert it into the set. Increment its frequency in the map.
- **Query 2 (`op == 2`)**: Remove an element `x` from the bag. Decrement its frequency in the map. If the frequency becomes zero, remove it from the set.
- **Query 3 (`op == 3`)**: Print the size of the set, which represents the number of unique elements in the bag.

This approach ensures that all operations are efficient, with the set providing average O(log n) time complexity for insertion and deletion, and the map providing constant time complexity for updating frequencies.
