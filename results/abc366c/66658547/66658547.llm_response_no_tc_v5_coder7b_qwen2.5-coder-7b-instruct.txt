```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
    unordered_set<int> st;
    int q; cin >> q;
    while(q--){
        int type, x; cin >> type >> x;
        if(type == 1){
            st.insert(x);
        } else if(type == 2){
            st.erase(x);
        } else if(type == 3){
            cout << st.size() << "\n";
        }
    }
}
```

### Explanation of Changes
- Changed the data structure from `set<int>` to `unordered_set<int>`: This change improves the time complexity for insertions and deletions to O(1) average case, making the solution more efficient for large inputs as specified in the constraints.
- Combined the conditions for handling query types into a single `if-else` structure for better readability and maintainability.
- Removed redundant variable declarations and kept the code concise and straightforward.
