```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios_base::sync_with_stdio(false);
  cin.tie(0);
  cout.tie(0);

  int Q; cin >> Q;
  set<int> a;

  while(Q--) {
    int t; cin >> t;
    if(t == 1) {
      int x; cin >> x;
      a.insert(x);
    } else if(t == 2) {
      int x; cin >> x;
      a.erase(a.find(x));
    } else if(t == 3) {
      cout << a.size() << '\n';
    }
  }

  return 0;
}
```

### Explanation
1. **Input Optimization**: The input/output optimization (`ios_base::sync_with_stdio(false);`, `cin.tie(0);`, `cout.tie(0);`) is applied to speed up I/O operations.
2. **Loop for Queries**: Changed the loop to use `while(Q--)` for better readability and performance.
3. **Erasing Element**: Used `a.erase(a.find(x))` instead of just `a.erase(x)` because the latter would not work correctly if multiple elements with the same value exist in the set. Since we know there is at least one such element, using `find()` ensures correct behavior.
