```cpp
                if (count[x] == 0) distinct--;
            }

            cout << distinct << '\n';
        }
    }
}
```

2. Maintains a distinct counter that's incremented when a new integer is added (count goes from 0 to 1)
3. Decrements the distinct counter when an integer's count drops to 0
