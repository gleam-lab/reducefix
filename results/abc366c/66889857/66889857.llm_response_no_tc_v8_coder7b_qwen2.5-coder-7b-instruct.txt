```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  int q; cin >> q;
  map<int, int> count;
  
  while(q--) {
    int op; cin >> op;
    if(op == 1) {
      int x; cin >> x;
      count[x]++;
    } 
    else if(op == 2) {
      int x; cin >> x;
      count[x]--;
      if(count[x] == 0)
        count.erase(x);
    } 
    else if(op == 3) {
      cout << count.size() << endl;
    }
  }

  return 0;
}
```

This solution uses a `map` instead of a `set`. The `map` allows us to keep track of how many times each number appears in the bag (`count`). When we add a number (`op == 1`), we increment its count. When we remove a number (`op == 2`), we decrement its count and erase it from the map if its count becomes zero. For the third operation (`op == 3`), we simply output the size of the map, which represents the number of unique numbers currently in the bag. This approach efficiently handles the constraints and ensures correct results.
