```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
  ios_base::sync_with_stdio(false);
  cin.tie(0);

  int q; cin >> q;
  map<int, int> countMap;
  int uniqueCount = 0;

  while(q--) {
    int t; cin >> t;
    if(t == 1) {
      int x; cin >> x;
      if(countMap[x] == 0) uniqueCount++;
      countMap[x]++;
    } 
    else if(t == 2) {
      int x; cin >> x;
      countMap[x]--;
      if(countMap[x] == 0) uniqueCount--;
    } 
    else if(t == 3) {
      cout << uniqueCount << '\n';
    }
  }

  return 0;
}
```

Explanation of changes made:
1. Changed `set<int> a` to `map<int, int> countMap`. This allows us to keep track of the frequency of each element.
2. Added `ios_base::sync_with_stdio(false); cin.tie(0);` to speed up I/O operations.
3. Used a while loop instead of a for loop for iterating over the queries. This makes the code more efficient as we don't need to know the total number of queries beforehand.
4. In the case of a query of type 1, we increment the count of the element in the `countMap`. If its count was previously zero, we increment `uniqueCount`.
5. In the case of a query of type 2, we decrement the count of the element in the `countMap`. If its count becomes zero, we decrement `uniqueCount`.
6. In the case of a query of type 3, we simply output `uniqueCount`, which gives the number of distinct elements in the bag.
