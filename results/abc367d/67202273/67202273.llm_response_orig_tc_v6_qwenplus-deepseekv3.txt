### Solution Code

```cpp
#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
const int MAXN=4e5+10;
ll a[MAXN], sum[MAXN];
unordered_map<ll, ll> cnt;

int main(){
    ll n, m, ans = 0;
    scanf("%lld %lld", &n, &m);
    for(ll i = 1; i <= n; i++){
        scanf("%lld", &a[i]);
        a[i + n] = a[i];
    }
    sum[0] = 0;
    cnt[0] = 1;
    for(ll i = 1; i <= 2 * n; i++){
        sum[i] = (sum[i-1] + a[i]) % m;
        if(i <= n){
            cnt[sum[i]]++;
        }
    }
    for(ll i = n + 1; i <= 2 * n; i++){
        cnt[sum[i - n]]--;
        ans += cnt[sum[i]];
        cnt[sum[i]]++;
    }
    printf("%lld\n", ans);
    return 0;
}
```

### Explanation
1. **Initialization**: We initialize arrays `a` to store the elements and `sum` to store prefix sums modulo `m`. A hash map `cnt` is used to count occurrences of each modulo value.
2. **Input Reading**: The input values for `n`, `m`, and the array `a` are read. The array `a` is duplicated to handle the circular nature (stored in `a[i + n]`).
3. **Prefix Sum Calculation**: The prefix sums are computed modulo `m` to ensure they remain within manageable limits and to facilitate the identification of subarrays with sum modulo `m` equal to zero.
4. **Counting Subarrays**: The `cnt` map is populated with the modulo values of the first `n` elements. For each subsequent element in the circular array, we adjust the count by removing the oldest prefix sum and adding the current one, then increment the answer by the count of the current modulo value.
5. **Output**: The total count of valid subarrays is printed.

This approach ensures that we correctly handle the circular nature of the array and efficiently count the valid subarrays using prefix sums and modulo operations, avoiding overflow and incorrect counts.
