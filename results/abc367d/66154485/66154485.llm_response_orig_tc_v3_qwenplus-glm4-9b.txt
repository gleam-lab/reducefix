```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, m, ans = 0;
    cin >> n >> m;
    vector<int> arr(n);
    for (int i = 0; i < n; i++) {
        cin >> arr[i];
    }

    // Using a frequency array to store the frequency of remainders
    vector<int> freq(m, 0);
    freq[0] = 1; // The frequency of 0 remainder is initially 1 (representing the sum of zero elements)

    // Calculate prefix sums
    for (int i = 0; i < n; i++) {
        arr[i] = (arr[i] % m + m) % m; // Make sure the elements are positive modulo m
    }

    long long current_sum = 0;
    for (int i = 0; i < n; i++) {
        current_sum = (current_sum + arr[i]) % m;

        // Increment the count of valid subsequences for the current remainder
        ans += freq[current_sum];

        // Update the frequency array
        freq[current_sum]++;
    }

    cout << ans;
    return 0;
}
```

## Explanation:
1. **Modular Arithmetic**: Each element is adjusted to be positive modulo `m` to simplify the calculations.
2. **Prefix Sum**: We keep track of running sum modulo `m`.
3. **Frequency Array**: This array stores the frequency of each possible remainder (from 0 to m-1) encountered so far.
4. Whenever a remainder `r` is encountered, it means that all the previous subsequences with sum `d * m - r` (where `d` is any non-negative integer) are valid subsequences of the current subsequence. This is why `freq[current_sum]` is added to `ans`.
5. After processing each element, the frequency of the remainder is updated to include the current subsequence.
