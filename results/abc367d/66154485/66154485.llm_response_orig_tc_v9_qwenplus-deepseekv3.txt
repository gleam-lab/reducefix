2. **Frequency Map**: Use a frequency map to count occurrences of each modulo value in the prefix sum array. For each modulo value `k`, the number of valid subarrays is given by the combination formula `C(freq[k], 2)`, which is `freq[k] * (freq[k] - 1) / 2`.
3. **Initial Adjustment**: The frequency of `0` is initially set to `1` because a prefix sum that is `0` modulo `m` indicates a subarray from the start of the array up to that index is valid.

This approach reduces the time complexity to O(N), making it efficient for large inputs.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, m;
    cin >> n >> m;
    vector<int> arr(n);
    for (int i = 0; i < n; ++i) {
        cin >> arr[i];
    }

    vector<long long> prefix(n + 1, 0);
    for (int i = 0; i < n; ++i) {
        prefix[i + 1] = prefix[i] + arr[i];
    }

    for (int i = 0; i <= n; ++i) {
        prefix[i] %= m;
        if (prefix[i] < 0) {
            prefix[i] += m;
        }
    }

    unordered_map<int, int> freq;
    freq[0] = 1;
    long long ans = 0;
    for (int i = 1; i <= n; ++i) {
        ans += freq[prefix[i]];
        freq[prefix[i]]++;
    }

    cout << ans << endl;
    return 0;
}
```

### Explanation
1. **Prefix Sum Array**: We first compute the prefix sum array where each element at index `i` represents the sum of all elements from the start of the array up to, but not including, the current index. This helps in efficiently calculating the sum of any subarray.
2. **Modulo Operation**: Each element in the prefix sum array is taken modulo `m` to transform the problem into finding subarrays where the difference in prefix sums is `0` modulo `m`.
3. **Frequency Map**: A frequency map keeps track of how many times each modulo value has been encountered. For each encountered modulo value, the number of valid subarrays ending at the current index is equal to the number of previous indices with the same modulo value.
4. **Initial Adjustment**: The frequency of `0` is initialized to `1` because a prefix sum of `0` (mod `m`) at any index indicates that the subarray from the start to that index is divisible by `m`.
5. **Result Calculation**: The result is accumulated by adding the frequency of the current modulo value before incrementing the frequency in the map. This ensures all possible pairs are counted efficiently.

This approach ensures that we efficiently count all valid subarrays in linear time, making it suitable for large input sizes.
