To fix this code, we need to optimize the algorithm to efficiently count the number of subarrays whose sum is divisible by `m`. The original approach uses a nested loop, which results in a time complexity of O(N^2), making it too slow for large inputs (e.g., N = 23762).

### Approaches
The optimal approach involves using prefix sums and modular arithmetic to reduce the problem to counting pairs of indices where the prefix sums have the same remainder when divided by `m`. This approach ensures an O(N) time complexity, which is efficient for large inputs.

1. **Prefix Sum and Modulo**: Compute the prefix sums of the array and take each sum modulo `m`. This transforms the problem into finding pairs of indices where the prefix sums have the same remainder.
2. **Count Remainders**: Use a hash map to count the occurrences of each remainder of the prefix sums modulo `m`.
3. **Combination Counting**: For each remainder that occurs `c` times, the number of valid subarrays is given by the combination formula `c choose 2`, i.e., `c * (c - 1) / 2`. Additionally, we include the count of prefix sums that are directly divisible by `m` (remainder 0).

### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, m;
    cin >> n >> m;
    vector<int> arr(n);
    for (int i = 0; i < n; ++i) {
        cin >> arr[i];
    }

    vector<long long> prefix_sum(n + 1, 0);
    for (int i = 0; i < n; ++i) {
        prefix_sum[i + 1] = prefix_sum[i] + arr[i];
    }

    vector<int> mod_counts(m, 0);
    for (int i = 0; i <= n; ++i) {
        int mod = (prefix_sum[i] % m + m) % m;
        mod_counts[mod]++;
    }

    long long ans = 0;
    for (int i = 0; i < m; ++i) {
        if (mod_counts[i] >= 2) {
            ans += (long long)mod_counts[i] * (mod_counts[i] - 1) / 2;
        }
    }

    cout << ans << endl;
    return 0;
}
```

### Explanation
1. **Prefix Sum Calculation**: The prefix sum array `prefix_sum` is computed where each element at index `i` represents the sum of the first `i` elements of the original array. This helps in efficiently calculating the sum of any subarray.
2. **Modulo Handling**: For each element in the prefix sum array, we compute its remainder when divided by `m` (adjusted to be non-negative). This remainder is stored in `mod_counts` which keeps track of how many times each remainder occurs.
3. **Counting Valid Subarrays**: For each remainder in `mod_counts`, if it occurs `c` times, the number of valid subarrays where the sum is divisible by `m` is given by the combination formula `c choose 2`. This is because any two prefix sums with the same remainder when divided by `m` will have a difference that is divisible by `m`.
4. **Output the Result**: The total count of such valid subarrays is printed as the result.

This approach efficiently reduces the problem to a linear pass through the array and a constant time operation for each remainder, making it suitable for large input sizes.
