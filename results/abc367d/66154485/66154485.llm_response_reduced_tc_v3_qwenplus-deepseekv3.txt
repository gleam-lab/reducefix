```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {

    vector<int> arr(n);
    for (int i = 0; i < n; i++) {
        cin >> arr[i];
    }

    vector<long long> prefix(n + 1, 0);
    for (int i = 0; i < n; i++) {
        prefix[i + 1] = prefix[i] + arr[i];
    }

    vector<int> mod_count(m, 0);
    mod_count[0] = 1; // for prefix[0] which is 0

    long long ans = 0;
    for (int i = 1; i <= n; i++) {
        int mod = prefix[i] % m;
        if (mod < 0) mod += m; // handle negative mod
        ans += mod_count[mod];
        mod_count[mod]++;
    }

    cout << ans;
    return 0;
}
```

This solution:
1. Computes prefix sums of the array
2. Uses modular arithmetic to count how many times each remainder occurs
3. For each prefix sum, it checks how many previous prefix sums had the same remainder (which means their difference is divisible by m)
4. The initial count of 0 is set to 1 because prefix[0] = 0

This approach has O(N) time complexity and O(M) space complexity, which is efficient enough for large inputs. It correctly handles all cases, including when m=1 (where every subarray sum is divisible by 1).

The key insight is that if two prefix sums have the same remainder when divided by m, then the subarray between them will have a sum divisible by m. This mathematical property allows us to count all valid subarrays efficiently.
