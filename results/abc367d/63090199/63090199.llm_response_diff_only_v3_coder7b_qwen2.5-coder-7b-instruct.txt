```cpp
#include<bits/stdc++.h>
using namespace std;

int n, m;
int main() {
    cin >> n >> m;
    vector<int> nums(n);
    for (int i = 0; i < n; i++) {
        cin >> nums[i];
    }

    vector<int> prefixSum(n + 1, 0);
    for (int i = 0; i < n; i++) { // Calculate prefix sum
        prefixSum[i + 1] = prefixSum[i] + nums[i];
    }

    int count = 0;
    // Find possible (s, t) pairs
    for (int s = 0; s <= n - 1; s++) {
        for (int t = s + 1; t <= n - 1; t++) {
            int step = prefixSum[t + 1] - prefixSum[s];
            if (step % m == 0) {
                count++;
            }
        }
    }

    cout << count << endl;
    return 0;
}
```

### Explanation of Fixes:
1. **Initialization of `s` in the outer loop**: Changed from `for(int s=1;s<=n;s++)` to `for(int s=0;s<=n-1;s++)`. This ensures that the starting index is within the valid range of the array indices.
2. **Loop condition for `t` in the inner loop**: Changed from `for(int t=1;t<=n;t++)` to `for(int t=s+1;t<=n-1;t++)`. This ensures that `t` always starts from `s + 1`, avoiding duplicate pairs and self-comparison.
3. **Prefix sum calculation**: Correctly calculated the prefix sum using `prefixSum[t + 1] - prefixSum[s]` to get the sum of elements between `s` and `t`.
4. **Edge case handling**: Ensured that all possible pairs `(s, t)` where `s < t` are considered without missing any combinations.
