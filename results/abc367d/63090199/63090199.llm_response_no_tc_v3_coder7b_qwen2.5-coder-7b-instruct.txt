```cpp
#include<bits/stdc++.h>
using namespace std;

int n, m;
vector<int> nums;
vector<int> prefixSum;

void calculatePrefixSum() {
    prefixSum.resize(n + 1, 0);
    for (int i = 0; i < n; ++i) {
        prefixSum[i + 1] = prefixSum[i] + nums[i];
    }
}

int main() {
    cin >> n >> m;
    nums.resize(n);
    for (int i = 0; i < n; ++i) {
        cin >> nums[i];
    }

    calculatePrefixSum();

    int count = 0;
    for (int s = 0; s < n; ++s) {
        for (int t = s + 1; t < n; ++t) {
            int step = prefixSum[t + 1] - prefixSum[s];
            if (step % m == 0) {
                count++;
            }
        }
    }

    cout << count << endl;
    return 0;
}
```
### Explanation of Changes:
1. **Initialization**: Properly initialize `nums` and `prefixSum` vectors before using them.
2. **Prefix Sum Calculation**: Extracted the prefix sum calculation into a separate function `calculatePrefixSum()` to improve readability and maintainability.
3. **Loop Bounds**: Corrected the loop bounds for `t` to start from `s + 1` to avoid unnecessary comparisons with the same element.
4. **Indexing Correction**: Used `t + 1` in `prefixSum[t + 1]` as the array is zero-indexed.
5. **Edge Case Handling**: The original code already handled the case where `s == t` by continuing the loop, which is correct. No changes were made here.
