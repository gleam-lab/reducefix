```cpp
#include<bits/stdc++.h>
using namespace std;

int n, m;

int main() {
    cin >> n >> m;
    vector<int> nums(n);
    for (int i = 0; i < n; i++) {
        cin >> nums[i];
    }

    vector<int> prefixSum(n + 1, 0);
    for (int i = 0; i < n; i++) { // 求前缀和
        prefixSum[i + 1] = prefixSum[i] + nums[i];
    }

    int count = 0;
    // 寻找有可能的(s, t)对
    for (int s = 1; s <= n; s++) {
        for (int t = 1; t <= n; t++) {
            if (s == t) continue;
            int step = 0;
            if (s < t) {
                step = prefixSum[t] - prefixSum[s];
            } else {
                step = prefixSum[n] - prefixSum[s] + prefixSum[t];
            }
            if (step % m == 0) {
                count++;
            }
        }
    }

    cout << count << endl;
    return 0;
}
```

### Explanation of Fixes
1. **Initialization**: The `prefixSum` array is initialized with size `n + 1` to accommodate the index `n`.
2. **Prefix Sum Calculation**: The loop for calculating the prefix sum correctly accumulates the values from `nums`.
3. **Step Calculation**: The logic for calculating the step between indices `s` and `t` is correct, considering both cases where `s < t` and `s > t`.
4. **Modulo Check**: The condition `if (step % m == 0)` correctly checks if the step is divisible by `m`.
