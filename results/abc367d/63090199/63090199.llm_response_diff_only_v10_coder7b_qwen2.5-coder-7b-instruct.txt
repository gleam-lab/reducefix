```cpp
#include<bits/stdc++.h>
using namespace std;

int main() {
    int n, m;
    cin >> n >> m;
    vector<int> nums(n);
    for (int i = 0; i < n; ++i) {
        cin >> nums[i];
    }

    // Initialize prefix sum array with an extra space to handle edge cases easily
    vector<int> prefixSum(n + 1, 0);
    for (int i = 0; i < n; ++i) {
        prefixSum[i + 1] = prefixSum[i] + nums[i];
    }

    int count = 0;
    // Iterate over all possible pairs (s, t)
    for (int s = 0; s < n; ++s) {
        for (int t = s + 1; t < n; ++t) {
            int step = prefixSum[t + 1] - prefixSum[s]; // Correctly calculate the step using prefix sums
            if (step % m == 0) {
                count++;
            }
        }
    }

    cout << count << endl;
    return 0;
}
```

### Explanation of Changes
1. **Initialization of `prefixSum`**: Added an extra element at the end to simplify index handling.
2. **Correct Calculation of `step`**: Used `prefixSum[t + 1] - prefixSum[s]` to correctly compute the difference between the sums up to indices `t` and `s`.
3. **Loop Adjustment**: Changed inner loop to start from `s + 1` to avoid redundant checks where `s` equals `t`.

These changes ensure that the code correctly calculates the number of valid `(s, t)` pairs where the difference between the prefix sums is divisible by `m`.
