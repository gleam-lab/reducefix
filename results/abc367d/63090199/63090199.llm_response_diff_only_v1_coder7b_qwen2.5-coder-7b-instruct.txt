```cpp
#include<bits/stdc++.h>
using namespace std;

int n, m;

int main() {
    cin >> n >> m;
    vector<int> nums(n);
    for (int i = 0; i < n; i++) {
        cin >> nums[i];
    }

    vector<int> prefixSum(n + 1, 0);
    for (int i = 0; i < n; i++) { // 计算前缀和
        prefixSum[i + 1] = prefixSum[i] + nums[i];
    }

    int count = 0;
    // 寻找可能的 (s, t) 对
    for (int s = 0; s < n; s++) {
        for (int t = s + 1; t <= n; t++) {
            int step = prefixSum[t] - prefixSum[s];
            if (step % m == 0) {
                count++;
            }
        }
    }

    cout << count << endl;
    return 0;
}
```

### Explanation of Fixes
1. **Initialization of `prefixSum`**: The initial value of `prefixSum[0]` is already set to 0, so no change needed here.
2. **Loop Range for `t`**: Changed the inner loop to start from `s + 1` instead of `1`. This ensures that `s` and `t` are distinct indices.
3. **Correct Calculation of `step`**: Simplified the calculation of `step` to always use the formula `prefixSum[t] - prefixSum[s]`, which correctly handles both cases (`s < t` and `s > t`).
4. **Edge Case Handling**: The original code had an off-by-one error in the range of the outer loop for `s` and inner loop for `t`. Adjusting these loops ensures all valid pairs `(s, t)` are considered.
