```cpp
#include<bits/stdc++.h>
using namespace std;

int n, m;
int main() {
    cin >> n >> m;
    vector<int> nums(n + 1);
    for (int i = 1; i <= n; ++i) {
        cin >> nums[i];
        nums[i] += nums[i - 1]; // Correctly compute the prefix sum
    }

    int count = 0;
    for (int s = 1; s <= n; ++s) {
        for (int t = s; t <= n; ++t) { // Start t from s to avoid duplicate counting
            int step = nums[t] - nums[s - 1];
            if (step % m == 0) {
                count++;
            }
        }
    }

    cout << count << endl;
    return 0;
}
```
In this solution:
- The prefix sum array `prefixsum` is correctly initialized with size `n+1`.
- When computing the prefix sum, we ensure that `nums[0]` is 0 to handle the case when `s` is 1.
- The inner loop starts from `s` to `n` to avoid counting the same pair `(s, t)` multiple times.
- The logic for calculating `step` is simplified and corrected.
