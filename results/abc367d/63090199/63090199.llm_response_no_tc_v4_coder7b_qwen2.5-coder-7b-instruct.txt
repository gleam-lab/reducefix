```cpp
#include<bits/stdc++.h>
using namespace std;

int n, m;

int main() {
    cin >> n >> m;
    vector<int> nums(n);
    for (int i = 0; i < n; i++) {
        cin >> nums[i];
    }

    // Calculate prefix sum array
    vector<int> prefixSum(n + 1, 0);
    for (int i = 0; i < n; i++) {
        prefixSum[i + 1] = prefixSum[i] + nums[i];
    }

    int count = 0;
    // Iterate over all possible pairs (s, t)
    for (int s = 0; s < n; s++) {
        for (int t = s + 1; t < n; t++) {
            int step = prefixSum[t + 1] - prefixSum[s];
            if (step % m == 0) {
                count++;
            }
        }
    }

    cout << count << endl;
    return 0;
}
```
### Explanation of Changes:
1. **Prefix Sum Calculation**: The loop to calculate the prefix sum should go up to `i + 1` to include the current element in the sum.
2. **Pair Iteration**: Changed the inner loop to start from `s + 1` instead of `1` to avoid redundant comparisons with the same index.
3. **Step Calculation**: Simplified the calculation of steps between indices `s` and `t`.
4. **Edge Cases**: Ensured that the loop boundaries correctly handle the range `[0, n)` for both outer and inner loops.
