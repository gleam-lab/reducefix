To fix this code, we need to correctly count the number of subarrays where the sum of elements modulo M is equal to zero. The original code had some logical issues in how it was counting these subarrays. 

### Approaches

1. **Prefix Sum and Modulo**: We use prefix sums to efficiently compute the sum of any subarray. The key insight is that if the difference between two prefix sums is congruent to zero modulo M, then the subarray between these two indices has a sum divisible by M.

2. **Hash Map for Counting**: We maintain a hash map (or dictionary) to count the occurrences of each modulo result of the prefix sums. For each prefix sum modulo M, we check how many times this modulo result has been seen before. This count gives the number of valid subarrays ending at the current index.

3. **Initialization**: We initialize the hash map with a zero modulo result at index -1 (or before the first element) to account for subarrays starting from the beginning of the array.

### Solution Code

```cpp
#include<bits/stdc++.h>

using namespace std;
using ll=long long;
typedef pair<int, int>PII;
const int N=2e5+10,mod=1e9+7;

void solve()
{
    int n, M;
    cin >> n >> M;
    vector<int> a(n);
    for (int i = 0; i < n; ++i) {
        cin >> a[i];
    }
    
    vector<ll> prefix(n + 1, 0);
    for (int i = 0; i < n; ++i) {
        prefix[i+1] = prefix[i] + a[i];
    }
    
    map<ll, ll> mod_counts;
    mod_counts[0] = 1; // To account for subarrays starting from the beginning
    
    ll ans = 0;
    for (int i = 1; i <= n; ++i) {
        ll mod_val = (prefix[i] % M + M) % M;
        ans += mod_counts[mod_val];
        mod_counts[mod_val]++;
    }
    
    cout << ans << endl;
}

int main()
{
    cin.tie(0);
    ios::sync_with_stdio(0);
    ll t=1;
    while(t--){
        solve();
    }
    return 0;
}
```

### Explanation

1. **Prefix Sum Calculation**: We compute the prefix sums of the array to allow efficient sum queries of any subarray. The prefix sum at index `i` (1-based) is the sum of the first `i` elements of the array.

2. **Modulo Operation**: For each prefix sum, we compute its modulo with `M` to find the remainder when the sum up to that index is divided by `M`. This helps in identifying subarrays where the sum is divisible by `M`.

3. **Counting Valid Subarrays**: Using a hash map, we keep track of how many times each modulo result has been encountered. For each current modulo result, if it has been seen before, it means there are subarrays ending at the current index that satisfy the condition (sum divisible by `M`). The count of such subarrays is incremented by the number of times the current modulo result has been seen before.

4. **Initialization**: The hash map is initialized with `mod_counts[0] = 1` to handle the case where the subarray starts from the beginning of the array (i.e., the prefix sum itself is divisible by `M`).

This approach ensures that we efficiently count all valid subarrays in O(N) time with O(N) space complexity, leveraging the properties of prefix sums and modulo arithmetic.
