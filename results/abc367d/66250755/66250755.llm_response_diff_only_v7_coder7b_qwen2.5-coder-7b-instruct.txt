```cpp
#include<bits/stdc++.h>
using namespace std;

int n, m, a[800005], t[1000005], an, w[800005];

int main() {
    cin >> n >> m;
    for (int i = 1; i <= n; ++i) {
        cin >> a[i];
        a[i + n] = a[i];
    }
    for (int i = 1; i < 2 * n; ++i) {
        w[i + 1] = (w[i] + a[i]) % m;
        if (i > n) --t[w[i - n]];
        if (i >= n) {
            an += t[w[i]];
        }
        ++t[w[i]];
    }
    cout << an;
    return 0;
}
```
### Explanation of Changes:
1. **Array Size Adjustment**: The array size was increased to `800005` from `400005` to accommodate the doubled input array and ensure it does not overflow.
2. **Initialization Correction**: Removed the initialization of `an` as it should be calculated dynamically within the loop.
3. **Efficiency Improvement**: The logic inside the loop ensures that only necessary calculations are performed, maintaining high efficiency.
4. **Edge Case Handling**: The solution now handles the edge case where `i` is less than or equal to `n` correctly, ensuring no out-of-bounds access.
