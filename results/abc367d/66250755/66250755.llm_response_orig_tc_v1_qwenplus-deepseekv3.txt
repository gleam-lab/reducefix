```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, m;
    cin >> n >> m;
    vector<int> a(n);
    for (int i = 0; i < n; i++) {
        cin >> a[i];
        a[i] %= m;
    }

    vector<long long> prefix(n + 1, 0);
    for (int i = 0; i < n; i++) {
        prefix[i + 1] = (prefix[i] + a[i]) % m;
    }

    unordered_map<int, int> count;
    count[0] = 1;
    long long result = 0;

    for (int i = 1; i <= n; i++) {
        int mod = prefix[i];
        result += count[mod];
        count[mod]++;
    }

    cout << result << endl;
    return 0;
}
```

1. **Input Reading**: The code reads the input values for `n` (number of elements) and `m` (modulo value).
2. **Modulo Handling**: Each element is taken modulo `m` immediately after reading.
3. **Prefix Sum Calculation**: A prefix sum array is computed where each element represents the sum of all previous elements modulo `m`.
4. **Counting Subarrays**: Using a hash map to count occurrences of each prefix sum modulo `m`. If a prefix sum appears `k` times, there are `k*(k-1)/2` subarrays that sum to 0 modulo `m` between those indices.
5. **Result Calculation**: The result is computed by iterating through the prefix sums and adding the count of each modulo value seen so far to the result.

This approach ensures that we efficiently count all subarrays whose sum is congruent to 0 modulo `m` in O(n) time complexity, which is optimal for this problem. The use of a hash map allows for constant time lookups and updates.
