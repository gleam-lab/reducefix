### Approaches

2. **Frequency Counting**: Use a hash map (or frequency array) to keep track of how many times each modulo value has been encountered. This allows us to count the number of valid subarrays ending at each position efficiently.
3. **Sliding Window or Frequency Array**: Given the constraints, a frequency array might be more efficient than a hash map for large `m`, but since `m` can be up to 1e6, a frequency array of size `m` is feasible.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, m;
    cin >> n >> m;
    vector<int> a(n);
    for (int i = 0; i < n; ++i) {
        cin >> a[i];
        a[i] %= m;
    }

    vector<long long> prefix(n + 1, 0);
    for (int i = 0; i < n; ++i) {
        prefix[i + 1] = (prefix[i] + a[i]) % m;
    }

    unordered_map<int, long long> freq;
    freq[0] = 1;
    long long result = 0;

    for (int i = 1; i <= n; ++i) {
        int mod = prefix[i];
        result += freq[mod];
        freq[mod]++;
    }

    cout << result << endl;
    return 0;
}
```

### Explanation
1. **Input Handling**: The code reads the number of elements `n` and the modulus `m`, followed by the array elements. Each element is immediately taken modulo `m` to simplify further calculations.
2. **Prefix Sum Calculation**: The prefix sum array is computed where each entry at index `i` represents the sum of the first `i` elements modulo `m`. This helps in identifying subarrays whose sum modulo `m` is zero.
3. **Frequency Map**: A hash map `freq` is used to count how many times each modulo value has been encountered in the prefix sum array. The initial value `freq[0] = 1` accounts for the sum of the empty prefix (sum 0).
4. **Counting Valid Subarrays**: For each position in the prefix sum array, the code checks how many times the current modulo value has been seen before. Each occurrence indicates a subarray ending at the current position that sums to zero modulo `m`. The result is accumulated by adding these counts.
5. **Output**: The total count of valid subarrays is printed.

This approach efficiently counts the valid subarrays in O(n) time with O(m) space complexity, leveraging modular arithmetic and prefix sums to avoid brute-force checks.
