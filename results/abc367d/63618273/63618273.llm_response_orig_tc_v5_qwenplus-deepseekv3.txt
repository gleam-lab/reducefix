### Correct Approach

2. **Frequency Map**: Use a map to keep track of how many times each remainder has been seen so far.
3. **Counting Valid Subarrays**: For each remainder, if it has been seen before, it means there exists a subarray between the previous occurrence and the current index whose sum modulo `M` is zero.

#include <bits/stdc++.h>

using namespace std;
using ll = long long;
typedef pair<int, int> PII;
const int N = 2e5 + 10, mod = 1e9 + 7;

void solve() {
    int n, M;
    cin >> n >> M;
    vector<int> a(n + 10);
    for (int i = 1; i <= n; ++i) {
        cin >> a[i];
    }
    vector<ll> s(n + 10);
    for (int i = 1; i <= n; ++i) {
        s[i] = s[i - 1] + a[i];
    }
    map<ll, ll> mp;
    mp[0] = 1;
    ll ans = 0;
    for (int i = 1; i <= n; ++i) {
        ll rem = (s[i] % M + M) % M;
        ans += mp[rem];
        mp[rem]++;
    }
    cout << ans << endl;
}

int main() {

    ios::sync_with_stdio(0);

### Explanation
1. **Prefix Sum Calculation**: The prefix sum array `s` is computed where `s[i]` is the sum of the first `i` elements.
2. **Modulo Remainder Tracking**: For each prefix sum, we compute its remainder when divided by `M` (handling negative values by adding `M` before taking modulo).
3. **Frequency Map**: A map `mp` is used to count how many times each remainder has been encountered. This helps in identifying how many subarrays ending at the current index have a sum divisible by `M`.
4. **Counting Valid Subarrays**: For each remainder at position `i`, if the same remainder was seen before at position `j`, then the subarray from `j+1` to `i` has a sum divisible by `M`. The count of such subarrays is incremented by the frequency of the remainder before updating the map.

This approach ensures that all valid subarrays are counted efficiently with a time complexity of O(N), which is optimal for the problem constraints.
