### Analysis
1. **Problem Understanding**: We need to count the number of subarrays where the sum of elements modulo M is zero. This can be rephrased as finding pairs of indices (i, j) where the prefix sum at j minus the prefix sum at i is congruent to 0 modulo M, i.e., (sum[j] - sum[i]) % M == 0.

2. **Prefix Sum Approach**: Calculate the prefix sums and keep track of the modulo of these sums with M. For each position, we need to find how many previous prefix sums have the same modulo value, as this indicates a subarray that sums to a multiple of M.

3. **Map Usage**: Use a map to store the frequency of each modulo value encountered in the prefix sums. For each current prefix sum modulo M, the number of valid subarrays ending at the current position is the count of previous prefix sums with the same modulo value.

4. **Initialization**: The map should initially contain a zero count for modulo 0 to account for the prefix sum before the first element.

#include<bits/stdc++.h>

using namespace std;
using ll=long long;
typedef pair<int, int>PII;
const int N=2e5+10,mod=1e9+7;

void solve()
{
    int n,M;

    vector<int>a(n+10);

        cin>>a[i];
    }
    vector<ll>s(n+10,0);
    for(int i=1; i<=n; i++){
        s[i]=s[i-1]+a[i];
    }

    mp[0]=1;
    ll ans=0;
    for(int i=1; i<=n; i++){
        int mod_val=(s[i]%M + M)%M;
        ans+=mp[mod_val];
        mp[mod_val]++;
    }
    cout<<ans;
}

int main()
{

### Explanation
1. **Prefix Sum Calculation**: The prefix sum array `s` is computed where `s[i]` represents the sum of the first `i` elements of the array.
2. **Modulo Handling**: For each prefix sum `s[i]`, compute its modulo value with `M` to get `mod_val`. This value is adjusted to be non-negative by adding `M` and taking modulo `M` again.
3. **Counting Valid Subarrays**: Using a map `mp` to keep track of how many times each modulo value has been encountered, we increment the answer by the count of previous prefix sums with the same modulo value. This works because if two prefix sums have the same modulo value, the subarray between them sums to a multiple of `M`.
4. **Initialization**: The map is initialized with `mp[0] = 1` to account for the prefix sum before the first element, which is zero.

This approach efficiently counts all valid subarrays in O(N) time with O(M) space complexity, leveraging the properties of prefix sums and modular arithmetic.
