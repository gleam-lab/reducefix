### Solution Code

#include <bits/stdc++.h>
using LL = long long;

void solve() {
    int n, m;
    std::cin >> n >> m;
    std::vector<int> a(n);

    for (int &x : a) {
        std::cin >> x;
        x %= m;
    }

    std::unordered_map<int, int> prefix_counts;
    prefix_counts[0] = 1;
    int current_sum = 0;
    LL result = 0;

    for (int x : a) {
        current_sum = (current_sum + x) % m;
        if (current_sum < 0) current_sum += m; // Ensure it's positive
        result += prefix_counts[current_sum];
        prefix_counts[current_sum]++;
    }

    std::cout << result << '\n';
}

int main() {
    std::ios::sync_with_stdio(false);
    std::cin.tie(nullptr);
    std::cout.tie(nullptr);

    int T = 1;
    // std::cin >> T;
    while (T--) {
        solve();
    }

    return 0;
}

### Explanation
1. **Input Handling**: The code reads the number of elements `n` and the modulo `m`, followed by the array elements. Each element is immediately taken modulo `m` to simplify further calculations.
2. **Prefix Sum Calculation**: The `current_sum` variable keeps track of the cumulative sum of elements modulo `m` as we iterate through the array.
3. **Counting Subarrays**: The `prefix_counts` map keeps track of how many times each modulo value has been encountered in the prefix sums. For each element, if the current modulo value has been seen before, it means there are subarrays ending at the current index that sum to a multiple of `m`.
4. **Result Calculation**: The result is accumulated by adding the count of previous occurrences of the current modulo value from the map, and then updating the map with the current modulo value.
5. **Output**: The total count of such subarrays is printed.

This approach efficiently counts the number of subarrays with sum divisible by `m` in O(n) time with O(m) space complexity, leveraging the properties of prefix sums and modular arithmetic.
