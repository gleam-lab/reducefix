2. **Key Insight**: The sum of a subarray from index `i` to `j` is `prefix[j] - prefix[i-1]`. For this sum to be divisible by `m`, `(prefix[j] - prefix[i-1]) % m == 0`, which implies `prefix[j] % m == prefix[i-1] % m`. Thus, we can use a map to count occurrences of each prefix sum modulo `m`.

3. **Algorithm Choice**: 
   - Compute prefix sums modulo `m`.
   - For each prefix sum, check how many times the same value has appeared before (using a map).
   - The count of such occurrences gives the number of valid subarrays ending at the current index.

4. **Edge Cases**:
   - Handle the case where the entire array is divisible by `m`.
   - Ensure the initial prefix sum (0) is counted before processing the array.

5. **Complexity**: The algorithm runs in O(N) time with O(N) space due to the use of a hash map to store prefix sums.

#include <bits/stdc++.h>
using LL = long long;

void solve() {
    int n, m; std::cin >> n >> m;
    std::vector<int> a(n);

    for (int &x : a) {
        std::cin >> x;
        x %= m;
    }

    std::unordered_map<int, int> prefix_counts;
    prefix_counts[0] = 1; // To handle subarrays starting from index 0
    int current_prefix = 0;
    LL res = 0;

    for (int x : a) {
        current_prefix = (current_prefix + x) % m;
        res += prefix_counts[current_prefix];
        prefix_counts[current_prefix]++;
    }

    std::cout << res << '\n';
    return;
}
