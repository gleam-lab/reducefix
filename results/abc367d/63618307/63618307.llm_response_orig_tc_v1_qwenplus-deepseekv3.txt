```cpp
#include<bits/stdc++.h>

using namespace std;
using ll=long long;
typedef pair<int, int>PII;
const int N=2e5+10,mod=1e9+7;

void solve()
{
    int n,M;
    cin>>n>>M;
    vector<int>a(n+10);
    for(int i=0; i<n; i++){
        cin>>a[i];
    }
    vector<ll>s(n+10);
    for(int i=0; i<n; i++){
        s[i+1]=s[i]+a[i];
    }
    map<int, int>mp;
    mp[0]=1; // Initialize with 0 to account for prefix sums that are 0 mod M
    ll ans=0;
    for(int i=1; i<=n; i++){
        int mod_val = (s[i]%M + M) % M; // Ensure positive modulo
        ans += mp[mod_val];
        mp[mod_val]++;
    }
    cout<<ans;
}

int main()
{
    cin.tie(0);
    ios::sync_with_stdio(0);
    ll t=1;
    while(t--){
        solve();
    }
    return 0;
}
```

### Explanation
1. **Prefix Sum Calculation**: We first compute the prefix sums `s` where `s[i]` is the sum of the first `i` elements of the array.
2. **Modulo Handling**: For each prefix sum, we compute its modulo with `M` to find `mod_val`. This value is adjusted to be positive if necessary.
3. **Map Usage**: We use a map `mp` to keep track of the counts of each `mod_val` encountered. The key idea is that if two prefix sums have the same `mod_val`, the subarray between them will have a sum divisible by `M`.
4. **Initialization**: The map is initialized with `mp[0] = 1` to account for prefix sums that are exactly divisible by `M` from the start.
5. **Counting Subarrays**: For each `mod_val`, we add the current count of that `mod_val` from the map to `ans` and then increment the count in the map for future matches.

This approach ensures that we efficiently count all valid subarrays in O(N) time with O(N) space complexity, which is optimal for the problem constraints.
