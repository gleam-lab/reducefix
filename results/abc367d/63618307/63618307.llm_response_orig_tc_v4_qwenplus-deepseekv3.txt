### Approach
1. **Prefix Sum and Modulo**: Calculate the prefix sums of the array and compute each prefix sum modulo M. This helps in finding subarrays whose sum modulo M is zero or another specific value.
2. **Counting Modulo Values**: Use a map to keep track of how many times each modulo value has been encountered. For each prefix sum modulo M, the number of valid subarrays ending at the current position can be determined by checking how many times a complementary value (current_mod - target_mod) has been seen before.
3. **Initialization**: Initialize the map with the modulo value of zero to account for the prefix sum before the array starts (sum of zero elements).

#include<bits/stdc++.h>

using namespace std;
using ll=long long;
typedef pair<int, int>PII;
const int N=2e5+10,mod=1e9+7;

{
    int n,M;

    vector<int>a(n+10);
    for(int i=0; i<n; i++){
        cin>>a[i];
    }
    vector<ll>s(n+10);
    for(int i=0; i<n; i++){
        s[i+1]=s[i]+a[i];
    }
    map<int, int>mp;
    mp[0]=1;
    ll ans=0;
    for(int i=1; i<=n; i++){
        int mod_val = (s[i] % M + M) % M;
        ans += mp[mod_val];
        mp[mod_val]++;
    }

}

int main()
{
    cin.tie(0);

### Explanation
1. **Prefix Sum Calculation**: The prefix sum array `s` is computed where `s[i]` represents the sum of the first `i` elements of the array. This allows efficient calculation of the sum of any subarray.
2. **Modulo Handling**: For each prefix sum, the modulo with `M` is computed. This helps in identifying subarrays where the sum modulo `M` is zero.
3. **Map Initialization**: The map `mp` is initialized with `mp[0] = 1` to account for the prefix sum of zero elements, which is necessary for subarrays starting from the beginning of the array.
4. **Counting Valid Subarrays**: For each prefix sum modulo `M`, the number of valid subarrays ending at the current position is the count of how many times this modulo value has been seen previously. This count is added to `ans`, and the current modulo value is then incremented in the map.
5. **Output**: The total count of valid subarrays is printed.

This approach efficiently counts the number of subarrays with a sum modulo `M` equal to zero using prefix sums and a hash map, ensuring optimal performance with a time complexity of O(N).
