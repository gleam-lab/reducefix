```cpp
#include<bits/stdc++.h>

using namespace std;
using ll=long long;
typedef pair<int, int>PII;
const int N=2e5+10,mod=1e9+7;

void solve()
{
	int n,M;
	cin>>n>>M;
	vector<int>a(n+10);
	for(int i=0; i<n; i++){
		cin>>a[i];
	}
	vector<ll>s(n+10);
	for(int i=0; i<n; i++){
		s[i+1]=s[i]+a[i];
	}
	map<int, int>mp;
	mp[0]=1; // To account for subarrays starting from index 0
	ll ans=0;
	for(int i=1; i<=n; i++){
		int rem=(s[i]%M + M)%M; // Ensure positive remainder
		ans+=mp[rem];
		mp[rem]++;
	}
	cout<<ans;
}

int main()
{
	cin.tie(0);
	ios::sync_with_stdio(0);
	ll t=1;
	while(t--){
		solve();
	}
	return 0;
}
```

### Explanation
1. **Prefix Sum Calculation**: The prefix sum array `s` is computed where `s[i]` represents the sum of the first `i` elements of the array. This allows efficient calculation of subarray sums.

2. **Remainder Tracking**: For each prefix sum `s[i]`, the remainder when divided by `M` is computed. This remainder helps in identifying subarrays whose sum modulo `M` equals zero.

3. **Map Initialization**: The map `mp` is initialized with `mp[0] = 1` to account for subarrays that start from the beginning of the array and whose sum is divisible by `M`.

4. **Counting Valid Subarrays**: For each remainder encountered, the number of valid subarrays ending at the current position is given by the count of the same remainder in the map. This count is added to `ans`, and the current remainder's count in the map is incremented.

5. **Efficiency**: The algorithm efficiently counts all valid subarrays in O(N) time by leveraging prefix sums and a map to track remainders, ensuring optimal performance even for large input sizes.
