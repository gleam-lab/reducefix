```cpp
#include <vector>
#include <iostream>
#include <algorithm>
#include <cmath>
#include <string>
#include <numeric>

using namespace std;

#define INF (long long)1e18 + 7

int main(){
    int n;
    long long dp0, dp1, x, temp;
    dp0 = 0;
    dp1 = -INF;
    cin >> n;

    for (int i = 0; i < n; i++) {
        cin >> x;
        temp = dp0;
        dp0 = max(dp1 + 2 * x, dp0);
        dp1 = max(temp + x, dp1);
    }

    cout << max(dp0, dp1) << endl;
    return 0;
}
```

### Explanation of Fixes
1. **Constant Definition**: Changed `INF` to `(long long)1e18 + 7` to ensure it fits within the range of `long long` and provides a sufficiently large negative value.
2. **Code Formatting**: Improved readability by adding spaces around operators and aligning variable declarations.
3. **Variable Naming**: Ensured variable names are descriptive and consistent.
