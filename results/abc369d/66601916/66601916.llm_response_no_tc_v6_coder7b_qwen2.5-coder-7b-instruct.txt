```cpp
#include <vector>
#include <iostream>
#include <algorithm>
#include <cmath>
#include <string>
#include <numeric>

using namespace std;

#define INF (long long)1e18 + 7

int main() {
    int n;
    long long dp0 = 0, dp1 = -INF, x, temp;
    cin >> n;

    for (int i = 0; i < n; ++i) {
        cin >> x;
        temp = dp0;
        dp0 = max(dp1 + 2 * x, dp0);
        dp1 = max(temp + x, dp1);
    }

    cout << max(dp0, dp1) << endl;
    return 0;
}
```

### Explanation of Changes:
1. **Initialization**: Changed `dp0` to `0` and `dp1` to `-INF` as per the problem statement.
2. **Loop Counter Increment**: Added `++i` in the loop to correctly iterate through the array elements.
3. **Constant Definition**: Changed `INF` from `(long long)1e1+8` to `(long long)1e18 + 7` to handle larger values more effectively.
4. **Code Formatting**: Ensured consistent formatting for better readability.
