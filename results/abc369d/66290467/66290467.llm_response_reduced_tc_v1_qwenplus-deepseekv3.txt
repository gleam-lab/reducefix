### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
#define rep(i, n) for (int(i) = 0; (i) < (n); ++(i))
#define drep(i, n) for (int(i) = (n) - 1; (i) >= 0; --(i))

int main() {
    int n;
    cin >> n;
    vector<ll> a(n);
    rep(i, n) cin >> a[i];

    if (n == 1) {
        cout << a[0] << endl;
        return 0;
    }

    vector<vector<ll>> dp(n, vector<ll>(2, 0));
    dp[0][0] = a[0];
    dp[0][1] = 0; // cannot skip before first

    if (n >= 2) {
        dp[1][0] = a[0] + a[1];
        dp[1][1] = 2 * a[1];
    }

    for (int i = 2; i < n; ++i) {
        dp[i][0] = max(dp[i - 1][0], dp[i - 1][1]) + a[i];
        dp[i][1] = max(dp[i - 2][0], dp[i - 2][1]) + 2 * a[i];
    }

    cout << max(dp[n - 1][0], dp[n - 1][1]) << endl;

    return 0;
}
```

### Explanation
1. **DP State Definition**: 
   - `dp[i][0]` represents the maximum experience up to the i-th monster without skipping the (i-1)-th monster.
   - `dp[i][1]` represents the maximum experience up to the i-th monster by skipping the (i-1)-th monster, thus getting a 2x bonus for the i-th monster.

2. **Initialization**: 
   - For the first monster, you cannot skip any previous monster, so `dp[0][0] = a[0]` and `dp[0][1] = 0`.
   - For the second monster, you either defeat both normally (`dp[1][0] = a[0] + a[1]`) or skip the first to get a 2x bonus on the second (`dp[1][1] = 2 * a[1]`).

3. **Recurrence Relation**: 
   - `dp[i][0]` is derived by taking the maximum of the previous states (either skipped or not) and adding the current monster's value normally.
   - `dp[i][1]` is derived by taking the maximum of the states two steps back (since you skip the (i-1)-th monster) and adding 2x the current monster's value.

4. **Result**: The maximum value between `dp[n-1][0]` and `dp[n-1][1]` gives the optimal experience.

This approach correctly handles all cases, including the provided test case, by considering both possibilities (skipping or not skipping the previous monster) at each step.
