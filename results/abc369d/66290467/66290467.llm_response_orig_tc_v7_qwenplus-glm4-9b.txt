The issue occurs in the line:
```cpp
rep(i, n) for (int(i) = 0; (i) < (n); ++(i))
```
The variable `i` is being declared within the for loop, which is not allowed in C++. The correct way to declare the variable `i` is outside the for loop header.

#define drep(i, n) for (int i = n - 1; i >= 0; --i)
