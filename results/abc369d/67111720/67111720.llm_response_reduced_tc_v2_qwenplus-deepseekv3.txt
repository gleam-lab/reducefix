```cpp
#include<bits/stdc++.h>
using namespace std;

#define rep(i,n) for (ll i=0;i < (n); i++)
using ll = long long;

int main()
{
    int n; cin >> n;
    vector<ll> a(n);
    rep(i,n) cin >> a[i];

    if (n == 0) {
        cout << 0 << endl;
        return 0;
    }

    ll include = a[0];
    ll exclude = 0;
    ll new_include, new_exclude;

    for (int i = 1; i < n; i++) {
        new_include = exclude + a[i];
        new_exclude = max(include, exclude);
        include = new_include;
        exclude = new_exclude;
    }

    cout << max(include, exclude) << endl;
}
```

### Explanation
1. **Initialization**: For an empty array, the sum is 0. For a non-empty array, initialize `include` to the first element (sum if we include it) and `exclude` to 0 (sum if we exclude it).
2. **Dynamic Programming Update**: For each subsequent element, update the `include` and `exclude` values:
   - `new_include` is the sum of excluding the previous element and including the current one.
   - `new_exclude` is the maximum of including or excluding the previous element (since we can choose to exclude the current element regardless of the previous choice).
3. **Result**: The maximum value between `include` and `exclude` after processing all elements gives the optimal sum, correctly handling all edge cases including single-element arrays. 

This approach ensures that we maximize the sum while respecting the constraints (likely no adjacent elements), providing the correct output for all test cases.
