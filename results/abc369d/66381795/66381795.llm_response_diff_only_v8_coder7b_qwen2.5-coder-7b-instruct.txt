```cpp
#include <bits/stdc++.h>
using namespace std;

#define rep(i, n) for (int i = 0; i < (int)(n); i++)
using ll = long long;

int main() {
    int n;
    cin >> n;
    vector<int> data(n);
    rep(i, n) {
        cin >> data[i];
    }

    ll sum = 0;
    for (int i = 0; i < n - 1; i++) {
        if (data[i + 1] * 2 > data[i] + data[i + 1]) {
            sum += data[i + 1] * 2;
            i++; // Skip the next element as it's already counted
        } else {
            sum += max(data[i], data[i + 1]);
        }
    }
    // Add the last element if it's not included yet
    if (n % 2 != 0) {
        sum += data[n - 1];
    }

    cout << sum << endl;
    return 0;
}
```

### Explanation of Fixes
1. **Loop Correction**: The original loop was incorrectly iterating over `i` from 0 to `n`, which caused an out-of-bounds access when `i` reached `n-1`. The corrected loop iterates from 0 to `n-2`.
2. **Sum Calculation**: The condition to check if `data[i+1] * 2 > data[i] + data[i+1]` was incorrect. It should compare the doubled value of the next element with the sum of the current and next elements. If true, both elements are added twice; otherwise, only the maximum of the two is added.
3. **Edge Case Handling**: Added a check at the end to add the last element if `n` is odd, ensuring no element is left unaccounted for.
4. **Variable Initialization**: Removed the unnecessary `cnt` variable since it wasn't used effectively in the logic.
