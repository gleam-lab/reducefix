### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;
#define rep(i, n) for (int i = 0; i < (int)(n); i++)
using ll = long long;

int main() {
    int n;
    cin >> n;
    vector<int> data(n);
    rep(i, n) {
        cin >> data[i];
    }
    ll sum = 0;
    rep(i, n) {
        sum += data[i];
    }
    cout << sum << endl;
    return 0;
}
```

### Explanation
1. **Input Handling**: The code reads the number of elements `n` and then reads `n` integers into a vector `data`.
2. **Sum Calculation**: The code initializes a `sum` variable of type `long long` to accumulate the total sum of all elements. It iterates through each element in the vector and adds it to `sum`.
3. **Output**: The total sum is printed.

This approach ensures that all elements are summed correctly, handling the sequence of large and small numbers appropriately. The complexity is linear (O(N)), which is optimal for this problem. No conditions or special cases are needed, making the solution both simple and efficient.
