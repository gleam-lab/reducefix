### Solution Code

```cpp
#include <bits/stdc++.h>
using namespace std;

#define INF (long long)1e18

int main() {
    int n;
    long long dp0, dp1, x;
    dp0 = 0;
    dp1 = -INF;
    cin >> n;

    for (int i = 0; i < n; i++) {
        cin >> x;
        long long new_dp0 = max(dp0, dp1 + x);
        long long new_dp1 = max(dp1, dp0 - x);
        dp0 = new_dp0;
        dp1 = new_dp1;
    }

    cout << max(dp0, dp1) << endl;
    return 0;
}
```

### Explanation
1. **Initialization**: We initialize `dp0` to 0 (starting with no elements) and `dp1` to negative infinity to represent an initially invalid state where no elements have been doubled yet.
2. **Reading Input**: For each element in the input sequence, we read the value `x`.
3. **State Transitions**:
   - **`new_dp0`**: This state is updated by taking the maximum between the previous `dp0` (no operation) or adding `x` to the previous `dp1` (which was doubled).
   - **`new_dp1`**: This state is updated by taking the maximum between the previous `dp1` (no operation) or subtracting `x` from the previous `dp0` (since doubling `x` would have added `x` previously, and now we need to offset it).
4. **Output**: After processing all elements, the result is the maximum value between `dp0` and `dp1`, which gives the optimal sum under the given operations.

This approach ensures that we correctly handle all cases, including edge cases like single elements, by dynamically tracking the best possible sums at each step without doubling or with doubling operations.
