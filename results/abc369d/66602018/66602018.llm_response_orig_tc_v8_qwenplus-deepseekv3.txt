### Solution Code
```cpp
#include <bits/stdc++.h>
using namespace std;

#define INF (long long)1e18

int main() {
    int n;
    long long dp0, dp1, x, temp;
    dp0 = 0;
    dp1 = -INF;
    cin >> n;

    for (int i = 0; i < n; i++) {
        cin >> x;
        temp = dp0;
        dp0 = max(dp1 + x, dp0);
        dp1 = max(temp - x, dp1);
    }

    cout << max(dp0, dp1) << endl;
    return 0;
}
```

### Explanation
1. **Initialization**: We initialize `dp0` to 0, representing the maximum value achievable with no prior operations, and `dp1` to negative infinity, representing an initially impossible state.
2. **Processing Each Element**: For each element in the sequence:
   - **Temp Variable**: Store the current value of `dp0` to use in the next step.
   - **Update dp0**: The new `dp0` is the maximum of either adding the current element to `dp1` (representing a prior operation) or keeping `dp0` as is.
   - **Update dp1**: The new `dp1` is the maximum of either subtracting the current element from the previous `dp0` (stored in `temp`) or keeping `dp1` as is.
3. **Output**: The result is the maximum value between `dp0` and `dp1` after processing all elements.

This approach ensures that we correctly track the maximum values achievable by considering both possible operations at each step, leading to the correct result for all test cases.
