The issue with your original code was that you were checking `~dp[i][parity]` instead of `dp[i][parity] != -1`. This is because `~dp[i][parity]` checks if `dp[i][parity]` is not zero, which is incorrect since `dp[i][parity]` should be checked against `-1` to determine if it has been computed before. The corrected line is `if (dp[i][parity] != -1)` to correctly check if the value has already been computed and stored in the DP table.
