```python
n = int(input())
a = list(map(int, input().split()))

# dp[i][0]: 最も最近のモンスターを通常で倒したときの最大経験値（i体目まで）
# dp[i][1]: 最も最近のモンスターをボーナス付きで倒したときの最大経験値（i体目まで）
# ボーナス付きは、奇数個の倒したモンスターの後に来なければならない（つまり直前の行動が倒し・非ボーナス）

dp = [[0, 0] for _ in range(n + 1)]

for i in range(1, n + 1):
    # 現在のモンスターを倒す（ボーナスなし）: 前回がボーナスあり or なし どちらでもOK
    dp[i][0] = max(dp[i-1][0], dp[i-1][1]) + a[i-1]
    
    # 現在のモンスターを倒す（ボーナスあり）: これは「偶数番目の倒したモンスター」になる必要がある
    # → 直前の状態は「奇数番目の倒し（= ボーナスなし）」でなければならない
    # つまり、直前の状態は dp[i-1][0] のみから遷移可能
    dp[i][1] = dp[i-1][0] + 2 * a[i-1]

print(max(dp[n]))
```
