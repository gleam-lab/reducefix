```python
n = int(input())
a = list(map(int, input().split()))

# dp[i][0]: maximum points up to i-th monster where the last defeated monster is at an odd position (1st, 3rd, ...)
# dp[i][1]: maximum points up to i-th monster where the last defeated monster is at an even position (2nd, 4th, ...)
dp = [[0, 0] for _ in range(n + 1)]

for i in range(1, n + 1):
    # Option 1: Skip the current monster
    skip = max(dp[i-1][0], dp[i-1][1])
    
    # Option 2: Take current monster as an odd-positioned defeat (normal points)
    take_odd = dp[i-1][1] + a[i-1]  # coming from even -> now odd
    
    # Option 3: Take current monster as an even-positioned defeat (double points)
    # This requires that previous was odd
    take_even = dp[i-1][0] + 2 * a[i-1]
    
    dp[i][0] = max(skip, take_odd)
    dp[i][1] = take_even

print(max(dp[n][0], dp[n][1]))
```
