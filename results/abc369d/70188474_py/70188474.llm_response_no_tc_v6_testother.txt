```python
n = int(input())
a = list(map(int, input().split()))

# dp[i][0]: 最も最近の倒したモンスターが偶数番目でない（奇数番目）時の最大経験値
# dp[i][1]: 最も最近の倒したモンスターが偶数番目である時の最大経験値
# ただし、「最も最近の倒したモンスター」の順序は、倒した順における相対的な位置を指す

# 別の解釈: 倒したモンスターの総数が奇数か偶数かによってボーナスが決まる
# 実際には、i体目まで処理して、今までに倒したモンスターの数が偶数か奇数かが重要

# dp[i][j]: i体目のモンスターまで処理し、倒したモンスターの数が j (mod 2) のときの最大経験値
dp = [[0, 0] for _ in range(n+1)]
# 初期状態: 0体倒した → 倒した数は0（偶数）
dp[0][0] = 0
dp[0][1] = -10**18  # 不可能状態

for i in range(1, n+1):
    # i-1 番目の要素 (0-indexed) を見る
    x = a[i-1]
    # 選択1: モンスターを倒さない
    no_fight = max(dp[i-1][0], dp[i-1][1])
    dp[i][0] = max(dp[i][0], no_fight)
    dp[i][1] = max(dp[i][1], no_fight)
    
    # 選択2: モンスターを倒す
    # 前に倒した数が偶数なら、今のは奇数番目（通常ポイント）
    if dp[i-1][0] != -10**18:
        new_val = dp[i-1][0] + x
        # 新たに倒したので、倒した数の偶奇が切り替わる
        dp[i][1] = max(dp[i][1], new_val)
    
    # 前に倒した数が奇数なら、今のは偶数番目（2倍ポイント）
    if dp[i-1][1] != -10**18:
        new_val = dp[i-1][1] + 2 * x
        dp[i][0] = max(dp[i][0], new_val)

print(max(dp[n][0], dp[n][1]))
```
