```python
n = int(input())
a = list(map(int, input().split()))

# dp[i][0]: maximum points up to i-th monster where the i-th is taken normally (odd position in defeated sequence)
# dp[i][1]: maximum points up to i-th monster where the i-th is taken with double (even position in defeated sequence)
dp = [[0, 0] for _ in range(n + 1)]

for i in range(1, n + 1):
    # Option 1: Take monster i as a normal defeat (adds to odd positions in defeated sequence)
    # Can come from either state at i-1
    dp[i][0] = max(dp[i-1][0], dp[i-1][1]) + a[i-1]
    
    # Option 2: Take monster i as a double defeat (adds to even positions in defeated sequence)
    # This requires that the previous defeated monster was at an odd position
    # So we can only transition from dp[i-1][0] (where last defeat was odd)
    dp[i][1] = dp[i-1][0] + 2 * a[i-1]

print(max(dp[n]))
```
