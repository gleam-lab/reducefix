```python
n, k = map(int, input().split())
a = list(map(int, input().split()))

# Sort the array to consider optimal removals
a.sort()

# We are removing exactly K elements, so we keep (n - k) elements
# To minimize (max - min) of the remaining sequence, we should choose a contiguous segment of (n-k) elements
# because any non-contiguous selection can be improved by closing gaps in sorted order

min_diff = float('inf')
length = n - k

# Try all possible contiguous subarrays of length (n - k)
for i in range(n - length + 1):
    j = i + length - 1
    diff = a[j] - a[i]
    if diff < min_diff:
        min_diff = diff

print(min_diff)
```
