```python
n, k = map(int, input().split())
a = list(map(int, input().split()))

# Sort the array to consider elements in order
a.sort()

# We are removing exactly K elements, so we keep (n - k) elements
# To minimize (max - min) of the remaining sequence, we should choose a contiguous subsequence of length (n - k)
# Because if the kept elements are not contiguous in sorted order, we can always get a better or equal result by making them contiguous

min_diff = float('inf')
length = n - k

# Try all possible contiguous segments of length (n - k)
for i in range(n - length + 1):
    j = i + length - 1
    diff = a[j] - a[i]
    min_diff = min(min_diff, diff)

print(min_diff)
```
