```python
n, k = map(int, input().split())
a = list(map(int, input().split()))

# Sort the array to consider optimal removals
a.sort()

# We are removing exactly K elements, so we keep (n - k) elements.
# To minimize (max - min) of the remaining subsequence,
# we want to choose a contiguous segment of (n - k) elements in the sorted array.
# Because if the remaining elements are not contiguous in sorted order,
# we could get a smaller range by closing the gap.

min_range = float('inf')
length = n - k

for i in range(n - length + 1):
    current_min = a[i]
    current_max = a[i + length - 1]
    min_range = min(min_range, current_max - current_min)

print(min_range)
```
