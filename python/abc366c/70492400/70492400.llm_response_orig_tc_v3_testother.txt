The issue with the provided code is that it's tracking the total number of balls in the bag (`l`) instead of the number of distinct integers. When a ball is added or removed, we need to track which specific numbers are present.

We should use a dictionary (or defaultdict) to count occurrences of each number, and maintain a separate counter for the number of distinct integers.

```python
q = int(input())
bag = {}
distinct_count = 0

for _ in range(q):
    query = list(map(int, input().split()))
    
    if query[0] == 1:
        x = query[1]
        if x not in bag:
            bag[x] = 0
            distinct_count += 1
        bag[x] += 1
        
    elif query[0] == 2:
        x = query[1]
        bag[x] -= 1
        if bag[x] == 0:
            del bag[x]
            distinct_count -= 1
            
    else:  # query[0] == 3
        print(distinct_count)
```
