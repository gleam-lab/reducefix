```python
n = int(input())
a = list(map(int, input().split()))

# dp[i][0]: 最も最近の defeated モンスターが奇数番目 (1st, 3rd, ...) で終わっている場合の最大経験値
# dp[i][1]: 最も最近の defeated モンスターが偶数番目 (2nd, 4th, ...) で終わっている場合の最大経験値
# 状態の意味: dp[i][0] は i 番目までのモンスターを見て、最後に倒したモンスターがそのセッションで奇数番目（つまり次に倒せばボーナス）であるときの最大値

# 別の解釈: 
# dp[i][0]: i 番目までで、倒したモンスターの総数が偶数（つまり次に倒すと奇数個目 → ボーナスなし）
# dp[i][1]: i 番目までで、倒したモンスターの総数が奇数（つまり次に倒すと偶数個目 → ボーナス付き）

# しかし問題は「偶数回目に倒したモンスターにボーナス」なので、倒した順番での偶奇が重要。

# 正しい解釈:
# dp[i][0]: i 番目まで見て、今まで倒したモンスターの数が偶数（次の倒すのは奇数番目 → ボーナスなし）
# dp[i][1]: i 番目まで見て、今まで倒したモンスターの数が奇数（次の倒すのは偶数番目 → ボーナスあり）

dp = [[0, 0] for _ in range(n+1)]

for i in range(1, n+1):
    # モンスター i を倒さない場合
    no_defeat = max(dp[i-1][0], dp[i-1][1])
    
    # モンスター i を倒す場合
    # 現在の倒した数が偶数なら（次は奇数個目）、ボーナスなし
    # 現在の倒した数が奇数なら（次は偶数個目）、ボーナスあり
    defeat_without_bonus = dp[i-1][0] + a[i-1]  # 前が偶数個 -> 今倒すと奇数個目（ボーナスなし）
    defeat_with_bonus = dp[i-1][1] + 2 * a[i-1]  # 前が奇数個 -> 今倒すと偶数個目（ボーナスあり）
    
    # 更新
    dp[i][0] = max(no_defeat, defeat_without_bonus)  # 今倒した後、合計倒した数は奇数（偶数 +1）
    dp[i][1] = defeat_with_bonus  # 今倒した後、合計倒した数は偶数（奇数 +1）→ ただし、倒さない選択肢はない？
    
    # 上の更新は間違い。dp[i][1] にも「倒さない」選択肢を含める必要がある
    
    # 正しくは：
    # dp[i][0]: 最終状態が「偶数個倒した」状態
    # dp[i][1]: 最終状態が「奇数個倒した」状態
    #
    # 遷移：
    # - 倒さない: dp[i][*] += max(dp[i-1][0], dp[i-1][1]) のどちらか
    # - 倒す: 
    #   - 前が偶数個 -> 今倒すと奇数個目（ボーナスなし）-> dp[i][1] を更新
    #   - 前が奇数個 -> 今倒すと偶数個目（ボーナスあり）-> dp[i][0] を更新

# 再定義:
# dp[i][0]: i 番目までで、倒したモンスターの総数が偶数（0,2,4,...）
# dp[i][1]: i 番目までで、倒したモンスターの総数が奇数（1,3,5,...）

# 初期化
dp = [0, 0]  # dp[0]: 偶数個, dp[1]: 奇数個

for i in range(n):
    new_dp = [0, 0]
    # 倒さない
    base = max(dp[0], dp[1])
    new_dp[0] = base  # 倒さないので偶奇はそのまま？いや、状態は引き継ぐ
    new_dp[1] = base  # 実際は、倒さないなら前の状態がそのまま生きるが、max取った時点で統合される
    
    # 正しくは、倒さない場合は各状態がそのまま遷移
    new_dp[0] = max(new_dp[0], dp[0])  # 前が偶数個で倒さない -> 今も偶数個
    new_dp[1] = max(new_dp[1], dp[1])  # 前が奇数個で倒さない -> 今も奇数個
    
    # 倒す
    # 前が偶数個 -> 今倒すと奇数個目（ボーナスなし）-> 新たな奇数個状態
    new_dp[1] = max(new_dp[1], dp[0] + a[i])
    # 前が奇数個 -> 今倒すと偶数個目（ボーナスあり）-> 新たな偶数個状態
    new_dp[0] = max(new_dp[0], dp[1] + 2 * a[i])
    
    dp = new_dp

print(max(dp))
```
