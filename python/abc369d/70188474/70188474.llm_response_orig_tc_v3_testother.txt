```python
n = int(input())
a = list(map(int, input().split()))

# dp[i][0]: maximum points up to i-th monster where the i-th monster is defeated but not doubled
# dp[i][1]: maximum points up to i-th monster where the i-th monster is defeated and doubled
# We need to track whether the current defeat is even-numbered (doubled) or odd-numbered (normal)

# Instead of tracking parity by position, we track by count of defeats
# Let dp[i][j] = max total experience after processing first i monsters with j defeats (j mod 2 determines if next defeat will be doubled)
# But simpler: we can use two states: 
#   dp0 = max experience when number of defeats so far is even (so next defeat gets doubled)
#   dp1 = max experience when number of defeats so far is odd (so next defeat gets normal)

dp0 = 0  # even number of defeats so far (0 is even)
dp1 = -10**18  # odd number of defeats so far (initially impossible)

for i in range(n):
    x = a[i]
    # Option 1: skip monster i
    new_dp0 = dp0
    new_dp1 = dp1
    
    # Option 2: defeat monster i
    # If current count is even (dp0), then this will be an odd-numbered defeat -> normal points
    if dp0 != -10**18:
        new_dp1 = max(new_dp1, dp0 + x)
    
    # If current count is odd (dp1), then this will be an even-numbered defeat -> double points
    if dp1 != -10**18:
        new_dp0 = max(new_dp0, dp1 + 2 * x)
    
    dp0, dp1 = new_dp0, new_dp1

print(max(dp0, dp1))
```
